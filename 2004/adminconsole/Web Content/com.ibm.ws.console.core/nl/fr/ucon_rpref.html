Paramètres de préférences

Cette page des préférences permet d'indiquer si vous voulez que l'espace de travail de la console d'administration soit régénéré automatiquement après les modifications, que la portée par défaut corresponde au noeud de la console d'administration, que des boîtes de dialogue de confirmation s'affichent et que la bannière et les descriptions de l'espace de travail s'affichent.

Pour afficher cette page de la console d'administration, cliquez sur Préférences dans la barre des tâches de la console.

Activer la régénération automatique de l'espace de travail
Indique si vous souhaitez que l'espace de travail de la console d'administration soit automatiquement redessiné une fois la console d'administration modifiée.

Par défaut, l'espace de travail est automatiquement redessiné. Si vous demandez à la console de créer par exemple, une instance de serveur d'applications, la page Serveurs d'applications est automatiquement régénérée et affiche le nom du nouveau serveur dans la collection de serveurs.

Si vous indiquez que l'espace de travail ne doit pas être automatiquement redessiné, cela signifie que vous devez accéder de nouveau à une page en cliquant dans l'arborescence de navigation de la console ou sur les liens des pages de la collection pour voir les changements apportés à la configuration administrative.

Type de données Elément booléen
Valeur par défaut true
Ne pas confirmer les suppressions dans l'espace de travail
Indique si des boîtes de dialogue de confirmation s'affichent après une demande de suppression d'objet. Par défaut, les messages de confirmation ne s'affichent pas.
Type de données Elément booléen
Valeur par défaut false
Utiliser la portée par défaut
Indique si la portée par défaut est le noeud de la console d'administration. Par défaut, la portée n'est pas le noeud de la console.
Type de données Elément booléen
Valeur par défaut false
Masquer/afficher la bannière
Indique si la bannière WebSphere Application Server située dans la partie supérieure de la console d'administration doit s'afficher. Par défaut, la bannière s'affiche.
Type de données Elément booléen
Valeur par défaut true
Afficher/masquer les descriptions
Indique si des informations relatives aux pages et aux zones de la console doivent s'afficher. Les icônes de masquage/affichage des descriptions de zone et de page situées dans la partie droite de la barre de tâches offrent la même fonction. Par défaut, les descriptions des pages et des zones s'affichent.
Type de données Elément booléen
Valeur par défaut true
