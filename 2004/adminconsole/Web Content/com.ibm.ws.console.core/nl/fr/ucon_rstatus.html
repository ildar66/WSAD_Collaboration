Paramètres d'état WebSphere

Utilisez la zone d'état WebSphere de la console d'administration pour afficher les messages d'erreurs et d'événements d'exécution renvoyés par WebSphere Application Server.

La zone Etat WebSphere s'affiche en bas de la console et reste visible à mesure que vous naviguez de la page d'accueil (Home) WebSphere vers les autres pages. La zone comporte deux cadres : Problèmes de configuration dans WebSphere et Messages d'exécution WebSphere. Cliquez sur Précédent ou Suivant pour passer d'un cadre à l'autre.

Cliquez sur l'icône dans la partie supérieure droite de la zone pour régénérer les informations affichées. L'intervalle de régénération automatique peut être ajusté dans les paramètres Préférences.

Problèmes de configuration dans WebSphere
Affiche le nombre de fichiers de l'espace de travail. Ce panneau affiche également le nombre d'incidents rencontrés dans la configuration d'administration pour l'ID utilisateur.

Cliquez sur le nombre pour afficher des informations détaillées relatives aux incidents.

Message d'exécution WebSphere
Affiche le nombre de messages renvoyés par WebSphere Application Server ainsi que le nombre de messages d'erreur (icône x), de messages d'avertissement (icône ! ) et les messages d'information (icône i).

Cliquez sur le nombre de messages pour afficher les détails.
