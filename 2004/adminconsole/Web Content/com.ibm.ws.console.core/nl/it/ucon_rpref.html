Impostazioni preferenze

Utilizzare la pagina Preferenze per specificare se si desidera che l'area di lavoro della console di gestione venga aggiornata automaticamente dopo le modifiche, che l'ambito predefinito sia il nodo della console di gestione e che vengano visualizzate le finestre di dialogo di conferma, l'intestazione e le descrizioni dell'area di lavoro.

Per visualizzare questa pagina della console di gestione, fare clic su Preferenze sulla barra delle applicazioni della console.

Abilitazione dell'aggiornamento automatico dell'area di lavoro
Specifica se si desidera che l'area di lavoro della console di gestione venga aggiornata automaticamente dopo le modifiche alla configurazione di gestione.

Il valore predefinito è quello di aggiornare automaticamente l'area di lavoro. Se la console viene utilizzata per creare una nuova istanza, ad esempio di un server delle applicazioni, la pagina Server delle applicazioni viene aggiornata automaticamente visualizzando il nome del nuovo server nella raccolta dei server.

Se si specifica di non aggiornare automaticamente l'area di lavoro, sarà necessario accedere di nuovo alla pagina facendo clic sull'albero di navigazione della console o sui collegamenti alle pagine di raccolta per controllare le modifiche effettuate alla configurazione di gestione.

Tipo dati Valore booleano
Predefinito vero
Non confermare Annulla WorkSpace
Specifica se verranno visualizzate o meno le finestre di dialogo dopo una richiesta di cancellazione di un oggetto. Il valore predefinito è quello di non visualizzare le finestre di dialogo di conferma.
Tipo dati Valore booleano
Predefinito false
Utilizzo dell'ambito predefinito
Specifica se l'ambito predefinito sarà il nodo della console di gestione. Il valore predefinito indica che l'ambito non deve essere il nodo della console.
Tipo dati Valore booleano
Predefinito false
Nascondi/Visualizza intestazione
Specifica se l'intestazione di WebSphere Application Server viene visualizzata o meno lungo la parte superiore della console di gestione. Il valore predefinito è la visualizzazione dell'intestazione.
Tipo dati Valore booleano
Predefinito vero
Nascondi/Visualizza descrizioni
Specifica se, all'interno delle pagine, vengono visualizzate o meno le informazioni sui campi e sulle pagine della console. Le icone Nascondi/Visualizza descrizioni di campi e pagine, sulla destra della barra delle applicazioni, offre la stessa funzione. Il valore predefinito è quello di visualizzare le descrizioni di campi e pagine.
Tipo dati Valore booleano
Predefinito vero
