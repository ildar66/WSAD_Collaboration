Impostazioni di login

Utilizzare questa pagina per specificare l'utente della console di gestione di WebSphere Application Server. Se si sta utilizzando la sicurezza globale, è necessario specificare anche una password.

La specifica di un utente consente di poter riprendere il lavoro effettuato in precedenza nel prodotto. Dopo aver digitato un ID utente, fare clic su OK per passare alla pagina successiva e accedere alla console di gestione.

Per visualizzare questa pagina, avviare la console di gestione.

ID utente
Specifica una stringa che identifica l'utente. L'ID utente deve essere univoco sul server di gestione.

Il lavoro elaborato con il prodotto e salvato prima della chiusura verrà salvato in una configurazione identificata dall'ID utente immesso. Per accedere successivamente al lavoro con tale ID utente, specificare l'ID utente nella pagina Login.

Se si sta effettuando il collegamento da una macchina client che ha attualmente un browser aperto sulla console di gestione e viene contattato lo stesso server, le due connessioni potrebbero condividere lo stesso oggetto sessione HTTP, indipendentemente dal fatto che il collegamento avvenga utilizzando ID utente diversi. È necessario utilizzare due diversi tipi di browser (ad esempio, Microsoft Internet Explorer e Netscape Communicator) per collegarsi alla console di gestione dalla stessa macchina client, per poter lavorare da due oggetti sessione HTTP diversi, a prescindere dall'utilizzo o meno dello stesso ID utente. In altre parole, per utilizzare due sessioni simultanee sulla stessa macchina client, accedere alla console di gestione da due tipi diversi di browser.

Notare che è possibile utilizzare lo stesso tipo di browser per collegarsi due volte alla console ma è necessario collegarsi da host fisici diversi e utilizzare ID utente differenti. Ad esempio, si supponga che l'utente Joe si colleghi alla console di gestione utilizzando l'ID utente joe sull'host fisico myhost1 utilizzando il browser Internet Explorer. Joe può collegarsi alla stessa console una seconda volta, utilizzando il browser Internet Explorer ma deve utilizzare un diverso ID utente (ad esempio peggy) e un host fisico differente (come myhost2).

Un altro utente è attualmente collegato con lo stesso identificativo
Indica se disconnettere l'utente e proseguire con lo stesso ID utente specificato, se tornare alla pagina Login e specificare un altro ID utente oppure attendere fino a quando l'utente non chiuderà la connessione.

Questo campo viene visualizzato se l'utente ha chiuso un browser Web mentre eseguiva una ricerca sulla console di gestione, senza terminare il collegamento, ha riaperto un nuovo browser e ha cercato di accedere alla console di gestione.

Uso della configurazione principale
Quando abilitato, indica che l'utente desidera utilizzare la configurazione di gestione predefinita e non quella utilizzata più di recente per la sessione dell'utente. Le modifiche apportate alla sessione dell'utente, dopo l'ultimo salvataggio della configurazione di gestione, andranno perdute.

Questo campo viene visualizzato soltanto se l'utente ha modificato la configurazione di gestione e ha chiuso la connessione senza salvare le modifiche.

Tipo dati Valore booleano
Predefinito false
Ripristino delle modifiche effettuate nella sessione precedente
Quando abilitato, indica che l'utente desidera utilizzare la stessa configurazione di gestione utilizzata nell'ultima sessione dell'utente. Ripristina le modifiche effettuate dall'utente dopo l'ultimo salvataggio della configurazione di gestione per la sessione dell'utente.

Questo campo viene visualizzato soltanto se l'utente ha modificato la configurazione di gestione e ha chiuso la connessione senza salvare le modifiche.
