Azioni della barra delle applicazioni della console di gestione

Utilizzare la barra delle applicazioni della console di gestione per tornare alla pagina Home della console, salvare le modifiche apportate alle configurazioni di gestione, specificare le preferenze di console, chiudere la connessione della console e accedere alle informazioni sul prodotto.

Per visualizzare la barra delle applicazioni, andare alla console di gestione di WebSphere Application Server e controllare la barra orizzontale accanto al lato superiore della console. La barra delle applicazioni consente di eseguire numerose azioni.

Home
Visualizza la pagina Home della console di gestione nell'area di lavoro. L'area di lavoro si trova sul lato destro della console. La pagina Home fornisce i collegamenti alle informazioni relative all'utilizzo del prodotto WebSphere Application Server.
Salva
Visualizza la pagina Salva, che l'utente può utilizzare per salvare il lavoro realizzato nella console di gestione.

Le modifiche apportate alle configurazioni di gestione vengono salvate in un contenitore principale che comprende file di configurazione .xml. Per controllare le modifiche che sono state salvate, espandere Visualizza voci con modifiche. Salvare periodicamente per prevenire inavvertite perdite delle modifiche apportate.

Preferenze
Visualizza la pagina Preferenze, in cui è possibile specificare se si desidera aggiornare automaticamente l'area di lavoro della console di gestione dopo aver effettuato delle modifiche, visualizzare le finestre di dialogo di conferma e stabilire il nodo della console di gestione come ambito predefinito.
Chiudi sessione
Chiude la sessione della console di gestione e visualizza la pagina Login. Se la configurazione di gestione è stata modificata dopo l'ultimo salvataggio della configurazione nel contenitore principale, viene visualizzata la pagina Salva, prima di tornare alla pagina Collegamento. Fare clic su Salva per salvare le modifiche nel contenitore principale, su Annulla per chiudere la sessione senza salvare le modifiche o su Chiudi sessione per chiudere la sessione senza salvare le modifiche ma con la possibilità di ripristinarle al prossimo utilizzo della console.
Menu ?
Apre un nuovo browser Web sulla guida in linea del prodotto WebSphere Application Server.
Attiva/Disattiva descrizioni di campi e pagine
Consente di selezionare la visualizzazione o meno delle informazioni sulle pagine e sui campi della console. Le icone sul lato destro della barra delle applicazioni consentono l'attivazione e la disattivazione.
