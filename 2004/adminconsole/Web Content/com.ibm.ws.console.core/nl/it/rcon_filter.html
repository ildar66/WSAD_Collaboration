Impostazioni di filtro della console di gestione

Utilizzare le impostazioni Filtro per specificare il modo di filtrare le voci visualizzate in una vista di raccolta.

Selezionare la colonna da filtrare e immettere i criteri di filtro.

Colonna da filtrare
Utilizzare l'elenco a discesa per selezionare la colonna da filtrare. Quando il filtro viene applicato, la tabella di raccolta viene modificata di conseguenza.

Ad esempio, selezionare Server delle applicazioni se si desidera immettere i criteri con i quali filtrare i nomi dei server delle applicazioni.

Criteri di filtro
Nel campo accanto all'elenco a discesa, immettere una stringa che deve essere ricercata nel nome di una voce di raccolta, per qualificare la voce da visualizzare nella tabella di raccolta. La stringa può contenere i simboli %, * o ? come caratteri jolly. Ad esempio, immettere *App* per ricercare qualunque server delle applicazioni il cui nome contiene la stringa App.

Far precedere ogni carattere ( ) ^ * % { } \ + $ dal simbolo \, per fare in modo che il motore dell'espressione regolare che esegue la ricerca segua esattamente i criteri di ricerca forniti. Ad esempio, per ricercare tutti i provider JDBC contenenti (XA) nel nome provider, specificare quanto segue:

*\(XA\)
