Azioni dell'albero di navigazione della console di gestione

Utilizzare l'albero di navigazione della console di gestione per accedere alle pagine per creare e gestire i server, le applicazioni, le risorse e altri componenti.

Per visualizzare l'albero di navigazione, andare alla console di gestione di WebSphere Application Server e controllare l'albero sul lato sinistro della console. L'albero consente di visualizzare le informazioni sul runtime e sulle attività di configurazione. Gli argomenti principali disponibili sull'albero sono visualizzati di seguito. Per utilizzare l'albero, espandere un'argomento principale e selezionare una voce dall'elenco espanso, per visualizzare una pagina su cui completare le operazioni.

Server
Consente di configurare i server di gestione e altri tipi di server, come i server JMS.
Applicazioni
Consente di installare le applicazioni sui server e di gestire le applicazioni installate.
Risorse
Consente di configurare le risorse e di visualizzare le informazioni sulle risorse esistenti nella cella di gestione.
Sicurezza
Accede al Centro sicurezza; da utilizzare per proteggere le applicazioni e i server e per gestire gli utenti della console di gestione.
Ambiente
Consente di configurare host, server Web, variabili ed altri componenti.
Risoluzione dei problemi  
Consente di verificare e di tenere traccia dei problemi e degli errori di configurazione.
