管理コンソールのナビゲーション・ツリー・アクション

管理コンソールのナビゲーション・ツリーを使用して、 サーバー、アプリケーション、 リソースおよびその他のコンポーネントの作成と管理を行うためのページにアクセスします。

ナビゲーション・ツリーを表示するには、WebSphere Application Server の管理コンソールに移動して、 コンソールの左側にあるツリーを確認します。 ツリーでは、構成タスクとランタイム情報をナビゲートできます。 ツリーに提供されている主なトピックは以下のとおりです。 ツリーを使用するには、作業を実行するページを表示するために、必要となる主なトピックを展開し、 展開したリストから項目を選択します。

サーバー
管理サーバーおよび他のタイプのサーバー (JMS サーバーなど) を構成することができます。
アプリケーション
アプリケーションをサーバーにインストールして、 インストールしたアプリケーションを管理することができます。
リソース
リソースを構成して、管理セルに入っているリソースに関する情報を表示することができます。
セキュリティー
セキュリティー・センターにアクセスします。 セキュリティー・センターを使用して、アプリケーションおよびサーバーを保護し、管理コンソールのユーザーを管理します。
環境
ホスト、Web サーバー、変数、およびその他のコンポーネントを構成することができます。
トラブルシューティング  
構成エラーおよびさまざまな問題を検査し、追跡することができます。
