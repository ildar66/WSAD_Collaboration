관리 콘솔 작업 표시줄 조치

콘솔 홈 페이지로 리턴하고 관리 구성에 수행한 변경사항을 저장하고 콘솔 환경 설정을 지정하고 콘솔을 로그아웃하고 제품 정보에 액세스하려면 관리 콘솔의 작업 표시줄을 사용하십시오.

작업 표시줄을 보려면 WebSphere Application Server 관리 콘솔로 이동하여 콘솔의 맨 위 근처에 있는 가로 막대를 찾으십시오. 작업 표시줄은 여러 조치를 제공합니다.

홈
관리 콘솔 홈 페이지, 작업영역 표시. 작업영역이 콘솔의 오른쪽에 있습니다. 홈 페이지는 WebSphere Application Server 제품의 사용에 관한 정보에 대해 링크를 제공합니다.
저장
관리 콘솔에서 수행한 작업을 저장하는 데 사용하는 저장 페이지를 표시합니다.

관리 구성에 대한 변경사항이 .xml 구성 파일로 구성된 마스터 저장소에 저장됩니다. 저장된 변경사항을 확인하려면, 변경 항목 보기를 펼치십시오. 작성한 변경사항을 실수로 유실하지 않도록 주기적으로 작업을 저장합니다.

환경 설정
환경 설정 페이지를 표시합니다. 여기에서 관리 콘솔 작업공간 페이지가 변경 후 자동으로 새로 고쳐질지, 확인 대화 상자가 표시될지 및 기본 범위가 관리 콘솔 노드가 될지 지정할 수 있습니다.
로그아웃
관리 콘솔 세션을 로그아웃하고 로그인 페이지를 표시합니다. 구성을 마스터 저장소에 마지막으로 저장한 이후에 관리 구성을 변경한 경우 로그인 페이지로 리턴하기 전에 저장 페이지가 표시됩니다. 저장을 클릭하여 변경사항을 마스터 저장소에 저장하고 버리기를 클릭하여 변경사항을 저장하지 않고 세션을 종료하거나 로그아웃을 클릭하여 변경사항을 저장하지 않고 세션을 종료하지만 기회를 사용하여 콘솔을 리턴할 때 변경사항을 복구하십시오.
도움말
WebSphere Application Server 제품에 대한 온라인 도움말의 새 Web 브라우저를 엽니다.
필드 및 페이지 설명 토글을 숨기기/표시
페이지 내에서 콘솔 페이지 및 필드에 대한 정보를 표시할지 여부를 선택할 수 있습니다. 작업 표시줄의 오른쪽에 있는 아이콘은 토글을 제공합니다.
