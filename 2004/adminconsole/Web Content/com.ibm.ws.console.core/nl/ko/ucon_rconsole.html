관리 콘솔 영역

관리 콘솔을 사용하여 자원, 응용프로그램 및 서버를 작성하고 관리하거나 제품 메시지를 볼 수 있습니다.

관리 콘솔을 보려면 관리 콘솔에 대한 Application Server가 실행 중인지 확인하십시오. 웹 브라우저에서 관리 콘솔에 대한 URL을 지시한 후 로그인 페이지에 사용자 ID 및 필요한 경우 암호를 입력하십시오.

콘솔에는 원하는대로 크기 조정할 수 있는 다음의 기본 영역이 있습니다.

작업 표시줄
작업 표시줄은 콘솔 홈 페이지로 리턴하거나 관리 구성에 대한 변경사항을 저장하거나 콘솔 전체의 환경 설정을 지정하거나 콘솔을 로그 아웃하거나 제품 정보에 액세스하는 옵션을 제공합니다.
탐색 트리
콘솔의 왼쪽에 있는 탐색 트리는 WebSphere Application Server 셀의 구성요소를 작성 및 관리할 수 있는 콘솔 페이지로의 링크를 제공합니다.

트리 폴더 또는 항목 옆에 있는 +를 누르면 폴더 또는 항목에 대한 트리가 펼쳐집니다. 폴더 또는 항목에 대한 트리를 접으려면 -를 클릭하십시오. 트리 보기에서 항목을 두 번 클릭하면 펼치기와 접기 간의 상태가 토글됩니다.

WorkSpace
콘솔 오른쪽에 있는 작업영역에는 서버 및 자원과 같은 구성 오브젝트를 작성 및 관리하는 데 사용하는 페이지가 있습니다. 탐색 트리내의 링크를 클릭하여 구성된 오브젝트의 다른 유형을 볼 수 있습니다. 작업영역에서 구성된 오브젝트를 클릭하여 구성, 런타임 상태 및 옵션을 볼 수 있습니다. 작업 표시줄의 홈을 클릭하여 WebSphere Application Server 제품 사용시 정보로의 링크를 포함하는 작업영역 홈 페이지를 표시할 수 있습니다.
WebSphere 상태
콘솔 맨 아래의 상태 메시지 영역에서는 관리 구성에서의 문제점과 관련하여 WebSphere Application Server에 의해 리턴되는 메시지는 물론 런타임 이벤트에 대한 메시지에 관한 정보를 제공합니다. 구성 문제점 및 런타임 이벤트에 관한 표시 사이에서 토글하려면 이전 또는 다음을 클릭하십시오.

환경 설정에서 자동 새로 고치기 사이의 간격을 조정할 수 있습니다.
