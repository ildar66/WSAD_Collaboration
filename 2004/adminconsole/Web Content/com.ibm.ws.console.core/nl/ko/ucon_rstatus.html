WebSphere 상태 설정

관리 콘솔의 WebSphere 상태 영역을 사용하여 WebSphere Application Server가 리턴하는 오류 및 런타임 이벤트 메시지를 보십시오.

WebSphere 상태 영역은 콘솔의 맨 아래에 가로로 표시되며 WebSphere 홈 페이지에서 다른 페이지로 이동할 때 계속 보입니다. 이 영역은 WebSphere 구성 문제점 및 WebSphere 런타임 메시지의 두 프레임을 표시합니다. 프레임 사이에서 토글하려면 이전 또는 다음을 클릭하십시오.

이 영역의 오른쪽 위에 있는 아이콘을 클릭하여 표시된 정보를 새로 고치십시오. 환경 설정 설정에서 자동 새로 고치기 사이의 간격을 조정할 수 있습니다.

WebSphere 구성 문제점
작업공간 파일의 수를 표시합니다. 이 프레임 옆에 사용자 ID에 대한 관리 구성 문제점의 번호가 표시됩니다.

문제점에 대한 자세한 정보를 보려면 번호를 클릭하십시오.

WebSphere 런타임 메시지
WebSphere Application Server에서 리턴한 메시지 수 뿐아니라 오류 메시지(x 아이콘 ), 경고 메시지(! 아이콘) 및 정보 메시지(i 아이콘)의 수를 표시합니다.

세부사항을 보려면 메시지의 번호를 클릭하십시오.
