管理控制台任务栏操作

使用管理控制台的任务栏返回控制台主页、保存所做的管理配置更改、指定控制台首选项、注销控制台和访问产品信息。

要查看任务栏，转至 WebSphere Application Server 管理控制台并查看靠近控制台顶部的水平栏。任务栏提供几项操作。

起始对象
显示工作台中的管理控制台主页。控制台的右侧是工作台。主页提供到有关于使用 WebSphere Application Server 产品的信息的链接。
保存
显示“保存”页面，您可以用它来保存在管理控制台中完成的工作。

对管理配置所做的更改保存到包含 .xml 配置文件的主库。要查看已保存的更改，展开查看更改项。定期保存您的工作，确保您不会不经意地丢失您所做的更改。

首选项
显示“首选项”页面，在其中您可以指定是否希望管理控制台工作台页面在更改后自动刷新、显示确认对话框和作为管理控制台节点的缺省作用域。
注销
让您注销管理控制台会话并显示“登录”页面。如果您已更改自上次保存的主库配置后的管理配置，在返回“登录”页面前显示“保存”页面。单击保存保存对主库的更改、废弃不保存更改退出会话，或注销不保存更改退出会话但当您返回到控制台时您可以恢复您的更改。
帮助
打开新的 web 浏览器，浏览关于 WebSphere Application Server 产品的联机帮助信息。
隐藏／显示字段和页面描述切换
使您能够选择是否要显示控制台页面上的信息和页面中的字段。任务栏右侧的图标提供切换。
