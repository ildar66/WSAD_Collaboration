管理控制台导航树操作

使用管理控制台的导航树访问创建和管理服务器、应用程序、资源和其它组件的页面。

要查看导航树，转至 WebSphere Application Server 管理控制台并查看控制台左侧的树。树提供到配置任务和运行时信息的导航。树中可用的主要主题在下面显示。要使用树，展开感兴趣的主要主题并从展开的列表中选择一项，以显示可以完成您的工作的页面。

服务器
使您能够配置管理服务器和其它类型的服务器（如 JMS 服务器）。
应用程序
使您能够安装应用程序到服务器并管理安装的应用程序。
资源
使您能够配置资源并查看管理单元中存在的资源信息。
安全性
访问安全性中心，您可以用它来保护应用程序和服务器，以及管理管理控制台的用户。
环境
使您能够配置主机、Web 服务器、变量和其它组件。
故障诊断  
使您能够检查和跟踪配置错误和问题。
