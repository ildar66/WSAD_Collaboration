首选项设置

使用“首选项”页面指定是否希望管理控制台工作区更改后自动刷新、缺省作用域作为管理控制台节点、显示确认对话框，以及是否显示工作区标题和描述。

要查看此“管理控制台”页面，在控制台任务栏上单击首选项。

启用工作空间自动刷新
指定是否希望管理控制台工作空间在管理配置更改后自动重绘。

缺省为自动重绘工作空间。如果控制台直接创建新的实例（如应用程序服务器），“应用程序服务器”页面自动刷新并显示服务器集合中新的服务器名。

指定工作空间不自动重绘意味着：必须通过单击控制台导航树重新访问页面，或集合页面上的链接查看管理配置更改。

数据类型 Boolean
缺省 true
不确认工作空间废弃
指定是否在请求删除对象后显示确认对话框。缺省为不显示确认对话框。
数据类型 Boolean
缺省 false
使用缺省作用域
指定缺省作用域是否是管理控制台节点。缺省是对于作用域不是控制台节点。
数据类型 Boolean
缺省 false
隐藏／显示标题
指定是否沿着管理控制台顶部显示 WebSphere Application Server 标题。缺省为要显示标题。
数据类型 Boolean
缺省 true
隐藏／显示描述
指定是否要显示控制台页面上的信息和页面中的字段。任务栏右侧的“隐藏／显示字段和页面描述”图标提供相同的功能。缺省为要显示页面和字段描述。
数据类型 Boolean
缺省 true
