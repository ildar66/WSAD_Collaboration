WebSphere 状态设置

使用管理控制台的 WebSphere 状态区域查看 WebSphere Application Server 返回的错误和运行时事件消息。

WebSphere 状态区域在控制台底部显示，并且在您从 WebSphere 主页浏览到其它页面时仍为可视的。区域显示两个框架：WebSphere 配置问题和 WebSphere 运行时消息。单击上一个或下一个，在框架之间进行切换。

单击该区域右上角的图标刷新显示的信息。您可以调整首选项设置中的自动刷新之间的间隔。

WebSphere 配置问题
显示工作台文件数。此框架还显示了用户标识的管理配置的问题号。

单击数字以查看问题的详细信息。

WebSphere 运行时消息
显示 WebSphere Application Server 返回的消息数，以及错误消息数（x 图标）、警告消息数（! 图标）和信息性消息数（i 图标）。

单击消息号码查看详细信息。
