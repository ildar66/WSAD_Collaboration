管理控制台区域

使用管理控制台创建和管理资源、应用程序和服务器或查看产品消息。

要查看管理控制台，确保管理控制台的应用程序服务器在运行。将 Web 浏览器指向管理控制台的 URL，在“登录”页面上输入您的用户标识和密码（如果需要）。

控制台有下列主要区域，可按需要调整大小：

任务栏
任务栏提供的选项有：返回到控制台主页、保存对管理配置的更改、指定控制台范围的首选项、注销控制台和访问产品信息。
导航树
控制台左边的导航树提供至控制台页面的链接，您可以用这些控制台页面创建和管理 WebSphere Application Server 管理单元中的组件。

单击树文件夹或项旁边的 + 展开文件夹或项的树。单击 - 折叠文件夹或项的树。在树视图中双击一项可以使它的状态在展开和折叠之间切换。

工作台
控制台右侧的工作台包含您可以用于创建和管理如服务器和资源的对象的页面。单击导航树中的链接查看已配置对象的不同类型。在工作台内，单击已配置对象来查看其配置、运行时状态和选项。单击任务栏中的起始对象显示工作台主页，其包含到使用 WebSphere Application Server 产品的信息的链接。
WebSphere 状态
控制台底部的状态消息区域提供了有关由 WebSphere Application Server 返回的消息的信息，这些信息有关于您的管理配置中的问题以及关于运行时事件的消息。单击前一步或下一步切换配置问题和运行时事件的显示。

您可以在“首选项”设置中调整自动刷新的间隔。
