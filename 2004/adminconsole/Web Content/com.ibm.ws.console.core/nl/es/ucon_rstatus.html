Valores de estado de WebSphere

Utilice el área Estado de WebSphere de la consola administrativa para ver los mensajes de error y de sucesos de tiempo de ejecución devueltos por WebSphere Application Server.

El área Estado de WebSphere se muestra a lo largo de la parte inferior de la consola y permanece visible mientras se desplace desde la página principal de WebSphere a otras páginas. El área muestra dos marcos: Problemas de configuración de WebSphere y Mensajes de tiempo de ejecución de WebSphere. Pulse Anterior o Siguiente para conmutar entre los marcos.

Pulse el icono que aparece en la parte superior izquierda del área para renovar la información mostrada. Puede ajustar el intervalo entre renovaciones automáticas en los valores de Preferencias.

Problemas de configuración de WebSphere
Muestra el número de archivos de espacio de trabajo. En este marco, también se muestra el número de problemas con la configuración administrativa para el ID de usuario.

Pulse el número para ver información detallada de los problemas.

Mensajes de tiempo de ejecución de WebSphere
Muestra el número de mensajes devueltos por WebSphere Application Server, además del número de mensajes de error (icono x), mensajes de aviso (icono !) y mensajes informativos (iconoi).

Pulse el número de mensajes para ver los detalles.
