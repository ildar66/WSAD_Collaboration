Valores de preferencias

Utilice la página Preferencias para especificar si desea que el espacio de trabajo de la consola administrativa se renueve automáticamente después de los cambios, el ámbito por omisión sea el nodo de la consola administrativa, se muestren diálogos de confirmación y se visualicen las descripciones y el mensaje de cabecera del espacio de trabajo.

Para ver esta página de la consola administrativa, pulse Preferencias en la barra de tareas.

Habilitar la renovación automática del espacio de trabajo
Especifica si desea que el espacio de trabajo de la consola administrativa se vuelva a trazar automáticamente después de que cambie la configuración administrativa.

El valor por omisión es que el espacio de trabajo se vuelva a trazar automáticamente. Si dirige la consola para crear una nueva instancia, por ejemplo, de un servidor de aplicaciones, la página Servidores de aplicaciones se renueva automáticamente y muestra el nuevo nombre del servidor en la colección de servidores.

Si especifica que el espacio de trabajo no se vuelva a trazar automáticamente, significa que debe volver a acceder a la página pulsando en el árbol de navegación de la consola o en los enlaces de las páginas de la colección para ver los cambios realizados en la configuración administrativa.

Tipo de datos Boolean
Valor por omisión true
No confirmar los descartes del espacio de trabajo
Especifica si deben aparecer diálogos de confirmación después de una petición para suprimir un objeto. El valor por omisión es que no aparezcan diálogos de confirmación.
Tipo de datos Boolean
Valor por omisión false
Utilizar ámbito por omisión
Especifica si el ámbito por omisión es el nodo de la consola administrativa. El valor por omisión es que el ámbito no sea el nodo de la consola.
Tipo de datos Boolean
Valor por omisión false
Mostrar/ocultar mensaje de cabecera
Especifica si se visualizará el mensaje de cabecera de WebSphere Application Server en la parte superior de la consola administrativa. El valor por omisión es que se visualice el mensaje de cabecera.
Tipo de datos Boolean
Valor por omisión true
Mostrar/ocultar descripciones
Especifica si se visualizará la información de las páginas de la consola y los campos de las páginas. Los iconos de mostrar/ocultar descripciones de campo y de página que aparecen en la parte derecha de la barra de tareas proporcionan la misma función. El valor por omisión es mostrar las descripciones de campo y de página.
Tipo de datos Boolean
Valor por omisión true
