Áreas de la consola administrativa

Utilice la consola administrativa para crear y gestionar recursos, aplicaciones y servidores o bien para ver los mensajes del producto.

Para ver la consola administrativa, asegúrese de que el servidor de aplicaciones de la consola administrativa se esté ejecutando. Apunte a un navegador Web en el URL de la consola administrativa, entre el ID de usuario y, si es necesario, una contraseña en una página de inicio de sesión.

La consola tiene las siguientes áreas principales, que puede redimensionar según lo desee:

Barra de tareas
La barra de tareas ofrece opciones para volver a la página principal de la consola, guardar los cambios en las configuraciones administrativas, especificar preferencias en toda la consola, terminar la sesión de la consola y acceder a la información de productos.
Árbol de navegación
El árbol de navegación de la parte izquierda de la consola proporciona enlaces a las páginas de la consola que se utilizan para crear y gestionar componentes de una célula administrativa de WebSphere Application Server.

Pulse el signo + que aparece junto a un elemento o carpeta de árbol para expandir el árbol de la carpeta o elemento. Pulse el signo - para contraer el árbol de la carpeta o elemento. Pulse dos veces un elemento de la vista de árbol para conmutar entre los estados expandido y contraído.

Espacio de trabajo
El espacio de trabajo de la parte derecha de la consola contiene páginas que se utilizan para crear y gestionar objetos de configuración tales como los servidores y recursos. Pulse los enlaces del árbol de navegación para ver los distintos tipos de objetos configurados. En el espacio de trabajo, pulse los objetos configurados para ver las configuraciones, estados de tiempo de ejecución y opciones correspondientes. Pulse Inicio en la barra de tareas para visualizar la página principal del espacio de trabajo, que contiene enlaces con información sobre cómo utilizar el producto WebSphere Application Server.
Estado de WebSphere
El área de mensajes de estado situada en la parte inferior de la consola ofrece información sobre los mensajes devueltos por WebSphere Application Server relacionados con problemas de la configuración administrativa así como con sucesos de tiempo de ejecución. Pulse Anterior o Siguiente para conmutar entre las pantallas de los problemas de configuración y los sucesos de tiempo de ejecución.

Puede ajustar el intervalo entre renovaciones automáticas en los valores de Preferencias.
