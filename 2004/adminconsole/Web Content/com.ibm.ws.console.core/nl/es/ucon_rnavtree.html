Acciones del árbol de navegación de la consola administrativa

Utilice el árbol de navegación de la consola administrativa para acceder a las páginas para crear y gestionar servidores, aplicaciones, recursos y otros componentes.

Para ver el árbol de navegación, vaya a la consola administrativa de WebSphere Application Server y busque el árbol a la izquierda de la consola. El árbol permite el desplazamiento a las tareas de configuración y la información de tiempo de ejecución. Los temas principales que están disponibles en el árbol se muestran a continuación. Para utilizar el árbol, expanda un tema principal de interés y seleccione un elemento de la lista ampliada para visualizar una página en la que pueda finalizar el trabajo.

Servidores
Permite configurar los servidores de administración y otros tipos de servidores, como por ejemplo, los servidores JMS.
Aplicaciones
Permite instalar aplicaciones en servidores y gestionar las aplicaciones instaladas.
Recursos
Permite configurar recursos y ver información sobre los recursos que existen en la célula de administración.
Seguridad
Sirve para acceder al Centro de Seguridad, que se utiliza para proteger las aplicaciones y servidores y gestionar los usuarios de la consola administrativa.
Entorno
Permite configurar sistemas principales, servidores Web, variables y otros componentes.
Resolución de problemas  
Permite comprobar y hacer un seguimiento de errores y problemas de configuración.
