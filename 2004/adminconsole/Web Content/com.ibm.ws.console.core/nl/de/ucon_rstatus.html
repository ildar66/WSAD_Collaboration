Einstellungen des WebSphere-Status

Im WebSphere-Statusbereich der Administrationskonsole können Sie Fehlernachrichten und Nachrichten zu Laufzeitereignissen, die von WebSphere Application Server zurückgegeben werden, anzeigen.

Der WebSphere-Statusbereich befindet sich im unteren Bereich der Konsole und bleibt sichtbar, während Sie auf der WebSphere-Homepage zu anderen Seiten navigieren. Im Bereich werden zwei Rahmen angezeigt: WebSphere-Konfigurationsfehler und WebSphere-Laufzeitnachrichten. Klicken Sie auf Zurück oder Weiter, um zwischen den Anzeigen hin- und herzuschalten.

Klicken Sie auf das Symbol oben rechts, um die angezeigten Informationen zu aktualisieren. In den Einstellungen für die Vorgaben können Sie das Intervall zwischen den automatischen Aktualisierungen ändern.

WebSphere-Konfigurationsfehler
Zeigt die Anzahl der Arbeitsbereichsdateien an. In diesem Rahmen wird auch die Anzahl der Fehler in der Administrationskonfiguration für diese Benutzer-ID angezeigt.

Klicken Sie auf die Zahl, um detaillierte Informationen zu den Fehlern anzuzeigen.

WebSphere-Laufzeitnachrichten
Zeigt die Anzahl der von WebSphere Application Server zurückgegebenen Nachrichten sowie die Anzahl der Fehlernachrichten (Symbol x), Warnnachrichten (Symbol (! ) und Informationsnachrichten (Symbol i) an.

Klicken Sie auf die entsprechende Zahl, um detaillierte Informationen anzuzeigen.
