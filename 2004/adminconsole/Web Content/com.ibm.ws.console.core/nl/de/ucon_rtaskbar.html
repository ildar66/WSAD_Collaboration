Aktionen in der Task-Leiste der Administrationskonsole

Sie können über die Task-Leiste der Administrationskonsole zur Homepage der Konsole zurückkehren, Änderungen an Administrationskonfigurationen speichern, Konsolenvorgaben definieren, sich von der Konsole abmelden und auf Produktinformationen zugreifen.

Die Task-Leiste ist im oberen Abschnitt der Administrationskonsole von WebSphere Application Server horizontal angeordnet. Die Task-Leiste ermöglicht verschiedene Aktionen.

Home
Zeigt die Homepage der Administrationskonsole im Arbeitsbereich an. Der Arbeitsbereich befindet sich auf der rechten Seite der Konsole. Die Homepage enthält Links mit Informationen zur Verwendung von WebSphere Application Server.
Speichern
Zeigt die Seite "Speichern" an, auf der Sie die in der Administrationskonsole durchgeführte Arbeit speichern können.

Die Änderungen, die Sie an der Administrationskonfiguration vorgenommen haben, werden im Master-Repository gespeichert, das sich aus .xml-Konfigurationsdateien zusammensetzt. Wenn Sie sehen möchten, welche Änderungen gespeichert werden, blenden Sie Geänderte Einträge anzeigen ein. Sie müssen Ihre Arbeit in regelmäßigen Abständen speichern, um sicherzustellen, dass Änderungen, die Sie vorgenommen haben, nicht versehentlich verloren gehen.

Vorgaben
Zeigt die Seite "Vorgaben" an, auf der Sie angeben können, ob die Seite Arbeitsbereichsseite der Administrationskonsole nach Änderungen automatisch aktualisiert, Bestätigungsdialoge angezeigt und der Standardbereich als Knoten der Administrationskonsole verwendet werden sollen.
Abmelden
Führt die Abmeldung von der Sitzung der Administrationskonsole durch und zeigt die Seite "Anmelden" an. Wenn Sie die Administrationskonfiguration seit dem letzten Speichern der Konfiguration im Master-Repository geändert haben, wird die Seite "Speichern" angezeigt, bevor die Seite "Anmelden" wieder erscheint. Klicken Sie auf Speichern, um die Änderungen am Master-Repository zu speichern, auf Löschen, um die Sitzung ohne Speichern der Änderungen zu beenden, oder auf Abmelden, um die Sitzung zu verlassen, ohne die Änderungen zu speichern. Im letzten Fall haben Sie die Möglichkeit, die Änderungen wiederherzustellen, wenn Sie in die Konsole zurückkehren.
Hilfe
Öffnet einen neuen Webbrowser für die Onlinehilfe von WebSphere Application Server.
Feld- und Seitenbeschreibungen anzeigen/verdecken
Mit dieser Option können Sie angeben, ob Informationen zu Konsolseiten und -felder auf der Seite angezeigt werden. Mit den Symbolen auf der rechten Seite der Task-Leiste können Sie hin- und herschalten.
