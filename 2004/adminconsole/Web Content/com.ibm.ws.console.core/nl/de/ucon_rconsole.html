Bereiche der Administrationskonsole

Mit der Administrationskonsole können Sie Ressourcen, Anwendungen und Server erstellen und verwalten sowie Produktnachrichten anzeigen.

Wenn Sie die Administrationskonsole anzeigen möchten, müssen Sie sich vergewissern, dass der Anwendungsserver für die Administrationskonsole ausgeführt wird. Geben Sie dann in einem Webbrowser den URL für die Administrationskonsole ein. Geben Sie auf einer Anmeldeseite Ihre Benutzer-ID und, falls erforderlich, ein Kennwort ein.

Die Konsole setzt sich aus folgenden Bereichen zusammen, deren Größe Sie wie gewünscht anpassen können:

Task-Leiste
Die Task-Leiste enthält Optionen für die Rückkehr zur Homepage der Konsole, das Speichern von Änderungen, die Sie an den Administrationskonfigurationen vorgenommen haben, die Angabe von Konsolenvorgaben, das Abmelden von der Konsole und den Zugriff auf Produktinformationen.
Navigationsstruktur
Die Navigationsstruktur auf der linken Seite der Konsole enthält Links zu Konsolseiten, auf denen Sie Komponenten in einer Administrationszelle des WebSphere Application Server erstellen und verwalten können.

Klicken Sie auf das Pluszeichen (+) neben einem Ordner oder Element, um dessen Struktur einzublenden. Klicken Sie auf das Minuszeichen (-) neben einem Ordner oder Element, um dessen Struktur auszublenden. Klicken Sie doppelt auf ein Element in der Baumstruktur, um zwischen dem Status "Eingeblendet" und dem Status "Ausgeblendet" hin- und herzuschalten.

WorkSpace
Der Arbeitsbereich auf der rechten Seite der Konsole enthält Seiten, auf denen Sie Konfigurationsobjekte, wie z. B. Server und Ressourcen, erstellen und verwalten können. Klicken Sie in der Navigationsstruktur auf die entsprechenden Links, um die verschiedenen Typen konfigurierter Objekte anzuzeigen. Klicken Sie im Arbeitsbereich auf konfigurierte Objekte, um deren Konfigurationen, Laufzeitstatus und Optionen anzuzeigen. Klicken Sie in der Task-Leiste auf Home, um die Homepage des Arbeitsbereichs anzuzeigen, auf der sich Links zur Verwendung des WebSphere Application Server befinden.
WebSphere-Status
Im Bereich für die Statusnachrichten im unteren Bereich der Konsole werden Informationen zu den vom WebSphere Application Server zurückgegebenen Nachrichten, zu Fehlern in der Administrationskonfiguration sowie zu Laufzeitereignissen angezeigt. Klicken Sie auf Zurück oder Weiter, um zwischen der Anzeige für Konfigurationsfehler und der Anzeige für Laufzeitereignisse hin- und herzuschalten.

In den Einstellungen für die Vorgaben können Sie das Intervall zwischen den automatischen Aktualisierungen ändern.
