Aktionen in der Navigationsstruktur der Administrationskonsole

Über die Navigationsstruktur der Administrationskonsole können Sie auf Seiten zugreifen, um Server, Anwendungen, Ressourcen und andere Komponenten zu erstellen und zu verwalten.

Zum Anzeigen der Navigationsstruktur rufen Sie die Administrationskonsole von WebSphere Application Server auf, und prüfen Sie die Baumstruktur auf der linken Seite der Konsole. In der Baumstruktur können Sie zu Konfigurations-Tasks und Laufzeitinformationen navigieren. Die wichtigsten verfügbaren Topics sind in der unten angezeigten Baumstruktur verfügbar. Möchten Sie die Baumstruktursicht verwenden, blenden Sie ein Haupt-Topic ein, und wählen Sie eine Option aus der eingeblendeten Liste aus. Es wird dann eine Seite angezeigt, auf der Sie die erforderlichen Aktionen durchführen können.

Server
Ermöglicht die Konfiguration von Administrationsservern und anderen Servertypen, wie z. B. JMS-Servern.
Anwendungen
Ermöglicht die Installation von Anwendungen auf Servern und die Verwaltung installierter Anwendungen.
Ressourcen
Ermöglicht die Konfiguration von Ressourcen und die Anzeige von Informationen zu Ressourcen, die in der Administrationszelle vorhanden sind.
Sicherheit
Greift auf das Sicherheitscenter zu, mit dem Sie Anwendungen und Server sichern und Benutzer der Administrationskonsole verwalten können.
Umgebung
Ermöglicht die Konfiguration von Hosts, Webservern, Variablen und anderen Komponenten.
Fehlerbehebung  
Bietet Ihnen die Möglichkeit, nach Konfigurationsfehlern und anderen Fehlern zu suchen und diese zu protokollieren.
