Vorgaben für die Administrationskonsole

Geben Sie in den Vorgaben an, wie Informationen auf einer Administrationskonsolseite angezeigt werden sollen.

Maximale Zeilenanzahl
Die Maximalanzahl der Zeilen, die pro Seite angezeigt werden sollen, wenn die Objektgruppe umfangreich ist.
Filterprotokoll
Gibt an, ob für die Anzeige dieser Seite beim nächsten Aufruf der Seite dieselben Filterkriterien verwendet werden sollen.

Wählen Sie das Markierungsfeld Filterkriterien beibehalten aus, um die zuletzt eingegebenen Filterkriterien beizubehalten. Wenn Sie später auf die Seite zurückkehren, steuern die beibehaltenen Filterkriterien die Liste der Anwendungen, die in der Tabelle erscheint.
