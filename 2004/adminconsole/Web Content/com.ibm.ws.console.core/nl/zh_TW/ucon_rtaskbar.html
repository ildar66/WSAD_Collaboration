管理主控台工作列動作

請利用管理主控台工作列來返回主控台首頁、儲存管理配置的變更、指定主控台喜好設定、登出主控台，以及存取產品資訊。

如果要檢視工作列，請移至 WebSphere Application Server 管理主控台，查看接近主控台頂端的水平列。 工作列提供數個動作。

首頁
在工作區中顯示管理主控台首頁。 工作區在主控台右側。 首頁有多個鏈結可讓您取得如何使用 WebSphere Application Server 產品的相關資訊。
儲存
顯示儲存頁面，您可以使用它來儲存您已在管理主控台中進行的工作。

對管理配置所做的變更將儲存到由 .xml 配置檔所組成的主要儲存庫。如果要知道儲存了哪些變更，請展開檢視變更項目。 定期地儲存您的工作，以確定您不會不慎地失去您所做的變更。

喜好設定
顯示「喜好設定」頁面，讓您指定管理主控台工作區頁面在變更之後要不要自動重新整理、要不要顯示確認對話框，以及要不要使預設範圍成為管理主控台節點。
登出
登出管理主控台階段作業並顯示「登入」頁面。 如果從上次將配置儲存到主要儲存庫之後，您又變更了管理配置，在返回「登入」頁面之前，會顯示「儲存」頁面。 請按一下儲存將變更儲存在主要儲存庫中、按一下捨棄結束階段作業而不儲存變更，或按一下登出來結束階段作業，不儲存變更，但在返回主控台時有機會回復變更。
說明
開啟新的 Web 瀏覽器來顯示 WebSphere Application Server 產品的線上說明。
「隱藏/顯示欄位和頁面說明」切換
可讓您選取是否要顯示主控台頁面及頁面內欄位上的資訊。 工作列右手邊的圖示提供切換動作。
