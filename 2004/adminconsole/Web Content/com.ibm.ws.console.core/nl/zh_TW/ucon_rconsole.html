管理主控台區

請利用管理主控台來建立和管理資源、應用程式和伺服器，或檢視產品訊息。

如果要檢視管理主控台，請確定管理主控台的應用程式伺服器在執行中。 將 Web 瀏覽器指向管理主控台的 URL，在「登入」頁面中輸入您的使用者 ID，必要時，也要輸入密碼。

主控台有下列主要區域，您可以依照需要來調整它們的大小：

作業列
工作列提供各種選項，讓您返回主控台首頁、儲存管理配置的變更、指定全主控台的喜好設定、登出主控台，以及存取產品資訊。
導覽樹狀結構
主控台左側的導覽樹狀結構提供主控台頁面的鏈結，您可以使用這些頁面來建立及管理 WebSphere Application Server 管理 Cell 中的元件。

按一下樹狀結構資料夾或項目旁的 + 會展開資料夾或項目的樹狀結構。 按一下 - 會收合資料夾或項目的樹狀結構。 按兩下樹狀視圖中的項目，在已展開與已收合之間切換它的狀態。

工作區
主控台右側的工作區有若干頁面，您可以利用這些頁面來建立和管理伺服器和資源之類的配置物件。 請按一下導覽樹狀結構中的鏈結來檢視不同類型的配置物件。 在工作區內按一下配置的物件，會檢視它們的配置、執行時期狀態和選項。 在工作列中按一下首頁會顯示工作區首頁，其中有通往如何使用 WebSphere Application Server 產品的相關資訊之鏈結。
WebSphere 狀態
主控台底端的狀態訊息區會提供 WebSphere Application Server 傳回的有關管理配置問題之訊息的相關資訊，以及執行時期事件訊息的相關資訊。 請按上一頁或下一頁來切換配置問題和執行時期事件的顯示畫面。

您可以在喜好設定中調整自動重新整理的間隔。
