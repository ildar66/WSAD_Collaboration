喜好設定

請利用「喜好設定」頁面來指定管理主控台工作區在變更之後要不要自動重新整理、預設範圍要不要成為管理主控台節點、要不要顯示確認對話框，以及要不要顯示工作台橫幅和說明。

如果要檢視這個管理主控台頁面，請按一下主控台工作列上的喜好設定。

啟用工作區自動重新整理
指定在管理配置變更之後，管理主控台工作區要不要自動重畫。

預設值就是工作區要自動重畫。 如果您引導主控台建立例如應用程式伺服器的新實例，「應用程式伺服器」頁面就會自動重新整理，且會在伺服器集合中顯示新伺服器的名稱。

指定工作區不自動重畫表示您必須重新存取頁面，您要按一下主控台導覽樹狀結構或集合頁面中的鏈結來查看對管理配置的變更。

資料類型 Boolean
預設值 true
不確認工作區捨棄
指定在要求刪除物件時，要不要顯示確認對話框。 預設值是不顯示確認對話框。
資料類型 Boolean
預設值 false
使用預設範圍
指定預設範圍是管理主控台節點。 預設值是範圍不是主控台節點。
資料類型 Boolean
預設值 false
隱藏/顯示橫幅
指定要不要在管理主控台頂端顯示 WebSphere Application Server 橫幅。預設值是要顯示橫幅。
資料類型 Boolean
預設值 true
隱藏/顯示說明
指定要不要顯示主控台頁面和頁面內欄位的相關資訊。 工作列右側的「隱藏/顯示欄位」和「頁面說明」圖示提供相同的功能。 預設值是顯示頁面和欄位說明。
資料類型 Boolean
預設值 true
