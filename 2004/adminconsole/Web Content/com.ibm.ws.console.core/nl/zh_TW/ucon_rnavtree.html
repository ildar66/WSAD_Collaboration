管理主控台導覽樹狀結構動作

請利用管理主控台的導覽樹狀結構來存取建立和管理伺服器、應用程式、資源及其他元件時所用的頁面。

如果要檢視導覽樹狀結構，請進入 WebSphere Application Server 管理主控台，再檢視主控台左側的樹狀結構。 這個樹狀結構可用來導覽配置作業和執行時期資訊。 樹狀結構中可用的主要主題顯示在下面。 如果要使用樹狀結構，請展開您想要的主要主題，再從展開的清單中選取一個項目來顯示您可以在其中完成工作的頁面。

伺服器
可讓您配置管理伺服器及 JMS 伺服器之類的其他伺服器類型。
應用程式
可讓您將應用程式安裝到伺服器並管理已安裝的應用程式。
資源
可讓您配置資源並檢視存在於管理 Cell 之資源的相關資訊。
安全
存取用來維護應用程式和伺服器安全及管理「管理主控台」使用者的「安全中心」。
環境
可讓您配置主機、Web 伺服器、變數及其他元件。
疑難排解  
可讓您檢查是否有配置錯誤及問題，並追蹤它們。
