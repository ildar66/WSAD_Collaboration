WebSphere 狀態設定

請利用管理主控台的「WebSphere 狀態區」來檢視 WebSphere Application Server 傳回的錯誤和執行時期事件訊息。

「WebSphere 狀態區」會單獨顯示於主控台的底端，當您從「WebSphere 首頁」導覽到其他頁面時，您還是會一直看到它。該區域會顯示兩個 頁框：WebSphere 配置問題和 WebSphere 執行時期訊息。您可按上一頁或下一頁來切換頁框。

按一下區域右上方會重新整理所顯示的資訊。您可以在喜好設定中調整自動重新整理的間隔。

WebSphere 配置問題
顯示工作區檔案數目。這個頁框也會顯示使用者 ID 的管理配置問題數目。

按一下該數目可以檢視這些問題的詳細資訊。

WebSphere 執行時期訊息
顯示 WebSphere Application Server 傳回的訊息數目以及錯誤訊息（x 圖示）、警告訊息（! 圖示）和參考訊息（i 圖示）的數目。

按一下訊息的數目可以檢視詳細資料。
