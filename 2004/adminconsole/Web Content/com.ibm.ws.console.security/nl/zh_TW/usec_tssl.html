Secure Socket Layer 配置儲存庫設定

請利用這個頁面來定義新的 Secure Socket Layer (SSL) 別名。 當使用 SSL 配置儲存庫時，管理者可以定義任意數目的 SSL 設定，以便用來配置 SSL 超文字轉送通訊協定 (HTTPS)、SSL 網際網路交互 ORB 通訊協定 (IIOPS) 或 SSL 輕裝備目錄存取通訊協定 (LDAPS) 連線。 您可以從管理主控台內任何支援 SSL 連線的位置來選取這裡所定義的 SSL 設定之一。 這簡化了 SSL 配置程序，因為只要在很多位置指定別名，您就可以重複使用這些 SSL 配置。

如果要檢視這個管理主控台頁面，請按一下安全 > SSL。

請按一下新建來建立新的 SSL 配置儲存庫別名。

請按一下 刪除 來移除 SSL 配置儲存庫別名。 如果在配置中參照 SSL 配置別名，但在這裡刪除它，當存取刪除的別名時，SSL 連線會失敗。

別名
指定特定 SSL 設定的名稱。
