加密記號設定

請利用這個頁面來配置加密記號設定。

如果要檢視這個管理主控台頁面，請按一下安全 > SSL > alias_name > 加密記號。

「配置」標籤

記號類型
指定在加密記號中實作的內建金鑰儲存庫檔類型，如 PKCS#11。

WebSphere Application Server 會利用 Java Secure Socket Extension (JSSE) 實作，以 Secure Socket Layer (SSL) 來支援加密記號。 支援不同的加密裝置。SSL 伺服器有下列支援裝置：

  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
SSL 用戶端有下列支援裝置：
  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
  • IBM Security Kit Smartcard
  • GemPlus Smartcards
  • Rainbow iKey 1000/2000（USB "Smartcard" 裝置）
  • Eracom CSA8000
請遵循裝置所附文件的說明來安裝您的加密記號。

資料類型： 字串
程式庫檔
指定實作加密記號裝置介面的動態鏈結程式庫 (DLL) 或共用物件。

資料類型： 字串
密碼
指定加密記號裝置的密碼。
資料類型： 字串
