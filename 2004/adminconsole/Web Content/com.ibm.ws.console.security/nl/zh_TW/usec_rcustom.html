自訂使用者登錄設定

您可利用這個頁面來配置自訂使用者登錄。

如果要檢視這個管理主控台頁面，請按一下安全 > 使用者登錄 > 自訂。

一旦在這個畫面設定內容，請按一下套用。請利用內容畫面來處理自訂登錄所需要的其他內容。 當啟用安全且任何這些內容有了改變時，請進入廣域安全畫面，按一下套用來驗證變更。

「配置」標籤

伺服器使用者 ID
指定為了安全而在伺服器執行時採用的使用者 ID。

這個伺服器 ID 代表自訂登錄中的有效使用者。

資料類型： 字串
伺服器使用者密碼
指定安全伺服器 ID 所對應的密碼。
資料類型： 字串
自訂登錄類別名稱
指定實作 com.ibm.websphere.security.UserRegistry 介面的帶點類別名稱。

請將自訂登錄類別名稱放在類別路徑中。 建議的位置是 %install_root%/lib/ext 目錄。 雖然自訂登錄實作 com.ibm.websphere.security.UserRegistry 介面，但為了與舊版相容，使用者登錄也可以另行實作 com.ibm.websphere.security.CustomRegistry 介面。

資料類型： 字串
預設值： com.ibm.websphere.security.FileRegistrySample
不區分大小寫
可指定執行不區分大小寫授權檢查。
預設值： 啟用
範圍： 啟用或停用
請利用自訂內容鏈結來新增起始設定自訂登錄所需要的其他內容。 以下是是產品預先定義的內容；請只在必要時才設定這個內容：
  • WAS_UseDisplayName--當設為 true 時，getCallerPrincipal()、getUserPrincipal()、getRemoteUser() 等方法會傳回顯示名稱。 依預設，會傳回使用者的 securityName。 此舉主要是為了支援與 4.0 版自訂登錄的反向相容性。
