信任關聯設定

信任關聯會啟用 IBM WebSphere Application Server 安全和協力廠商安全伺服器的整合。 更明確地說，反向 Proxy 伺服器可以作為前端鑑別伺服器，產品會將它本身的權限原則套用在 Proxy 伺服器所傳遞的結果憑證上。 請利用這個頁面來配置信任關聯設定。

如果要檢視這個管理主控台頁面，請按一下安全中心 > 鑑別機制 > LTPA > 信任關聯。

當啟用安全且任何這些內容有了改變時，請進入廣域安全畫面，按一下套用來驗證變更。

「配置」標籤

啟用
指定要不要啟用信任關聯。
資料類型： Boolean
預設值： 停用
範圍： 啟用或停用
