Java 2 連接器鑑別資料項目設定

請利用這個頁面作為中央位置，供管理者定義鑑別資料，其中包括使用者身分和密碼。 資源配接器、資料來源和需要鑑別資料的其他配置都可以利用別名來參照這些鑑別資料項目。

您可以從 JAAS 配置頁面或使用 J2C 鑑別資料項目之資源的其他頁面中，直接顯示這個頁面。 比方說，如果要檢視這個管理頁面中，您可以按一下安全 > JAAS 配置 > J2C 鑑別資料項目或資源 > WebSphere JMS Provider > WebSphere Queue Connection Factory > connection_factory > J2C 鑑別資料項目。

附註: 當刪除鑑別資料項目時，請特別小心。 如果其他配置使用刪除的鑑別資料，起始設定資源程序就會失敗。

按一下新建來定義新的鑑別資料項目。

別名
指定鑑別資料項目的名稱。
資料類型： 字串
單位： 字串
預設值： 無
使用者 ID
指定使用者身分。
資料類型： 字串
說明
指定鑑別資料項目的選擇性說明。 比方說，這個鑑別資料項目用來連接 DB2。
資料類型： 字串
