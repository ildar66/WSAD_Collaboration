小型認證機構設定

請利用這個頁面來配置小型認證機構 (LTPA) 設定。

如果要檢視這個管理主控台頁面，請按一下安全 > 鑑別機制 > LTPA。

如果您是第一次配置安全性，只有密碼是必要的。 輸入密碼之後，請按一下套用。 請按一下單一登入 (SSO)，再輸入領域名稱。 確定已啟用 SSO。 請按一下套用。 如果要完成安全設定，請確定您設定了適當的登錄，再按一下「廣域安全」畫面中的套用。 當啟用安全且任何這些內容有了改變時，請進入「廣域安全」畫面，按一下套用來驗證變更。

「配置」標籤

產生金鑰
指定伺服器要不要產生新的 LTPA 金鑰。

當安全性第一次開啟並以 LTPA 作為鑑別機制時，系統會自動以 畫面中輸入的密碼產生 LTPA 金鑰。 如果您需要使用之前設定的密碼，來產生新的金鑰集，請按一下產生金鑰。 如果已經使用新密碼，就不要按這個按鈕。 當您輸入新密碼，並且按一下確定或套用時，就會產生新的金鑰集。 每當新的金鑰集產生時，除非您儲存它們，否則不會使用它們。

匯入金鑰
指定伺服器要不要匯入新的 LTPA 金鑰。

如果要跨越多個 WebSphere 領域 (Cell) 來支援 WebSphere 產品中的單一登入 (SSO)，這些領域要共用 LTPA 金鑰和密碼。 您可以利用匯入金鑰按鈕，從其他領域匯入 LTPA 金鑰。LTPA 金鑰應該已經從其中一個 Cell 匯出到某個檔案。 為了匯入新的 LTPA 金鑰集，請輸入適當的密碼和 LTPA 金鑰所在的檔名。 接著只要按一下匯入金鑰即可，不要按一下 確定或套用。

匯出金鑰
指定伺服器要不要匯出 LTPA 金鑰。

如果要跨越多個 WebSphere 領域 (Cell) 來支援 WebSphere 產品中的單一登入 (SSO)，這些領域要共用 LTPA 金鑰和密碼。 您可以利用匯出金鑰按鈕，將 LTPA 金鑰匯出至其他領域。

如果要匯出 LTPA 金鑰，系統必須啟用安全，而且使用 LTPA。 然後在金鑰檔名稱欄位中輸入檔案名稱，接著按一下匯出金鑰。 加密的金鑰會儲存在指定的檔案中。

密碼
指定用來加密和解密 LTPA 金鑰的密碼。 當匯入這些金鑰到其他 WebSphere Application Server 管理領域配置（如果有的話）時，以及在配置 Domino 伺服器的 SSO 時，請使用這個密碼。

產生或匯入金鑰之後，會利用它們來加密及解密 LTPA 記號。 每當密碼變更時，只要您按一下確定或套用，就會自動產生新的 LTPA 金鑰集。 這些新的金鑰集只會在您儲存它們時，才會派上用場。

資料類型 字串
確認密碼
指定用來加密和解密 LTPA 金鑰的確認密碼。

將這些金鑰匯入其他 WebSphere Application Server 管理領域配置（如果有的話）時，以及在配置 SSO for Domino Server 時，請使用這個密碼。

資料類型 字串
逾時值
指定 LTPA 記號到期之前的時間長度（分鐘）。請確定這個時間長度比「廣域安全」畫面中配置的快取逾時值長。
資料類型 整數
單位 分鐘
預設值 120
金鑰檔名稱
指定匯入或匯出金鑰時所用的檔案名稱。

請輸入完整的金鑰檔名稱，然後再按一下匯入金鑰或匯出金鑰。
