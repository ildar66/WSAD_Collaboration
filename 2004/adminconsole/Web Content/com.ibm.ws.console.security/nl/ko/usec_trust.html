신뢰 연관 설정

신뢰 연관을 사용하면 IBM WebSphere Application Server 보안 및 타사 보안 서버의 통합이 가능해집니다. 더 구체적으로, 반전 프록시 서버가 프론트 엔드 인증 서버 역할을 수행하는 반면, 이 제품은 자체 권한 정책을 프록시 서버에서 전달한 결과 신임에 적용합니다. 이 페이지를 참조하여 신뢰 연관 설정을 구성하십시오.

이 관리 콘솔 페이지를 보려면 보안 센터 > 인증 메커니즘 > LTPA > 신뢰 연관을 클릭하십시오.

보안이 사용 가능하고 이 등록 정보 중 하나가 변경되는 경우 글로벌 보안 패널로 이동하여 적용을 클릭하여 변경사항의 유효성을 검증하십시오.
