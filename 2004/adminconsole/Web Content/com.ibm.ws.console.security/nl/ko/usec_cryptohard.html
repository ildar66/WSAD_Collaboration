암호 토큰 설정

이 페이지를 참조하여 암호 토큰 설정을 구성하십시오.

이 관리 콘솔 페이지를 보려면 보안 > SSL > alias_name > 암호 토큰을 클릭하십시오.

구성 탭

토큰 유형
PKCS#11과 같은 암호 토큰으로 구현된 내장 keystore 파일의 유형을 지정합니다.

WebSphere Application Server는 JSSE(Java Secure Socket Extension)를 사용하여 SSL(Secure Socket Layer)에서의 암호화 토큰을 지원합니다. 다른 암호 장치가 지원됩니다. SSL 서버의 경우, 다음 장치가 지원됩니다.

  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
SSL 클라이언트의 경우, 다음 장치가 지원됩니다.
  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
  • IBM Security Kit Smartcard
  • GemPlus Smartcards
  • Rainbow iKey 1000/2000(USB "Smartcard" 장치)
  • Eracom CSA8000
암호화 토큰을 설치하려면 사용자 장치에 해당하는 문서를 따르십시오.

데이터 유형: 문자열
라이브러리 파일
암호화 토큰 장치에 대한 인터페이스를 구현하는 DLL(Dynamic Link Library) 또는 공유 오브젝트를 지정합니다.

데이터 유형: 문자열
암호
암호화 토큰 장치의 암호를 지정합니다.
데이터 유형: 문자열
