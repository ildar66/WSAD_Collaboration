Java 2 커넥터 인증 데이터 항목 설정

관리자에 대한 중앙 위치로서 이 페이지를 참조하여 사용자 ID 및 암호를 포함하는 인증 데이터를 정의하십시오. 이들은 자원 어댑터, 데이터소스 및 별명을 사용하는 인증 데이터가 필요한 기타 구성으로 인증 데이터 항목을 참조할 수 있습니다.

JAAS 구성 페이지에서 직접 또는 J2C 인증 데이터 항목을 사용하는 자원에 대한 기타 페이지에서 이 페이지를 표시할 수 있습니다. 예를 들어, 이 관리 페이지를 보려면 보안 > JAAS 구성 > J2C 인증 데이터 항목을 클릭하거나 자원 > WebSphere JMS 프로바이더 > WebSphere 대기열 연결 팩토리 > connection_factory > J2C 인증 데이터 항목을 클릭하십시오.

참고: 인증 데이터 항목 제거 시 주의하십시오. 기타 구성이 제거된 인증 데이터를 사용하는 경우, 초기화 자원 프로세스는 실패합니다.
