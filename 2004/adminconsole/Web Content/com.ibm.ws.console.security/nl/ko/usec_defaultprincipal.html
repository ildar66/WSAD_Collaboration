응용프로그램 로그인 구성 설정

응용프로그램 로그인 구성을 구성하려면 이 페이지를 참조하십시오.

이 관리 콘솔 페이지를 보려면 보안 > JAAS 구성 > 응용프로그램 로그인 > alias_name을 클릭하십시오.

변경사항을 저장하고 원래의 별명 앞에 여분의 노드 이름을 추가하려면 적용을 클릭하십시오. 확인을 눌러도 새로운 변경사항이 security.xml 파일에 저장되지 않습니다.

별명
응용프로그램 로그인의 별명을 지정하십시오.

JAAS 로그인 구성 항목을 정의할 때 별명에 슬래시 기호(/)를 사용하지 마십시오. JAAS 로그인 구성 구문 분석기는 슬래시 기호를 처리할 수 없습니다.

데이터 유형: 문자열
