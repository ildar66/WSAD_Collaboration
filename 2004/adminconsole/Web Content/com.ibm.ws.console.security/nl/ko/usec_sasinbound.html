Secure Association Service 전송 인바운드 설정

이 페이지를 참조하여 SAS(Secure Association Service) 인증 프로토콜을 사용하여 이 서버가 승인하는 연결에 대한 전송 설정을 지정하십시오. 프로토콜은 이전 WebSphere Application Server 릴리스를 사용하여 안전하게 Enterprise Bean에 통신하는 데 사용됩니다.

이 관리 콘솔 페이지를 보려면 보안 > 인증 프로토콜 > SAS 인바운드 전송을 클릭하십시오.

구성 탭

SSL 설정
인바운드 연결을 위해 선택할 사전 정의된 SSL 설정값 목록을 지정합니다. 이 설정은 SSL 레퍼토리 패널에서 구성됩니다.
데이터 유형: 문자열
기본값: DefaultSSLSettings
