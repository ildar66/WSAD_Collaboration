SSL(Secure Socket Layer) 구성 레퍼토리 설정

이 페이지를 참조하여 새 SSL(Secure Socket Layer) 별명을 정의하십시오. SSL(Secure Socket Layer) 구성 레퍼토리를 통해 관리자가 HTTPS(Hypertext Transfer Protocol with SSL), IIOPS(Internet InterORB Protocol with SSL) 또는 LDAPS(Lightweight Directory Access Protocol with SSL) 연결을 구성하는 데 사용할 SSL(Secure Socket Layer) 설정을 원하는 만큼 정의할 수 있습니다. 여기서 정의된 SSL 설정값 중 하나를 SSL 연결을 지원하는 관리 콘솔 내의 임의의 위치에서 선택할 수 있습니다. 여러 위치에서 별명을 지정하여 이 SSL 구성 중 여러 개를 다시 사용할 수 있으므로 이 기능은 SSL 구성 프로세스를 단순화합니다.

이 관리 콘솔 페이지를 보려면 보안 > SSL을 클릭하십시오.

새로 작성을 클릭하여 새 SSL 구성 레퍼토리 별명을 작성하십시오.

삭제를 클릭하여 SSL 구성 레퍼토리 별명을 제거하십시오. SSL 구성 별명이 구성에서 참조되고 여기에 삭제된 다음 SSL 별명이 액세스될 때 SSL 연결에 실패합니다
