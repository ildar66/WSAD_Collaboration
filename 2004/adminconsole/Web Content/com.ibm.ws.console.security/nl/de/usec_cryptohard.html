Einstellungen für Verschlüsselungs-Token

Auf dieser Seite können Sie die Einstellungen für Verschlüsselungs-Token konfigurieren.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Sicherheit > SSL > Aliasname > Verschlüsselungs-Token.

Register "Konfiguration"

Token-Typ
Der Typ der integrierten Keystore-Datei, die im Verschlüsselungs-Token implementiert ist, z. B. PKCS#11.

WebSphere Application Server verwendet eine Implementierung von Java Secure Socket Extension (JSSE) für die Unterstützung von Verschlüsselungs-Token mit Secure Sockets Layer (SSL). Verschiedene Verschlüsselungseinheiten werden unterstützt. Für einen SSL-Server werden die folgenden Einheiten unterstützt:

  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
Für einen SSL-Client werden die folgenden Einheiten unterstützt:
  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
  • IBM Security Kit Smartcard
  • GemPlus Smartcards
  • Rainbow iKey 1000/2000 (USB "Smartcard" device)
  • Eracom CSA8000
Folgen Sie den Anweisungen in der Dokumentation zu Ihrer Einheit, um das Verschlüsselungs-Token zu installieren.

Datentyp String
Bibliotheksdatei
