Einstellungen für Trust Association

Trust Association ermöglicht die Integration der Sicherheit von IBM WebSphere Application Server mit Sicherheitsservern von Fremdanbietern. Insbesondere kann ein Reverse-Proxy-Server als Front-End-Authentifizierungsserver arbeiten, während das Produkt seine eigene Autorisierungsrichtlinie auf die sich ergebenden Berechtigungsnachweise anwendet, die vom Proxy-Server weitergegeben werden. Auf dieser Seite können Sie Einstellungen für Trust Associations konfigurieren.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Sicherheitscenter > Authentifizierungsverfahren > LTPA > Trust Association.

Wenn bei aktivierter Sicherheitseinrichtung eines dieser Merkmale geändert wird, müssen Sie in die Anzeige Globale Sicherheit wechseln und auf Anwenden klicken, um die Änderungen zu überprüfen.
