Einstellungen des SAS-Transports (Secure Association Service) für abgehende Verbindungen

Auf dieser Seite können Sie die Transporteinstellungen für Verbindungen angeben, die von diesem Server anhand des SAS-Authentifizierungsprotokolls (Secure Association Service) akzeptiert werden. Verwenden Sie das Protokoll SAS, um sicher mit Enterprise-Beans aus vorherigen Releases von WebSphere Application Server zu kommunizieren.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Sicherheit > Authentifizierungsprotokoll > Abgehender SAS-Transport.
