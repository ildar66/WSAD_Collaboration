Einstellungen für Serversicherheit

Auf dieser Seite können Sie die Sicherheitseinstellungen des Servers konfigurieren und die globalen Sicherheitseinstellungen außer Kraft setzen. Falls Sie die globalen Standardeinstellungen für die Sicherheit wiederherstellen müssen, wählen Sie in der Administrationskonsole einfach das entsprechende Markierungsfeld ab.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Server > Anwendungsserver > Servername > Serversicherheit. Diese Einstellungen gelten nur für Network Deployment.

Sie können die Sicherheits für einzelne Anwendungsserver zwar inaktivieren, wenn die globale Sicherheit aktiviert ist. aber Sie können die Sicherheitseinrichtung für einen Anwendungsserver nicht aktivieren, wenn die globale Sicherheit inaktiviert ist. Wenn die Sicherheit des Anwendungsserver für Benutzeranforderungen inaktiviert ist, bleibt die Sicherheit für Administrations- und Namensservices für diesen Anwendungsserver weiterhin aktiviert, damit die Administrations- und Namensinfrastruktur geschützt ist. Vergewissern Sie sich, dass die Sicherheit für Namensservices die Zugriffsberechtigung Jeder für die Namensfunktion hat, die Sie im Benutzercode verwenden. Sie müssen die administrative Sicherheit nicht konfigurieren, weil der Benutzercode nicht direkt auf die Administrationsfunktionen zugreift. Der Benutzercode greift über die unterstützten Scripting-Tools auf die Administrationsfunktionen zu.

Sicherheit auf Serverebene
Gibt an ob der Server die Zellenstandardwerte für die Sicherheit außer Kraft setzen soll.

Zum Wiederherstellen der Zellenstandardwerte für die Sicherheit auf Serverebene klicken Sie auf Zellensicherheit verwenden. Klicken Sie auf Anwenden und wählen Sie dann Speichern aus, um die Änderungen auf Serverebene zu validieren.

Standardwert False
CSI
Gibt an ob der Server die Zellenstandardwerte für das CSI-Protokoll außer Kraft setzen soll.

Zum Wiederherstellen der Zellenstandardwerte für das Protokoll CSI klicken Sie auf Zellen-CSI verwenden . Klicken Sie auf Anwenden und wählen Sie dann Speichern aus, um die Änderungen auf Serverebene zu validieren.

Standardwert False
SAS
Gibt an ob der Server die Zellenstandardwerte für das Protokoll außer Kraft setzen soll.

Zum Wiederherstellen der Zellenstandardwerte für das Protokoll SAS klicken Sie auf Zellen-SAS verwenden . Klicken Sie auf Anwenden und wählen Sie dann Speichern aus, um die Änderungen auf Serverebene zu validieren.

Standardwert False
