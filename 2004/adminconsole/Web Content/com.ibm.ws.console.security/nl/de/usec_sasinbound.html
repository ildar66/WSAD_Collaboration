Einstellungen des SAS-Transports (Secure Association Service) für eingehende Verbindungen

Auf dieser Seite können Sie die Transporteinstellungen für Verbindungen angeben, die von diesem Server anhand des SAS-Authentifizierungsprotokolls (Secure Association Service) akzeptiert werden. Das SAS-Protokoll wird verwendet, um bei älteren Versionen des WebSphere Application Server eine sichere Datenübertragung zu Enterprise-Beans zu gewährleisten.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Sicherheit > Authentifizierungsprotokoll > Eingehender SAS-Transport.
