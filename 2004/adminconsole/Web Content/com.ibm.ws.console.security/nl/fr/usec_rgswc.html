Paramètres de sécurité du serveur

Cette page permet de configurer la sécurité du serveur et de remplacer les paramètres de sécurité globaux. Pour revenir aux paramètres de sécurité globaux par défaut, désélectionnez la case à cocher appropriée dans la console d'administration.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > nom_serveur > Sécurité du serveur. Ces paramètres s'appliquent uniquement à Network Deployment.

Vous pouvez désactiver la sécurité au niveau des serveurs d'applications individuels lorsque la sécurité globale est activée. Toutefois, vous ne pouvez pas activer la sécurité au niveau d'un serveur d'application individuel lorsque la sécurité globale est activée. Lorsque la sécurité du serveur d'applications est désactivée pour les demandes utilisateurs, la sécurité d'attribution de nom et d'administration est toujours activée pour le serveur d'applications afin que l'infrastructure d'attribution de nom et d'administration reste sécurisée. Pour éviter d'éventuels incidents, vérifiez que la sécurité de nommage dispose de l'accès Tous les utilisateurs pour la fonction de nommage que vous utilisez dans le code utilisateur. Vous n'avez pas besoin de configurer la sécurité d'administration car le code utilisateur n'accède pas directement aux fonctions d'administration. Le code utilisateur accède aux fonctions d'administration via les outils de scriptage pris en charge.

Sécurité au niveau du serveur
Indique si le serveur remplace les paramètres par défaut de cellule pour la sécurité.

Pour revenir aux paramètres par défaut des appels de la sécurité au niveau du serveur, cliquez surUtiliser la sécurité de la cellule. Cliquez sur Valider, puis sélectionnez Sauvegarder pour valider les modifications au niveau serveur.

Valeur par défaut False
CSI
Indique si le serveur remplace les paramètres par défaut de cellule pour le protocole CSI.

Pour revenir aux paramètres par défaut de cellule pour le protocole CSI, cliquez sur Utiliser la fonction CSI de la cellule . Cliquez sur Valider, puis sélectionnez Sauvegarder pour valider les modifications au niveau serveur.

Valeur par défaut False
SAS
Indique si le serveur remplace les paramètres par défaut de cellule pour le protocole .

Pour revenir aux paramètres par défaut de cellule pour le protocole SAS, cliquez sur Utiliser la fonction SAS de la cellule. Cliquez sur Valider, puis sélectionnez Sauvegarder pour valider les modifications au niveau serveur.

Valeur par défaut False
