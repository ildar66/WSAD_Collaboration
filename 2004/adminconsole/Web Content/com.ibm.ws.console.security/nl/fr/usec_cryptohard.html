Paramètres des jetons de chiffrement

Cette page permet de configurer les paramètres des jetons de chiffrement.

Pour afficher cette page de la console d'administration, cliquez sur Sécurité > SSL > nom_alias > Jeton de chiffrement.

Onglet Configuration

Type de jeton
Indique le type de fichier de clés intégré mis en oeuvre dans le jeton de chiffrement, tel que PKCS#11.

WebSphere Application Server utilise une implémentation de JSSE (Java Secure Socket Extension) pour prendre en charge le jeton de chiffrement avec SSL (Secure Sockets Layer). Les différents périphériques de chiffrement sont pris en charge. Pour un serveur SSL, les éléments suivants sont pris en charge :

  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
Pour un client SSL, les éléments suivants sont pris en charge :
  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
  • IBM Security Kit Smartcard
  • GemPlus Smartcards
  • Rainbow iKey 1000/2000 (dispositif USB "Smartcard")
  • Eracom CSA8000
Reportez-vous à la documentation fournie avec votre périphérique afin d'installer le jeton de chiffrement.

Type de données : Chaîne
Fichier bibliothèque
Indique la bibliothèque DLL dynamique ou l'objet partagé qui implémente l'interface associé au dispositif du jeton de chiffrement.

Type de données : Chaîne

Indique le mot de passe du périphérique de jeton de chiffrement.
