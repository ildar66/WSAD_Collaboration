Paramètres des communications entrantes SAS (Secure Association Service)

Cette page permet définir les paramètres de transport pour les connexions qui sont acceptées par ce serveur à l'aide du protocole d'authentification SAS (Secure Association Service). Le protocole SAS permet de communiquer en toute sécurité avec les beans enterprise créés avec des versions antérieures de WebSphere Application Server.

Pour afficher cette page de la console d'administration, cliquez sur Sécurité > Protocole d'authentification > Transport des communications entrantes SAS.
