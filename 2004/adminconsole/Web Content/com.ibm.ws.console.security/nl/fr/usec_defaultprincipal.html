Paramètres de configuration de la connexion de l'application

Cette page permet de configurer les configurations de connexion de l'application.

Pour afficher cette page de la console d'administration, cliquez sur Sécurité > > Configuration JAAS > Connexions de l'application > nom_alias.

Cliquez sur Appliquer pour sauvegarder les modifications et ajouter le nom de noeud supplémentaire qui précède le nom d'alias d'origine. Si vous cliquez sur OK, les nouvelles modifications ne sont pas sauvegardées dans le fichier security.xml.

Alias
Indique le nom de l'alias de la connexion de l'application.

N'utilisez pas le caractère de barre oblique (/) dans le nom de l'alias lors de la définition des entrées de configuration de connexion JAAS. Le programme d'analyse de configuration de connexion JAAS ne peut pas gérer le caractère de barre oblique.

Type de données : Chaîne
