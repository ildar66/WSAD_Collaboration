Paramètres du répertoire de configuration SSL (Secure Sockets Layer)

Cette page permet de définir un nouvel alias SSL (Secure Sockets Layer). Par le biais du répertoire de configuration SSL, les administrateurs peuvent définir autant de paramètres SSL qu'ils le souhaitent pour configurer des connexions HTTPS (Hypertext Transfer Protocol with SSL), IIOPS (Internet InterORB Protocol with SSL) ou LDAPS (Lightweight Directory Access Protocol with SSL). Vous pouvez extraire un des paramètres SSL définis ici à partir d'un emplacement de la console d'administration qui prend en charge les connexions SSL. Cette opération simplifie le processus de configuration, étant donné que vous réutilisez un grand nombre de configurations SSL en indiquant l'alias à plusieurs emplacements.

Pour afficher cette page de la console d'administration, cliquez sur Sécurité > SSL.

Cliquez sur Nouveau pour créer un alias de répertoire de configuration SSL.

Cliquez sur Supprimer pour supprimer un alias de répertoire de configuration SSL. Si un alias de configuration SSL est référencé dans la configuration et supprimé ici, la connexion SSL échoue au moment où vous accédez à l'alias supprimé.

Alias
Indique le nom du paramètre SSL spécifique.
