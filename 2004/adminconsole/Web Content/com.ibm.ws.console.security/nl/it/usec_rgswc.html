Impostazioni di sicurezza server

Utilizzare questa pagina per configurare la sicurezza server e sovrapporre le impostazioni di sicurezza globale. Se si desidera tornare alle impostazioni predefinite della sicurezza globale, deselezionare la casella di controllo appropriata nella console di gestione.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Sicurezza server. Tenere presente che queste impostazioni si applicano solo a Network Deployment.

È possibile disabilitare la sicurezza su singoli server delle applicazioni mentre la sicurezza globale è abilitata.Tuttavia, non è possibile abilitare la sicurezza su un singolo server delle applicazioni mentre la sicurezza globale è disattivata. Mentre è disabilitata la sicurezza del server delle applicazioni per le richieste dell'utente, la sicurezza di denominazione e di gestione è ancora abilitata per quel server delle applicazioni; in questo modo l'infrastruttura di denominazione e gestione rimane sicura. Per evitare problemi, verificare che la sicurezza di denominazione abbia l'accesso Ognuno alla funzione di denominazione che si sta utilizzando all'interno del codice utente. Non è necessario configurare la sicurezza di gestione perché il codice utente non ha accesso diretto alle funzioni di gestione. Il codice utente ha accesso alle funzioni di gestione attraverso gli strumenti di script supportati.
