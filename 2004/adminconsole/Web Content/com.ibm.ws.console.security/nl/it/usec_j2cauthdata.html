Impostazioni voci dati di autenticazione di Java 2 Connector

Utilizzare questa pagina come posizione principale degli amministratori per poter definire i dati di autenticazione che includono le identità utente e le password. Queste possono riferirsi alle voci dei dati di autenticazione mediante gli adattatori risorse, le origini dati e altre configurazioni che richiedono i dati di autenticazione mediante un alias.

È possibile visualizzare questa pagina direttamente dalla pagina di configurazione JAAS o da altre pagine per risorse che utilizzano voci di dati di autenticazione J2C. Ad esempio, per visualizzare questa pagina di gestione, è possibile fare clic su Sicurezza > Configurazione JAAS > Voci dati di autenticazione J2C oppure Risorse > Provider JMS WebSphere > Produzioni connessioni code WebSphere > produzione_connessioni > Voci dati di autenticazione J2C.

Nota: Fare attenzione quando si cancellano le voci dei dati di autenticazione. Se i dati cancellati vengono utilizzati da altre configurazioni, l'inizializzazione del processo risorse non riesce.

Definire una nuova voce dati di autenticazione facendo clic su Nuovo.

Alias
Specifica il nome della voce dati di autenticazione.
Tipo dati: Stringa
Unità: Stringa
Predefinito: Nessuna
ID utente
Specifica l'identità utente.
Tipo dati: Stringa
Descrizione
Specifica una descrizione facoltativa della voce dati di autenticazione. Ad esempio, questa voce dati di autenticazione viene utilizzata per connettersi a DB2.
Tipo dati: Stringa
