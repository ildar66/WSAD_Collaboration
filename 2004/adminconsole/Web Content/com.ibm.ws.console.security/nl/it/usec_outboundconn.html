Impostazioni di trasporto in uscita di CSI (Common secure interoperability)

Utilizzare questa pagina per specificare le impostazioni dei trasporti e di SSL, che il server utilizza durante le comunicazioni con i server downstream per le richieste in uscita.

Per visualizzare questa pagina della console di gestione, fare clic su Sicurezza > Protocollo di autenticazione > Trasporto in uscita CSI.

Scheda Configurazione

Trasporto
Specifica se i processi del client si collegano al server utilizzando uno dei trasporti collegati del server.

È possibile scegliere di utilizzare SSL, TCP/IP o entrambi come trasporto in uscita supportato da un server. Se si specifica TCP/IP, il server supporta solo TCP/IP e non può inizializzare le connessioni SSL con i server downstream. Se si specifica SSL supportato, questo server può inizializzare le connessioni TCP/IP o SSL. Se si specifica SSL obbligatorio, il server deve utilizzare SSL per inizializzare le connessioni ai server downstream. Quando viene specificato SSL, decidere quali gruppi di impostazioni di configurazione SSL si desidera utilizzare per la configurazione in uscita. Tale decisione stabilisce quale file di chiavi e quale file sicuro utilizzare per le connessioni in uscita sui server downstream. Considerare, ad esempio, quanto segue:

TCP/IP
Questo server apre solo le connessioni TCP/IP con i server downstream.
SSL Supportato
Questo server apre le connessioni SSL con qualsiasi server downstream che le supporta e le connessioni TCP/IP con qualsiasi server downstream che non le supporta.
SSL obbligatorio
Questo server apre sempre le connessioni SSL con server downstream.

Predefinito: SSL-Supportato
Intervallo: TCP/IP, SSL-Obbligatorio, SSL-Supportato
Impostazioni SSL
Specifica un elenco di impostazioni SSL predefinite per le connessioni in uscita. Queste impostazioni vengono configurate nel pannello Repertori di configurazione SSL.
Tipo dati: Stringa
Predefinito: DefaultSSLSettings
Intervallo: Qualsiasi impostazione SSL configurata nel pannello Repertori di configurazione SSL
