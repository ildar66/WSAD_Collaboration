Impostazioni di trasporto in uscita di SAS (Secure Association Service)

Utilizzare questa pagina per specificare le impostazioni di trasporto per le connessioni accettate da questo server tramite il protocollo di autenticazione SAS (Secure Association Service). Utilizzare il protocollo SAS per comunicare in modo sicuro sui bean enterprise con le precedenti release di WebSphere Application Server.

Per visualizzare questa pagina della console di gestione, fare clic su Sicurezza > Protocollo di autenticazione > Trasporto in uscita SAS.
