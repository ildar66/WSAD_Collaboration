Impostazioni LTPA (Lightweight Third Party Authentication)

Utilizzare questa pagina per configurare le impostazioni LTPA (Lightweight Third Party Authentication).

Per visualizzare questa pagina della console di gestione, fare clic su Sicurezza > Meccanismi di autenticazione > LTPA.

Se si sta configurando per la prima volta la sicurezza, è necessaria solo la password. Una volta immessa quest'ultima, fare clic su Applica. Fare clic su SSO (Single Sign On) ed inserire il nome del dominio. Assicurarsi che l'SSO sia abilitato. Fare clic su Applica. Per completare l'impostazione della sicurezza, verificare che sia configurato il registro appropriato e fare clic su Applica sul pannello Sicurezza globale. Quando viene abilitata la sicurezza, ed una qualsiasi di queste proprietà viene modificata, andare al pannello Sicurezza Globale e fare clic su Applica per convalidare le modifiche.

Scheda Configurazione

Crea chiavi
Specifica se il server creerà nuove chiavi LTPA.

Quando la sicurezza viene attivata per la prima volta con LTPA come meccanismo di autenticazione, le chiavi LTPA vengono create automaticamente immettendo la password nel pannello. Se si desidera una nuova serie di chiavi, da creare mediante la password impostata in precedenza, fare clic su Crea chiavi. Se si utilizza una nuova password, non selezionare questo pulsante. Una volta immessa la nuova password e aver selezionato OK o Applica, si crea una nuova serie di chiavi. Ogni volta che si genera un nuovo gruppo di chiavi, queste non vengono utilizzate fino a quando non sono salvate.

Importa chiavi
Specifica se il server importerà delle nuove chiavi LTPA.

Per supportare SSO (Single Sign-On) nel prodotto WebSphere attraverso più domini WebSphere (celle), condividere le chiavi LTPA e la password tra i domini. Il pulsante Importa chiavi può essere utilizzato per importare le chiavi LTPA da altri domini. Le chiavi LTPA devono essere state esportate in precedenza da una delle celle su un file. Per importare una nuova serie di chiavi LTPA, immettere la password appropriata e il nome del file in cui si trovano tali chiavi. Selezionare solo Importa chiavi e non fare clic su OK o Applica.

Esporta chiavi
Specifica se il server esporterà le chiavi LTPA.

Per supportare SSO (single sign on) nel prodotto WebSphere attraverso più domini WebSphere (celle), condividere le chiavi LTPA e la password tra i domini. Il pulsante Esporta chiavi può essere utilizzato per esportare le chiavi LTPA su altri domini.

Per esportare le chiavi LTPA, verificare che il sistema sia in esecuzione con la sicurezza abilitata e che stia utilizzando LTPA. Immettere il nome file nel campo Nome file di chiavi e fare clic su Esporta chiavi. Le chiavi codificate verranno memorizzate nel file specificato.

Password
Specifica la password per codificare e decodificare le chiavi LTPA. Utilizzare questa password quando si importano tali chiavi in altre configurazioni di dominio di gestione di WebSphere Application Server (se presenti) e quando si configura SSO per il server Domino.

Dopo aver creato o importato le chiavi, queste vengono utilizzate per codificare e decodificare il token LTPA. Ogni volta che la password viene modificata, si crea automaticamente una nuova serie di chiavi LTPA facendo clic su OK o Applica. Questo nuovo gruppo di chiavi verrà utilizzato dopo essere stato salvato.

Tipo dati Stringa
Conferma password
Specifica la password confermata utilizzata per codificare e decodificare le chiavi LTPA.

Utilizzare questa password quando si importano tali chiavi in altre configurazioni di dominio di gestione di WebSphere Application Server (se presenti) e quando si configura SSO per il server Domino.

Tipo dati Stringa
Timeout
Specifica il periodo di tempo in minuti dopo il quale un token LTPA scadrà. Controllare che questo periodo di tempo sia maggiore del timeout della cache configurato nel pannello Sicurezza globale.
Tipo dati Numero intero
Unità Minuti
Predefinito 120
Nome file di chiavi
Specifica il nome del file utilizzato durante l'importazione o l'esportazione di chiavi.

Immettere un nome file di chiavi completo e fare clic su Importa chiavi o Esporta chiavi.
