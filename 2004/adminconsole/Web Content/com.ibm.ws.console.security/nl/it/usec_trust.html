Impostazioni associazioni sicure

L'associazione sicura consente l'integrazione della sicurezza di IBM WebSphere Application Server e dei server di terze parti. In particolare, consente a un server proxy inverso di agire come server di autenticazione front-end mentre il prodotto applica le politiche di autorizzazione sulle credenziali risultanti inviate dal server proxy. Utilizzare questa pagina per configurare le impostazioni delle associazioni sicure.

Per visualizzare questa pagina della console di gestione, fare clic su Centro sicurezza > Meccanismi di autenticazione > LTPA > Associazione sicura.

Quando viene abilitata la sicurezza ed una qualsiasi di queste proprietà viene modificata, andare al pannello Sicurezza globale e fare clic su Applica per convalidare le modifiche.
