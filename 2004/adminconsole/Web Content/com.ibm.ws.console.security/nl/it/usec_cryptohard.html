Impostazioni del token crittografico

Utilizzare questa pagina per configurare le impostazioni del token crittografico.

Per visualizzare questa pagina della console di gestione, fare clic su Sicurezza> SSL > nome_alias > Token crittografico.

Scheda Configurazione

Tipo di token
Specifica il tipo di file keystore incorporato che viene implementato nel token crittografico, ad esempio PKCS#11.

WebSphere Application Server utilizza un'implementazione di JSSE per supportare il token crittografico con SSL (Secure Sockets Layer). Sono supportate anche diverse unità crittografiche. Per un server SSL, vengono supportate le seguenti unità:

  • IBM 4758-23
  • nCipher nForce
  • Cryptoswift Rainbow
Per un client SSL, vengono supportate le seguenti unità:
  • IBM 4758-23
  • nCipher nForce
  • Cryptoswift Rainbow
  • Smartcard del kit di sicurezza IBM
  • Smartcard GemPlus
  • Rainbow iKey 1000/2000 (unità "Smartcard" USB)
  • Eracom CSA8000
Per installare il token crittografico, seguire la documentazione fornita insieme all'unità.

Tipo dati: Stringa
File libreria
Specifica il DLL (dynamic link library) o l'oggetto condiviso che implementa l'interfaccia per l'unità del token crittografico.

Tipo dati: Stringa
Password
Specifica la password dell'unità token crittografico.
