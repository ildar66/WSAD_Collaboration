Impostazioni di configurazione login di applicazione

Utilizzare questa pagina per impostare le configurazioni di login dell'applicazione.

Per visualizzare questa pagina ella console di gestione, fare clic su Sicurezza > Configurazione JAAS > Login dell'applicazione > nome_alias.

Fare clic su Applica per salvare le modifiche e aggiungere il nome nodo extra che precede il nome alias originale. Selezionando OK non si salvano le nuove modifiche nel file security.xml.

Alias
Specifica il nome alias del login dell'applicazione.

Non utilizzare la barra (/) nel nome alias durante la definizione delle voci di configurazione login JAAS. L'analizzatore della configurazione login JAAS non è in grado di gestire la barra.

Tipo dati: Stringa
