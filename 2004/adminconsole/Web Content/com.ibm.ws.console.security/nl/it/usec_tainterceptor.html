Raccolta intercettatore associazioni sicure

Utilizzare questa pagina per specificare le informazioni di garanzia per i server proxy a sicurezza inversa.

Per visualizzare questa pagina della console di gestione, fare clic su Sicurezza > Meccanismi di autenticazione > LTPA > Associazione sicura > Intercettatori.

Quando viene abilitata la sicurezza, ed una qualsiasi di queste proprietà viene modificata, andare al pannello Sicurezza globale e fare clic su Applica per convalidare le modifiche.
