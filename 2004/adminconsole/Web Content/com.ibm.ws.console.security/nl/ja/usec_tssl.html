Secure Sockets Layer (SSL) 構成レパートリー設定

このページを使用して、新規の Secure Sockets Layer (SSL) のエイリアスを定義します。管理者は、SSL 構成 レパートリーを介して、任意の数の SSL 設定を定義し、Hypertext Transfer Protocol と SSL (HTTPS)、Internet InterORB Protocol と SSL (IIOPS) または Lightweight Directory Access Protocol と SSL (LDAPS) 接続を構成する場合に使用できます。SSL 接続をサポートする管理コンソール内の任意の場所から、 ここで定義した SSL 設定のいずれかを選択できます。これによって、複数の場所で単にエイリアスを指定することにより、 これらの多くの SSL 構成を再利用することができるため、SSL 構成プロセスが単純化されます。

管理コンソール・ページを表示するには、「セキュリティー」>「SSL」をクリックします。

「新規」をクリックして、「SSL 構成レパートリー」で新しいエイリアスを作成します。

「削除」をクリックして、「SSL 構成レパートリー」からエイリアスを除去します。SSL 構成エイリアスが構成で 参照されている場合にこれを削除すると、削除されたエイリアスにアクセスしたときに SSL 接続で障害が発生します。

エイリアス
特定の SSL 設定の名前を指定します。
