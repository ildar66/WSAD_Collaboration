暗号トークン設定

このページを使用して、暗号トークン設定を構成します。

この管理コンソール・ページを表示するには、「セキュリティー」>「SSL」>「alias_name」>「Cryptographic Token」をクリックします。

「構成」タブ

トークン・タイプ
PKCS#11 など、暗号トークンにインプリメントされている組み込み鍵ストアのタイプを指定します。

WebSphere Application Server は、Java Secure Socket Extension (JSSE) インプリメンテーションを使用して、 Secure Sockets Layer (SSL) を用いる暗号トークンをサポートします。さまざまな暗号デバイスがサポートされています。 SSL サーバーには、以下のデバイスがサポートされます。

  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
SSL クライアントには、以下のデバイスがサポートされます。
  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
  • IBM Security Kit Smartcard
  • GemPlus Smartcards
  • Rainbow iKey 1000/2000 (USB "Smartcard" デバイス)
  • Eracom CSA8000
暗号トークンをインストールするには、ご使用のデバイスに添付された資料の説明に従ってください。

データ型: ストリング
ライブラリー・ファイル
暗号トークン・デバイスに対するインターフェースをインプリメントする ダイナミック・リンク・ライブラリー (DLL) または共用オブジェクトを指定します。

データ型: ストリング
パスワード
暗号トークン・デバイスのパスワードを指定します。
