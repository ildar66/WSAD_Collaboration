トラスト・アソシエーション設定

トラスト・アソシエーションにより、IBM WebSphere Application Server セキュリティーとサード・パーティー製 セキュリティー・サーバーとを統合することができます。もっと具体的に述べると、リバース・プロキシー・サーバーがフロントエンド認証サーバーとして機能できるようになる一方で、この製品は、製品自体の認証ポリシーを、プロキシー・サーバーに渡される信任状に適用します。 このページを使用して、トラスト・アソシエーション設定を構成します。

この管理コンソール・ページを表示するには、「セキュリティー・センター」>「認証メカニズム」>「LTPA」>「トラスト・アソシエーション」をクリックします。

セキュリティーが使用可能になっていて、これらのプロパティーのいずれかが変更 されている場合は、「グローバル・セキュリティー」パネルに移動し、「 適用」をクリックして、変更を確認します。

「構成」タブ

使用可能
トラスト・アソシエーションを使用可能にするかどうかを指定します。
データ型: ブール
デフォルト: 使用不可
範囲: 使用可能または使用不可
