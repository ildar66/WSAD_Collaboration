Valores de seguridad del servidor

Utilice esta página para configurar la seguridad del servidor y alterar temporalmente los valores de seguridad global. Si necesita regresar a los valores de seguridad global por omisión, deseleccione el recuadro de selección correspondiente de la consola administrativa.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Seguridad de servidor. Tenga en cuenta de estos valores sólo son aplicables a Network Deployment.

Se puede inhabilitar la seguridad en servidores de aplicaciones individuales mientras está habilitada la seguridad global. No obstante, no se puede habilitar la seguridad en el servidor de aplicaciones individual mientras la seguridad global esté inhabilitada. Aunque la seguridad del servidor de aplicaciones esté inhabilitada para las peticiones de usuario, la seguridad de administración y de nombres continúa habilitada para ese servidor de aplicaciones, por lo que la infraestructura de administración y de nombres permanece segura. Para evitar problemas, verifique que la seguridad de denominación tenga el tipo de acceso Todos a la función de denominación que se utiliza en el código de usuario. No es necesario que configure la seguridad de administración, ya que el código de usuario no accede directamente a las funciones administrativas. El código de usuario accede a las funciones administrativas a través de las herramientas de scripts soportadas.

Seguridad a nivel de servidor

Pera volver a los valores por omisión de Seguridad a nivel de servidor, pulse Utilizar seguridad de células. Pulse Aplicar y seleccione Guardar para validar los cambios a nivel de servidor.
