Valores de configuración del inicio de sesión de la aplicación

Utilice esta página para establecer las configuraciones de inicio de sesión de la aplicación.

Para ver esta página de la consola administrativa, pulse Seguridad > Configuración JAAS > Inicios de sesión de la aplicación > nombre_alias.

Pulse Aplicar para guardar los cambios y añadir el nombre de nodo adicional que precede al nombre de alias original. Si pulsa Aceptar no se guardarán los nuevos cambios en el archivo security.xml.

Alias
Especifica el nombre de alias del inicio de sesión de la aplicación.

No utilice la barra inclinada (/) en el nombre de alias cuando defina las entradas de configuración de inicio de sesión JAAS. El analizador de configuración de inicio de sesión JAAS no acepta el carácter de barra inclinada.

Tipo de datos: String
