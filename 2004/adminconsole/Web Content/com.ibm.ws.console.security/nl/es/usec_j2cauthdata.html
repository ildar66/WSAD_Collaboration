Valores de entrada de datos de autenticación del conector Java 2

Utilice esta página como punto principal para que los administradores definan los datos de autenticación, que incluyen las contraseñas e identidades de usuario. Estos datos pueden hacer referencia a entradas de datos de autenticación de adaptadores de recursos, orígenes de datos y otras configuraciones que requieran datos de autenticación mediante un alias.

Puede visualizar esta página directamente desde la página de configuración de JAAS o desde otras páginas para recursos que utilicen entradas de datos de autenticación de J2C. Por ejemplo, para ver esta página de la consola administrativa, puede pulsar Seguridad > Configuración JAAS > Entradas de datos de autenticación de J2C o Recursos > Proveedor de JMS de WebSphere > Fábricas de conexiones de cola WebSphere > fábrica-conexiones > Entradas de datos de autenticación de J2C.

Nota: Tenga cuidado al suprimir las entradas de datos de autenticación. Si otras configuraciones utilizan los datos de autenticación suprimidos, el proceso de recursos de inicialización no se realiza adecuadamente.

Para definir una nueva entrada de datos de autenticación, pulse Nueva.

Alias
Especifica el nombre de la entrada de datos de autenticación.
Tipo de datos: String
Unidades: String
Valor por omisión: Ninguno
ID de usuario
Especifica la identidad de usuario.
Tipo de datos: String
Descripción
Especifica una descripción opcional de la entrada de datos de autenticación. Por ejemplo, esta entrada de datos de autenticación se utiliza para conectarse a DB2.
Tipo de datos: String
