Valores de LTPA (Lightweight Third Party Authentication)

Utilice esta página para configurar los valores de LTPA (Lightweight Third Party Authentication).

Para ver esta página de la consola administrativa, pulse Seguridad > Mecanismos de autenticación > LTPA.

Si está configurando la seguridad por primera vez, sólo necesita la contraseña. Una vez entrada la contraseña, pulse Aplicar. Pulse Inicio de sesión único (SSO) y entre el nombre de dominio. Asegúrese de que SSO esté habilitado. Pulse Aplicar. Para completar la configuración de seguridad, asegúrese de que el registro apropiado esté configurado y pulse Aplicar en el panel Seguridad Global. Cuando se habilite la seguridad y se modifique una de estas propiedades, vaya al panel Seguridad global y pulse Aplicar para validar los cambios.

Pestaña Configuración

Generar claves
Especifica si el servidor generará nuevas claves LTPA.

Cuando se activa la seguridad por primera vez con LTPA como mecanismo de autenticación, las claves se generan automáticamente al entrar la contraseña en el panel. Si necesita que se genere un nuevo conjunto de claves mediante la contraseña establecida anteriormente, pulse Generar claves. Si se utiliza una nueva contraseña, no pulse este botón. Una vez que entre la nueva contraseña y pulse Aceptar o Aplicar, se generará un nuevo conjunto de claves. Siempre que se genere un nuevo conjunto de claves, éstas no se utilizarán hasta que las guarde.

Importar claves
Especifica si el servidor importará nuevas claves LTPA.

Para dar soporte al inicio de sesión único (SSO) del producto WebSphere en varios dominios de WebSphere (células), las claves LTPA y la contraseña se deben compartir entre los dominios. El botón Importar claves puede utilizarse para importar las claves LTPA de otros dominios. Las claves LTPA se deben haber exportado anteriormente de una de las células a un archivo. Para importar un nuevo conjunto de claves LTPA, entre la contraseña correcta y el nombre de archivo donde se encuentren las claves LTPA. Pulse únicamente Importar claves; no pulse Aceptar o Aplicar.

Exportar claves
Especifica si el servidor exportará claves LTPA.

Para dar soporte al inicio de sesión único (SSO) del producto WebSphere en varios dominios de WebSphere (células), las claves LTPA y la contraseña se deben compartir entre los dominios. Se puede utilizar el botón Exportar claves para exportar las claves LTPA a otros dominios.

Para exportar las claves LTPA, asegúrese de que el sistema se esté ejecutando con la seguridad habilitada y esté utilizando LTPA. Entre el nombre de archivo en el campo Nombre de archivo de claves y pulse Exportar claves. Las claves cifradas se almacenarán en el archivo especificado.

Contraseña
Especifica la contraseña para cifrar y descifrar las claves LTPA. Utilice esta contraseña cuando se importen estas claves a otras configuraciones del dominio administrativo de WebSphere Application Server (si existen) y cuando se configure SSO para Domino Server.

Una vez generadas o importadas las claves, se utilizan para cifrar o descifrar el símbolo LTPA. Siempre que se modifique la contraseña, automáticamente se generará un nuevo conjunto de claves LTPA al pulsar Aceptar o Aplicar. Estos conjuntos de claves nuevos se utilizarán únicamente cuando realice la operación de guardar.

Tipo de datos String
Confirmar contraseña
Especifica la contraseña confirmada para cifrar y descifrar las claves LTPA.

Utilice esta contraseña cuando se importen estas claves a otras configuraciones del dominio administrativo de WebSphere Application Server (si existen) y cuando se configure SSO para Domino Server.

Tipo de datos String
Tiempo de espera
Especifica el intervalo en minutos durante el cual caduca un símbolo LTPA. Asegúrese de que este periodo de tiempo sea mayor que el tiempo de espera de antememoria configurado en el panel Seguridad Global.
Tipo de datos Integer
Unidades Minutos
Valor por omisión 120
Nombre de archivo de claves
Especifica el nombre del archivo utilizado al importar o exportar claves.

Especifique un nombre de archivo de claves plenamente cualificado, y pulse Importar claves o Exportar claves.
