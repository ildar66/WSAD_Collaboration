Valores de símbolos criptográficos

Utilice esta página para configurar los valores de símbolos criptográficos.

Para ver esta página de la consola administrativa, pulse Seguridad > SSL > nombre_alias > Símbolo criptográfico.

Pestaña Configuración

Tipo de símbolo
Especifica el tipo de almacén de claves incorporado que se implementa en el símbolo criptográfico, como por ejemplo, PKCS#11.

WebSphere Application Server utiliza la implementación JSSE (Java Secure Socket Extension) para dar soporte a los símbolos criptográficos con SSL (Secure Sockets Layer). Se da soporte a diferentes dispositivos criptográficos. En un servidor SSL, se da soporte a los siguientes dispositivos:

  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
En un cliente SSL, se da soporte a los siguientes dispositivos:
  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
  • IBM Security Kit Smartcard
  • GemPlus Smartcards
  • Rainbow iKey 1000/2000 (dispositivo USB "Smartcard")
  • Eracom CSA8000
Siga las instrucciones de la documentación que se adjunta con el dispositivo para instalar el símbolo criptográfico.

Tipo de datos: String
Archivo de biblioteca
Especifica la DLL (Biblioteca de enlaces dinámicos) o el objeto compartido que implementa la interfaz con este dispositivo de símbolos criptográficos.

Tipo de datos: String
Contraseña
Especifica la contraseña del dispositivo de símbolos criptográficos.
Tipo de datos: String
