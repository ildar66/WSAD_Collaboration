Valores de salida de transporte CSI (Common secure interoperability)

Utilice esta página para especificar qué transportes y valores SSL (Secure Sockets Layer) utiliza este servidor cuando se comunica con los servidores en sentido descendente para las peticiones de salida.

Para ver esta página de la consola administrativa, pulse Seguridad > Protocolo de autenticación > Transporte de salida CSI.

Pestaña Configuración

Transporte
Especifica si los procesos de cliente se conectan al servidor utilizando uno de los transportes conectados del servidor.

Puede elegir si desea utilizar SSL, TCP/IP o Ambos tipos de conexión como transporte de salida al que da soporte el servidor. Si especifica TCP/IP, el servidor sólo da soporte a TCP/IP y no puede iniciar conexiones SSL con los servidores en sentido descendente. Si especifica SSL soportado, este servidor puede iniciar las conexiones TCP/IP o SSL indistintamente. Si especifica SSL necesario, dicho servidor debe utilizar SSL para iniciar las conexiones con servidores en sentido descendente. Cuando especifique SSL, decida qué conjunto de valores de configuración SSL desea utilizar para la configuración de salida. Esta decisión determina qué archivo de claves y archivo de confianza se utilizan para las conexiones de salida con el servidor en sentido descendente. Por ejemplo, tenga en cuenta lo siguiente:

TCP/IP
Este servidor sólo abre conexiones TCP/IP con servidores en sentido descendente.
SSL soportado
Este servidor abre las conexiones SSL con cualquier servidor en sentido descendente que las soporte, además de las conexiones TCP/IP con servidores en sentido descendente que no las soporte.
SSL necesario
Este servidor siempre abre conexiones TCP/IP con servidores en sentido descendente.

Valor por omisión: SSL soportado
Rango: TCP/IP, SSL necesario, SSL soportado
Valores SSL
Especifica una lista de valores SSL predefinidos para las conexiones de salida. Estos valores se configuran en el panel Repertorios de configuración SSL.
Tipo de datos: String
Valor por omisión: DefaultSSLSettings
Rango: Cualquier valor SSL que esté configurado en panel Repertorios de configuración SSL
