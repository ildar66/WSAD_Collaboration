Colección de interceptor de asociación de confianza

Utilice esta página para especificar la información de confianza de los servidores proxy de seguridad de retroceso.

Para ver esta página de la consola administrativa, pulse Seguridad > Mecanismos de autenticación > LTPA > Asociación de confianza > Interceptores.

Cuando se habilite la seguridad y se modifique una de estas propiedades, vaya al panel Seguridad Global y pulse Aplicar para validar los cambios.
