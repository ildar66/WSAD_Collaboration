Valores del repertorio de configuraciones SSL (Secure Sockets Layer)

Utilice esta página para definir un nuevo alias SSL. Mediante el repertorio de configuración SSL, los administradores pueden definir cualquier cantidad de valores SSL que se van a utilizar para configurar las conexiones HTTPS (Hypertext Transfer Protocol with SSL), IIOPS (Internet InterORB Protocol with SSL) o LDAPS (Lightweight Directory Access Protocol with SSL). Puede seleccionar uno de los valores SSL definidos aquí desde cualquier ubicación de la consola administrativa que de soporte a las conexiones SSL. Esto simplifica el proceso de configuración SSL ya que se pueden volver a utilizar muchas de estas configuraciones SSL simplemente especificando el alias en varios lugares.

Para ver esta página de la consola administrativa, pulse Seguridad > SSL.

Pulse Nuevo para crear un nuevo alias del repertorio de configuración SSL.

Pulse Suprimir para suprimir un alias del repertorio de configuración SSL. Si se hace referencia en la configuración a un alias de configuración SSL, y si está suprimido en la misma, cuando se acceda al alias suprimido la conexión SSL fallará.

Alias
Especifica el nombre del valor SSL específico.
