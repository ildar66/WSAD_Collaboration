信任关联设置

信任关联启用 IBM WebSphere Application Server 安全性和第三方安全性服务器的集成。更特别地，当产品应用它自已的授权策略到由代理服务器传递的结果凭证时，逆向代理服务器可以担当前端认证服务器。使用此页面配置信任关联设置。

要查看此管理控制台页面，单击安全性中心 > 认证机制 > LTPA > 信任关联。

当启用了安全性并且更改任何这些属性时，请转至全局安全性面板并单击应用来确认更改。

“配置”选项卡

启用的
指定是否启用信任关联。
数据类型： Boolean
缺省： 禁用
范围： 启用或禁用
