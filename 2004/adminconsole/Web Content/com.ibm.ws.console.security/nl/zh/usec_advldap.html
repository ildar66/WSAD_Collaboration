轻量级目录访问协议高级设置

使用此页面配置当用户和组驻留在外部 LDAP 目录中时使用的高级轻量级目录访问协议（LDAP）用户注册表设置。

要查看此管理页面，单击安全性 > 用户注册表 > LDAP 高级LDAP 设置。

所有用户和相关组过滤器的缺省值已在相应的字段中完成。可以根据您的要求更改这些值。这些缺省值基于在 LDAP 设置面板中所选的 LDAP 服务器类型。如果类型更改（例如从 NETSCAPE 改为 SECUREWAY），缺省过滤器自动更改。当缺省过滤器值更改时，LDAP 服务器类型更改为定制以表明正在使用定制过滤器。当启用了安全性并且更改任何这些属性时，转至全局安全性面板并单击应用或确定以确认更改。

“配置”选项卡

用户过滤器
指定 LDAP 搜索用户注册表的用户过滤器。

此选项通常用于安全性角色到用户指定。它指定用于在目录服务中查找用户的属性。例如，要根据用户的用户标识查找用户，指定 (ampersand(uid=%v)(objectclass=inetOrgPerson)，其中 ampersand 是 and 符号（&）。要获得更多的关于该语法的信息，请参阅 LDAP 目录服务文档。

数据类型： String
组过滤器
指定 LDAP 搜索组注册表的组过滤器。

在安全性角色到组的指定中通常会使用此选项。它指定用于在目录服务中查找组的属性。要获得更多的关于该语法的信息，请参阅 LDAP 目录服务文档。

数据类型： String
用户标识映射
指定将用户的缩写名映射到 LDAP 条目的 LDAP 过滤器。

指定在显示用户时代表用户的一条信息。例如，要按用户的标识显示类型对象类 = inetOrgPerson 的条目，指定 inetOrgPerson:uid。此字段用分号（;）来定界多个对象类:属性对。

数据类型： String
组标识映射
指定将组的缩写名映射到 LDAP 条目的 LDAP 过滤器。

指定在显示组时表示组的一条信息。例如，要按组名显示组，指定 *:cn。在这种情况下，星号（*）是搜索任何对象类的通配符。此字段采用多个 objectclass:property 对，以分号（;）定界。

数据类型： String
组成员标识映射
指定标识用户与组之间关系的过滤器。

对于目录类型 SecureWay、Netscape 和 Domino，此字段用分号（;）来定界多个 objectclass:property 对。在 objectclass:property 对中，objectclas 值与“组过滤器”中定义的 objectclas 相同，而属性是成员属性。如果 objectclass 值不匹配组过滤器中的 objectclass，如果组映射到安全性角色那么授权可能失败。要获得有关此语法的更多信息，请参阅 LDAP 目录服务文档。

对于 IBM Directory Server、iPlanet Directory Server 和 Active Directory，此字段用分号（;）定界多个（组 attribute:member 属性）对。它们用于查找用户的组成员资格，这是通过枚举由给定用户处理的所有组属性实现的。例如，Active Directory 使用的属性对 (memberof:member)，和 IBM Directory Server 使用的属性对 (ibm-allGroup:member)。此字段也指定了对象类的哪个属性存储属于由此对象类代表的组的成员列表。 对于受支持的 LDAP directory server，请参阅信息中心文章“受支持的目录服务”。

数据类型： String
证书映射方式
指定通过 EXACT_DN 还是 CERTIFICATE_FILTER 将 X.509 证书映射到 LDAP 目录。指定 CERTIFICATE_FILTER 使用指定的证书过滤器进行映射。
数据类型： String
证书过滤器
指定是否使用过滤器证书映射属性，指定用于将客户机证书中的属性映射到 LDAP 注册表中条目的 LDAP 过滤器。

要启用此字段选择，为证书映射单击 CERTIFICATE_FILTER。如果运行时多个 LDAP 条目与过滤器规范匹配，则认证失败，因为它导致模糊匹配。 此过滤器的语法或结构是：LDAP attribute=${Client certificate attribute}（例如，uid=${SubjectCN}）。过滤器规范的左侧是 LDAP 属性，其取决于 LDAP 服务器配置使用的模式。过滤器规范的右侧是客户机证书的公共属性之一。右侧必须以美元符号（$）和左大括号（{）开始，并以右大括号（}）结束。您可使用下列在过滤器规范的右侧使用的证书属性值。字符串的大小写是重要的。

  • ${UniqueKey}
  • ${PublicKey}
  • ${PublicKey}
  • ${Issuer}
  • ${NotAfter}
  • ${NotBefore}
  • ${SerialNumber}
  • ${SigAlgName}
  • ${SigAlgOID}
  • ${SigAlgParams}
  • ${SubjectCN}
  • ${Version}

数据类型： String
