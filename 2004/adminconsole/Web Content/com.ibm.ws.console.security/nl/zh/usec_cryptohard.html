加密令牌设置

使用此页面配置加密令牌设置。

要查看此管理控制台页面，单击安全性 > SSL > alias_name >加密令牌。

“配置”选项卡

令牌类型
指定加密令牌中实现的内置密钥存储的类型，如 PKCS#11。

WebSphere Application Server 使用 Java 安全套接字扩展（JSSE）的实现支持带有安全套接字层（SSL）的加密令牌。支持不同的加密设备。对于 SSL 服务器，支持以下设备：

  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
对于 SSL 客户机，支持以下设备：
  • IBM 4758-23
  • nCipher nForce
  • Rainbow Cryptoswift
  • IBM Security Kit Smartcard
  • GemPlus Smartcards
  • Rainbow iKey 1000/2000（USB“Smartcard”设备）
  • Eracom CSA8000
请按照设备附带的文档，以安装您的加密令牌。

数据类型： String
库文件
指定实现到加密令牌设备接口的动态链接库（DLL）或共享对象。

数据类型： String
密码
指定加密令牌设备的密码。
数据类型： String
