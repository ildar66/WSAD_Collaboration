安全关联服务传输出站设置

使用此页面为连接指定传输设置，这些连接是使用安全关联服务（SAS）认证协议的这台服务器接受的连接。SAS 协议用于安全地通信到具有 WebSphere Application Server 前发行版的企业 bean。

要查看此管理控制台页面，单击安全性 > 认证协议 > SAS 出站传输。

“配置”选项卡

SSL 设置
指定预定义安全套接字层（SSL）设置列表，供选择用于出站连接。这些设置在“SSL 指令表”面板上配置。
数据类型： String
缺省： DefaultSSLSettings
