安全套接字层配置指令表设置

使用此页面定义新的安全套接字层（SSL）别名。通过 SSL 配置指令表，管理员可定义任何数量的 SSL 设置，以用于配置带有 SSL 的超文本传输协议（HTTPS）、带有 SSL 的因特网 ORB 间协议（IIOPS）或带有 SSL 的轻量级目录访问协议（LDAPS）连接。从支持 SSL 连接的管理控制台中的任何位置，您都可以选择在此定义的一个 SSL 设置。这简化了 SSL 配置过程，因为您可以通过在多个位置指定别名来重用很多这些 SSL 配置。

要查看此管理控制台页面，单击安全性 > SSL。

单击新建创建新的 SSL 配置指令表别名。

单击删除除去 SSL 配置指令表别名。如果在配置中引用 SSL 配置别名，而且它在这里被删除，则访问此已删别名时 SSL 连接失败。

别名
指定特定 SSL 设置的名称。
