Java 认证授权服务登录模块设置

使用此页面定义 Java 认证授权服务（JAAS）登录配置的登录模块。

要查看此管理页面，单击安全性 > JAAS 配置 > 应用程序登录 > alias_name > JAAS 登录模块。

“配置”选项卡

模块类名
指定给定的登录模块的类名。

由 WebSphere 产品定义的缺省登录模块使用代理 LoginModule 类，com.ibm.ws.security.common.auth.module.WSLoginModuleProxy。此代理类用线程上下文类装入程序装入 WebSphere 登录模块并委托所有操作到实际的登录模块实现。实际的登录模块实现指定为选项配置中的委托选项。代理类是必需的，因为 WebSphere 产品类装入程序对于开发者工具箱应用程序类装入程序是不可见的。

数据类型： String
认证策略
指定认证行为是沿登录模块堆栈继续向下认证。

JAAS 认证提供程序提供认证策略。在 JAAS 中，认证策略通过 LoginModule 接口实现。

数据类型： String
缺省： 必需的
范围： 必需的、必要的、足够的和可选的

在选项其它属性中指定附加的选项。在初始化期间，这些名称和值对传递到登录模块。这是用于传递信息到登录模块的一种机制。
