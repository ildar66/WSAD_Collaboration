Java 2 连接器认证数据条目设置

管理员将此页面用作中央位置来定义认证数据，它包括用户身份和密码。这些可以通过资源适配器、数据源和其它需要使用别名的认证数据的配置来引用认证数据条目。

您可用直接从 JAAS 配置页面或从使用 J2C 认证数据条目的资源的其它页面显示此页面。例如，要查看此管理页面，您可以单击安全性 > JAAS 配置 > J2C 认证数据条目或资源 > WebSphere JMS 提供程序 > WebSphere 队列连接工厂 > connection_factory > J2C 认证数据条目。

注: 在删除认证数据条目时请小心。如果删除的认证数据是由其它配置使用的，那么初始化资源进程将失败。

通过单击新建定义新的认证数据条目。

别名
指定认证数据条目的名称。
数据类型： String
单位： String
缺省： 无
用户标识
指定用户身份。
数据类型： String
描述
指定认证数据条目的可选描述。例如，此认证数据条目用于连接 DB2。
数据类型： String
