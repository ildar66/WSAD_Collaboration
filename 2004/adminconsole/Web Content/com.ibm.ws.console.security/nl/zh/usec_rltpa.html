轻量级第三方认证设置

使用此页面配置轻量级第三方认证（LTPA）设置。

要查看此管理控制台页面，单击安全性 > 认证机制 > LTPA。

如果首次配置安全性，则仅密码是必需的。一旦输入了密码，单击应用。单击单次注册（SSO）并输入域名。确保启用了 SSO。 单击应用。 要完成安全性设置，确保设置了相应的注册表，并从“全局安全性”面板中单击应用。当启用了安全性并且更改任 何这些属性时，请转至“全局安全性”面板并单击应用确认这些更改。

“配置”选项卡

生成密钥
指定服务器是否将生成新的 LTPA 密钥。

第一次用 LTPA 打开安全性作为认证机制时，用面板中输入的密码自动生成 LTPA 密钥。如果您需要使用先前设置的密码生成新的密钥集，那么单击生成密钥。如果使用新密码，那么不要单击此按钮。一旦输入新密码并单击确定或应用，就生成新的密钥集。无论新的密钥集是何时生成的，在您保存之前都不会使用它们。

导入密钥
指定服务器是否将导入新的 LTPA 密钥。

要支持跨多个 WebSphere 域（单元）的 WebSphere 产品中的单次注册（SSO），在这些域中共享 LTPA 密钥和密码。导入密钥按钮可用于从其它域导入 LTPA 密钥。先前应该已从一个单元导出 LTPA 密钥到文件。要导入新的 LTPA 密钥集，输入找到 LTPA 密钥处的适当的密码和文件名。 仅单击导入密钥并且不要单击确定或应用。

导出密钥
指定服务器是否将导出 LTPA 密钥。

要支持跨多个 WebSphere 域（单元）的 WebSphere 产品中的单次注册（SSO），在这些域中共享 LTPA 密钥和密码。导出密钥按钮可用于将 LTPA 密钥导出到其它域。

要导出 LTPA 密钥，确保系统正在以启用了安全性的方式运行，并且正在使用 LTPA。在密钥文件名字段中输入文件名并单击导出密钥。加密的密钥将存储在指定的文件中。

密码
指定对 LTPA 密钥进行加密和解密的密码。将这些密钥导入其它 WebSphere Application Server 管理域配置（如果有的话）并在配置 Domino 服务器的 SSO 时，使用此密码。

一旦生成或导入了密钥，这些密钥用于加密和解密 LTPA 令牌。只要更改了密码，当您单击确定或应用时就自动生成新的 LTPA 密钥集。仅当您保存时才使用这些新的密钥集。

数据类型 String
确认密码
指定用于对 LTPA 密钥进行加密和解密的确认密码。

将这些密钥导入其它 WebSphere Application Server 管理域配置（如果有的话）并在配置 Domino Server 的 SSO 时，使用此密码。

数据类型 String
超时
指定 LTPA 令牌将到期的时间周期（以分钟为单位）。确保此时间周期比在“全局安全性”面板中配置的高速缓存超时长。
数据类型 整型
单位 分钟
缺省 120
密钥文件名
指定导入或导出密钥时使用的文件名。

输入全限定密钥文件名，并单击导入密钥或导出密钥。

数据类型 String
