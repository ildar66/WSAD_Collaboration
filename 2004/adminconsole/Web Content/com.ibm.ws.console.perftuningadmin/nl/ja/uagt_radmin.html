管理サービス設定

このページを使用して、管理サービスの構成を表示および変更します。

この管理コンソール・ページを表示するには、「 サーバー」>「アプリケーション・サーバー」>「server_name」>「Administration Services」をクリックします。

「構成」タブ

スタンドアロン
サーバー・プロセスが Network Deployment セルの参加者であるかどうかを指定します。 true である場合、サーバーは分散管理に参加していません。false である場合、サーバーは Network Deployment システムに参加しています。

基本 WebSphere Application Server インストール・システムのデフォルト値は true です。addNode が実行され、サーバーが Network Deployment セルに取り込まれると、 この値は false に切り替えられます。

データ型 ブール
デフォルト true
優先コネクター
優先される JMX コネクターのタイプを指定します。 SOAPConnector や RMIConnector などの使用可能なオプションは、「 JMX コネクター」ページを使用して定義されます。
データ型 ストリング
