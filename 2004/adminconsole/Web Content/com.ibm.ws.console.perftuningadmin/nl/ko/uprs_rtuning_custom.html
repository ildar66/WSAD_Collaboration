매개변수 사용자 정의 설정 조정

분배 세션에 대한 조정 매개변수를 사용자 정의하려면 이 페이지를 참조하십시오.

이 관리 콘솔 페이지를 보려면 서버 > Application Servers > server_name > 웹 컨테이너 > 세션 관리 > 분배 환경 설정 > 조정 매개변수 사용자 정의 > 설정 사용자 정의를 클릭하십시오.

쓰기 빈도
지속적 저장소에 세션을 쓸 시기를 지정합니다.

Servlet 서비스 종료 세션이 데이터베이스에 쓰거나 Servlet 이후 다른 WebSphere Application Server 인스턴스가 실행을 완료합니다.
수동 갱신 IBMSession 오브젝트의 프로그램 동기화는 데이터베이스 또는 WebSphere Application Server 인스턴스에 세션 데이터를 기록해야 합니다.
기본 시간 세션이 지정된 쓰기 간격 값에 따라 데이터베이스 또는 다른 WebSphere Application Server 인스턴스에 씁니다. 기본값: 10초
쓰기 내용
변경되었는지 여부와 상관없이 갱신된 속성만 외부 위치에만 쓸지 혹은 모든 세션 속성을 외부 위치에 쓸지 지정합니다. 외부 위치는 데이터베이스 또는 다른 Application Server가 될 수 있습니다.
갱신된 속성만 지속적 저장소에 갱신된 속성만 씁니다.
모든 세션 속성 지속적 저장소에 모든 속성을 씁니다.
저장 정리하기 스케줄
유효하지 않은 세션을 데이터베이스나 다른 Application Server 인스턴스에서 정리할 시기를 지정합니다.
분배 세션 정리 스케줄 지정 무효화된 HTTP 세션을 외부 위치에서 정리하는 데 스케줄된 무효화 프로세스를 사용할 수 있습니다. 이 옵션을 사용 가능하게 하여 HTTP 세션의 활성화를 유지하는 데 필요한 데이터베이스 또는 Application Server 서버의 갱신 수를 줄이십시오. 이 옵션을 사용 가능하게 하지 않으면, 무효화 프로세스가 몇 분마다 실행되어 무효화된 HTTP 세션을 제거합니다.

이 옵션이 사용 가능해지면 이 프로세스가 외부 위치에서 무효화된 세션을 정리하는 데 하루 두 시간을 지정하십시오. Application Server의 활동이 최소인 시기를 지정합니다. 외부 위치는 데이터베이스나 다른 Application Server 인스턴스가 될 수 있습니다.

하루 중 첫 번째 시간(0 - 23) 그리니치 평균시(GMT)에서 무효화된 세션이 외부 위치에서 정리되는 첫 번째 시간을 표시합니다. 이 값을 0-23 사이의 양의 정수로 지정합니다. 이 값은 스케줄 무효화가 사용 가능할 때에만 유효합니다.
하루 중 두 번째 시간(0 - 23) 그리니치 평균시(GMT)에서 무효화된 세션이 외부 위치에서 정리되는 시간을 표시합니다. 이 값을 0-23 사이의 양의 정수로 지정합니다. 이 값은 스케줄 무효화가 사용 가능할 때에만 유효합니다.
