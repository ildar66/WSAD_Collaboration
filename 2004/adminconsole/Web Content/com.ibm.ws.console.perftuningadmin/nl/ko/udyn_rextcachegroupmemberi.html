외부 캐시 그룹 구성원 설정

이 페이지를 참조하여 캐시 그룹의 구성원인 특정 캐시를 구성하십시오.

이 관리 콘솔 페이지를 보려면 서버 > Application Servers > server > 동적 캐시 서비스 > 외부 캐시 그룹 > External_cache_group > 외부 캐시 그룹 구성원 > External_cache_group_members를 클릭하십시오.

구성 탭

Address
외부 캐시에 연결하기 위해 외부 캐시 어댑터 Bean이 사용한 구성 문자열을 지정합니다.
AdapterBeanName
어댑터 Bean 이름을 지정합니다.
AFPA
AdapterBeanName: com.ibm.ws.cache.servlet.Afpa
주소: afpa가 청취하는 포트
ESI
AdapterBeanName: com.ibm.websphere.servlet.cache.ESIInvalidatorServlet
주소: 로컬 호스트
WTE
AdapterBeanName: com.ibm.websphere.edge.dynacache.WteAdapter
주소: 호스트이름:포트 (WTE가 청취 중인 호스트 이름 및 포트)
