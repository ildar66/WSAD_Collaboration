성능 모니터링 서비스 설정

이 페이지를 참조하여 성능 모니터링 사용 가능, PMI 모듈 선택 및 모니터링 레벨 선택을 포함한 성능 모니터링을 위한 설정을 지정하십시오.

이 관리 콘솔 페이지를 보려면 서버 > Application Server > server > 성능 모니터링을 클릭하십시오.

구성 탭

시작
Application Server가 지정된 서비스 시작 시도 여부를 지정합니다. 성능 모니터링 서비스가 사용 불가능한 경우에 Application Server가 시작되면 서버를 다시 시작하여 이를 사용 가능하게 해야 합니다.
초기 스펙 레벨
서버의 모든 구성요소에 대해 PMI(Performance Monitoring Infrastructure) 스펙 레벨을 저장하는 PMI 문자열을 지정합니다(예를 들어, 모듈 레벨).

없음, 표준 또는 사용자 정의 선택란을 선택함으로써 PMI 스펙 레벨을 설정할 수 있습니다. 없음을 선택할 경우, 모든 PMI 모듈이 없음 레벨로 설정됩니다. 표준을 선택하는 경우, 모든 PMI 모듈이 고로 설정되고 메소드 레벨 데이터 및 JVMPI 데이터를 제외한 모든 PMI 데이터를 사용 가능하게 할 수 있습니다. 사용자 정의를 선택한 경우에는 각 개별적인 PMI 모듈에 대하여 레벨을 변경하는 옵션을 제공합니다. 사용자는 레벨을 N, L, M, H 또는 X(없음, 저, 중, 고 및 최대)로 설정할 수 있습니다. 모듈 이름을 변경하지 않음에 주의하십시오.

런타임 탭

스펙
사용자가 설정한 PMI 모듈과 모니터링 레벨을 지정합니다.

없음, 표준 또는 사용자 정의 선택란을 선택함으로써 PMI 스펙 레벨을 설정할 수 있습니다. 없음을 선택할 경우, 모든 PMI 모듈이 없음 레벨로 설정됩니다. 표준을 선택하는 경우, 모든 PMI 모듈이 고로 설정되고 메소드 레벨 데이터 및 JVMPI 데이터를 제외한 모든 PMI 데이터를 사용 가능하게 할 수 있습니다. 사용자 정의를 선택한 경우에는 각 개별적인 PMI 모듈에 대하여 레벨을 변경하는 옵션을 제공합니다. 사용자는 레벨을 N, L, M, H 또는 X(없음, 저, 중, 고 및 최대)로 설정할 수 있습니다. 모듈 이름을 변경하지 않음에 주의하십시오.
