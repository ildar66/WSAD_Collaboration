외부 캐시 그룹 구성원 콜렉션

이 페이지를 참조하여 캐시 그룹의 구성원인 특정 캐시를 정의하십시오.

이 관리 콘솔 페이지를 보려면 서버 > Application Servers > server > 동적 캐시 서비스 > 외부 캐시 그룹 > external_cache_group > 외부 캐시 그룹 구성원을 클릭하십시오.

AdapterBeanName
어댑터 Bean 이름을 지정합니다.

WebSphere Application Server에서 지원되는 어댑터 Bean 이름 예는 다음과 같습니다.
AFPA
AdapterBeanName: com.ibm.ws.cache.servlet.Afpa
주소: afpa가 청취하는 포트
ESI
AdapterBeanName: com.ibm.websphere.servlet.cache.ESIInvalidatorServlet
주소: 로컬 호스트
WTE
AdapterBeanName: com.ibm.websphere.edge.dynacache.WteAdapter
주소: 호스트 이름:포트 (WTE가 청취 중인 호스트 이름 및 포트)
