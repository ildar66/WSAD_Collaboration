관리 콘솔 필터 설정

필터 설정을 사용하여 필터 항목이 콜렉션 보기에 표시되는 방법을 지정합니다.

필터할 열을 선택한 다음 필터 기준을 입력하십시오.

필터할 열
드롭 다운 목록을 사용하여 필터할 열을 선택하십시오. 필터를 적용할 때, 콜렉션 테이블이 적절하게 수정됩니다.

예를 들어, Application Server 이름을 필터하는 기준을 입력할 계획인 경우, Application Server를 선택하십시오.

필터 기준
드롭 다운 목록 옆의 필드에서, 콜렉션 항목의 이름에 있어야 하는 문자열을 입력하여 콜렉션 테이블에 나타낼 항목을 규정하십시오. 문자열에는 %, * 또는 ? 기호가 와일드 카드 문자로서 포함될 수 있습니다. 예를 들어, *App*를 입력하여 이름에 App 문자열이 포함된 Application Server를 찾으십시오.

정확하게 검색을 수행하는 정규 표현식 엔진이 제공된 검색 기준에 일치하도록 ( ) ^ * % { } \ + $ 문자 각각에 접두부 \를 붙이십시오. 예를 들어, 프로바이더 이름에 (XA)를 포함하는 모든 JDBC 프로바이더를 검색하려면 다음과 같이 지정하십시오.

*\(XA\)
