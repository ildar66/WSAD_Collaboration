서비스 세부사항 디버깅

이 페이지를 참조하여 디버깅 서비스에서 사용하는 설정을 보고 수정하십시오.

이 관리 콘솔 페이지를 보려면 서버 > Application Servers > server > 디버깅 서비스를 클릭하십시오.

구성 탭

시작
서버 시작 시 서버가 디버그 서비스를 시작할지 여부를 지정합니다.
JVM 디버그 포트
디버그 연결을 위해 Java 가상 시스템이 청취할 포트를 지정합니다.
JVM 디버그 인수
디버그 모드로 JVM을 시작하는 데 사용되는 디버깅 인수 문자열을 지정합니다.
디버그 클래스 필터
디버그 시 무시할 클래스 배열을 지정합니다. 단계별 모드에서 실행하는 경우, 필터 항목과 일치하는 클래스에서는 디버거가 중지되지 않습니다.
BSF 디버그 포트
BSF 디버그 관리자가 청취하는 포트를 지정합니다.
BSF 로깅 레벨
BSF 디버그 관리자가 제공하는 로깅 레벨을 지정합니다. 올바른 범위는 0 - 3이며, 최대 로깅 레벨은 3입니다.
