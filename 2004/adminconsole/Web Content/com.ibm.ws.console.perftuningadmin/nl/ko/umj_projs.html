WebSphere 내부 JMS 서버 설정

WebSphere Application Server 관리 도메인 내의 노드에 있는 JMS 함수는 해당 노드의 JMS 서버에서 제공합니다. 선택한 JMS 서버의 구성 등록 정보를 보거나 변경하려면 이 패널을 사용하십시오.

이 패널을 사용하여 임베드된 WebSphere JMS 프로바이더에 대해 자동으로 구성된 기본 등록 정보 값에 추가되는 JMS 서버 등록 정보의 일반 설정을 구성할 수 있습니다.

참고: 일반적으로 WebSphere MQ 등록 정보의 기본값은 WebSphere 내부 JMS 서버에 적절합니다. 그러나 로드가 큰 메시징을 실행하는 경우, 일부 WebSphere MQ 등록 정보(예: 로그 파일 위치, 파일 페이지 및 버퍼 페이지에 대한 등록 정보)를 변경해야 합니다. WebSphere MQ 등록 정보 구성에 대한 자세한 정보는 WebSphere MQ System Administration 서적, SC33-1873을 참조하십시오. 이 서적은    IBM Publications Center 또는 WebSphere MQ collection kit, SK2T-0730 서적에서 사용 가능합니다.
