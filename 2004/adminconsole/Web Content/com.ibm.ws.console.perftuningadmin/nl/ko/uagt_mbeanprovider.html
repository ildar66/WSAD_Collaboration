Extension MBean Providers 콜렉션

이 페이지를 참조하여 JMX extension MBean provider를 보고 변경하십시오.

JMX Extension MBean Provider를 구성하여 코어 관리 시스템의 기존 WebSphere 관리 자원을 확장하는데 사용할 수 있습니다. 각각의 MBean 프로바이더는 JMX MBean 및 MBean XML 설명자 파일의 구현을 포함하는 라이브러리입니다.

이 관리 콘솔 페이지를 보려면, 서버 > Application Servers > server_name > 관리 서비스 > Extension MBean Provider를 클릭하십시오.

클래스 경로
MBean Descriptor가 위치할 수 있는 프로바이더 라이브러리의 클래스 경로.
설명
Extension MBean Provider 구성을 위한 임의의 설명 텍스트.
이름
Extension MBean Provider 라이브러리를 식별하기 위해 사용되는 이름.
