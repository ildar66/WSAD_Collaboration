HTTP 전송 콜렉션

이 페이지를 참조하여 HTTP 전송을 보거나 관리하십시오. 전송은 웹 서버에 대한 WebSphere 플러그인과 응용프로그램의 웹 모듈이 상주하는 웹 컨테이너 사이의 요청 대기열을 제공합니다. 사용자가 웹 브라우저에서 응용프로그램을 요청할 때 요청은 웹 서버에 전달된 후 전송을 따라 웹 컨테이너에 전달됩니다.

이 관리 콘솔 페이지를 보려면 서버 > Application Servers > server_name > 웹 컨테이너 > HTTP 전송을 클릭하십시오.

호스트
전송을 위해 바인드할 호스트 IP 주소를 지정합니다. Application Server가 로컬 시스템에 있는 경우, 호스트 이름은 localhost일 수 있습니다.
포트
전송하기 위해 바인드할 포트를 지정합니다. 포트 번호는 현재 시스템에서 사용하지 않는 임의의 포트일 수 있습니다. 지정된 시스템에서 각 Application Server 인스턴스에 대한 포트 번호는 고유해야 합니다.
SSL 사용 가능
WebSphere 플러그인과 SSL(Secure Sockets Layer)을 사용하는 Application Server 간의 연결을 보호할지 여부를 지정합니다. 기본값은 SSL을 사용하지 않는 것입니다.
