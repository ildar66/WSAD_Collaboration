HTTP 전송 설정값

이 페이지를 참조하여 HTTP 전송을 보고 구성하십시오. 페이지의 이름은 DefaultSSLSettings 같은 SSL 설정의 이름일 수 있습니다.

이 관리 콘솔 페이지를 보려면 서버 > Application Servers > server_name > 웹 컨테이너 > HTTP 전송 > host_name을 클릭하십시오.

구성 탭

호스트
전송을 위해 바인드할 호스트 IP 주소를 지정합니다.

Application Server가 로컬 시스템에 있는 경우, 호스트 이름은 localhost일 수 있습니다.

데이터 유형 문자열
포트
전송하기 위해 바인드할 포트를 지정합니다. 1 - 65535 사이의 포트 번호를 지정하십시오. 포트 번호는 지정된 시스템에서 각 Application Server에 대해 고유해야 합니다.
데이터 유형 정수
범위 1 - 65535
SSL 사용 가능
WebSphere 플러그인과 SSL(Secure Sockets Layer)을 사용하는 Application Server 간의 연결을 보호할지 여부를 지정합니다. 기본값은 SSL을 사용하지 않는 것입니다.
데이터 유형 부울
기본값 false
SSL
WebSphere 플러그인 Application Server 간 연결에 대해 SSL(Secure Sockets Layer) 설정 유형을 지정합니다. 옵션은 보안 센터에 정의된 하나 이상의 SSL 설정을 포함합니다. 예: DefaultSSLSettings, ORBSSLSettings 또는 LDAPSSLSettings.
데이터 유형 문자열
기본값 보안 센터에 정의된 SSL 설정.
