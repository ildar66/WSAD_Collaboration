엔드포인트 설정

이것을 사용하여 한 프로세스 내에서 실행 중인 런타임 구성요소가 사용하는 통신 끝점에 대한 구성을 보고 변경하십시오. 끝점은 서버의 호스트 및 포트 스펙을 제공합니다.

이 관리 콘솔 페이지를 보려면, 서버 > Application Servers > server_name > 엔드포인트> end_point_name을 클릭하십시오.

구성 탭

엔드포인트 이름
엔드포인트의 이름을 지정합니다. 이름은 서버 내에서 고유해야 합니다.

이 필드는 Application Server에 대한 엔드포인트를 정의하는 경우에만 표시됩니다.

데이터 유형 문자열
호스트
클라이언트가 자원(예: 네임 서비스, 관리 서비스 또는 JMS 브로커)을 요청하기 위해 사용하는 IP 주소, 도메인 이름 확장자를 갖는 DNS 호스트 이름 또는 그냥 DNS 호스트 이름을 지정합니다.

예를 들어, 호스트 이름이 myhost인 경우, 완전한 DNS 이름은 myhost.myco.com이 될 수 있고 IP 주소는 155.123.88.201이 될 수 있습니다.

데이터 유형 문자열
기본값 *
포트
클라이언트 요청을 승인하도록 서비스가 구성된 포트를 지정하십시오. 호스트 이름과 함께 포트 값이 사용됩니다.
데이터 유형 정수
기본값 없음
