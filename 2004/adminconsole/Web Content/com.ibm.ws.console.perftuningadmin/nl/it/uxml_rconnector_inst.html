Impostazioni del connettore JMX

Utilizzare questa pagina per visualizzare la configurazione del connettore JMX (Java Management Extensions).

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Servizi di gestione > Connettori JMX > tipo_connettore

Tipo dati Enum

Intervallo
SOAPConnector
Per connessioni JMX utilizzando SOAP (Simple Object Access Protocol).
RMIConnector
Per connessioni JMX utilizzando RMI (Remote Method Invocation).
HTTPConnector
Per connessioni JMX utilizzando HTTP.
JMSConnector
Per connessioni JMX utilizzando JMS (Java Messaging Service).
