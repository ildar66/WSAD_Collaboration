Impostazioni log di servizio IBM

Utilizzare questa pagina per configurare il log di servizio IBM, noto anche come log delle attività.

Per visualizzare questa pagina della console di gestione, fare clic su Risoluzione dei problemi > Log e traccia > server > Log di servizio IBM.

Scheda Configurazione

Abilita log servizio
Specifica la creazione di un file di log da parte del log di servizio IBM.
Nome file
Specifica il nome del file utilizzato dal log di servizio IBM.
Dimensione massima file
Specifica la dimensione massima in megabyte del file di log del servizio. Il valore predefinito è 2 megabyte.

Una volta raggiunta questa dimensione, il log di servizio viene allineato nella posizione corretta. Tale log non esegue il rollover su un file di log nuovo, come i log JVM.

Filtro dei messaggi
Specifica il livello del filtro dei messaggi sullo stato desiderato. È possibile utilizzare questa opzione per filtrare o sopprimere alcune categorie di messaggi. Questo valore filtro viene applicato a tutti i log e non solo a quello di servizio.
Abilita ID di correlazione
Specifica la creazione di un ID di correlazione collegato ad ogni messaggio.

È possibile utilizzare l'ID di correlazione per collegare l'attività a una richiesta client particolare o per correlare le attività su più server delle applicazioni.
