Impostazioni endpoint

Utilizzare questa pagina per visualizzare e modificare la configurazione di un endpoint di comunicazione utilizzato dai componenti di runtime in esecuzione in un processo. Un endpoint fornisce le specifiche di porta e host di un server.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Endpoint > nome_end_point

Scheda Configurazione

Nome endpoint
Specifica il nome dell'endpoint. Il nome deve essere univoco all'interno del server.

Notare che questo campo viene visualizzato solo durante la definizione di un endpoint di un server delle applicazioni.

Tipo dati Stringa
Host
Specifica l'indirizzo IP, il nome host DNS (Domain Name Server) con il suffisso nome dominio o soltanto il nome host DNS utilizzato da un client per richiedere una risorsa (come il servizio di denominazione, il servizio di gestione o il broker JMS).

Ad esempio, se il nome host è myhost, il nome DNS completo può essere myhost.myco.com e l'indirizzo IP 155.123.88.201.

Tipo dati Stringa
Predefinito *
Porta
Specifica la porta per cui il servizio è stato configurato per accettare richieste client. Il valore della porta è utilizzato in unione con il nome host.
Tipo dati Numero intero
Predefinito Nessuna
