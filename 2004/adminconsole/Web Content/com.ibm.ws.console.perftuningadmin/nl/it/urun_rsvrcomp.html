Raccolta Componenti server

Utilizzare questa pagina per gestire i tipi di componenti server come i server delle applicazioni, i server di messaggistica o i server dei nomi e per visualizzarne le informazioni.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Componenti server.
