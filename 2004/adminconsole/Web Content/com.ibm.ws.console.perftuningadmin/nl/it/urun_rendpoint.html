Raccolta di endpoint

Utilizzare questa pagina per visualizzare e gestire gli endpoint di comunicazione utilizzati dai componenti di runtime in esecuzione all'interno di un processo. Gli endpoint forniscono le specifiche di porta e host di un server.

Per visualizzare la pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Endpoint.

Notare che questa pagina viene visualizzata solo durante la gestione degli endpoint dei server delle applicazioni.

Nome endpoint
Specifica il nome di un endpoint. Ogni nome deve essere univoco all'interno del server.
