Raccolta membri del gruppo cache esterno

Utilizzare questa pagina per definire le cache specifiche che fanno parte di un gruppo cache.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > server > Servizio cache dinamica > Gruppi cache esterni > gruppi_cache_esterni > Membro del gruppo cache esterno.

AdapterBeanName
Specifica il nome del bean adattatore.

Nomi di bean adattatore di esempio supportati in WebSphere Application Server sono:
AFPA
AdapterBeanName: com.ibm.ws.cache.servlet.Afpa
Indirizzo: porta su cui è in ascolto afpa
ESI
AdapterBeanName: com.ibm.websphere.servlet.cache.ESIInvalidatorServlet
Indirizzo: host locale
WTE
AdapterBeanName: com.ibm.websphere.edge.dynacache.WteAdapter
Indirizzo: nome host:porta (nome host e porta su cui WTE è in ascolto)
