Impostazioni del servizio Administration

Utilizzare questa pagina per visualizzare e modificare la configurazione di un servizio di gestione.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Servizi di gestione

Scheda Configurazione

Autonomo
Specifica se il processo server è un partecipante in una cella di Network Deployment o meno. Se il valore della condizione è true, il server non partecipa alla gestione distribuita. Se il valore della condizione è false, il server partecipa al sistema Network Deployment.

Il valore predefinito delle installazioni di WebSphere Application Server di base è true. Quando addNode è in esecuzione per incorporare il server in una cella Network Deployment, il valore viene commutato in false.

Tipo dati Valore booleano
Predefinito vero
Connettori preferiti
Specifica il tipo di connettore JMX desiderato. Le opzioni disponibili, come SOAPConnector o RMIConnector, vengono definite utilizzando la pagina Connettori JMX.
Tipo dati Stringa
