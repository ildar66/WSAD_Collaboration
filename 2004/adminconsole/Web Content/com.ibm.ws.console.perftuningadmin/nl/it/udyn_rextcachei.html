Impostazioni gruppo cache esterno

Utilizzare questa pagina per configurare i gruppi di cache esterne controllati da WebSphere Application Server sui server Web, quali IBM Edge Server e IBM HTTP Server.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > server > Servizio cache dinamica > Gruppi cache esterni > gruppo_cache_esterno.

Scheda Configurazione

Nome
Specifica il nome del gruppo cache esterno.

Il nome del gruppo cache esterno deve corrispondere alla proprietà externalcache come definito nel file cachespec.xml JSP (JavaServer Pages) o servlet.

Quando la memorizzazione nella cache esterna è abilitata, la cache associa le pagine alle URI e invia le pagine associate alla cache esterna. Le voci possono quindi essere supportate dalla cache esterna, invece che dal server delle applicazioni. Questa opzione consente di migliorare le prestazioni.

Tipo
Specifica il tipo di gruppo cache esterno.
