Impostazioni di parametri di ottimizzazione

Utilizzare questa pagina per impostare i parametri di ottimizzazione per le sessioni distribuite.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Contenitore Web > Gestione sessioni > Impostazioni ambiente distribuito > Parametri di ottimizzazione personalizzati.

Scheda Configurazione

Livello di ottimizzazione
Specifica che la funzione Gestione sessioni fornisce alcune impostazioni predefinite che influiscono sulle prestazioni.

Selezionare una delle impostazioni predefinite o personalizzarne una. Per personalizzare un'impostazione, selezionare una delle impostazioni predefinite che più assomiglia all'impostazione desiderata, fare clic su Impostazioni personalizzate, apportare le modifiche, quindi fare clic su OK.
