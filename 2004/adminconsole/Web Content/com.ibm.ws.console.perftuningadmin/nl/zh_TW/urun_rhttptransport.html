HTTP 傳輸集合

您可利用這個頁面來檢視或管理 HTTP 傳輸。傳輸會提供 Web 伺服器之 WebSphere 外掛程式和應用程式之 Web 模組所在之 Web 儲存區之間的 要求佇列。當您在 Web 瀏覽器中要求一個應用程式時，要求將傳遞到 Web 伺服器，再傳輸到 Web 儲存區。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > Web 儲存區 > HTTP 傳輸。

主機
指定傳輸要連結的主機 IP 位址。 如果應用程式伺服器位在本端機器，則主機名稱可能是 localhost。
埠
指定傳輸要連結的埠。 埠號可以是系統目前不在使用中的任何埠。 給定的機器上的每一個應用程式伺服器都必須有一個唯一的埠號。
SSL 已啟用
指定要不要利用 Secure Socket Layer (SSL) 來保護在 WebSphere 外掛程式和應用程式伺服器之間的連線。預設不會使用 SSL。
