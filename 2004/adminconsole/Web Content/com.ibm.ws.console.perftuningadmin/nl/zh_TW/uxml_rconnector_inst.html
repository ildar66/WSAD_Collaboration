JMX 連接器設定

您可利用這個頁面來檢視 Java 管理延伸規格 (JMX) 連接器的配置。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > 管理服務 > JMX 連接器 > connector_type

「配置」標籤

Type
指定 JMX 連接器的類型。
資料類型 列舉
預設值 SOAPConnector
範圍
SOAPConnector
用於使用簡式物件存取通訊協定 (SOAP) 的 JMX 連線。
RMIConnector
用於使用遠端方法呼叫 (RMI) 的 JMX 連線。
HTTPConnector
用於使用 HTTP 的 JMX 連線。
JMSConnector
用於使用 Java 傳訊服務 (JMS) 的 JMX 連線。
