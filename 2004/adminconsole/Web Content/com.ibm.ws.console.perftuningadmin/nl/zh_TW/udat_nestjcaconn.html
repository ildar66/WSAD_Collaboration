連接器模組設定

請利用這個頁面來檢視連接器模組。

如果要檢視這個管理主控台頁面，請按一下應用程式 > 企業應用程式 > application > 連接器模組 > connector_module。

「配置」標籤

Uri
指定 WebSphere Application Server 要服務的資源之邏輯路徑。
資料類型 字串
名稱
指定連接器模組的顯示名稱。
資料類型 字串
altDD
可指定連接器模組的另一個 DD。

給定模組的另一個 DD URI。

資料類型 字串
啟動加權
指定連接器模組相對於其他模組的啟動優先順序。

如果應用程式含有多個模組時，您所指定的起始份量，會決定在伺服器啟動時，這個模組相對於其他模組的啟動優先順序。 啟動順序較低的模組，會優先啟動。
