外部快取群組集合

請利用這個頁面來定義在 IBM Edge Server 和 IBM HTTP Server 之類的 Web 伺服器中，WebSphere Application Server 所控制的各組外部快取。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server > 動態快取服務 > 外部快取群組。

名稱
指定外部快取群組名稱。

外部快取群組名稱必須符合 Servlet 或 JSP cachespec.xml檔中所定義的 externalcache 內容。

當啟用外部快取時，快取記憶體會比對頁面及其 URI，再將相符的頁面推到外部快取。 之後，就可以從外部快取記憶體提供項目，而不是從應用程式伺服器提供。

類型
指定外部快取群組類型。
