管理服務設定

請利用這個頁面來檢視和變更管理服務的配置。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > 管理服務。

「配置」標籤

獨立式
指定伺服器程序是否為 Network Deployment Cell 中的參與者。如果是 true，則伺服器不會參與分散式管理。 如果是 false，伺服器會參與 Network Deployment 系統。

基本 WebSphere Application Server 安裝架構的預設值是 true。 當執行 addNode 以將伺服器納入 Network Deployment Cell 時，值會切換成 false。

資料類型 Boolean
預設值 true
偏好的連接器
指定偏好的 JMX 連接器類型。可用的選項，如 SOAPConnector 或 RMIConnector，是使用「JMX 連接器」頁面來定義的。
資料類型 字串
