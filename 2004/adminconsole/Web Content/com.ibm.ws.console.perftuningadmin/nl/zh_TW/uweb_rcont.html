Web 儲存區設定

請利用這個頁面來配置儲存區設定。

請在管理主控台中遍訪下列路徑來存取這個頁面：

伺服器 > 應用程式伺服器 > server_instance > Web 儲存區

配置 - 一般內容

預設虛擬主機
指定虛擬主機是能使單一主機機器變成類似多部主機機器的配置。 虛擬主機所關聯的資源無法與另一虛擬主機所關聯的資源共用資料，即使它們共用相同的實體機器也一樣。

請選取虛擬主機選項：

預設主機
產品提供一個預設虛擬主機，含有若干通用別名，如機器的 IP 位址、短的主機名稱及完整的主機名稱。 別名構成了 Servlet 這類資源之存取路徑的第一部分。 比方說，在 http://localhost:9080/myServlet 要求中，它是 localhost:9080。
管理主機
這是應用程式伺服器的另一個名稱；在基本安裝架構中，也稱為 server1。 這個程序支援利用管理主控台。
Servlet 快取
指定如果呼叫 Servlet，且它產生要快取的輸出，就建立一個快取項目，不但包含輸出，同時也包含呼叫的邊際作用。 這些邊際作用可能包括呼叫其他 Servlet 或 Java Server Pages (JSP) 檔，以及項目的 Meta 資料，其中包括逾時和項目優先順序資訊。
