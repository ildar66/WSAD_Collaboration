自訂服務設定

請利用這個頁面來配置在應用程式伺服器中執行的服務。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > 自訂服務 > custom_service_name。

「配置」標籤

啟動
指定當服務所在的程序（伺服器）啟動時，伺服器要不要嘗試啟動和起始設定服務。 依預設，當服務所在的程序啟動時，不會啟用服務。如果要啟用服務，請選取這個勾選框。
資料類型 Boolean
預設值 false
外部配置 URL
指定自訂服務配置檔的 URL。

如果您的自訂服務類別需要配置檔，請針對此值指定該配置檔的完整路徑名稱。這個檔案名稱會傳給您的自訂服務類別。

資料類型 字串
單位 URL
類別名稱
指定服務實作方式的類別名稱。這個類別必須實作「自訂服務」介面。
資料類型 字串
單位 Java 類別名稱
顯示名稱
指定服務的名稱。
資料類型 字串
說明
說明自訂服務。
資料類型 字串
類別路徑
指定用來尋找這個服務的類別和 JAR 檔的類別路徑。
資料類型 字串
單位 類別路徑
