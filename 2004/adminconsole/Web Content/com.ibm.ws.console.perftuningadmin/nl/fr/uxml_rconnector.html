Connecteurs Java Management Extensions

Cette page permet d'afficher et de modifier la configuration des connecteurs JMX (Java Management Extensions).

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > nom_serveur > Services d'administration > Connecteurs JMX.

Les connecteurs Java Management Extensions (JMX) communiquent avec WebSphere Application Server lorsque vous appelez un processus de scriptage. Il n'existe pas de valeur par défaut pour le type et les paramètres d'un connecteur. Le fichier wsadmin.properties désigne le connecteur SOAP (Simple Object Access Protocol) et fournit un numéro de port approprié. Vous pouvez également utiliser le connecteur RMI (Remote Method Invocation).

Pour sélectionner le type et les attributs du connecteur, utilisez l'une des méthodes suivantes :

  • Définissez des propriétés dans un fichier de propriétés.
  • Entrez des options sur la ligne de commande.
