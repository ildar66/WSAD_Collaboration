Paramètres de définition des processus

Cette page permet d'afficher ou de modifier les paramètres d'une définition de processus, qui fournit des informations de ligne de commande relatives au démarrage ou à l'initialisation d'un processus.

Pour afficher cette page de la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur > Définition du processus.

Onglet Configuration

Nom de l'exécutable
Indique le nom de l'exécutable du processus.
Type de données Chaîne
Arguments du fichier exécutable
Indique les commandes de l'exécutable qui s'exécutent lors du démarrage du processus.

Par exemple, le programme cible de l'exécutable accepte les trois arguments suivants : arg1 arg2 arg3.

Type de données Chaîne
Unité Arguments de la ligne de commande Java
Répertoire de travail
Indique le répertoire du système de fichiers dans lequel le processus va s'exécuter.

Ce répertoire permet de déterminer l'emplacement des fichiers d'entrée et de sortie avec les noms de chemins d'accès relatifs.

Les beans enterprise rendus passifs sont placés dans le répertoire de travail en cours du serveur d'applications sur lequel les beans s'exécutent. Assurez-vous que le répertoire de travail est un répertoire connu sous le répertoire principal de WebSphere Application Server.

Type de données Chaîne
