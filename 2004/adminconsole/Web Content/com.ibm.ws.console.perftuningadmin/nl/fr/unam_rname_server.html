Paramètres du serveur de noms

Utilisez cette page pour configurer les paramètres de fournisseur de service de nommage pour le serveur d'applications.

Pour afficher cette page de la console administrative console, cliquez sur Serveurs > Serveurs d'applications > nom_serveur > Composants serveur > Serveur de noms.
