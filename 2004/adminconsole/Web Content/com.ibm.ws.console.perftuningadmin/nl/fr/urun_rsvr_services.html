Collection de services personnalisés

Cette page permet d'afficher la liste des services disponibles sur le serveur d'applications et de savoir s'ils sont activés. Un service personnalisé offre la possibilité de se connecter à un serveur d'applications WebSphere pour définir un code qui s'exécute au démarrage ou à l'arrêt du serveur.

Pour afficher la page de la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur> Services personnalisés.

URL de la configuration externe
Indique l'URL du fichier de configuration d'un service personnalisé.

Si la classe des services personnalisés requiert un fichier de configuration, la valeur fournit un nom de chemin complet à ce fichier de configuration. Ce nom de fichier est transmis à la classe de service personnalisé.

Nom de classe
Indique le nom de classe de l'implémentation du service. Cette classe doit implémenter l'interface du service personnalisé.
Nom affiché
Indique le nom du service.
Démarrage
Indique si le serveur tente de démarrer et d'initialiser le service lorsque le processus qui le contient (le serveur) démarre. Par défaut, le service n'est pas activé lorsque le processus qui le contient démarre.
