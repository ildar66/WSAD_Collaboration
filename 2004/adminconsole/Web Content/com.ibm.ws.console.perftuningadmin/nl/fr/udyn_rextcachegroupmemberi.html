Paramètres des membres de groupes de caches externes

Cette page permet de configurer des caches spécifiques qui font partie d'un groupe de caches.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > serveur > Service de cache dynamique > Groupes de caches externes > groupe_caches_externes > Membres du groupe de caches externes > membres_groupe_caches_externes.

Onglet Configuration

Adresse
Indique une chaîne de configuration utilisée par le bean adaptateur de cache externe pour se connecter à la mémoire cache externe.
Nom du bean adaptateur
Indique le nom du bean adaptateur.
AFPA
AdapterBeanName : com.ibm.ws.cache.servlet.Afpa
Address : port d'écoute d'afpa
ESI
AdapterBeanName : com.ibm.websphere.servlet.cache.ESIInvalidatorServlet
Address : hôte local
WTE
AdapterBeanName : com.ibm.websphere.edge.dynacache.WteAdapter
Address : nom d'hôte:port (nom d'hôte et port sur lequel WTE effectue l'écoute
