Paramètres d'optimisation

Cette page permet de définir des paramètres d'optimisation pour les sessions réparties.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > nom_serveur > Conteneur Web > Gestion de session > Paramètres d'environnement répartis > Paramètres d'optimisation personnalisés.

Onglet Configuration

Niveau d'optimisation
Indique que l'utilitaire de gestion de session fournit certains paramètres prédéfinis qui ont une incidence sur les performances.

Sélectionnez un de ces paramètres prédéfinis ou personnalisez un paramètre. Pour personnaliser un paramètre, sélectionnez un des paramètres prédéfinis le plus proche de celui que vous voulez, cliquez sur Paramètres personnalisés, apportez vos modifications, puis cliquez sur OK.
