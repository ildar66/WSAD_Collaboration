Collection de références de bibliothèques

Cette page permet d'afficher et de gérer des références de bibliothèques qui définissent l'utilisation des bibliothèques générales. Vous pouvez par exemple utiliser cette page pour associer des fichiers de bibliothèque partagée à une application déployée.

Pour afficher la page de la console d'administration, sélectionnez Applications > Applications d'entreprise > nom_application > Bibliothèques.

Nom de bibliothèque
Indique le nom d'une référence de bibliothèque.
