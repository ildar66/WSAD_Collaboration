Paramètres du service de référentiel

Cette page permet d'afficher et de modifier la configuration d'un référentiel de service d'administration.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > nom_serveur > Services d'administration> Service de référentiel.

Onglet Configuration

Audit activé
Indique si les mises à jour du référentiel dans le fichier journal doivent faire l'objet d'un audit. Par défaut, les mises à jour du référentiel font l'objet d'un audit.
Type de données Booléen
Valeur par défaut true
