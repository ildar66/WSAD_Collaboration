Collection d'applications d'entreprise

Cette page permet d'afficher et de gérer des applications d'entreprise.

Pour afficher la page de la console d'administration, sélectionnez Applications > Applications d'entreprise.

Nom
Indique le nom de l'application installée (ou déployée). Les noms d'application doivent être uniques dans une cellule et ne contenir aucun caractère non autorisé dans les noms d'objet.
Etat
Indique si l'application déployée sur un serveur d'applications est lancée, arrêtée ou non disponible.

Une application à l'état non disponible peut, en réalité, être exécutée mais posséder cet état en raison du fait que le serveur sur lequel est exécutée la console d'administration ne peut pas communiquer avec le serveur sur lequel est exécutée l'application.
