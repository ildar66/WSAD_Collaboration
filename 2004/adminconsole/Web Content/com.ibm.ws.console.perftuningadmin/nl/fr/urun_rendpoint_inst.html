Collection de noeuds finaux

Cette page permet d'afficher et de modifier la configuration d'un noeud final de communication utilisé par les composants d'exécution s'exécutant dans un processus. Un noeud final fournit des spécifications relatives au port et à l'hôte pour un serveur.

Pour afficher la page de la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur > Noeuds finaux > nom_noeud_final.

Onglet Configuration

Nom du noeud final
Indique le nom du noeud final. Ce nom doit être unique sur le serveur.

Cette zone s'affiche uniquement lorsque vous définissez un noeud final pour un serveur d'applications.

Type de données Chaîne
Hôte
Indique l'adresse IP, le nom d'hôte DNS (Domain Name Server) avec préfixe de nom de domaine ou uniquement le nom d'hôte DNS utilisé par un client pour demander une ressource (telle qu'un service de nommage, un service d'administration ou JMS Broker).

Par exemple, si le nom d'hôte est monhôte, le nom DNS complet peut être monhôte.myco.com et l'adresse IP peut être 155.123.88.201.

Type de données Chaîne
Valeur par défaut *
Port
Indique le port pour lequel le service est configuré pour accepter les demandes du client. La valeur du port est utilisée avec le nom d'hôte.
Type de données Entier
Valeur par défaut Aucune
