Paramètres du service d'administration

Cette page permet d'afficher et de modifier la configuration d'un service d'administration.

Pour afficher la page de la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur > Services d'administration

Onglet Configuration

Autonome
Indique si le processus serveur fait partie d'une cellule de déploiement en réseau ou non. Si la valeur est true, cela signifie que le serveur ne fait pas partie de l'administration répartie. Si la valeur est false, cela signifie que le serveur fait partie d'un système Network Deployment.

La valeur par défaut des installations de base de WebSphere Application Server est true. Lorsque addNode est exécuté pour incorporer le serveur dans une cellule de Network Deployment, la valeur devient false.

Type de données Booléen
Valeur par défaut true
Connecteur préféré
Indique le type de connecteur JMX préféré. Les options disponibles, telles que SOAPConnector ou RMIConnector, peuvent être définies à l'aide de la page Connecteurs JMX.
Type de données Chaîne
