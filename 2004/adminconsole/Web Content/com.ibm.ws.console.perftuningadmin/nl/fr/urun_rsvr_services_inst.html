Paramètres des services personnalisés

Cette page permet de configurer un service qui s'exécute sur un serveur d'applications.

Pour afficher la page de la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur > Services personnalisés >nom_service_personnalisé.

Onglet Configuration

Démarrage
Indique si le serveur tente de démarrer et d'initialiser le service lorsque le processus qui le contient (le serveur) démarre. Par défaut, le service n'est pas activé lorsque le processus qui le contient démarre. Pour activer le service, cochez la case appropriée.
Type de données Booléen
Valeur par défaut false
URL de la configuration externe
Indique l'URL du fichier de configuration d'un service personnalisé.

Si la classe de vos services personnalisés nécessite un fichier de configuration, spécifiez le nom du chemin d'accès complet à ce fichier de configuration comme valeur. Ce nom de fichier est transmis à la classe de service personnalisé.

Type de données Chaîne
Unité URL
Nom de classe
Indique le nom de classe de l'implémentation du service. Cette classe doit implémenter l'interface du service personnalisé.
Type de données Chaîne
Unité Nom de classe Java
Nom affiché
Indique le nom du service.
Type de données Chaîne
Description
Décrit le service personnalisé.
Type de données Chaîne
Chemin d'accès aux classes
Indique le chemin d'accès aux classes permettant de situer les classes et les fichiers JAR pour ce service.
Type de données Chaîne
Unité Chemin d'accès aux classes
