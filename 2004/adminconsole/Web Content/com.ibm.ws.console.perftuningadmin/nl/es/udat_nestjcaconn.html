Valores de módulos de conector

Utilice esta página para ver los módulos de conector.

Para ver esta página de la consola administrativa, pulse Aplicaciones > Aplicaciones de empresa > aplicación > Módulos de conector > módulo_conector.

Pestaña Configuración

URI
Especifica la vía de acceso lógica al recurso que recibe servicios de WebSphere Application Server.
Tipo de datos String
Nombre
Especifica el nombre de visualización del módulo de conector.
Tipo de datos String
altDD
Especifica el DD alternativo del módulo de conector.

URI de DD alternativo de un módulo determinado.

Tipo de datos String
Peso inicial
Especifica la prioridad de arranque del módulo de conector sobre los demás.

Cuando la aplicación contiene varios módulos, el peso inicial que especifique determina la prioridad de arranque de este módulo sobre los demás módulos durante el arranque del servidor. Los módulos con orden de arranque más bajo se inician primero.

Tipo de datos String
