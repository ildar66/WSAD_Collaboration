Valores de componentes de servidor

Utilice esta página para ver o configurar una instancia de componente de servidor.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Componentes de servidor > nombre_componente_servidor.

Pestaña Configuración

Nombre
Especifica el nombre del componente.
Tipo de datos String
Estado inicial
Especifica el estado deseado del componente cuando se inicia el proceso del servidor. Las opciones son: Iniciado y Detenido. El valor por omisión es Iniciado.
Tipo de datos String
Valor por omisión Iniciado
