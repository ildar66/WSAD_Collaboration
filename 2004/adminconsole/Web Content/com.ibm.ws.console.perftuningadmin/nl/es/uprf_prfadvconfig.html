Valores de configuración del Asesor de rendimiento de tiempo de ejecución

Utilice esta página para especificar valores del Asesor de rendimiento de tiempo de ejecución.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Configuración del Asesor de rendimiento de tiempo de ejecución.

Pestaña Configuración

Habilitación del Asesor de rendimiento de tiempo de ejecución
Especifica si se ejecutará el Asesor de rendimiento de tiempo de ejecución.
Intervalo de cálculo
Los datos PMI se toman durante un periodo de tiempo y se calcula el promedio a fin de proporcionar un aviso. El intervalo de cálculo especifica la duración del tiempo durante el cual los datos se toman para este aviso. Por lo tanto, los detalles de los mensajes de aviso aparecerán como el promedio durante este intervalo.
Secuencia de aviso máxima
Especifica el número de avisos consecutivos antes de que se actualice el umbral.
Número de procesadores
Especifica el número de procesadores en el servidor.

Reiniciar (botón)
La selección de Reiniciar en la pestaña Tiempo de ejecución vuelve a reiniciar el Asesor de rendimiento de tiempo de ejecución utilizando la última información guardada en el disco. Tenga en cuenta que esta acción también restablece el estado del Asesor de rendimiento de tiempo de ejecución. Por ejemplo, el recuento actual de avisos se restablece a cero para cada mensaje.
