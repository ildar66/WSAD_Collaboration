Valores del servicio de supervisión del rendimiento

Utilice esta página para especificar valores para la supervisión del rendimiento, por ejemplo, para habilitar la supervisión del rendimiento, seleccionar el módulo PMI y establecer niveles de supervisión.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > servidor > Supervisión del rendimiento.

Pestaña Configuración

Arranque
Especifica si el servidor de aplicaciones intentará iniciar el servicio especificado. Si un servidor de aplicaciones se inicia cuando está inhabilitado el servicio de supervisión del rendimiento, tendrá que reiniciar el servidor para poder habilitarlo.
Nivel de especificación inicial
Especifica una serie de PMI (Performance Monitoring Infrastructure) que almacena los niveles de especificación de PMI, por ejemplo, los niveles de módulos, para todos los componentes del servidor.

Para establecer los niveles de especificación de PMI, seleccione el recuadro de selección ninguna, estándar o personalizada. Si selecciona ninguna, todos los módulos PMI están establecidos en el nivel ninguno. Si se selecciona estándar, se establecen todos los módulo PMI en alto y se habilitan todos los datos PMI, a excepción de los datos de nivel de método y los datos de JVMPI. Si selecciona personalizada, se le proporciona la opción de cambiar el nivel de cada uno de los módulos PMI. Puede establecer el nivel en N, L, M, H o X (ninguno, bajo, medio, alto y máximo). Tenga en cuenta que no debe modificar los nombres de módulo.
