Valores de ejecución de proceso

Utilice esta página para ver o modificar la información de línea de mandatos para iniciar o inicializar un proceso UNIX.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Definición de proceso > Ejecución de proceso.

Pestaña Configuración

Prioridad de proceso
Especifica la prioridad del sistema operativo para el proceso. Sólo los usuarios root pueden modificar este valor.
Tipo de datos Integer
Valor por omisión 1000 para WebSphere Application Server en la mayoría de sistemas operativos. En OS/400, el valor por omisión es 25.
UMASK
Especifica la máscara de usuario con la que se ejecuta el proceso (máscara de permiso de la modalidad de archivo).
Tipo de datos Integer
Ejecutar como usuario
Especifica el usuario como el que se ejecuta el proceso.
Tipo de datos String
Ejecutar como grupo
Especifica el grupo del que es miembro el proceso y como el que se ejecuta.

En OS/400, se ignora el valor de Ejecutar como grupo.

Tipo de datos String
Ejecutar en grupo de proceso
Especifica un grupo de proceso específico para el proceso. Este grupo de proceso es útil en operaciones como la partición de procesadores. Un administrador del sistema puede asignar un grupo de proceso para que se ejecute, por ejemplo, en 6 de 12 procesadores. El valor por omisión (0) no asigna el proceso a ningún grupo específico.

En OS/400, se ignora el valor Ejecutar en grupo de proceso.

Tipo de datos Integer
Valor por omisión 0
