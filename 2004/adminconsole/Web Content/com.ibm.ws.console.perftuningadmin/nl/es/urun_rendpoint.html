Colección de puntos finales

Utilice esta página para ver y gestionar los puntos finales de comunicación utilizados por los componentes de tiempo de ejecución que se ejecutan en un proceso. Los puntos finales proporcionan especificaciones de sistema principal y de puerto para un servidor.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Puntos finales.

Tenga en cuenta que esta página sólo se muestra cuando esté trabajando con puntos finales para servidores de aplicaciones.

Nombre de punto final
Especifica el nombre de un punto final. Cada nombre debe ser exclusivo en el servidor.
