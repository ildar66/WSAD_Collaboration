Valores del conector JMX

Utilice esta página para ver la configuración de conectores JMX (Extensiones de gestión de Java).

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Servicios de administración> Conectores JMX > tipo_conector.

Pestaña Configuración

Tipo
Especifica el tipo del conector JMX.
Tipo de datos Enum
Valor por omisión SOAPConnector
Rango
SOAPConnector
Para conexiones JMX que utilizan el protocolo SOAP (Protocolo de acceso a objetos sencillo).
RMIConnector
Para conexiones JMX que utilizan RMI (Invocación a método remoto).
HTTPConnector
Para conexiones JMX que utilizan HTTP.
JMSConnector
Para JMX conexiones que utilizan JMS (Servicio de mensajes Java).
