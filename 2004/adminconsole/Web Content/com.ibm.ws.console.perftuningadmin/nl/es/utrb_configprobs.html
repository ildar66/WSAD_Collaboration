Valores de los problemas de configuración

Utilice esta página para identificar y ver los problemas que se han producido en la configuración actual.

Para ver esta página de la consola administrativa, pulse Resolución de problemas > Problemas de configuración en el árbol de navegación de la consola.

Pulse un problema de configuración en la tabla Problemas de configuración para ver más información sobre el problema.

Validación de documentos de configuración
Utilice estos campos para especificar el nivel de validación que se va a realizar en los documentos de configuración.
Validación cruzada
Habilita la validación cruzada de documentos de configuración.

Al habilitar la validación cruzada, se permite la comparación de documentos de configuración para identificar valores en conflicto.

Problemas de configuración

Muestra los mensajes de error del problema de configuración actual. Pulse un mensaje para obtener información detallada sobre el problema.

Ámbito

Ordena la lista de problemas de configuración según el archivo de configuración donde ocurre cada error. Pulse un mensaje para obtener información detallada sobre el problema.

Campos que explican cada problema de configuración:

Los siguientes campos de información aparecen cuando pulsa un mensaje de problema de configuración.

Mensaje
Muestra el mensaje devuelto por el validador.
Explicación
Una breve descripción del problema.
Acción del usuario
Especifica la acción recomendada para corregir el problema.
Objeto de destino
Identifica el objeto de configuración donde se ha producido el error de validación.
Gravedad
Indica la gravedad de un error de configuración, siendo 1 un error grave. La gravedad disminuye a medida que aumenta el descriptor de gravedad.
URI local
Especifica el URI local del archivo de configuración donde se ha producido el error.
URI completo
Especifica el URI completo del archivo de configuración donde se ha producido el error.
Nombre de clase de validador
Nombre de clase del validador que notifica el problema.
