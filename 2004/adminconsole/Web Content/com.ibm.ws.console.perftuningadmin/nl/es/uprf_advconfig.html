Valores de configuración de avisos

Utilice esta página para seleccionar el aviso que desea habilitar o inhabilitar.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Configuración del Asesor de rendimiento de tiempo de ejecución.

Pestaña Configuración

Nombre del aviso
Especifica el aviso que puede habilitar o inhabilitar.
Aviso aplicado al componente
Especifica el componente de WebSphere Application Server al que se aplica el aviso de rendimiento de tiempo de ejecución.
Estado del aviso
Especifica si se detiene o se inicia el aviso.

Sólo existen dos valores -- Iniciado y Detenido. Iniciado indica que el aviso se ejecuta si se aplica el aviso. Detenido indica que el aviso no se ejecuta.

Pestaña Tiempo de ejecución

Nombre del aviso
Especifica el aviso que puede habilitar o inhabilitar.
Aviso aplicado al componente
Especifica el componente de WebSphere Application Server al que se aplica el aviso de rendimiento de tiempo de ejecución.
Estado del aviso
Especifica si se detiene, se inicia o no está disponible el aviso.

El estado del aviso tiene uno de los tres valores siguientes: Iniciado, Detenido o No disponible. Iniciado indica que se aplica el aviso. Detenido indica que no se aplica el aviso. No disponible indica que el aviso no se aplica a la configuración actual (como por ejemplo, el aviso Tamaño de la sesión persistente en una configuración sin sesiones persistentes).
