Valores de definición de proceso

Utilice esta página para ver o modificar los valores de una definición de proceso, que proporciona información de línea de mandatos para iniciar o inicializar un proceso.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Definición de proceso.

Pestaña Configuración

Nombre ejecutable
Especifica el nombre ejecutable del proceso.
Tipo de datos String
Argumentos ejecutables
Especifican los mandatos ejecutables que se ejecutan cuando se inicia el proceso.

Por ejemplo, el programa de destino ejecutable puede requerir tres argumentos: arg1 arg2 arg3.

Tipo de datos String
Unidades Argumentos de línea de mandatos de Java
Directorio de trabajo
Especifica el directorio del sistema de archivos donde se ejecutará el proceso.

Este directorio se utiliza para determinar las ubicaciones de los archivos de entrada y salida con nombres de vías de acceso relativas.

Los enterprise beans desactivados se sitúan en el directorio de trabajo activo del servidor de aplicaciones en el que se están ejecutando los beans. Asegúrese de que el directorio de trabajo es un directorio conocido situado bajo el directorio raíz del producto WebSphere Application Server.

Tipo de datos String
