Colección de servicios personalizados

Utilice esta página para ver una lista de los servicios disponibles para el servidor de aplicaciones y comprobar si estos servicios están habilitados. Un servicio personalizado proporciona la posibilidad de conectarse a un servidor de aplicaciones de WebSphere y definir el código que se ejecuta cuando se inicia el servidor o se cierra.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor> Servicios personalizados.

URL de configuración externo
Especifica el URL de un archivo de configuración de un servicio personalizado.

Si la clase de servicios personalizados requiere un archivo de configuración, este valor proporciona un nombre plenamente cualificado de la vía de acceso a ese archivo de configuración para el valor. Este nombre de archivo se pasa a la clase de servicio personalizado.

Classname
Especifica un nombre de clase de la implementación de servicio. Esta clase debe implementar HttpServlet.
Nombre de visualización
Especifica el nombre del servicio.
Arranque
Especifica si el servidor intentará iniciar e inicializar el servicio cuando se inicie el proceso (el servidor) que contiene. Por omisión, el servicio no está habilitado cuando se inicia el proceso que contiene.
