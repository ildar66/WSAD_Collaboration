Colección de puertos receptores de mensajes

Puertos receptores de mensajes configurados en el dominio administrativo.

Este panel muestra una lista de los puertos receptores de mensajes configurados en el dominio administrativo. Cada puerto receptor se utiliza con un bean controlado por mensajes para que reciba automáticamente mensajes de un destino de JMS asociado. Puede utilizar este panel para añadir puertos receptores nuevos o para modificar las propiedades de puertos receptores existentes. Para obtener más información sobre los campos de propiedades de puertos receptores, consulte Propiedades de puertos receptores.

Para ver esta página de la consola administrativa, pulse Servidores-> servidor_aplicaciones-> Servicio receptor de mensajes-> Puertos receptores
