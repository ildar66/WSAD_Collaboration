Valores de proveedores de MBean de extensión

Utilice esta página para ver y modificar la configuración de proveedores de MBean de extensión JMX.

Puede configurar bibliotecas que contienen implementaciones de MBeans JMX y sus correspondientes archivos descriptores de MBeans XML, con objeto de utilizarlos para ampliar los recursos existentes gestionados por WebSphere en el sistema administrativo central.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Servicios de administración > Proveedores de MBean de extensión> nombre_biblioteca_proveedor.

Pestaña Configuración

Classpath
Classpath contenida en la biblioteca del proveedor donde se puede localizar el descriptor de MBean. El cargador de clases necesita esta información para cargar y analizar el archivo descriptor de MBeans de extensión XML.
Tipo de datos String
Descripción
Texto descriptivo arbitrario para la configuración del proveedor de MBean de extensión. Utilice este campo para incluir texto que le ayude a identificar o diferenciar la configuración del proveedor.
Tipo de datos String
Nombre
Nombre que se utiliza para identificar la biblioteca del proveedor de MBean de extensión.
Tipo de datos String
