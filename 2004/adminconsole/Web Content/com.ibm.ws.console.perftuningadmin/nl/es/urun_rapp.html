Colección de aplicaciones de empresa

Utilice esta página para ver y gestionar aplicaciones de empresa.

Para ver esta página de la consola administrativa, pulse Aplicaciones > Aplicaciones de empresa.

Nombre
Especifica el nombre de la aplicación instalada (o desplegada). Los nombres de aplicación deben ser exclusivos en una célula y no deben contener caracteres no permitidos en los nombres de objeto.
Estado
Indica si la aplicación desplegada en el servidor de aplicaciones se ha iniciado, se ha detenido o no está disponible.

Es posible que una aplicación cuyo estado sea de no disponible se esté ejecutando a pesar de ello, ya que dicho estado sea debido a que el servidor que esté ejecutando la consola administrativa no se pueda comunicar con el servidor que esté ejecutando la aplicación.
