Valores de servicio personalizado

Utilice esta página para configurar un servicio que se ejecute en un servidor de aplicaciones.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Servicios personalizados >nombre_servicio_personalizado.

Pestaña Configuración

Arranque
Especifica si el servidor intentará iniciar e inicializar el servicio cuando se inicie el proceso (el servidor) que contiene. Por omisión, el servicio no está habilitado cuando se inicia el proceso que contiene. Para habilitar el servicio, coloque una marca de selección en el recuadro de selección.
Tipo de datos Boolean
Valor por omisión false
URL de configuración externo
Especifica el URL de un archivo de configuración de un servicio personalizado.

Si la clase de servicios personalizado requiere un archivo de configuración, especifique el nombre plenamente cualificado de la vía de acceso a ese archivo de configuración para el valor. Este nombre de archivo se pasa a la clase de servicio personalizado.

Tipo de datos String
Unidades URL
Classname
Especifica un nombre de clase de la implementación de servicio. Esta clase debe implementar HttpServlet.
Tipo de datos String
Unidades Nombre de clase Java
Nombre de visualización
Especifica el nombre del servicio.
Tipo de datos String
Descripción
Describe el servicio personalizado.
Tipo de datos String
Classpath
Especifica la classpath utilizada para localizar las clases y archivos JAR para este servicio.
Tipo de datos String
Unidades Classpath
