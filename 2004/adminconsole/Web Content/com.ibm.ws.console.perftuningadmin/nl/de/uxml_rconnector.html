JMX-Connector

Verwenden Sie diese Seite, um die Konfiguration von JMX-Connector (Java Management Extensions) anzuzeigen und zu ändern.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Server > Anwendungsserver > Servername > Administrationsservices > JMX-Connector.

JMX-Connector (JMX = Java Management Extensions) kommunizieren mit dem WebSphere Application Server, wenn Sie einen Scripting-Prozess aufrufen. Es gibt keinen Standard für Typ und Parameter eines Connectors. In der Datei wsadmin.properties sind der SOAP-Connector (Simple Object Access Protocol) und eine zugehörige Port-Nummer angegeben. Sie können auch den RMI-Connector (RMI = Remote Method Invocation) verwenden.

Verwenden Sie eine der folgenden Methoden, um Typ und Attribute des Connectors auszuwählen:

  • Geben Sie die Merkmale in einer Merkmalsdatei an.
  • Geben Sie Optionen in der Befehlszeile ein.
