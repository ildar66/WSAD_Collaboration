Einstellungen des Leistungsüberwachungsservice

Auf dieser Seite können Sie Einstellungen für die Leistungsüberwachung angeben: Aktivieren der Leistungsüberwachung, Auswahl des PMI-Moduls und Definition der Überwachungsstufen.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Server > Leistungsüberwachung.

Register "Konfiguration"

Start
Gibt an, ob der Anwendungsserver versucht, den angegebenen Service zu starten. Wenn ein Anwendungsserver bei inaktiviertem Leistungsüberwachungsservice gestartet wird, müssen Sie den Server erneut starten, um den Service zu aktivieren.
Ausgangsspezifikationsebene
Eine PMI-Zeichenfolge (Performance Monitoring Infrastructure), in der für alle Komponenten auf dem Server die Ebenen der PMI-Spezifikation, z. B. Modulebenen, angegeben sind.

Legen Sie die Ebenen der PMI-Spezifikation durch Auswahl des Markierungsfelds Keine, Standard oder Benutzerdefiniert fest. Wenn Sie Keine auswählen, wird für alle PMI-Module die Ebene Keine festgelegt. Bei Auswahl von Standard wird für alle PMI-Module die Ebene Hoch festgelegt. In diesem Fall werden alle PMI-Daten mit Ausnahme der Daten der Methodenebene und der JVMPI-Daten aktiviert. Bei Auswahl von "Benutzerdefiniert" haben Sie die Möglichkeit, die Ebene für jedes einzelne PMI-Modul zu ändern. Sie können die Ebene auf N (Keine, None), L (Niedrig, Low), M (Mittel, Medium), H (Hoch, High) oder X (Maximum) einstellen. Anmerkung: Sie dürfen die Modulnamen nicht ändern.
