Einstellungen des IBM Serviceprotokolls

Auf dieser Seite können Sie das IBM Serviceprotokoll (auch Aktivitätenprotokoll genannt) konfigurieren.

Klicken SIe zum Anzeigen dieser Seite der Administrationskonsole auf Fehlerbehebung > Protokolle und Trace > Server > IBM Serviceprotokolle.

Register "Konfiguration"

Serviceprotokoll aktivieren
Legt fest, dass eine Protokolldatei durch das IBM Serviceprotokoll erstellt wird.
Dateiname
Gibt den Namen der vom IBM Serviceprotokoll verwendeten Datei an.
Maximale Dateigröße
Gibt die maximale Größe der Serviceprotokolldatei in MB an. Der Standardwert ist 2 MB.

Wenn der Grenzwert erreicht ist, wird das Serviceprotokoll angezeigt. Beachten Sie, dass das Serviceprotokoll nicht in einer neuen Protokolldatei fortgeführt wird wie die JVM-Protokolle.

Nachrichtenfilter
Legt die Stufe des Nachrichtenfilters für den gewünschten Status fest. Sie können diese Option verwenden, um einige Nachrichtenkategorien herauszufiltern oder zu unterdrücken. Dieser Filter wird nicht nur für das Serviceprotokoll, sondern für alle Protokolle verwendet.
Korrelations-ID aktivieren
Gibt die Generierung einer Korrelations-ID an, die mit jeder Nachricht protokolliert wird.

Sie können die Korrelations-ID verwenden, um eine Aktivität zu einer bestimmten Clientanforderung bzw. Aktivitäten auf mehreren Anwendungsservern zueinander in Beziehung zu setzen.
