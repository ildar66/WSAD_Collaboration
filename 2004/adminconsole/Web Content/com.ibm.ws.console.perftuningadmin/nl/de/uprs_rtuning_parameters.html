Einstellungen für Optimierungsparameter

Auf dieser Seite können Sie Optimierungsparameter für verteilte Sitzungen festlegen.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Servername > Webcontainer > Sitzungsverwaltung > Einstellungen für verteilte Umgebung > Benutzerdefinierte Optimierungsparameter.

Register "Konfiguration"

Optimierungsebene
Gibt an, dass die Sitzungsverwaltung bestimmte vordefinierte Einstellungen bereitstellt, die die Leistung beeinflussen.

Wählen Sie eine dieser vordefinierten Einstellungen aus oder passen Sie eine Einstellung an. Möchten Sie eine Einstellung anpassen, müssen Sie eine der vordefinierten Einstellungen, die der gewünschten Einstellung am ähnlichsten ist, auswählen, auf Benutzerdefinierte Einstellungen klicken, Ihre Änderungen durchführen und anschließend auf OK klicken.
