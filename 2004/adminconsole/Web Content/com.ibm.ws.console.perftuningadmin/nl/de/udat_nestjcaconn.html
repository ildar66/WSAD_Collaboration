Einstellungen für Connector-Module

Auf dieser Seite können Sie Connector-Module anzeigen.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Anwendungen > Enterprise-Anwendungen > Anwendung > Connector-Module > Connector-Modul.

Register "Konfiguration"

URI
Gibt den logischen Pfad zu der von WebSphere Application Server verwalteten Ressource an.
Datentyp String
Name
Gibt den Anzeigenamen für das Connector-Modul an.
Datentyp String
altDD
Gibt den alternativen Deployment-Deskriptor (DD) für das Connector-Modul an.

Der URI des alternativen DD für ein bestimmtes Modul.

Datentyp String
Anfangswertigkeit
Gibt die Anfangspriorität des Connector-Moduls im Vergleich zu anderen Modulen an.

Wenn Ihre Anwendung mehrere Module enthält, bestimmt die festgelegte Anfangswertigkeit die Startpriorität des Moduls während des Serverstarts im Vergleich zu anderen Modulen. Module mit einer niedrigen Startpriorität werden zuerst gestartet.

Datentyp String
