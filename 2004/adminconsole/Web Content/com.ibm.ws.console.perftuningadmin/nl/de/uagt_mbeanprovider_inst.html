Einstellungen des MBean-Providers für Erweiterungen

Verwenden Sie diese Seite, um die Konfiguration eines MBean-Providers für JMX-Erweiterungen anzuzeigen und zu ändern.

Sie können eine Bibliothek konfigurieren, die eine Implementierung einer JMX-Bean und die zugehörige XML-Deskriptordatei enthält und die zur Erweiterung der vorhandenen, von WebSphere verwalteten Ressourcen im Hauptadministrationssystem verwendet werden soll.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Server > Anwendungsserver > Servername > Administrationsservices > MBean-Provider für Erweiterungen > Name_der_Providerbibliothek.

Register "Konfiguration"

Klassenpfad
Der Klassenpfad in der Providerbibliothek, unter dem der MBean-Deskriptor zu finden ist. Der Klassenlader benötigt diese Informationen, um die XML-Deskriptordatei der Erweiterungs-MBean zu laden und syntaktisch zu analysieren.
Datentyp String
Beschreibung
Ein beschreibender Text für die Konfiguration des MBean-Providers für Erweiterungen. Geben Sie in diesem Feld einen Text ein, der Ihnen hilft, die Providerkonfiguration wiederzuerkennen und von anderen zu unterscheiden.
Datentyp String
Name
Der Name der MBean-Providerbibliothek für Erweiterungen.
Datentyp String
