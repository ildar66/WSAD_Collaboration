Konfigurationseinstellungen für Empfehlung

Verwenden Sie diese Seite, um die zu aktivierende oder zu inaktivierende Empfehlung auszuwählen.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Server > Anwendungsserver > Servername > Konfiguration des Runtime Performance Advisor > Empfehlungskonfiguration.

Register "Konfiguration"

Name der Empfehlung
Die Empfehlung, die Sie aktivieren und inaktivieren können.
Auf Komponente angewendete Empfehlung
Die WAS-Komponente, für die die Runtime-Performance-Empfehlung gilt.
Empfehlungsstatus
Gibt an, ob die Empfehlung gestoppt oder gestartet ist.

Es gibt nur zwei Werte: Gestartet und Gestoppt. Gestartet bedeutet, dass die Empfehlung ausgeführt wird, wenn die Empfehlung zutrifft. Gestoppt bedeutet, dass die Empfehlung nicht ausgeführt wird.

Register "Laufzeit"

Name der Empfehlung
Die Empfehlung, die Sie aktivieren und inaktivieren können.
Auf Komponente angewendete Empfehlung
Die WAS-Komponente, für die die Runtime-Performance-Empfehlung gilt.
Empfehlungsstatus
Gibt an, ob die Empfehlung gestoppt, gestartet oder nicht verfügbar ist.

Es gibt drei gültige Empfehlungsstatus: Gestartet, Gestoppt und Nicht verfügbar. Gestartet bedeutet, dass die Empfehlung angewendet wird. Gestoppt bedeutet, dass die Empfehlung nicht angewendet wird. Nicht verfügbar bedeutet, dass die Empfehlung nicht auf die aktuelle Konfiguration zutrifft, z. b. eine Empfehlung für die Größe persistent gespeicherter Sitzungen für eine Konfiguration, in der keine persistent gespeicherten Sitzungen verwendet werden.
