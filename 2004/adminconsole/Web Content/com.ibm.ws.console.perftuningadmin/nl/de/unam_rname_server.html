Einstellungen des Namensservers

Auf dieser Seite können Sie die Einstellungen des Namensserviceproviders konfigurieren.

Klicken Sie zum Aufrufen dieser Seite der Administrationskonsole auf Server > Anwendungsserver > Servername > Serverkomponenten > Namensserver.

Register "Konfiguration"

Name
Gibt den Anzeigenamen für den Server an.
Datentyp String
Ausgangsstatus
Gibt den Ausführungsstatus an. Die Optionen lauten Gestartet und Gestoppt.
Datentyp String
Standardwert Gestartet
