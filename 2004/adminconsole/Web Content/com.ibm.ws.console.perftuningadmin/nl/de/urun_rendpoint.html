Endpoints

Auf dieser Seite können Sie Übertragungsendpunkte, die von Laufzeitkomponenten in einem Prozess verwendet werden, anzeigen und ändern. Endpoints stellen eine Host- und eine Port-Angabe für einen Server bereit.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Servername > Endpoints.

Beachten Sie, dass diese Seite nur angezeigt wird, wenn Sie mit Endpoints für Anwendungsserver arbeiten.

Name des Endpoint
Gibt den Namen eines Endpoint an. Jeder Name muss auf dem Server eindeutig sein.
