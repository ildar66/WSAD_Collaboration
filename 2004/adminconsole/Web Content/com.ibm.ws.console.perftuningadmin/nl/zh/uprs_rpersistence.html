分布式环境设置

使用本页面指定在分布式环境中保存会话的类型.

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > Web 容器 > 会话管理 > 分布式环境设置。

“配置”选项卡

分布式会话
指定用于保存会话的分布式环境类型。

无 指定当服务器关闭时会话管理设备废弃的会话数据。
数据库 指定会话管理设备将会话信息存储在由数据源连接设置所指定的数据源中。单击数据库，以更改这些数据源设置。
内存到内存复制 指定会话管理设施将会话信息存储在内存中的数据源中。复制会话信息到其它会话管理设施用于故障恢复。单击内存到内存复制指定要用来更改这些内存到内存设置的复制器。（仅用于 WebSphere Application Server Network Deployment。）
