流程定义设置

使用此页面查看或更改流程定义的设置，它提供starting or initializing 流程的命令行信息。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > 流程定义。

“配置”选项卡

可执行文件名称
指定进程的可执行（文件）名称。
数据类型 String
可执行自变量
指定当进程开始时运行的可执行命令。

例如，可执行文件目标程序可能期待三个自变量：arg1 arg2 arg3。

数据类型 String
单位 Java 命令行自变量
工作目录
指定进程运行的文件系统目录。

此目录用于确定具有相对路径名的输入和输出文件的位置。

将钝化的企业 bean 放置在运行 bean 的应用程序服务器的当前工作目录中。确保该工作目录是 WebSphere Application Server 产品根目录下的已知目录。

数据类型 String
