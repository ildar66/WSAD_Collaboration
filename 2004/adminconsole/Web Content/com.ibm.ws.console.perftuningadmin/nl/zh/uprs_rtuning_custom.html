调整参数定制设置

使用此页面定制分布式会话的调整参数。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > Web 容器 > 会话管理 > 分布式环境设置 > 定制调整参数 > 定制设置。

“配置”选项卡

写频率
指定写会话到持久性存储的时间。

结束 servlet 服务 在 servlet 执行完成后，会话写入数据库或另一个 WebSphere Application Server 实例。
手工更新 为了将会话数据写入数据库或另一个 WebSphere Application Server 实例，IBMSession 对象上有计划的同步是必需的。
基于时间的 会话数据根据指定的“写间隔”值写入数据库或另一个 WebSphere ApplicationServer 实例。缺省：10 秒
写内容
指定仅更新的属性写入外部位置还是所有会话属性（无论它们是否更改）都写入外部位置。外部位置可以是数据库也可以是另一个应用程序服务器实例。
仅更新的属性 仅将更新的属性写到持久性存储。
所有会话属性 将所有属性写到持久性存储。
调度会话清除
指定何时从数据库或另一个应用程序服务器实例清除失效会话。
指定分布式会话清除调度 允许调度失效进程从外部位置清除失效 HTTP 会话。启用此选项减少为了保持 HTTP 会话存活而必需对数据库或另一个应用程序服务器实例进行更新的数目。当禁用此选项时，失效者进程每几分钟运行一次以除去失效的 HTTP 会话。

当启用此选项时，为进程指定一天中的两个小时来清除外部位置中的失效会话。指定应用程序服务器中活动最少的时间。外部位置可以是数据库也可以是另一个应用程序服务器实例。
一天的第一时间（0 － 23） 表明从外部位置清除失效会话期间的第一个小时，以格林威治标准时间（GMT）表示。指定此值为 0 到 23 之间的正整数。此值仅当启用调度失效时才有效。
一天的第二时间（0 － 23） 表明从外部位置清除失效会话期间的第二个小时，以格林威治标准时间（GMT）表示。指定此值为 0 到 23 之间的正整数。此值仅当启用调度失效时才有效。
