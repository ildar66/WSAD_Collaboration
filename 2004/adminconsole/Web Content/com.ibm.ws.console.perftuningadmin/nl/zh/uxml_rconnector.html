Java 管理扩展连接器

使用此页面查看和更改 Java 管理扩展（JMX）连接器的配置。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > 管理服务 > JMX 连接器

当您调用脚本编制进程时，Java 管理扩展（JMX）连接器将与 WebSphere Application Server 进行通信。连接器的类型和参数没有缺省值。wsadmin.properties 文件指定简单对象访问协议（SOAP）连接器和一个相应的端口号。您还可使用远程方法调用（RMI）连接器。

使用下列方法之一来选择连接器类型和属性：

  • 指定属性文件中的属性。
  • 表明命令行的选项。

“配置”选项卡

类型
指定 JMX 连接器的类型。
数据类型 枚举
缺省 SOAPConnector
范围
SOAPConnector
对于使用简单对象访问协议（SOAP）的 JMX 连接。
RMIConnector
对于使用远程方法调用（RMI）的 JMX 连接。
HTTPConnector
对于使用 HTTP 的 JMX 连接。
JMSConnector
对于使用 Java 消息服务服务（JMS）的 JMX 连接。
