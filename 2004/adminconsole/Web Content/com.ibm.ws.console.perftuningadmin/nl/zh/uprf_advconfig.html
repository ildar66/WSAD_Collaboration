建议配置设置

使用此页面选择您希望启用或禁用的建议。

要查看此管理页面，单击服务器 > 应用程序服务器 > server_name > 运行时性能顾问程序配置 > 建议配置。

“配置”选项卡

建议名称
指定您可以启用或禁用的建议。
应用到组件的建议
指定运行时性能建议应用到的 WebSphere Application Server 组件。
建议状态
指定建议已停止还是已启动。

仅存在两个值 － 已启动和已停止。已启动意味着如果应用建议，则运行建议。已停止意味着不运行建议。

“运行时”选项卡

建议名称
指定您可以启用或禁用的建议。
应用到组件的建议
指定运行时性能建议应用到的 WebSphere Application Server 组件。
建议状态
指定建议已停止、已启动还是不可用。

建议状态具有三个值之一 － 已启动、已停止或不可用。已启动意味着正在应用建议。已停止意味着不应用建议。不可用意味着建议不应用于当前配置（如配置中“持久的会话大小”建议，无需持久会话）。
