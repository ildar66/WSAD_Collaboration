外部高速缓存组成员集合

使用此页面定义是高速缓存组成员的特定高速缓存。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server > 动态高速缓存服务 > 外部高速缓存组 > external_cache_group > 外部高速缓存组成员。

AdapterBeanName
指定适配器 bean 名称。

WebSphere Application Server 中支持的示例适配器 bean 名称是：
AFPA
AdapterBeanName：com.ibm.ws.cache.servlet.Afpa
地址：afpa 侦听的端口
ESI
AdapterBeanName：com.ibm.websphere.servlet.cache.ESIInvalidatorServlet
地址：本地主机
WTE
AdapterBeanName：com.ibm.websphere.edge.dynacache.WteAdapter
地址：hostname:port（WTE 正在侦听的主机名和端口）
