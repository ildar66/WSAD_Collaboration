密钥定位器集合

使用此页面查看可用密钥定位器列表。密钥定位器标识数字签名和加密所需的密钥。密钥定位器必需实现 com.ibm.wsspi.wssecurity.config.KeyLocator 接口。两个缺省实现为：com.ibm.wsspi.wssecurity.config.WSldKeyStoreMapKeyLocator 和 com.ibm.wsspi.wssecurity.config.KeyStoreKeyLocator。

要查看此管理控制台页面，完成以下步骤：

  1. 单击服务器 > 应用程序服务器 > server_name。
  2. 在“其它属性”下，单击 Web 服务：Web 服务安全性的缺省绑定 > 密钥定位器。
  3. 单击新建创建一个密钥定位器。单击删除删除一个密钥定位器。

使用此面板，完成以下步骤：

  1. 在面板上指定密钥定位器名称和密钥定位器类名
  2. 通过单击位于管理控制台顶部的保存来保存您的更改。当您单击保存时，返回到管理控制台主面板。
  3. 保存更改后，返回到密钥定位器集合面板用缺省绑定信息更新 Web 服务安全性运行时，该信息位于 ws_security.xml文件中。
  4. 要更新 Web 服务安全性运行时，单击更新运行时。 当您单击更新运行时时，对其它 Web 服务进行的配置更改也在 Web 服务安全性运行时中更新。
  5. 一旦您定义密钥定位器后，在其它属性下单击密钥定位器名，以指定其它属性和密钥。

密钥定位器名
指定密钥定位器的唯一名称。
密钥定位器类名
在密钥存储文件中指定密钥定位器的类名。
