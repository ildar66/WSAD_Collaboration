集合证书存储集合

使用此页面查看包含不可信的中间证书文件等待验证的证书存储列表。验证可包含检查证书是否在证书撤销列表（CRL）中、检查证书还未到期，并检查证书由可信签署者发出。

要查看此管理控制台页面，完成以下步骤：

  1. 单击服务器 > 应用程序服务器 server_name。
  2. 在相关项下，单击 Web 服务：Web 服务安全性的缺省绑定 > 集合证书存储。
  3. 单击新建以指定用于新的集合证书存储的存储名称和提供程序。
  4. 单击删除以删除集合证书存储。

使用此面板，完成以下步骤：

  1. 指定证书存储名称和证书存储供应商。
  2. 通过单击位于管理控制台顶部的保存来保存您的更改。当您单击保存时，您返回到管理控制台主面板。
  3. 返回到“集合证书存储集合”面板并单击更新运行时以更新带有缺省绑定信息的 Web 服务安全性运行时，该信息位于 ws_security.xml 文件中。

注: 当您单击更新运行时时，对其它 Web 服务进行的配置更改也在 Web 服务安全性运行时中更新。
