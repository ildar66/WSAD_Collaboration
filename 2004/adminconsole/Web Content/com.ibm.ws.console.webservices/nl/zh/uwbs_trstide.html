可信标识求值程序集合

使用此页面查看可信身份（标识）求值程序列表。 可信标识求值程序确定是否信任身份声明权限。一旦标识可信后，WebSphere Application Server 发出正确的凭证，其用于调用资源的下行调用。 可信标识求值程序实现 com.ibm.wsspi.wssecurity.id.TrustedIDEvaluator 接口。

要查看此管理控制台页面，完成以下步骤：

  1. 单击服务器 > 应用程序服务器 > server_name。
  2. 在“其它服务”下，单击 Web 服务：Web 服务安全性的缺省绑定 > 可信标识求值程序。

单击新建创建一个可信标识求值程序。

单击删除删除一个可信标识求值程序。

使用此面板，完成以下步骤：

  1. 指定可信标识求值程序名和可信标识求值程序类名。
  2. 通过单击位于管理控制台顶部的保存来保存您的更改。当您单击保存时，返回到管理控制台主面板。
  3. 返回到“可信标识求值程序”集合面板用缺省绑定信息更新 Web 服务安全性运行时，该信息位于 ws_security.xml 文件中。
  4. 单击更新运行时。对其它 Web 服务进行的配置更改也在 Web 服务安全性运行时中更新。

可信标识求值程序名
指定可信标识求值程序的唯一名称。
可信标识求值程序类名
指定可信标识求值程序的类名。
