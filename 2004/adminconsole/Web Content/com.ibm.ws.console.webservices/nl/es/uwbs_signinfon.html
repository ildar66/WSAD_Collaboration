Valores de configuración de la información de firmas

Utilice esta página para configurar los parámetros de firma nuevos.

Las especificaciones que figuran en esta página para el método de signatura y para los métodos de conversión y canonicalización se encuentran en el documento "XMLSignature Encryption Syntax and Specification: W3C Recommendation 12 Feb 2002".

Para ver esta página de la consola administrativa, pulse Aplicaciones de empresa > nombre_aplicación. En Elementos Relacionados, pulse Módulos Web > nombre_archivo_URI Servicios Web: Enlaces de seguridad del servidor. En la columna Enlace del receptor de peticiones, pulse Editar > Información de firmas > Nueva.

Pulse Nuevo para crear un parámetro de firma.

Pulse Suprimir para suprimir un parámetro de firma.

Método de signatura
Especifica el URI del algoritmo del método de signatura. Este método contiene el valor real de la signatura digital que se ha codificado con base64.

Se da soporte a los siguientes algoritmos:

  • http://www.w3.org/2000/09/xmldsig#rsa-sha1
  • http://www.w3.org/2000/09/xmldsig#dsa-sha1

Método de conversión
Especifica el URI del algoritmo del método de conversión.

Se da soporte al algoritmo http://www.w3.org/2000/09/xmldsig#sha1.

Método de canonicalización
Especifica el URI del algoritmo del método de canonicalización.

Se da soporte a los siguientes algoritmos:

  • http://www.w3.org/2001/10/xml-exc-c14n#
  • http://www.w3.org/2001/10/xml-exc-c14n#WithComments
  • http://www.w3.org/TR/2001/REC-xml-c14n-20010315
  • http://www.w3.org/TR/2001/REC-xml-c14n-20010315#WithComments

Clave de firma
Especifica la información de la clave que se utiliza para la firma. Estos campos se ignoran en la configuración del receptor.

Si especifica un Nombre de clave y una Referencia del localizador de claves, seleccione Ninguno como vía de acceso de certificados.

Vía de acceso de certificados
Especifica los valores para validar la vía de acceso de certificados. Cuando selecciona Confiar en cualquiera, se ignora esta validación y se confía en todos los certificados de entrada. Estos campos se ignoran en la configuración del remitente.

Si pulsa Confiar en cualquiera o selecciona un Ancla de confianza y un Almacén de certificados, seleccione Ninguna como clave de firma en el campo anterior.

Ancla de confianza

Las selecciones disponibles para el ancla de confianza se especifican pulsando Servidores > Servidores de aplicaciones > nombre_servidor. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security > Anclas de confianza.

Almacén de certificados

Las selecciones disponibles para el almacén de certificados de colecciones se especifican pulsando Servidores > Servidores de aplicaciones > nombre_servidor. En Elementos relacionados, pulse Servicios Web: Enlaces por omisión para Web Services Security > Almacén de certificados de colecciones.
