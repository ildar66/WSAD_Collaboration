Ámbito de implementación de los servicios Web

Utilice esta página para ver y gestionar el ámbito de los puertos de un bean del servicio Web.

Para ver esta página de la consola administrativa, pulse Aplicaciones > Aplicaciones de empresa > instancia_aplicación > Módulos Web > instancia_módulo >Ámbito de implementación de los servicios Web.

Puerto
Especifica un nombre de puerto para un servicio Web. Un módulo puede contener uno o más servicios Web, cada uno de los cuales puede contener uno o varios puertos.
