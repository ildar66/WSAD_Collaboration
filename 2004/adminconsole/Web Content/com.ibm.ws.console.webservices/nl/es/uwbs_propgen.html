Colección de propiedades de Web Services Security

Utilice esta página para ver una lista de propiedades adicionales para la configuración.

Hhay varios modos de ver un panel de valores de configuración de propiedades de Web Services Security. Realice los pasos siguientes para ver una de estas páginas de la consola administrativa:

  1. Pulse Servidores > Servidores de aplicaciones > nombre_servidor.
  2. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security > Localizadores de claves nombre_localizador_claves.
  3. En Propiedades adicionales, pulse Propiedades.
  4. Pulse Nuevo para crear una nueva propiedad.
  5. Pulse Suprimir para suprimir una propiedad que ha especificado anteriormente.

Nombre
Especifica el nombre de la propiedad.
Valor
Especifica el valor de la propiedad.
