Valores de configuración de claves

Utilice esta página para definir una correlación de un nombre lógico con un alias de clave en un archivo de almacén de claves.

Para ver esta página de la consola administrativa, realice los pasos siguientes:

  1. Pulse Servidores > Servidores de aplicaciones > nombre_servidor.
  2. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security > Localizadores de claves nombre_localizador_claves.
  3. En Propiedades adicionales, pulse Claves > Nueva.

Nombre de clave
Especifica el nombre del objeto de clave. Este nombre lo utiliza el localizador de claves para buscar la clave en el archivo de almacén de claves.
Alias de clave
Especifica el alias del objeto de clave del archivo de almacén de claves.
Contraseña de clave
Especifica la contraseña necesaria para acceder al objeto de clave en el archivo de almacén de claves.
