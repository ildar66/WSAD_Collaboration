Servicios Web: colección de enlaces de seguridad del cliente

Utilice esta página para ver una lista de configuraciones de enlaces del extremo del cliente para Web Services Security. Estos enlaces se utilizan cuando un servicio Web es cliente de otro servicio Web.

Para ver esta página de la consola administrativa, realice los pasos siguientes:

  1. Pulse Aplicaciones de empresa > nombre_aplicación.
  2. En Elementos relacionados, pulse Módulo Web > nombre_archivo_URI> Servicios Web: Enlaces de seguridad del cliente.

Puerto
Especifica el puerto que se utiliza para enviar y recibir mensajes desde un servidor.
Servicio Web
Especifica el nombre del servicio Web.
Enlaces del remitente de peticiones
Especifica la configuración de enlaces que se utiliza para enviar mensajes de petición al receptor de peticiones.

Pulse Editar para configurar la información de firma, la información de cifrado y los enlaces de inicio de sesión para el remitente de peticiones y para ver una lista de los localizadores de claves del almacén de claves.

La información de enlaces del remitente de peticiones especificada para el cliente debe coincidir con la información de enlaces del receptor de peticiones especificada para el servidor.

Enlace del receptor de respuestas
Especifica la configuración de enlaces que se utiliza para recibir mensajes de respuesta del remitente de respuestas.

Pulse Editar para configurar la información de firmas y de cifrado y para ver una lista de anclas de confianza, certificados intermedios ubicados en el almacén de certificados de colecciones y objetos de localizador de claves para el receptor de respuestas.

La información de enlaces del receptor de respuestas especificada para el cliente debe coincidir con la información de enlaces del remitente de respuestas especificada para el servidor.

Autenticación básica HTTP
Especifique el ID de usuario y la contraseña que se deben utilizar para este puerto con la autenticación básica a nivel de trasporte HTTP. Puede habilitar la seguridad de autenticación a nivel de transporte independientemente de la seguridad a nivel de mensaje.

Pulse Edita para configurar el ID de autenticación básica y la contraseña para la autenticación a nivel de transporte.

Configuración HTTP SSL
Habilita y configura la seguridad SSL a nivel de transporte para este puerto. Puede habilitar la seguridad SSL a nivel de transporte independientemente de la seguridad a nivel de mensaje.

Pulse Editar para especificar los valores para la configuración HTTP SSL a nivel de transporte para este puerto.
