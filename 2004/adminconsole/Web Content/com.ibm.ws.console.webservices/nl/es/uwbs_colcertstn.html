Valores de configuración del almacén de certificados de colecciones

Utilice esta página para especificar el nombre y el proveedor de un almacén de certificados.

Para ver esta página de la consola administrativa, realice los pasos siguientes:

  1. Pulse Servidores > Servidores de aplicaciones > nombre_servidor.
  2. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security > Almacén de certificados de colecciones > Nuevo.

Nombre del almacén de certificados
Especifica el nombre del almacén de certificados. El enlace de aplicación utiliza el nombre del almacén de certificados para hacer referencia al enlace definido previamente.
Proveedor del almacén de certificados
Especifica el proveedor de la implementación del almacén de certificados.
Tipo de datos String
Valor por omisión IBM CertPath
