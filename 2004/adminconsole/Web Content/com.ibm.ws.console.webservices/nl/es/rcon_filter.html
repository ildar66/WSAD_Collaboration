Valores de filtro de la consola administrativa

Utilice estos valores de filtro para especificar la forma de filtrar las entradas que se muestran en una vista de recopilación.

Seleccione la columna a filtrar, y luego entre un criterio de filtro.

Columna a filtrar
Utilice la lista desplegable para seleccionar la columna a filtrar. Cuando se aplica el filtro, la tabla de recopilación se modifica en consecuencia.

Por ejemplo, seleccione Servidores de aplicaciones si quiere entrar un criterio mediante el que filtrar los nombres de servidores de aplicaciones.

Criterio de filtro
En el campo junto a la lista desplegable, entre una serie a buscar en el nombre de una entrada de recopilación para aceptar la entrada y que aparezca en la tabla de recopilación. La serie puede incluir los caracteres comodines o símbolos siguientes: %, * o ?. Por ejemplo, escriba *Aplic* para buscar cualquier aplicación cuyo nombre contenta la serie Aplic.

A cada uno de los caracteres ( ) ^ * % { } \ + $ añádale el prefijo \ para que el motor de expresiones regulares que efectúa la búsqueda pueda encontrar coincidencias exactas con el criterio de búsqueda suministrado. Por ejemplo, para buscar todos los proveedores de JDBC que contienen (XA) en el nombre del proveedor, especifique lo siguiente:

*\(XA\)
