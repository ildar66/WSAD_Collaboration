Grupo de certificados X.509

Utilice esta página para ver una lista de certificados X.509.

Para ver esta página de la consola administrativa, realice los pasos siguientes:

  1. Pulse Servidores > Servidores de aplicaciones > nombre_servidor.
  2. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security > Almacén de certificados de colecciones.
  3. En la página Almacén de certificados de colecciones, pulse Certificado X.509 en Propiedades adicionales.

Pulse Nuevo para crear una nueva vía de acceso a un certificado X.509.

Pulse Suprimir para suprimir una vía de acceso a un certificado X.509.
