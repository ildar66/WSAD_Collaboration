Enlaces del cliente de servicios Web

Utilice esta página para especificar el archivo WSDL (Web Service Description Language) y las correlaciones de tipo de puerto por omisión para los servicios Web de un módulo.

Para ver esta página, pulse Aplicaciones > Aplicaciones de empresa > instancia_aplicación > Módulos Web > instancia_módulo >Enlaces del cliente de servicios Web.

Para módulos Web, pulse Aplicaciones > Aplicaciones de empresa > instancia_aplicación > Módulos EJB > instancia_módulo>Enlaces del cliente de servicios Web
