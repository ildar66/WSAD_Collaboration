Servicios Web: colección de enlaces de seguridad del servidor

Utilice esta página para ver una lista de configuraciones de enlaces del extremo del servidor para Web Services Security.

Para ver esta página de la consola administrativa, realice los pasos siguientes:

  1. Pulse Aplicaciones de empresa > nombre_aplicación.
  2. En Elementos relacionados, pulse Módulo Web > nombre_archivo_URIServicios Web: Enlaces de seguridad del servidor.

Puerto
Especifica el puerto en el que se reciben los mensajes del remitente de peticiones.
Servicio Web
Especifica el nombre del servicio Web.
Enlace del receptor de peticiones
Especifica la configuración de enlaces que se utiliza para recibir mensajes de petición del receptor de peticiones.

Pulse Editar para configurar la información de firmas y la información de cifrado y para ver una lista de anclas de confianza, certificados intermedios que se encuentran en el almacén de certificados de colecciones junto con localizadores de claves, evaluadores de ID de confianza y correlaciones de inicio de sesión.

La información de enlaces (del receptor de peticiones) especificada para el servidor debe coincidir con la información de enlaces (del remitente de peticiones) especificada para el cliente.

Enlace del remitente de respuestas
Especifica la configuración de enlaces que se utiliza para enviar mensajes de petición al receptor de respuestas.

Pulse Editar para configurar la información de firmas y la información de cifrado y para ver una lista de objetos de localizador de claves para el remitente de respuestas.

La información de enlaces (del remitente de respuestas) especificada para el servidor debe coincidir con la información de enlaces (del receptor de respuestas) especificada para el cliente.
