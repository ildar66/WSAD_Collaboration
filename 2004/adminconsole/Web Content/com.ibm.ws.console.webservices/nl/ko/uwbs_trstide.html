신뢰 ID 평가자 콜렉션

이 페이지를 참조하여 신뢰 ID 평가자의 목록을 보십시오. 신뢰 ID 평가자는 ID 확정 권한을 신뢰할지 여부를 판별합니다. ID가 신뢰되면, WebSphere Application Server는 자원을 호출하기 위한 다운스트림 호출에 사용되는 적절한 신임을 발행합니다. 신뢰 ID 평가자는 com.ibm.wsspi.wssecurity.id.TrustedIDEvaluator 인터페이스를 구현합니다.

이 관리 콘솔 페이지를 보려면, 다음 단계를 완료하십시오.

  1. 서버 > Application Servers > server_name을 클릭하십시오.
  2. 추가 서비스 아래에서 웹 서비스: 웹 서비스 보안을 위한 기본 바인딩 > 신뢰 ID 평가자를 클릭하십시오.

새로 작성을 클릭하여 신뢰 ID 평가자를 작성하십시오.

삭제를 클릭하여 신뢰 ID 평가자를 삭제하십시오.

이 패널을 사용하여 다음 단계를 완료하십시오.

  1. 신뢰 ID 평가자 이름 및 신뢰 ID 평가자 클래스 이름을 지정하십시오.
  2. 관리 콘솔의 맨 위에서 저장을 클릭하여 변경사항을 저장하십시오. 저장을 클릭하면, 관리 콘솔 홈 패널로 돌아갑니다.
  3. 신뢰 ID 평가자 콜렉션 패널로 돌아가 ws_security.xml 파일에 있는 기본 바인딩 정보로 웹 서비스 보안 런타임을 갱신하십시오.
  4. 런타임 갱신을 클릭하십시오. 다른 웹 서비스의 구성 변경사항도 웹 서비스 보안 런타임에서 갱신됩니다.
