웹 서비스: 서버 보안 바인딩 콜렉션

이 페이지를 참조하여 웹 서비스 보안을 위한 서버측 바인딩 구성의 목록을 보십시오.

이 관리 콘솔 페이지를 보려면, 다음 단계를 완료하십시오.

  1. 엔터프라이즈 응용프로그램 > application_name을 클릭하십시오.
  2. 관련 항목 아래에서 웹 모듈 > URI_file_name> 웹 서비스: 서버 보안 바인딩을 클릭하십시오.

포트
요청 송신자로부터 메시지가 수신되는 포트를 지정합니다
웹 서비스
웹 서비스의 이름을 지정합니다.
요청 수신자 바인딩
요청 송신자로부터 요청 메시지를 수신하는 데 사용되는 바인딩 구성을 지정합니다.

편집을 클릭하여 서명 정보 및 암호화 정보를 구성하고 신뢰 앵커의 목록, 콜렉션 인증 저장의 중간 인증, 키 위치 지정자, 신뢰 ID 평가자 및 로그인 맵핑을 보십시오.

서버에 지정된 바인딩 정보(요청 수신자의)는 클라이언트에 지정된 바인딩 정보(요청 전송자의)와 일치해야 합니다.

응답 송신자 바인딩
응답 수신자에 요청 메시지를 전송하는 데 사용되는 바인딩 구성을 지정합니다.

편집을 클릭하여 서명 및 암호화 정보를 구성하고, 응답 송신자에 대한 키 위치 지정자 오브젝트를 보십시오.

서버에 지정된 바인딩 정보(응답 전송자의)는 클라이언트에 지정된 바인딩 정보(응답 수신자의)와 일치해야 합니다.
