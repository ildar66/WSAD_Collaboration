Services Web : Collection de liaisons de sécurité du client

Cette page permet de visualiser une liste de configurations de liaisons côté client pour la sécurité des services Web. Ces liaisons sont utilisées lorsqu'un service Web est un client d'un autre service Web.

Pour afficher cette page de la console d'administration, suivez la procédure ci-après.

  1. Cliquez sur Applications d'entreprise > nom_application.
  2. Dans la section Articles liés, cliquez sur Module Web > nom_fichier_URI > Services Web : Liaisons de sécurité du client.

Port
Indique le port utilisé pour envoyer et recevoir des messages à partir d'un serveur.
Service Web
Indique le nom du service Web.
Liaison d'émetteur de demande
Indique la configuration de liaison utilisée pour envoyer les messages de demande au destinataire de la demande.

Cliquez sur Editer pour configurer les informations de signature et de chiffrement, ainsi que les liaisons de connexion pour l'émetteur de la demande, et visualiser une liste des localisateurs de clé dans le magasin de clés.

Les informations de liaison de l'émetteur de la demande indiquées pour le client doivent correspondre aux informations de liaison du récepteur de la demande indiquées pour le serveur.

Liaison de récepteur de réponse
Indique la configuration de liaison utilisée pour recevoir les messages de réponse envoyés par l'émetteur de la réponse.

Cliquez sur Editer pour configurer les informations de chiffrement et de signature et pour afficher une liste de points d'ancrage dignes de confiance, des certificats intermédiaires se trouvant dans le magasin de certificats de collection et les objets d'adresse de clé pour le récepteur de la réponse.

Les informations de liaison du récepteur de la demande indiquées pour le client doivent correspondre aux informations de liaison de l'émetteur de la demande indiquées pour le serveur.

Authentification HTTP de base
Indique l'ID utilisateur et le mot de passe pour ce port avec une authentification HTTP de base au niveau du transport. Vous pouvez activer la sécurité d'authentification au niveau du transport indépendamment de celle au niveau des messages.

Cliquez sur Editer pour configurer l'ID et le mot de passe d'authentification de base pour l'authentification au niveau du transport.

Configuration SSL HTTP
Active et désactive la sécurité SSL au niveau du transport pour ce port. Vous pouvez activer la sécurité SSL au niveau du transport indépendamment de la sécurité au niveau des messages.

Cliquez sur Editer pour spécifier les paramètres pour la configuration SSL HTTP au niveau du transport de ce port.
