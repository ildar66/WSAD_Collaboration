コレクション証明書ストア構成の設定

このページを使用して、証明書ストアの名前とプロバイダーを指定します。

この管理コンソール・ページを表示するには、以下のステップを実行します。

  1. 「サーバー」>「アプリケーション・サーバー」>「server_name 」をクリックします。
  2. 「追加プロパティー」の下の「Web Services: Default bindings for Web Services Security」>「Collection Certificate Store」>「新規」をクリックします。

Certificate Store Name
証明書ストアの名前を指定します。アプリケーション・バインディングは、この証明書ストアの名前を使用して、事前に定義されたバインディングを参照します。
Certificate Store Provider
証明書ストアのインプリメンテーションのプロバイダーを指定します。
データ型 ストリング
デフォルト IBM CertPath
