Web サービス: サーバー・セキュリティーのバインディング・コレクション

このページを使用して、Web サービス・セキュリティーの、 サーバー・サイドのバインディング構成のリストを表示します。

この管理コンソール・ページを表示するには、以下のステップを実行します。

  1. 「エンタープライズ・アプリケーション」>「application_name 」をクリックします。
  2. 「Related Items」の下の「Web モジュール」>「URI_file_name」>「Web Services: Server Security Bindings」をクリックします。

ポート
要求の送信側からのメッセージを受信するポートを指定します。
Web サービス
Web サービスの名前を指定します。
Request Receiver Binding
要求の送信側から要求メッセージを受信するために使用するバインディング構成を指定します。

「編集」をクリックして、署名情報と暗号化情報を構成し、トラスト・アンカー、コレクション証明書ストア内の中間証明書、鍵ロケーター、トラステッド ID エバリュエーター、およびログイン・マッピングのリストを表示します。

サーバーに対して指定されている、要求の受信側のバインディング情報は、 クライアントに対して指定されている、要求の送信側のバインディング情報と一致している必要があります。

Response Sender Binding
応答の受信側に応答メッセージを送信するために使用するバインディング構成を指定します。

「編集」をクリックして、署名情報と暗号化情報を構成し、応答の送信側の鍵ロケーター・オブジェクトのリストを表示します。

サーバーに対して指定されている、応答の送信側のバインディング情報は、 クライアントに対して指定されている、応答の受信側のバインディング情報と一致している必要があります。
