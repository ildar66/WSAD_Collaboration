鍵構成の設定

このページを使用して、論理名と鍵ストア・ファイル内の鍵のエイリアスの間のマッピングを定義します。

この管理コンソール・ページを表示するには、以下のステップを実行します。

  1. 「サーバー」>「アプリケーション・サーバー」>「server_name 」をクリックします。
  2. 「追加プロパティー」の下の「Web Services: Default bindings for Web Services Security」>「Key Locators」>「key_locator_name」をクリックします。
  3. 「追加プロパティー」の下の「Keys」>「新規」をクリックします。

Key Name
鍵オブジェクトの名前を指定します。この名前は、鍵ロケーターが鍵ストア・ファイル内の鍵を探し出す際に使用されます。
Key Alias
鍵ストア・ファイルに含まれている鍵オブジェクトのエイリアスを指定します。
鍵パスワード
鍵ストア・ファイル内の鍵オブジェクトにアクセスするために必要なパスワードを指定します。
