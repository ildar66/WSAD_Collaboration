Web サービス・インプリメンテーションの有効範囲

このページを使用して、Web サービス Bean で使用するポートの有効範囲を 表示および管理します。

この管理コンソール・ページを表示するには、「アプリケーション」>「エンタープライズ・アプリケーション」>「application_instance 」>「Web モジュール」 >「module_instance 」>「Web Services Implementation Scope」をクリックします。

Port
Web サービスのポート名を指定します。モジュールには、1 つ以上の Web サービスを含むことができ、各 Web サービスには 1 つ以上のポートを含むことができます。
