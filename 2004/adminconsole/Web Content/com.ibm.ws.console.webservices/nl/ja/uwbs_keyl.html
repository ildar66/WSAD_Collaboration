鍵ロケーターのコレクション

このページを使用して、使用可能な鍵ロケーターのリストを表示します。鍵ロケーターは、ディジタル署名および暗号化に必要な鍵を識別します。鍵ロケーターは、com.ibm.wsspi.wssecurity.config.KeyLocator インターフェースをインプリメントしている必要があります。com.ibm.wsspi.wssecurity.config.WSldKeyStoreMapKeyLocator および com.ibm.wsspi.wssecurity.config.KeyStoreKeyLocator という 2 つのデフォルトのインプリメンテーションがあります。

この管理コンソール・ページを表示するには、以下のステップを実行します。

  1. 「サーバー」>「アプリケーション・サーバー」>「server_name 」をクリックします。
  2. 「追加プロパティー」の下の「Web Services: Default bindings for Web Services Security」>「Key Locators」をクリックします。
  3. 鍵ロケーターを作成する場合は、「新規」をクリックします。鍵ロケーターを削除する場合は、「削除」をクリックします。

このパネルを使用して、次のステップを実行します。

  1. パネルで鍵ロケーター名および鍵ロケーター・クラス名を指定します。
  2. 管理コンソールの上部にある「保管」をクリックして変更を保管してください。「保管」をクリックすると、管理コンソールのホーム・パネルに戻ります。
  3. 変更を保管したら、「Key Locator」パネルに戻り、ws-security.xml ファイルに含まれているデフォルトのバインディング情報で Web サービス・セキュリティー・ランタイムを更新します。
  4. Web サービス・セキュリティー・ランタイムを更新するには、「Update runtime」をクリックします。「Update runtime」をクリックすると、他の Web サービスで変更された構成も、Web サービス・セキュリティー・ランタイムで更新されます。
  5. 鍵ロケーターを定義したら、鍵ロケーターの名前をクリックして、「追加プロパティー」の下に追加プロパティーと鍵を指定します。

Key Locator Name
鍵ロケーターの固有の名前を指定します。
Key Locator Classname
鍵ストア・ファイル内の鍵ロケーターのクラス名を指定します。
