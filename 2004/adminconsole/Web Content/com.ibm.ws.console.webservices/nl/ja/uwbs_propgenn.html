Web サービス・セキュリティー・プロパティー構成の設定

このページを使用して、追加のプロパティーを構成します。

「Web services security property configuration settings」パネルを表示する方法は、 複数あります。以下のステップを完了して、これらの管理コンソールページのいずれか 1 つを表示します。

  1. 「サーバー」>「アプリケーション・サーバー」>「server_name 」をクリックします。
  2. 「追加プロパティー」の下の「Web Services: Default bindings for Web Services Security」>「Key Locators」>「key_locator_name」をクリックします。
  3. 「追加プロパティー」の下の「プロパティー」>「新規」をクリックします。

プロパティー名
プロパティーの名前を指定します。

データ型: ストリング

プロパティー値
プロパティーの値を指定します。

データ型: ストリング
