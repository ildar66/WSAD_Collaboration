デフォルト・ポート・タイプ・マッピング・プロパティーの設定

このページを使用して、Web サービスに対するデフォルトのポート・タイプ・マッピングを表示および管理します。

管理コンソールのこのページを表示するには、「アプリケーション」>「エンタープライズ・アプリケーション」>「application_instance 」>「Web モジュール」>「module_instance 」>「Web Services Client Bindings」>「編集」>「default_port_instance 」をクリックします。

EJB モジュールの場合は、「アプリケーション」>「エンタープライズ・アプリケーション」>「application_instance 」>「EJB モジュール」>「module_instance 」>「Web Services Client Bindings」>「編集」>「default_port_instance 」をクリックします。

特定のポート・タイプが要求された場合は、サービスのデフォルト・ポートを指定します。 ポート・タイプは、そのローカル名とネーム・スペースで記述されます。 ポート・タイプのみを指定して getPort メソッドを実行すると、デフォルトのポート・ローカル名と ネーム・スペースで指定されたポートを取得します。

「構成」タブ

Port Type Local Name
マップされるポート・タイプのローカル名を指定します。
Port Type Namespace
マップされるポート・タイプのネーム・スペースを指定します。
Default Port Local Name
マップ先のポートのローカル名を指定します。
Default Port Namespace
マップ先のポートのネーム・スペースを指定します。
