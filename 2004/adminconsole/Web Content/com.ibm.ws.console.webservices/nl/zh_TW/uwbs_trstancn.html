信任錨點配置設定

請利用這項資訊來配置信任錨點。信任錨點指向含有最高授信使用者或自行簽章憑證的金鑰儲存庫。 您可以利用這項資訊來指定信任錨點的名稱和存取金鑰儲存庫所需要的資訊。 應用程式連結會利用這個名稱來參照連結檔中預先定義的信任錨點定義（或預設值）。

如果要檢視這個管理主控台頁面，請完成下列步驟：

  1. 按一下伺服器 > 應用程式伺服器 > server_name。
  2. 在「其他內容」之下，按一下 Web 服務：Web 服務安全的預設連結 > 信任錨點 > 新建。

信任錨點名稱
指定應用程式連結用來參照預設連結中預先定義的信任錨點定義的唯一名稱。
金鑰儲存庫密碼
指定存取金鑰儲存庫檔所需要的密碼。
金鑰儲存庫路徑
指定金鑰儲存庫檔案的位置。

請利用 ${USER_INSTALL_ROOT}，因為這個路徑會展開至您機器中的 WebSphere Application Server 路徑。

金鑰儲存庫類型
指定金鑰儲存庫檔的類型。

這個欄位的值是 JKS 或 JCEKS。

JKS
如果您沒有使用 Java Cryptography Extensions (JCE)，請指定這個選項。
JCEKS
如果您在使用 Java Cryptography Extensions，請指定這個選項。 雖然 JCEKS 金鑰儲存庫格式比較安全，但它會降低效能。

資料類型 字串
預設值 JKS
範圍 JKS, JCEKS
