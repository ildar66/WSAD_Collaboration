Web 服務：伺服器安全連結集合

請利用這個頁面來檢視 Web 服務安全的伺服器端連結配置清單。

如果要檢視這個管理主控台頁面，請完成下列步驟：

  1. 按一下企業應用程式 >application_name。
  2. 在「相關項目」之下，按一下 Web 模組 > URI_file_name > Web 服務：伺服器安全連結。

埠
指定接收要求傳送端發出之訊息的埠。
Web 服務
指定 Web 服務的名稱。
要求接收端連結
指定用來接收要求傳送端發出的要求訊息之連結配置。

請按一下編輯來配置簽章資訊和加密資訊，以及檢視信任錨點清單、在集合憑證儲存庫中的中間憑證、金鑰定位器、授信 ID 求值運算元和登入對映。

指定給伺服器的（要求接收端）連結資訊必須符合指定給用戶端的（要求傳送端）連結資訊。

回應傳送端連結
指定用來將要求訊息傳送給回應接收端的連結配置。

請按一下編輯來配置簽章和加密資訊，以及檢視回應傳送端的金鑰定位器物件清單。

指定給伺服器的（回應傳送端）連結資訊必須符合指定給用戶端的（回應接收端）連結資訊。
