授信 ID 求值運算元配置設定

請利用這項資訊來配置授信身分 (ID) 求值運算元。

如果要檢視這個管理主控台頁面，請完成下列步驟：

  1. 按一下伺服器 > 應用程式伺服器 > server_name。
  2. 在「其他內容」之下，按一下 Web 服務：Web 服務安全的預設連結 > 授信 ID 求值運算元 > 新建。

您必須指定預設授信 ID 求值運算元的名稱和值內容來建立要評估的授信 ID 清單。

授信 ID 求值運算元名稱
指定應用程式連結用來參照預設連結所定義授信身分 (ID) 求值運算元。

您必須以 trustedId_n 形式來指定授信 ID 求值運算元，其中 _n 是從 0 至 n 的整數。

授信 ID 求值運算元類別名稱
指定授信 ID 求值運算元的類別名稱。
預設值 com.ibm.wsspi.wssecurity.id.TrustedIDEvaluatorImpl
