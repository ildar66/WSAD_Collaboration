金鑰定位器集合

請利用這個頁面來檢視可用的金鑰定位器清單。 金鑰定位器用來識別數位簽章和加密所需要的金鑰。 金鑰定位器必須實作 com.ibm.wsspi.wssecurity.config.KeyLocator 介面。 預設實作有兩個：com.ibm.wsspi.wssecurity.config.WSldKeyStoreMapKeyLocator 和 com.ibm.wsspi.wssecurity.config.KeyStoreKeyLocator。

如果要檢視這個管理主控台頁面，請完成下列步驟：

  1. 按一下伺服器 > 應用程式伺服器 > server_name。
  2. 在「其他內容」之下，按一下 Web 服務：Web 服務安全的預設連結 > 金鑰定位器。
  3. 按一下新建來建立金鑰定位器。請按一下刪除來刪除金鑰定位器。

請利用這個畫面來完成下列步驟：

  1. 在畫面中指定金鑰定位器名稱和金鑰定位器類別名稱
  2. 按一下管理主控台頂端的儲存來儲存您的變更。 當您按一下儲存時，您會返回管理主控台起始畫面。
  3. 在儲存變更之後，返回金鑰定位器集合畫面以利用在 ws-security.xml 檔中找到的預設連結資訊來更新 Web 服務安全執行時期。
  4. 如果要更新 Web 服務安全執行時期，按一下更新執行時期。當您按一下更新執行時期時，也會在 Web 服務安全執行時期中更新其他 Web 服務的配置變更。
  5. 定義金鑰定位器之後，請按一下金鑰定位器名稱，在其他內容下指定適當的內容和金鑰。

金鑰定位器名稱
指定金鑰定位器的唯一名稱。
金鑰定位器類別名稱
指定金鑰儲存庫檔中之金鑰定位器的類別名稱。
