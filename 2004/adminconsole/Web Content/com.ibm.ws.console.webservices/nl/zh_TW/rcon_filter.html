管理主控台過濾器設定

請利用過濾器設定來指定如何過濾集合視圖中所顯示的項目。

請選取要過濾的直欄，再輸入過濾準則。

要過濾的直欄
請利用下拉清單來選取要過濾的直欄。 當您套用過濾器時，會據此修改集合表。

比方說，如果您要輸入用來過濾應用程式伺服器名稱的準則，請選取應用程式伺服器。

過濾準則
請在下拉清單旁的欄位中，輸入必須在集合項目名稱中找到的字串，以限定集合表所要顯示的項目。 這個字串可包含 %, * 或 ? 符號來作為萬用字元。 比方說，輸入 *App* 會找到名稱中含有 App 字中的任何應用程式伺服器。

將 ( ) ^ * % { } \ + $ 每個字元偕同 \ 放在前頭，以便正規表示式引擎正確地執行搜尋來符合所提供的搜尋準則。比方說，如果要針對所有 JDBC 提供者搜尋含 (XA) 的提供者名稱，請指定如下：

*\(XA\)
