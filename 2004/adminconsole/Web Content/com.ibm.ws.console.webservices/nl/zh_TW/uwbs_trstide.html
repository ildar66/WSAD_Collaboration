授信 ID 求值運算元集合

請利用這個頁面來檢視授信 ID 求值運算元清單。 授信 ID 求值運算元決定了應不應該信任 ID 確認管理中心。 ID 授信之後，WebSphere Application Server 會發出適當的認證，供下游呼叫用來呼叫資源。 授信 ID 求值運算元會實作 com.ibm.wsspi.wssecurity.id.TrustedIDEvaluator 介面。

如果要檢視這個管理主控台頁面，請完成下列步驟：

  1. 按一下伺服器 > 應用程式伺服器 > server_name。
  2. 在「其他內容」之下，按一下 Web 服務：Web 服務安全的預設連結 > 授信 ID 求值運算元。

按一下新建來建立授信 ID 求值運算元。

按一下刪除來刪除授信 ID 求值運算元。

請利用這個畫面來完成下列步驟：

  1. 指定授信 ID 求值運算元名稱和授信 ID 求值運算元類別名稱。
  2. 按一下管理主控台頂端的儲存來儲存您的變更。 當您按一下儲存時，您會返回管理主控台起始畫面。
  3. 返回「授信 ID 求值運算元」集合畫面，利用在 ws-security.xml 檔中找到的預設連結資訊來更新 Web 服務安全執行時期。
  4. 按一下更新執行時期。在 Web 服務安全執行時期中，也會更新其他 Web 服務的配置變更。

授信 ID 求值運算元名稱
指定授信 ID 求值運算元的唯一名稱。
授信 ID 求值運算元類別名稱
指定授信 ID 求值運算元的類別名稱。
