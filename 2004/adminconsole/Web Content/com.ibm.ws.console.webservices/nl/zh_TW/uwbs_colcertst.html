集合憑證儲存庫集合

請利用這個頁面來檢視含有等待驗證的非授信中間憑證檔的憑證儲存庫清單。 驗證可能包括下列各項檢查：憑證在不在憑證廢止清冊 (CRL) 中、憑證有沒有到期，以及憑證的發出者是不是授信簽章者。

如果要檢視這個管理主控台頁面，請完成下列步驟：

  1. 按一下伺服器 > 應用程式伺服器 > server_name。
  2. 在相關項目之下，按一下 Web 服務：Web 服務安全的預設連結 > 集合憑證儲存庫。
  3. 請按一下新建來指定新集合憑證儲存庫的儲存庫名稱和提供者。
  4. 請按一下刪除來刪除集合憑證儲存庫。

請利用這個畫面來完成下列步驟：

  1. 指定憑證儲存庫名稱和憑證儲存庫提供者。
  2. 按一下管理主控台頂端的儲存來儲存您的變更。 當您按一下儲存時，您會返回管理主控台起始畫面。
  3. 返回集合憑證儲存庫集合畫面，並按一下更新執行時期以利用在 ws-security.xml 檔中找到的預設連結資訊來更新 Web 服務安全執行時期。

附註: 當您按一下更新執行時期時，也會在 Web 服務安全執行時期中更新其他 Web 服務的配置變更。
