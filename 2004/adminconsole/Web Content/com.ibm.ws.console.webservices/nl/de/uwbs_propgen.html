Merkmale der Web-Services-Sicherheit

Verwenden Sie diese Seite, um eine Liste weiterer Merkmale für die Konfiguration anzuzeigen.

Es gibt verschiedene Möglichkeiten für die Anzeige eines Fensters mit Merkmalen der Web-Services-Sicherheit. Gehen Sie wie folgt vor, um eine dieser Seiten der Administrationskonsole anzuzeigen:

  1. Klicken Sie auf Server > Anwendungsserver > Servername.
  2. Klicken Sie unter "Weitere Merkmale" auf Web Services: Standardbindungen für die Sicherheit von Web Services > Key Locator > Name_des_Key_Locator.
  3. Klicken Sie unter "Zusätzliche Merkmale" auf Merkmale.
  4. Klicken Sie auf Neu, um ein neues Merkmal zu erstellen.
  5. Klicken Sie auf Löschen, um ein zuvor definiertes Merkmal zu löschen.
