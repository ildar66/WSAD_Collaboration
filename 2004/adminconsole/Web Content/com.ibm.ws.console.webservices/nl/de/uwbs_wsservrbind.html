Web Services: Sicherheitsbindungen des Servers

Verwenden Sie diese Seite, um eine Liste der serverseitigen Bindungskonfigurationen für die Web-Services-Sicherheit anzuzeigen.

Gehen Sie zum Anzeigen dieser Seite der Administrationskonsole wie folgt vor:

  1. Klicken Sie auf Enterprise-Anwendungen >Name_der_Anwendung.
  2. Klicken Sie unter "Zugehörige Elemente" auf Webmodule > Name_der_URI-Datei > Web Services: Sicherheitsbindungen des Servers.

Port
Der Port, an dem Nachrichten vom Anforderungssender empfangen werden.
Webservice
Der Name des Webservice.
Bindung für Anforderungsempfänger
Die Bindungskonfiguration, die für den Empfang von Anforderungsnachrichten vom Anforderungssender verwendet wird.

Klicken Sie auf Bearbeiten, um die Signaturdaten und die Verschlüsselungsdaten zu konfigurieren und eine Liste der Trust Anchor, der temporären Zertifikate im Collection Certificate Store, der Key Locator, der Truste ID Evaluator und der Anmeldezuordnungen anzuzeigen.

Die Bindungsdaten für den Antwortsender, die für den Server angegeben werden, müssen mit den Bindungsdaten für den Antwortempfänger, die für den Client angegeben werden, übereinstimmen.

Bindung für Antwortsender
Die Bindungskonfiguration, die zum Senden von Anforderungsnachrichten an den Antwortempfänger verwendet wird.

Klicken Sie auf Bearbeiten, um die Signatur- und Verschlüsselungsdaten zu konfigurieren und eine Liste der Key-Locator-Objekte für den Antwortsender anzuzeigen.

Die Bindungsdaten für den Antwortsender, die für den Server angegeben werden, müssen mit den Bindungsdaten für den Antwortempfänger, die für den Client angegeben werden, übereinstimmen.
