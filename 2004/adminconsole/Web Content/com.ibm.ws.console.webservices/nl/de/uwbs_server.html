Geltungsbereich der Web-Services-Implementierung

Verwenden Sie diese Seite, um den Geltungsbereich der Ports einer Web-Services-Bean anzuzeigen und zu verwalten.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Anwendungen >Enterprise-Anwendungen > Anwendungsinstanz > Webmodule > Modulinstanz > Geltungsbereich für die Implementierung von Web Services.

Port
Der Port-Name eines Webservice. Ein Modul kann einen oder mehrere Webservices enthalten, von denen jeder wiederum einen oder mehrere Ports enthalten kann.
