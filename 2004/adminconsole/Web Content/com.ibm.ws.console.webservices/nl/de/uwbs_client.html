Clientbindungen für Webservices

Verwenden Sie diese Seite, um den Namen der WSDL-Datei (Web Service Description Language) und die Standard-Port-Zuordnungen für die Webservices in einem Modul anzugeben.

Klicken Sie zum Anzeigen dieser Seite auf Anwendungen > Enterprise-Anwendungen > Anwendungsinstanz > Webmodule > Modulinstanz>Clientbindungen für Web Services.

Für EJB-Module klicken Sie auf Anwendungen >Enterprise-Anwendungen > Anwendungsinstanz > EJB-Module > Modulinstanz>Clientbindungen für Web Services.
