Konfigurationseinstellungen des Collection Certificate Store

Verwenden Sie diese Seite, um den Namen und den Provider eines Certificate Store anzugeben.

Gehen Sie zum Anzeigen dieser Seite der Administrationskonsole wie folgt vor:

  1. Klicken Sie auf Server > Anwendungsserver > Servername.
  2. Klicken Sie unter Weitere Merkmale auf Web Services: Standardbindungen für die Sicherheit von Web Services > Collection Certificate Store > Neu.

Name des Certificate Store
Der Name des Certificate Store. Die Anwendungsbindung verwendet den Namen des Certificate Store, um auf eine vordefinierte Bindung zu verweisen.
Certificate Store Provider
Der Provider der Certificate-Store-Implementierung.
Datentyp String
Standardwert IBM CertPath
