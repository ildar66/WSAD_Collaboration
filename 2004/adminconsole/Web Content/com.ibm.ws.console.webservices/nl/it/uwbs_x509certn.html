Impostazioni di configurazione del certificato X.509

Utilizzare questa pagina per specificare l'ubicazione dei certificati X.509.

Per visualizzare questa pagina della console di gestione, completare le seguenti operazioni:

  1. Fare clic su Server > Server delle applicazioni > nome_server.
  2. In Ulteriori proprietà, fare clic su Servizi Web: collegamenti predefiniti per la sicurezza dei servizi Web (WS-Security, Web Services Security) > Raccolta archivio certificati.
  3. Alla pagina Raccolta archivio certificati, in Ulteriori proprietà, fare clic su Certificati X.509 > Nuovo.
