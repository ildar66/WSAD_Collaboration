Impostazioni di configurazione delle chiavi

Utilizzare questa pagina per definire una mappatura di un nome logico su un alias chiave in un file di memorizzazione delle chiavi.

Per visualizzare questa pagina della console di gestione, completare le seguenti operazioni:

  1. Fare clic su Server > Server delle applicazioni > nome_server.
  2. In Ulteriori proprietà, fare clic su Servizi Web: collegamenti predefiniti per la sicurezza dei servizi Web (WS-Security, Web Services Security) > Indicatori di posizione della chiave > nome_indicacatoreposizione_chiave.
  3. In Ulteriori proprietà, fare clic su Chiavi > Nuovo.

Nome chiave
Specifica il nome dell'oggetto chiave. Questo nome viene utilizzato dall'indicatore di posizione della chiave per trovare la chiave all'interno del file di memorizzazione delle chiavi.
Alias chiave
Specifica l'alias per l'oggetto chiave contenuto nel file di memorizzazione delle chiavi.
Password chiave
Specifica la password necessaria per accedere all'oggetto chiave all'interno del file di memorizzazione delle chiavi.
