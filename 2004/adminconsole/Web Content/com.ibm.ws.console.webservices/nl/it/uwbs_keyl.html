Raccolta indicatori di posizione della chiave

Utilizzare questa pagina per visualizzare un elenco di indicatori di posizione della chiave disponibili. Gli indicatori di posizione della chiave identificano le chiavi necessarie per la codifica e la firma digitale. Un indicatore di posizione della chiave deve implementare l'interfaccia com.ibm.wsspi.wssecurity.config.KeyLocator. Le due implementazioni predefinite sono: com.ibm.wsspi.wssecurity.config.WSldKeyStoreMapKeyLocator e com.ibm.wsspi.wssecurity.config.KeyStoreKeyLocator.

Per visualizzare questa pagina della console di gestione, completare le seguenti operazioni:

  1. Fare clic su Server > Server delle applicazioni > nome_server.
  2. In Ulteriori proprietà, fare clic su Servizi Web: collegamenti predefiniti per la sicurezza dei servizi Web (WS-Security, Web Services Security) > Indicatore di posizione della chiave.
  3. Fare clic su Nuovo per creare un nuovo indicatore di posizione della chiave. Fare clic su Elimina per eliminare un indicatore di posizione della chiave.

Con questo pannello, completare le seguenti operazioni:

  1. Sul pannello, specificare un nome e una classe per gli indicatori di posizione della chiave
  2. Salvare le modifiche facendo clic su Salva nella parte superiore della console di gestione. Facendo clic su Salva, si torna al pannello principale della console di gestione.
  3. Dopo aver salvato le modifiche, tornare al pannello di raccolta Indicatore di posizione della chiave per aggiornare il runtime di sicurezza dei servizi Web (WS-Security, Web Services Security) con le informazioni di collegamento predefinite, contenute nel file ws_security.xml.
  4. Per aggiornare il runtime di sicurezza dei servizi Web (WS-Security, Web Services Security), fare clic su Aggiorna runtime. Quando si fa clic su Aggiorna runtime, anche le modifiche di configurazione apportate agli altri servizi Web vengono aggiornate nel runtime di sicurezza dei servizi Web (WS-Security, Web Services Security).
  5. Una volta definiti gli indicatori di posizione della chiave, fare clic sul nome dell'indicatore della chiave per specificare ulteriori proprietà e chiavi in Ulteriori proprietà.

Nome indicatore di posizione della chiave
Specifica il nome univoco dell'indicatore di posizione della chiave.
Nome classe indicatore di posizione della chiave
Specifica il nome della classe dell'indicatore di posizione della chiave nel file di memorizzazione delle chiavi.
