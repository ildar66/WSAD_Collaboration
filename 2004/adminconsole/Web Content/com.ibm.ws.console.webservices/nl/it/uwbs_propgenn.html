Impostazioni di configurazione delle proprietà per la sicurezza dei servizi Web (WS-Security, Web Services Security)

Utilizzare questa pagina per configurare ulteriori proprietà.

Sono disponibili vari modi per visualizzare il pannello Impostazioni di configurazione delle proprietà per la sicurezza dei servizi Web (WS-Security, Web Services Security). Completare le procedure riportate di seguito per visualizzare una di queste pagine della console di gestione:

  1. Fare clic su Server > Server delle applicazioni > nome_server.
  2. In Ulteriori proprietà, fare clic su Servizi Web: collegamenti predefiniti per la sicurezza dei servizi Web (WS-Security, Web Services Security) > Indicatori di posizione della chiave > nome_indicacatoreposizione_chiave.
  3. In Ulteriori proprietà, fare clic su Proprietà > Nuovo.

Nome proprietà
Specifica il nome della proprietà.

Valore proprietà
