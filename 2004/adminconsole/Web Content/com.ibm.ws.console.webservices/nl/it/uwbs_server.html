Ambito implementazione servizi Web

Utilizzare questa pagina per visualizzare e gestire l'ambito delle porte di un bean di servizio Web.

Per visualizzare questa pagina della console di gestione, fare clic su Applicazioni >Applicazioni enterprise > istanza_applicazione > Moduli Web > istanza_modulo >Ambito implementazione servizi Web.

Porta
Specifica un nome porta per un servizio Web. Un modulo può contenere uno o più servizi Web, ognuno dei quali può contenere una o più porte.
