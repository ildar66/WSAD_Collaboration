Raccolta delle autenticazioni di base HTTP

Utilizzare questa pagina per specificare un ID utente e una password per la sicurezza dell'autenticazione di base del livello di trasporto per questa porta. È possibile utilizzare questa configurazione quando un servizio Web funge da client per un altro servizio Web.

È possibile utilizzare la sicurezza del livello di trasporto per abilitare l'autenticazione di base. La sicurezza del livello di trasporto può essere abilitata o disabilitata indipendentemente dalla sicurezza del livello di messaggio. Poiché la sicurezza del livello di trasporto fornisce una sicurezza minima, quando la sicurezza è essenziale per l'applicazione dei servizi Web, utilizzare la sicurezza del livello di messaggio.

Per visualizzare questa pagina della console di gestione, completare le seguenti operazioni:

  1. Fare clic su Applicazioni > Applicazioni Enterprise > nome_applicazione.
  2. In Elementi correlati, fare clic su Modulo Web > nome_file_URI > Servizi Web: collegamenti sicurezza client.
  3. In Autenticazione di base HTTP, fare clic su Modifica.

Inoltre, è possibile specificare l'autenticazione di base SSL HTTP completando le seguenti procedure:

  1. Fare clic su Applicazioni > Applicazioni Enterprise > nome_applicazione.
  2. In Elementi correlati, fare clic su Modulo Web > nome_file_URI > Servizi Web: collegamenti sicurezza client.
  3. In Configurazione SSL HTTP, fare clic su Modifica.

ID autenticazione di base
Specifica l'ID utente per l'autenticazione di base HTTP per questa porta.
Password autenticazione di base
Specifica la password per l'autenticazione di base HTTP per questa porta.
