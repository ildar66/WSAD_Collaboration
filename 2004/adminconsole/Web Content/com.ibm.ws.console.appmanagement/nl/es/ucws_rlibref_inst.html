Valores de referencias de biblioteca

Utilice esta página para definir las referencias de biblioteca, que especifican cómo se utilizan las bibliotecas globales.

Para ver esta página de consola administrativa, pulse Aplicaciones > Aplicaciones > nombre_aplicación > Bibliotecas > nombre_referencia_biblioteca. Se debe definir una biblioteca compartida para ver esta página.

Pestaña Configuración

Nombre de biblioteca
Especifica el nombre de la biblioteca compartida que se debe utilizar para la referencia de biblioteca.
Tipo de datos String
