Valores de correlación de destino

Utilice esta página para correlacionar una aplicación o módulos desplegado con un servidor o clúster.

Para ver esta página de la consola administrativa, pulse Aplicaciones > Aplicaciones de empresa> nombre_aplicación > Correlaciones de destino > nombre_destino.

Pestaña Configuración

Destino
Indica el nombre del servidor de destino o clúster con el que se correlaciona la aplicación o módulo. Especifique el destino en la página de correlación de módulos con servidores de aplicaciones a la que se accede desde los valores de una aplicación.
Tipo de datos String
Habilitado
Indica si están iniciados (o habilitados) los módulos de aplicación instalados en el servidor de destino al iniciar el servidor. Establece el estado inicial de los módulos de aplicación. Un valor igual a true indica que los módulos correspondientes están habilitados y, por lo tanto, son accesibles al iniciar el servidor. Un valor igual a false indica que los módulos correspondientes no están habilitados y, por lo tanto, no son accesibles al iniciar el servidor.
Tipo de datos Boolean
Valor por omisión true
