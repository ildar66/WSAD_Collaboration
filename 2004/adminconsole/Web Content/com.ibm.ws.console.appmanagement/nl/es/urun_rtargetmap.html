Colección de correlaciones de destino

Utilice esta página para gestionar las correlaciones de aplicaciones o módulos desplegados con servidores o clústeres.

Para ver esta página de la consola administrativa, pulse Aplicaciones > Aplicaciones de empresa> nombre_aplicación > Correlaciones de destino.

Destino
Indica el nombre del servidor de destino o clúster con el que se correlaciona la aplicación o módulo. Especifique el destino en la página de correlación de módulos con servidores de aplicaciones a la que se accede desde los valores de una aplicación.
Nodo
Especifica el nombre de nodo si el destino es un servidor.
Estado
Indica si el estado de la aplicación en ejecución en el servidor de destino se ha iniciado, se ha detenido o no está disponible.
