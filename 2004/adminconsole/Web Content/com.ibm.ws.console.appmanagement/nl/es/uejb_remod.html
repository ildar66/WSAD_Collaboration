Valores de módulo EJB

Utilice esta página para configurar y gestionar un determinado módulo EJB desplegado.

Nota: No se puede iniciar o detener un único módulo EJB para modificarlo. Debe iniciar o detener la aplicación adecuada correctamente.

Para ver esta página de la consola administrativa, pulse Aplicaciones > Aplicaciones > nombreAplicación > Módulos EJB > nombreMódulo.

Pestaña Configuración

URI
Cuando se resuelve con respecto al URL de la aplicación, especifica la ubicación del contenido del archivador de módulos de un sistema de archivos. El URI debe coincidir con el URI de un URI ModuleRef en el descriptor de despliegue de la aplicación desplegada (EAR).
ADD alternativo
Especifica un descriptor de despliegue que se debe utilizar durante el tiempo de ejecución en lugar del que está instalado en el módulo.
Peso inicial
Especifica el orden en que se inician los módulos cuando se inicia el servidor. El módulo con el peso inicial menor se inicia primero.
Tipo de datos Integer
Valor por omisión 5000
Rango Mayor que 0
