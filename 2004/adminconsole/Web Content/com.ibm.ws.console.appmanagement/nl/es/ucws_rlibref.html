Colección de referencias de biblioteca

Utilice esta página para ver y gestionar las referencias de biblioteca que definen cómo se utilizan las bibliotecas globales. Por ejemplo, puede utilizar esta página para asociar archivos de biblioteca compartida con una aplicación desplegada.

Para ver esta página de la consola administrativa, pulse Aplicaciones de empresa > nombre_aplicación > Bibliotecas.

Nombre de biblioteca
Especifica un nombre para la referencia de biblioteca.
