Einstellungen für Bibliotheksreferenzen

Auf dieser Seite können Sie Bibliotheksreferenzen, die die Verwendung von globalen Bibliotheken angeben, definieren.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Anwendungen > Enterprise-Anwendungen > Anwendungsname > Bibliotheken > Name_der_Bibliotheksreferenz. Eine gemeinsam benutzte Bibliothek muss so definiert werden, dass diese Seite angezeigt wird.

Register "Konfiguration"

Bibliotheksname
Gibt den Namen der gemeinsam benutzten Bibliothek an, die für die Bibliotheksreferenz verwendet werden soll.
Datentyp String
