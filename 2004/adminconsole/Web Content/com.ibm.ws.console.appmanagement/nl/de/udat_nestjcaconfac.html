Connector-Module

Auf dieser Seite können Sie Einstellungen für Connector-Module anzeigen.

Für jedes Connector-Modul (RAR) in der Anwendung wird eine Instanz von ConnectorModuleDeployment erstellt.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Anwendungen > Enterprise-Anwendungen > Anwendung > Connector-Module.

URI
Gibt den logischen Pfad zu der vom Produkt verwalteten Ressource an.
Datentyp String
Name
Gibt den Anzeigenamen für das Connector-Modul an.
Datentyp String
