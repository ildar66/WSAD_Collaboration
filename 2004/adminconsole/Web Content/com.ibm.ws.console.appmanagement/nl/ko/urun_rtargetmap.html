대상 맵핑 콜렉션

이 페이지를 참조하여 서버나 클러스터에 전개된 응용프로그램 또는 모듈의 맵핑을 관리하십시오.

이 관리 콘솔 페이지를 보려면 응용프로그램 > 엔터프라이즈 응용프로그램 > application_name > 대상 맵핑을 클릭하십시오.

대상
응용프로그램 또는 모듈이 맵핑되는 대상 서버 또는 클러스터의 이름을 기술합니다. 맵 모듈의 대상을 응용프로그램의 설정에서 액세스한 Application Server 페이지에 지정합니다.
노드
대상이 서버인 경우 노드 이름을 지정합니다.
상태
대상 서버에서 실행 중인 응용프로그램의 상태가 시작됨, 중지됨 또는 사용 불가능인지 표시합니다.
