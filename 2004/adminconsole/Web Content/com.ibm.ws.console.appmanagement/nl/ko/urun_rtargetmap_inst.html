대상 맵핑 설정

이 페이지를 참조하여 전개된 응용프로그램 또는 모듈을 서버 또는 클러스터에 맵핑하십시오.

이 관리 콘솔 페이지를 보려면 응용프로그램 > 엔터프라이즈 응용프로그램 > application_name > 대상 맵핑> target_name을 클릭하십시오.

구성 탭

대상
응용프로그램 또는 모듈이 맵핑되는 대상 서버 또는 클러스터의 이름을 기술합니다. 맵 모듈의 대상을 응용프로그램의 설정에서 액세스한 Application Server 페이지에 지정합니다.
데이터 유형 문자열
사용 가능
대상 서버에 설치된 응용프로그램 모듈이 서버가 시작할 때 시작(또는 사용 가능)되는지 여부를 표시합니다. 이는 응용프로그램 모듈의 초기 상태를 설정합니다. true 값은 서버가 시작할 때 관련 모듈이 사용 가능하고 액세스 가능함을 표시합니다. false 값은 서버가 시작할 때 관련 모듈이 사용 가능하지 않고 액세스 가능하지 않음을 표시합니다.
데이터 유형 부울
기본값 true
