Impostazioni di mappature destinazione

Utilizzare questa pagina per mappare un modulo o un'applicazione distribuita con un server o un cluster.

Per visualizzare questa pagina della console di gestione, fare clic su Applicazioni > Applicazioni enterprise > nome_applicazione > Mappature destinazione > nome_destinazione.

Scheda Configurazione

Destinazione
Dichiara il nome del cluster o del server di destinazione con cui viene mappato il modulo o l'applicazione. Specificare la destinazione sulla pagina Mappa i moduli sui server delle applicazioni, accessibile dalle impostazioni di un'applicazione.
Tipo dati Stringa
Abilitato
Indica se i moduli dell'applicazione installati sul server di destinazione vengono avviati (o abilitati) o meno all'avvio del server. Ciò imposta lo stato iniziale dei moduli di applicazione. Un valore true indica che i moduli corrispondenti vengono abilitati e sono, quindi, accessibili all'avvio del server. Un valore false indica che i moduli corrispondenti non vengono abilitati e che non sono accessibili all'avvio del server.
Tipo dati Valore booleano
Predefinito vero
