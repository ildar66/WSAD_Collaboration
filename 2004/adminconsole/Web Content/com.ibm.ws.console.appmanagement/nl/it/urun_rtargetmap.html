Raccolta mappature destinazione

Utilizzare questa pagina per gestire le mappature dei moduli o delle applicazioni distribuite con i server o i cluster.

Per visualizzare questa pagina della console di gestione, fare clic su Applicazioni > Applicazioni enterprise > nome_applicazione > Mappature destinazione.

Destinazione
Dichiara il nome del cluster o del server di destinazione con cui viene mappato il modulo o l'applicazione. Specificare la destinazione sulla pagina Mappa i moduli sui server delle applicazioni, accessibile dalle impostazioni di un'applicazione.
Nodo
Specifica il nome nodo se la destinazione è un server.
Stato
Indica se lo stato dell'applicazione in esecuzione sul server di destinazione è avviato, arrestato o non disponibile.
