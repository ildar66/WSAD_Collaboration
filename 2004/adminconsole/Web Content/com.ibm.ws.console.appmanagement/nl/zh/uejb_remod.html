EJB 模块设置

使用此页面配置和管理特定的已部署 EJB 模块。

注: 您无法启动或停止单个 EJB 模块用于修改。您必须启动或停止整个相应的应用程序。

要查看此管理控制台页面，单击应用程序 > 企业应用程序 > applicationName > EJB 模块 > moduleName。

“配置”选项卡

URI
当解析的 URI 相对于应用程序 URL 时，它指定文件系统上模块归档内容的位置。此 URI 必须与已部署应用程序（EAR）的部署描述符中 ModuleRef URI 的 URI 匹配。
备用 DD
指定在运行时要使用的部署描述符，来取代模块中安装的部署描述符。
启动权值
指定服务器启动时模块启动的顺序。 具有最低启动权值的模块首先启动。
数据类型 整型
缺省 5000
范围 大于 0
