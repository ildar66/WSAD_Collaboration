EJB モジュール設定

このページを使用して、特定のデプロイ済み EJB モジュールの構成および管理を行います。

注: 変更を加えるために、個々の EJB モジュールを開始または停止することはできません。該当するアプリケーション全体を開始するか、停止する必要があります。

この管理コンソール・ページを表示するには、「 アプリケーション」>「エンタープライズ・アプリケーション」>「 applicationName」>「 EJB モジュール」>「moduleName」をクリックします。

「構成」タブ

URI
URI がアプリケーション URL に相対で解決されている場合は、これによって、 ファイル・システム上のモジュール・アーカイブ・コンテンツの場所を指定します。 この URI は、デプロイ済みアプリケーション (EAR) のデプロイメント記述子にある ModuleRef URI の URI と一致する必要があります。
代替 DD
モジュールにインストール済みのデプロイメント記述子に代わって、 実行時に使用されるデプロイメント記述子を指定します。
開始ウェイト
サーバー始動時のモジュールの開始順序を指定します。 開始ウェイトが一番低いモジュールから開始します。
データ型 整数
デフォルト 5000
範囲 0 より大
