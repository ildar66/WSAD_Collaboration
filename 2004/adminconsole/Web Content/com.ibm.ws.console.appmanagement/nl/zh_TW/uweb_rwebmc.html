Web 模組設定

請利用這個頁面來配置 Web 模組設定。

請在管理主控台中遍訪下列路徑來存取這個頁面：

企業應用程式 > 企業應用程式 > application_instance > Web 模組
URI
指定一個 URL，以在相對於應用程式 URL 而解析時，指定模組保存檔內容在檔案系統中的位置。 如果模組套裝在部署應用程式或企業保存檔 (EAR) 中，這個 URI 就必須符合應用程式部署描述子中的 ModuleRef URI。
名稱
指定模組的唯一顯示名稱。
替代 DD
指定用來取代模組 JAR 檔中的原始部署描述子檔之替代部署描述子檔名稱。

這個檔案是部署描述子檔的後組合版本。 您可以編輯原始部署描述子檔來解析相依性和安全資訊。 指定使用替代的部署描述子會將原始部署描述子檔保持不變。

替代 DD 內容的值必須是相對於模組根目錄的部署描述子檔的完整路徑名稱。 依照慣例，檔案會在 ALT-INF 目錄中。 如果沒有指定這個內容，就會讀取模組 JAR 檔中的部署描述子檔。

啟動加權
指定模組的啟動次序。 加權較低的模組比較高的模組先啟動。
喜好 WEB-INF 類別
指定在 WEB-INF 中，要在任何其他類別之前載入的類別。 建議您實作應用程式類別載入器，以便在載入常駐於全儲存區程式庫 JAR 檔中的類別和資源之前，先載入 WAR 檔內所套裝的類別和資源。
起始狀態
指定在伺服器啟動時，這個應用程式的預設狀態。
