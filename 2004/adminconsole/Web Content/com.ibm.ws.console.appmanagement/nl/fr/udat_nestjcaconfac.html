Collection de modules de connecteurs

Cette page permet d'afficher les paramètres de module de connecteur.

Une instance de ConnectorModuleDeployment est créée pour chaque module de connecteur (RAR) de l'application.

Pour afficher cette page de la console d'administration, cliquez sur Applications >Applications d'entreprise > application > Modules de connecteurs.

URI
Chemin d'accès logique à la ressource qui sera servie par le produit.
Type de données String (chaîne)
Nom
Nom complet du module de connecteur.
Type de données String (chaîne)
