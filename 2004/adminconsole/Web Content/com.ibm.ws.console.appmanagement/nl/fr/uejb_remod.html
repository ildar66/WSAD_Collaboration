Paramètres des modules d'EJB

Utilisez cette page pour configurer et gérer un module EJB déployé spécifique.

Important: Vous ne pouvez pas arrêter ou démarrer un module EJB particulier pour modification. Vous devez démarrer ou arrêter intégralement l'application adéquate.

Pour afficher cette vue de la console d'administration, cliquez sur Applications > Applications d'enterprise > nomApplication > Modules EJB > nomModule.

Onglet Configuration

URI
Lorsqu'il est converti par rapport à l'URL de l'application, cet URI indique l'emplacement du contenu archivé d'un module sur un système de fichiers. L'URI doit correspondre à l'URI d'une référence de module dans le descripteur de déploiement de l'application déployée (EAR).
Descripteur de déploiement secondaire
Spécifie un descripteur de déploiement à utiliser lors de l'exécution au lieu de celui qui est installé dans le module.
Pondération de démarrage
Indique l'ordre de lancement des modules lors du démarrage du serveur. Le module doté de la valeur de pondération la plus faible est lancé le premier.
Type de données Entier
Valeur par défaut 5000
Plage de valeurs Supérieur à 0
