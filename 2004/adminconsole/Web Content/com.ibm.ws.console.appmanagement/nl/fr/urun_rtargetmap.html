Collection de mappages cible

Cette page permet de gérer des mappages d'applications déployées ou de modules sur des serveurs ou dans des clusters.

Pour afficher la page de la console d'administration, sélectionnez Applications > Applications d'entreprise > nom_application > Mappages cible.

Cible
Indique le nom du serveur ou du cluster cible vers lequel l'application ou le module est mappé. Vous spécifiez la cible dans la page Mapper les modules vers les serveurs d'applications, accessible à partir des paramètres d'une application.
Noeud
Indique le nom du noeud si la cible est un serveur.
Etat
Indique si l'état de l'application exécutée sur le serveur cible est lancé, arrêté ou non disponible.
