Collection de modules EJB

Utilisez cette page pour gérer les modules EJB déployés dans une application spécifique.

Pour afficher cette page de la console d'administration, cliquez sur Applications > Applications d'entreprise > nomApplication > Modules EJB. Cochez les cases pour sélectionner un ou plusieurs modules EJB dans la collection.

URI
Lorsqu'il est converti par rapport à l'URL de l'application, cet URI indique l'emplacement du contenu archivé d'un module sur un système de fichiers. L'URI correspond à la balise <ejb> ou <web> dans la balise <module> du descripteur de déploiement.
