Paramètres de mappage cible

Cette page permet de mapper une application ou un module déployé vers un serveur ou un cluster.

Pour afficher la page de la console d'administration, sélectionnez Applications > Applications d'entreprise > nom_application > Mappages cible > nom_cible.

Onglet Configuration

Cible
Indique le nom du serveur ou du cluster cible vers lequel l'application ou le module est mappé. Vous spécifiez la cible dans la page Mapper les modules vers les serveurs d'applications, accessible à partir des paramètres d'une application.
Type de données Chaîne
Activé
Indique si les modules d'application installés sur le serveur cible sont lancés (ou activés) lors du démarrage du serveur. Il s'agit de l'état initial des modules d'application. La valeur true indique que les modules correspondants sont activés et accessibles lors du démarrage du serveur. La valeur false indique que les modules correspondants ne sont pas activés et ne sont donc pas accessibles lors du démarrage du serveur.
Type de données Booléen
Valeur par défaut true
