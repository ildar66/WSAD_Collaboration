데이터소스(버전 4)

이 페이지를 참조하여 버전 4.0 양식 데이터소스의 설정을 보십시오.

이 버전 4.0 데이터소스는 WebSphere Application Server 버전 4.0 연결 관리자 아키텍처를 사용합니다. 모든 EJB 1.1 모듈은 버전 4.0 데이터소스를 사용해야 합니다.

이 관리 콘솔 페이지를 보려면 자원 > JDBC 프로바이더 > JDBC_provider > 데이터소스(버전 4)를 클릭하십시오.
