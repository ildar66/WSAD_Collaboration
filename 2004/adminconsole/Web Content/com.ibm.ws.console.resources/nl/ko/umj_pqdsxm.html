WebSphere MQ 대기열 설정(MQ 구성)

이 패널을 사용하여 선택한 대기열 목적지에 대한 WebSphere MQ로 정의된 구성 등록 정보를 보거나 변경하십시오.

대기열 목적지는 JMS 대기열의 등록 정보를 구성하기 위해 사용됩니다. WebSphere MQ JMS 프로바이더에 사용하는 대기열에는 WebSphere MQ로 정의된 다음 추가 등록 정보가 있습니다.

참고:

  • 표시된 일부 등록 정보는 읽기 전용이며 변경할 수 없습니다.
  • 이 등록 정보를 보거나 변경할 수 있으려면 대기열이 상주하는 WebSphere MQ Queue Manager는 원격 관리용으로 구성되거나 실행 중이어야 합니다.
  • 사용자가 지정하는 등록 정보 값은 WebSphere MQ JMS 자원 구성 시 지정한 값과 일치해야 합니다. WebSphere MQ JMS 자원 구성에 대한 자세한 정보는 WebSphere MQ Using Java 서적을 참조하십시오. 예: WebSphere MQ 다중 플랫폼 라이브러리 웹 페이지는    http://www.ibm.com/software/ts/mqseries/library/manualsa/manuals/crosslatest.html .
  • WebSphere MQ에서 이름은 최대 48자이며, 최대 20자의 채널 예외를 갖습니다.

이 관리 콘솔 페이지를 보려면 자원-> WebSphere MQ JMS 프로바이더->(내용 분할창의 추가 등록 정보 아래) WebSphere MQ Queue 목적지-> destination_name->(내용 분할창의 추가 등록 정보 아래) MQ Config를 클릭하십시오.
