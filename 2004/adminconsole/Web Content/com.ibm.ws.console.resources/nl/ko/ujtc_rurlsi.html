URL 구성 설정

이 페이지를 참조하여 네트워크의 시스템에 있는 디렉토리 파일이나 데이터베이스에 저장된 문서와 같이 전자적으로 액세스할 수 있는 자원을 지정하는 URL(Uniform Resource Locator)을 구성하십시오.

이 관리 콘솔 페이지를 보려면 자원 > URL 프로바이더 > URL_provider > URL > URL을 클릭하십시오.

구성 탭

이름
자원의 표시 이름을 지정합니다.
JNDI 이름
JNDI 이름을 지정합니다.
설명
자원의 설명을 지정합니다.
카테고리
자원을 분류 또는 그룹화하는 데 사용할 수 있는 카테고리 문자열을 지정합니다.
스펙
URL을 형성하는 문자열을 지정합니다.
