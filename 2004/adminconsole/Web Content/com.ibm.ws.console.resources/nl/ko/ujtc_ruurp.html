URL 프로바이더 콜렉션

이 페이지를 참조하여 JDK(Java Developer Kit)에 의해 처리되지 않는 URL 프로토콜을 처리하는 새 URL 프로바이더를 작성할 수 있습니다(예, http, ftp 및 파일보다는 프로토콜). URL 프로바이더는 특정 URL 프로토콜에 대한 기능을 구현합니다. 이 프로바이더는 java.net.URLStreamHandler 및 java.net.URLConnection를 확장하는 두 개의 클래스로 구성되어 있습니다. 초기 제품 구성에 기본 URL 프로바이더가 포함되어 있습니다. 이 프로바이더는 JDK가 제공한 URL 지원을 이용합니다. HTTP 또는 FTP와 같은 Java 2 Standard Edition 1.3.1에 기초한 프로토콜을 사용하는 모든 URl 자원은 기본 URL 프로바이더를 이용할 수 있습니다.

이 관리 콘솔 페이지를 보려면 자원 > URL 프로바이더를 클릭하십시오.

이름
URL 프로바이더에 대한 관리 이름을 지정합니다.
설명
관리 레코드의 URL 프로바이더에 대해 설명합니다.
