WebSphere MQ JMS 프로바이더 설정

이 패널을 사용하여 WebSphere MQ JMS 프로바이더의 구성 등록 정보를 보십시오. 이 등록 정보는 WebSphere MQ를 WebSphere Application Server에 설치한 내부 JMS 프로바이더에 대한 JMS 프로바이더로 설치한 경우에만 적용됩니다. 이 등록 정보는 변경할 수 없습니다.

이 관리 콘솔 페이지를 보려면 자원-> WebSphere MQ JMS 프로바이더를 클릭하십시오.

이름
관리 목적의 WebSphere MQ JMS 프로바이더의 이름.
데이터 유형 문자열
기본값 WebSphereMQJMSProvider
설명
IBM WebSphere Application Server 내에서 관리 목적으로 JMS 프로바이더에 대한 설명.
데이터 유형 문자열
기본값 WebSphere MQ JMS 프로바이더
클래스 경로
JMS 프로바이더에 대한 Java 클래스 경로. JMS 프로바이더 클래스의 위치를 형성하는 경로 또는 JAR 파일 이름 목록.
데이터 유형 문자열
기본값 $MQJMS_LIB_ROOT
기본 라이브러리 경로
JMS 프로바이더에 대한 기본 라이브러리 경로. JMS 프로바이더에 필요한 모든 기본 라이브러리에 대한 선택적 경로.
데이터 유형 문자열
기본값 $MQJMS_LIB_ROOT

기본 라이브러리 경로 등록 정보는 WebSphere MQ Java 기능이 설치되어 있는 디렉토리로 설정됩니다.
