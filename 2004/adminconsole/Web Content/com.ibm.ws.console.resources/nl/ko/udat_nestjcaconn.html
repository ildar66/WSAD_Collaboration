커넥터 모듈 설정

이 페이지를 참조하여 커넥터 모듈을 보십시오.

이 관리 콘솔 페이지를 보려면 응용프로그램 > 엔터프라이즈 응용프로그램 > 응용프로그램 > 커넥터 모듈 > connector_module을 클릭하십시오.

구성 탭

URI
WebSphere Application Server가 제공하는 자원으로의 논리 경로를 지정합니다.
데이터 유형 문자열
이름
커넥터 모듈의 표시 이름을 지정합니다.
데이터 유형 문자열
altDD
커넥터 모듈의 대체 DD를 지정합니다.

해당 모듈의 대체 DD URI.

데이터 유형 문자열
시작 가중치
기타 모듈에 대한 커넥터 모듈의 시작 우선 순위를 지정합니다.

응용프로그램에 복수 모듈이 있는 경우, 사용자가 지정하는 시작 가중치가 서버 시작 시 다른 모듈에 대한 이 모듈의 시작 우선순위를 결정합니다. 시작 순서가 낮은 모듈이 먼저 시작됩니다.
