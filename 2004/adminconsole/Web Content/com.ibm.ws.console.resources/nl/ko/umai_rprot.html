프로토콜 프로바이더 콜렉션

이 페이지에서 새 프로토콜 프로바이더를 선택하거나 추가하여 소속 프로토콜에서 실행되는 JavaMail API 및 메일 서버와 대화하십시오. 예는 SMTP(Simple Mail Transfer Protocol)인데, 메일 전송을 위한 대중적인 전송 프로토콜입니다. JavaMail 응용프로그램은 SMTP 서버에 연결하고 프로토콜 프로바이더를 사용하여 해당 서버를 통해 메일을 전송할 수 있습니다.

이 관리 콘솔 페이지를 보려면 자원 > 메일 프로바이더 > mail_provider > 프로토콜 프로바이더를 클릭하십시오.

프로토콜
해당 프로토콜에 대한 프로토콜 프로바이더의 구성을 지정합니다.
클래스 이름
특정 프로토콜 프로바이더(JavaMail 서비스 프로바이더라고도 함)에 대한 구현 클래스를 지정합니다.
클래스 경로
특정 프로토콜 프로바이더(JavaMail 서비스 프로바이더라고도 함)에 대한 구현 클래스에 대한 경로를 지정합니다.
유형
프로토콜 프로바이더의 유형을 지정합니다. 올바른 옵션은 STORE 또는 TRANSPORT입니다.
