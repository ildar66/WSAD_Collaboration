JMS 프로바이더 설정

임베드된 WebSphere JMS 프로바이더나 WebSphere MQ JMS 프로바이더 이외의 JMS 프로바이더를 사용하려는 경우, JMS 프로바이더의 등록 정보를 구성하려면 이 패널을 사용하십시오.

이 관리 콘솔 페이지를 보려면, 자원-> 일반 JMS 프로바이더-> provider_name을 클릭하십시오.

이름
관리를 위한 JMS 프로바이더의 이름

기본값 널
설명
관리를 위한 JMS 프로바이더에 대한 설명

기본값 널
클래스 경로
JMS 프로바이더에 대한 Java 클래스 경로. JMS 프로바이더 클래스의 위치를 형성하는 경로 또는 JAR 파일 이름 목록.
데이터 유형 문자열
기본값 널
기본 라이브러리 경로
JMS 프로바이더에 대한 기본 라이브러리 경로. JMS 프로바이더에 필요한 모든 기본 라이브러리에 대한 선택적 경로.
데이터 유형 문자열
기본값 널
외부 초기 컨텍스트 팩토리
JMS 프로바이더에 대한 초기 컨텍스트 팩토리의 Java 클래스 이름.

예를 들어 LDAP 서비스 프로바이더에 대한 값은 com.sun.jndi.ldap.LdapCtxFactory 양식입니다.

데이터 유형 문자열
기본값 널
외부 프로바이더 URL
외부 JNDI 찾아보기에 대한 JMS 프로바이더 URL

예를 들어, JMS 프로바이더의 LDAP URL은 ldap://hostname.company.com/contextName 양식을 갖습니다.
