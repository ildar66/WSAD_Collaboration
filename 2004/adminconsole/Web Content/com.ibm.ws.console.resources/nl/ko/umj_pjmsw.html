WebSphere JMS 프로바이더 설정

이 패널을 사용하여 WebSphere Application Server와 함께 설치되는 임베드된 WebSphere JMS 프로바이더의 구성 등록 정보를 보십시오. 이 등록 정보를 변경할 수 없습니다.

관리 콘솔 페이지를 보려면 자원-> WebSpherehere JMS 프로바이더를 클릭하십시오.

이름
관리를 위한 JMS 프로바이더의 이름
데이터 유형 문자열
기본값 WebSphereJMSProvider
설명
IBM WebSphere Application Server 내에서 관리 목적으로 JMS 프로바이더에 대한 설명.
데이터 유형 문자열
기본값 내장 WebSphere JMS 프로바이더
