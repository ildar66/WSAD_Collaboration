URL 프로바이더 설정

이 페이지를 참조하여 새 URL 프로바이더를 작성하십시오.

이 관리 콘솔 페이지를 보려면 자원 > URL 프로바이더 > URL_provider를 클릭하십시오.

구성 탭

이름
URL 프로바이더에 대한 관리 이름을 지정합니다.
설명
관리 레코드의 URL 프로바이더에 대해 설명합니다.
클래스 경로
함께 자원 프로바이더 클래스에 대한 위치를 형성하는 경로 또는 JAR 파일 이름을 지정합니다.
스트림 핸들러 클래스 이름
FTP 같은 특정 URL 프로토콜에 대해 java.net.URLStreamHandler를 확장하는 사용자 정의 Java 클래스의 완전한 이름을 지정합니다.
프로토콜
이 스트림 핸들러가 지원하는 프로토콜을 지정합니다. 예를 들면, "nntp", "smtp", "ftp".
