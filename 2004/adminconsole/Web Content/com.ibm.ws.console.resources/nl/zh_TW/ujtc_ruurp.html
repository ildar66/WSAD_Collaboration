URL 提供者集合

請利用這個頁面建立新 URL 提供者來處理 Java Developer Kit (JDK) 所不處理的 URL 通訊協定，比方說，http、ftp 和 file 以外的通訊協定。 URL 提供者負責實作特定 URL 通訊協定的功能。 這個提供者由兩個類別組成，它們繼承 java.net.URLStreamHandler 和 java.net.URLConnection。 預設 URL 提供者併在起始產品配置中。 這個提供者會使用 JDK 所提供的 URL 支援。 任何通訊協定是以 Java 2 標準版 1.3.1 為基礎（如 HTTP 或 FTP）的 URL 資源，都可以使用預設 URL 提供者。

如果要檢視這個管理主控台頁面，請按一下資源 > URL 提供者。

名稱
指定 URL 提供者的管理名稱。
說明
說明管理記錄的 URL 提供者。
