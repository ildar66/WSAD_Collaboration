WebSphere JMS Provider 設定

請使用這個畫面來檢視 WebSphere Application Server 所安裝的內嵌 WebSphere JMS Provider 的配置內容。 您不能變更這些內容。

如果要檢視這個管理主控台頁面，請按一下資源 -> WebSphere JMS Provider
