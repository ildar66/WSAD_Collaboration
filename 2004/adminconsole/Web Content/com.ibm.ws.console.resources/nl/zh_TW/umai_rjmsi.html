郵件階段作業設定

請利用這個頁面來配置特定郵件提供者。

如果要檢視這個管理主控台頁面，請按一下資源 > 郵件提供者 > mail_provider > 郵件階段作業 > mail_session。

「配置」標籤

名稱
指定 JavaMail 階段作業物件的管理名稱。
JNDI 名稱
指定資源的 Java 命名和目錄介面 (JNDI) 名稱，其中包括任何命名子環境定義。

這個名稱提供用戶端應用程式部署描述子所定義之資源的平台連結資訊與平台將它連接到 JNDI 的實際資源之間的鏈結。

說明
指定管理記錄的選擇性說明。
種類
指定分類或分組階段作業的選用集合。
郵件傳輸主機
指定傳送郵件時所存取的伺服器。
郵件傳輸通訊協定
指定傳送郵件時所用的傳輸通訊協定。
郵件傳輸使用者
當郵件傳輸主機需要鑑別時，指定使用者 ID。

大部分的郵件伺服器很少使用這個設定。除非您使用需要使用者 ID 和密碼的郵件伺服器，否則，請將這個欄位留白。

郵件傳輸密碼
當郵件傳輸主機需要鑑別時，指定密碼。

大部分的郵件伺服器很少使用這個設定。除非您使用需要使用者 ID 和密碼的郵件伺服器，否則，請將這個欄位留白。

郵件來源
指定郵件發送端。

這個值代表在收到的訊息中，依預設會顯示為寄件人或收件人位址的網際網路電子郵件位址。 收件人的回信會送到這個位址。

郵件儲存庫主機
指定接收郵件時所存取的伺服器。

這個設定結合郵件儲存庫使用者 ID 和密碼，就代表有效的郵件帳號。 比方說，如果郵件帳戶是 john_william@my.company.com，郵件儲存庫主機就是 my.company.com。

郵件儲存庫通訊協定
指定在接收郵件時所用的通訊協定；它可以是 IMAP、POP3 或使用者已安裝提供者的任何儲存庫通訊協定。
郵件儲存庫使用者
指定給定郵件帳號的使用者 ID。

比方說，如果郵件帳號是 john_william@my.company.com，使用者就是 john_william。

郵件儲存庫密碼
指定給定郵件帳號的密碼。

比方說，如果郵件帳號是 john_william@my.company.com，請輸入 john_william 這個 ID 的密碼。

除錯
開啟和關閉這個郵件階段作業的除錯模式。
