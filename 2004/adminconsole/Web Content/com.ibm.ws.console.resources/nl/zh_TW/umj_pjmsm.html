WebSphere MQ JMS Provider 設定

請使用這個畫面來檢視 WebSphere MQ JMS Provider 的配置內容。您必須已將 WebSphere MQ 安裝成在 WebSphere Application Server 所安裝的內部 JMS 提供者之上 的 JMS 提供者，這些內容才適用。 您不能變更這些內容。

如果要檢視這個管理主控台頁面，請按一下資源 -> WebSphere MQ JMS Provider

名稱
用來識別 WebSphere MQ JMS Provider 供管理使用的名稱。
資料類型 字串
預設值 WebSphereMQJMSProvider
說明
JMS 提供者的說明，供 IBM WebSphere Application Server 內的管理使用。
資料類型 字串
預設值 WebSphere MQ JMS Provider
類別路徑
JMS 提供者的 Java 類別路徑。這是共同形成 JMS 提供者類別位置的路徑或 JAR 檔名稱的清單。
資料類型 字串
預設值 $MQJMS_LIB_ROOT
原生程式庫路徑
JMS 提供者的原生程式庫路徑。這是通往 JMS 提供者所需要的任何原生程式庫的選用路徑。
資料類型 字串
預設值 $MQJMS_LIB_ROOT

「原生程式庫路徑」內容設為 WebSphere MQ Java 特性的安裝路徑。
