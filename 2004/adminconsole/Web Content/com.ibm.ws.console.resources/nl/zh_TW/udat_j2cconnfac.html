J2C Connection Factory 集合

請利用這個頁面來選取代表一組連線配置值的 Connection Factory。

Enterprise Bean 之類的應用程式元件會有參照 Connection Factory 而不是參照資源配接器的資源參照描述子。 Connection Factory 實際上是配置內容清單保留區。 除了資源配接器供應商所定義的任意配置內容組之外，還有一些適用於 Connection Factory 的標準配置內容。這些標準內容供應用程式伺服器執行時期中的 Java 2 連接器連線儲存池管理程式使用，而且供應商提供的資源配接器程式碼不會知道它們。

如果要檢視這個管理主控台頁面，請按一下資源 > 資源配接器 > resource_adapter > J2C Connection Factory。
