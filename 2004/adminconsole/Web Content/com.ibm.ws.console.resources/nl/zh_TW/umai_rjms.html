郵件階段作業集合

請利用這個頁面來配置定義在母項郵件提供者之下的郵件階段作業內容。

如果要檢視這個管理主控台頁面，請按一下資源 > 郵件提供者 > mail_provider > 郵件階段作業。

名稱
指定 JavaMail 階段作業物件的管理名稱。
JNDI 名稱
指定資源的 Java 命名和目錄介面 (JNDI) 名稱，其中包括任何命名子環境定義。

這個名稱提供用戶端應用程式部署描述子所定義之資源的平台連結資訊與平台將它連接到 JNDI 的實際資源之間的鏈結。

說明
指定管理記錄的選擇性說明。
種類
指定分類或分組階段作業的選用集合。
