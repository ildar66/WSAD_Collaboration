通訊協定提供者集合

請利用這個頁面來選取或加入新的通訊協定提供者，以與 JavaMail API 和利用專屬通訊協定來執行的郵件伺服器交談。 簡易郵件轉送通訊協定 (SMTP) 就是範例，它是用來傳送郵件的通用傳輸通訊協定。 JavaMail 應用程式可以連接到 SMTP 伺服器，然後使用通訊協定提供者，透過它來傳送郵件。

如果要檢視這個管理主控台頁面，請按一下資源 > 郵件提供者 > mail_provider > 通訊協定提供者。

通訊協定
指定給定的通訊協定的通訊協定提供者配置。
類別名稱
指定特定通訊協定提供者（亦稱為 JavaMail 服務提供者）的實作類別。
類別路徑
指定特定通訊協定提供者（亦稱為 JavaMail 服務提供者）的實作類別路俓。
類型
指定通訊協定提供者的類型。有效的選項為 STORE 或 TRANSPORT。
