JMS 提供者集合

請使用這個畫面來列出 JMS 提供者，或選取一個 JMS 提供者來檢視或變更它的配置內容。

如果要檢視這個管理主控台頁面，請利用管理主控台來完成下列步驟：

  1. 在導覽窗格中，展開下列路徑之一：
    • 資源 -> WebSphere JMS Provider
    • 資源 -> WebSphere MQ JMS Provider
    • 資源 -> 一般 JMS 提供者

如果要檢視或變更 JMS 提供者或其資源的內容，請在顯示的清單中選取它的名稱。

如果要定義新的一般 JMS 提供者，請在資源 -> 一般 JMS 提供者頁面中，按一下新建。

如果要處理一或多個列出的 JMS 提供者，請按一下您要處理的物件名稱旁的勾選框，再使用所提供的管理主控台按鈕。
