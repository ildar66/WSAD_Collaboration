「WebSphere MQ 佇列」設定（MQ 配置）

使用這個畫面來檢視或變更定義給選取的 Queue Destination 之 WebSphere MQ 的配置內容。

Queue Destination 用來配置 JMS 佇列的內容。 搭配 WebSphere MQ JMS Provider 使用的佇列有下列定義給 WebSphere MQ 的額外內容。

注意事項:

  • 部分顯示的內容是唯讀的，不能改變。
  • 如果要能檢視或變更內容，佇列所在的 WebSphere MQ 佇列管理程式必須配置了遠端管理，且在執行中。
  • 您指定的內容值必須符合在配置 WebSphere MQ JMS 資源時所指定的值。 如果需要配置 WebSphere MQ JMS 資源的詳細資訊，請參閱 WebSphere MQ 使用 Java 一書； 比方說，造訪 WebSphere MQ 多平台書庫網頁，網址如下：   http://www.ibm.com/software/ts/mqseries/library/manualsa/manuals/crosslatest.html 。
  • 在 WebSphere MQ 中，名稱最多可有 48 個字元，但通道例外，它最多只能有 20 個字元。

如果要檢視這個管理主控台頁面，請按一下資源 -> WebSphere MQ JMS Provider->（在內容窗格的「其他內容」下）WebSphere MQ Queue Destination-> destination_name->（在內容窗格的「其他內容」）MQ 配置
