「JMS 提供者」設定

如果您要使用內嵌 WebSphere JMS Provider 或 WebSphere MQ JMS Provider 以外的 JMS 提供者，請使用這個畫面來配置 JMS 提供者的內容。

如果要檢視這個管理主控台頁面，請按一下資源 -> 一般 JMS 提供者 -> provider_name。

名稱
用來識別 JMS 提供者供管理使用的名稱。
資料類型 字串
預設值 空值
說明
JMS 提供者的說明，供管理使用。
資料類型 字串
預設值 空值
類別路徑
JMS 提供者的 Java 類別路徑。這是共同形成 JMS 提供者類別位置的路徑或 JAR 檔名稱的清單。
資料類型 字串
預設值 空值
原生程式庫路徑
JMS 提供者的原生程式庫路徑。這是通往 JMS 提供者所需要的任何原生程式庫的選用路徑。
資料類型 字串
預設值 空值
外部起始環境定義 Factory
JMS 提供者的起始環境定義 Factory 的 Java 類別名稱。

比方說，如果是 LDAP 服務提供者，值的格式如下：com.sun.jndi.ldap.LdapCtxFactory。

資料類型 字串
預設值 空值
外部提供者 URL
外部 JNDI 查閱的 JMS 提供者 URL。

例如，JMS 提供者的 LDAP URL 格式如下：ldap://hostname.company.com/contextName。

資料類型 字串
預設值 空值
