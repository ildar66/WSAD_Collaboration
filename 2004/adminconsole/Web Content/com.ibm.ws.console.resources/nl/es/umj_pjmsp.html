Valores del proveedor de JMS

Si desea utilizar un proveedor de JMS que no sea el proveedor de JMS incorporado de WebSphere ni el proveedor de JMS de WebSphere MQ, utilice este panel para configurar las propiedades del proveedor de JMS.

Para ver esta página de la consola administrativa, pulse Recursos-> Proveedores genéricos de JMS-> nombre_proveedor

Nombre
Nombre por el que se conoce el proveedor de JMS por razones administrativas.
Tipo de datos String
Valor por omisión Null
Descripción
Descripción del proveedor de JMS, por razones administrativas.
Tipo de datos String
Valor por omisión Null
Classpath
Classpath de Java para el proveedor de JMS. Lista de vías de acceso o nombres de archivos JAR que forman conjuntamente la ubicación de las clases de proveedores de JMS.
Tipo de datos String
Valor por omisión Null
Vía de acceso de biblioteca nativa
Vía de acceso de biblioteca nativa para el proveedor de JMS. Vía de acceso opcional a cualquier biblioteca nativa que necesite el proveedor de JMS.
Tipo de datos String
Valor por omisión Null
Fábrica externa de contexto inicial
Nombre de clase Java de la fábrica de contexto inicial del proveedor de JMS.

Por ejemplo, para un proveedor de servicio LDAP, el valor tiene el formato siguiente: com.sun.jndi.ldap.LdapCtxFactory.

Tipo de datos String
Valor por omisión Null
URL de proveedor externo
URL de proveedor de JMS para búsquedas de JNDI externas.

Por ejemplo, el URL LDAP para un proveedor de JMS tiene el formato siguiente: ldap://nombresistppal.enpresa.com/contextName.

Tipo de datos String
Valor por omisión Null
