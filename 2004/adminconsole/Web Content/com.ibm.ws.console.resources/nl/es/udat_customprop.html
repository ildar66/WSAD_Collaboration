Colección de propiedades personalizadas

Utilice esta página para ver las propiedades personalizadas.

Las propiedades personalizadas son exclusivas de cada recurso, y se necesitan cero o más.

Nota: Después de entrar en esta página, lo primero que debe hacer es pulsar en el campo Necesario para ordenar en orden descendente. De este modo, todos los valores necesarios (true) se ordenan al inicio de la página. Debe repetir este proceso hasta que estén establecidas todas las propiedades necesarias.

Para ver esta página de la consola administrativa, pulse Recursos >Proveedores JDBC > proveedor_JDBC > Orígenes de datos > origen_datos > Propiedades personalizadas.
