Nuevo proveedor JDBC

Utilice esta página para seleccionar el tipo de proveedor JDBC que desea crear.

Para ver esta página de la consola administrativa, pulse Recursos >Proveedores JDBC > Nuevo.

Proveedores JDBC
Especifica los nombres de los tipos de proveedor JDBC soportados.

Si la lista de tipos de proveedores JDBC soportados no incluye el proveedor JDBC que desea utilizar, seleccione el proveedor JDBC definido por el usuario. Es posible que necesite consultar la documentación del proveedor JDBC para obtener más información sobre las propiedades específicas que precisa dicho proveedor.
