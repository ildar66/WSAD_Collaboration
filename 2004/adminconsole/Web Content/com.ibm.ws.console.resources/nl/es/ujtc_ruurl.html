Valores del proveedor de URL

Utilice esta página para crear nuevos proveedores de URL.

Para ver esta página de la consola administrativa, pulse Recursos > Proveedores de URL > proveedor_URL.

Pestaña Configuración

Nombre
Especifica el nombre administrativo del proveedor de URL.
Descripción
Describe el proveedor de URL, para los registros de administración.
Classpath
Especifica las vías de acceso o nombres de archivos JAR que forman conjuntamente la ubicación de las clases de proveedores de recursos.
Nombre de clase de manejador de corriente de datos
Especifica el nombre plenamente cualificado de una clase Java definida por el usuario que amplía java.net.URLStreamHandler para un protocolo de URL determinado, como por ejemplo FTP.
Protocolo
Especifica el protocolo soportado por ese manejador de corrientes de datos. Por ejemplo, "nntp", "smtp" o "ftp".
