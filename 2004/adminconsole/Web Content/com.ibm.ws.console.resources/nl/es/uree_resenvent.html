Colección de entradas de entorno de recursos

Utilice esta página para ver las entradas de entorno de recursos.

Un recurso de entorno puede ser de un tipo arbitrario cualquiera. Para obtener más información sobre resource-env-refs y recursos de entorno, consulte la especificación EJB 2.0.

Para ver esta página de la consola administrativa, pulse Recursos >Proveedores de entorno de recursos > proveedor_entorno_recursos > Entradas de entorno de recursos.

Nombre
Especifica un identificador de texto que sirve para distinguir esta entrada de entorno de recursos de las demás.

Por ejemplo, puede utilizar Mi recurso como nombre.

Tipo de datos String
Nombre JNDI
Especifica la serie que se debe utilizar para buscar este recurso de entorno utilizando JNDI.

Ésta es la serie a que debe enlazar los descriptores de despliegue de resource-env-ref.

Tipo de datos String
Descripción
Especifica texto informativo que proporciona ayuda adicional para identificar y distinguir este recurso.
Tipo de datos String
Categoría
Especifica una categoría que puede utilizar para agrupar los recursos del entorno según una característica común.

Es estrictamente una propiedad organizativa y no afecta al funcionamiento del recurso de entorno.

Tipo de datos String
