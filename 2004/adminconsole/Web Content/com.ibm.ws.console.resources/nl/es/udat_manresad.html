Colección de adaptadores de recursos

Utilice esta página para seleccionar un adaptador de recursos, que representa un archivo archivador que contiene código que implementa una biblioteca para conectarse a un programa de fondo EIS (Enterprise Information System), como CICS, SAP o PeopleSoft.

Para ver esta página de la consola administrativa, pulse Recursos >Adaptadores de recursos.

Nombre
Especifica una lista de adaptadores de recursos disponibles.

Estos adaptadores de recursos puede suministrarlos un proveedor de otra compañía que no sea IBM. Por lo general, un único adaptador de recursos solamente puede conectarse con un tipo de sistema de programa de fondo (EIS), pero puede soportar muchas configuraciones distintas para establecer conexiones con ese EIS. El adaptador de recursos tiene muchas propiedades de configuración que se definen en la especificación de J2C y que establece el proveedor que proporciona el código.
