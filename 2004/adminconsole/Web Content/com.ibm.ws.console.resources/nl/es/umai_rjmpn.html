Valores del proveedor de correo

Utilice esta página para implementar JavaMail y crear sesiones de correo, que son una colección de proveedores de protocolos, como por ejemplo, el proveedor de correo incorporado de WebSphere Application Server que incluye tres proveedores de protocolos: SMTP (Simple Mail Transfer Protocol), IMAP (Internet Message Access Protocol) y POP3 (Post Office Protocol).

Para ver esta página de la consola administrativa, pulse Recursos > Proveedores de correo > proveedor_correo.

Pestaña Configuración

Ámbito
Especifica el ámbito del recurso configurado. Este valor indica la ubicación de configuración del archivo de configuración.
Nombre
Especifica el nombre del proveedor de recursos de JavaMail.
Descripción
Especifica la descripción del proveedor de correo.
