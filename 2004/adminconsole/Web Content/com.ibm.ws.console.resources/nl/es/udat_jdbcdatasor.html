Colección de orígenes de datos

Utilice esta página para crear o modificar un origen de datos versión 5.0.

Para ver esta página de la consola administrativa, pulse Recursos > Proveedores JDBC > proveedor_JDBC > Orígenes de datos.

Nombre
Especifica el nombre de visualización de este origen de datos.
Tipo de datos String
Nombre JNDI
Especifica el nombre de la interfaz JNDI (Java Naming and Directory Interface) correspondiente a este origen de datos.
Tipo de datos String
Descripción
Especifica una descripción de texto del origen de datos.
Tipo de datos String
Categoría
Especifica una serie que puede utilizarse para clasificar o agrupar un origen de datos.
Tipo de datos String
