Orígenes de datos (Versión 4)

Utilice esta página para ver los valores de un origen de datos de estilo de la versión 4.0.

Estos orígenes de datos de la versión 4.0 utilizan la arquitectura del gestor de conexiones de WebSphere Application Server Versión 4.0. Todos los módulos EJB 1.1 deben utilizar un origen de datos de la versión 4.0.

Para ver esta página de la consola administrativa, pulse Recursos > Proveedores JDBC > proveedor_JDBC > Orígenes de datos (Versión 4).

Nombre
Especifica un identificador de texto del origen de datos.
Tipo de datos String
Nombre JNDI
Especifica el nombre de la interfaz JNDI (Java Naming and Directory Interface) del origen de datos.
Tipo de datos String
Descripción
Especifica una descripción de texto del origen de datos.
Tipo de datos String
Categoría
Especifica una serie de texto que puede utilizarse para clasificar o agrupar el origen de datos.
Tipo de datos String
