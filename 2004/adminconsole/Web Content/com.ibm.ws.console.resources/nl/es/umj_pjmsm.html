Valores del proveedor de JMS de WebSphere MQ

Utilice este panel para ver las propiedades de configuración del proveedor de JMS de WebSphere MQ. Estas propiedades sólo se aplican si tiene instalado WebSphere MQ como proveedor de JMS en el proveedor de JMS interno instalado con WebSphere Application Server. No puede cambiar estas propiedades.

Para ver esta página de la consola administrativa, pulse Recursos-> Proveedor de JMS de WebSphere MQ.

Nombre
Nombre por el que se conoce el proveedor de JMS de WebSphere MQ por razones administrativas.
Tipo de datos String
Valor por omisión WebSphereMQJMSProvider
Descripción
Descripción del proveedor de JMS, para fines administrativos dentro de IBM WebSphere Application Server.
Tipo de datos String
Valor por omisión Proveedor de JMS de WebSphere MQ
Classpath
Classpath de Java para el proveedor de JMS. Lista de vías de acceso o nombres de archivos JAR que forman conjuntamente la ubicación de las clases de proveedores de JMS.
Tipo de datos String
Valor por omisión $MQJMS_LIB_ROOT
Vía de acceso de biblioteca nativa
Vía de acceso de biblioteca nativa para el proveedor de JMS. Vía de acceso opcional a cualquier biblioteca nativa que necesite el proveedor de JMS.
Tipo de datos String
Valor por omisión $MQJMS_LIB_ROOT

La propiedad Vía de acceso de biblioteca nativa se establece en el directorio donde está instalada la característica Java de WebSphere MQ.
