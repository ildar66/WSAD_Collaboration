Valores del proveedor de JMS de WebSphere

Utilice este panel para ver las propiedades de configuración del proveedor de JMS incorporado de WebSphere instalado con WebSphere Application Server. No puede cambiar estas propiedades.

Para ver esta página de la consola administrativa, pulse Recursos-> Proveedor de JMS de WebSphere.

Nombre
Nombre por el que se conoce el proveedor de JMS por razones administrativas.
Tipo de datos String
Valor por omisión Proveedor de JMS de WebSphere
Descripción
Descripción del proveedor de JMS, para fines administrativos dentro de IBM WebSphere Application Server.
Tipo de datos String
Valor por omisión Proveedor de JMS incorporado de WebSphere
