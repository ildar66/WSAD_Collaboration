Colección de proveedores JDBC

Utilice esta página para ver un proveedor JDBC.

Para ver esta página de la consola administrativa, pulse Recursos > Proveedores JDBC en el árbol de navegación de la consola.

Tenga en cuenta el Ámbito del proveedor JDBC. Si selecciona otro valor que no sea el valor por omisión de Nodo, es posible que no se pueda acceder al proveedor en otros contextos de ámbito.

Nombre
Especifica un identificador de texto de este proveedor.

Por ejemplo, este campo puede ser DB2 JDBC Provider (XA).

Tipo de datos String
Descripción
Especifica una serie de texto que describe este proveedor.
