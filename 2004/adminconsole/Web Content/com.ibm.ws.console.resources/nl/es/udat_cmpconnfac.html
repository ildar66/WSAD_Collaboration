Colección de fábricas de conexiones CMP

Utilice esta página para ver los valores de fábricas de conexiones CMP existentes.

Éstas son las fábricas de conexiones utilizadas por un bean CMP (Container Managed Persistence) para acceder a cualquier almacén de datos de programa de fondo. Las entidades de EJB modelo 2.0 con CMP versión 2.x utilizan una fábrica de conexiones CMP. Las fábricas de conexiones que se enumeran en esta página se crean automáticamente bajo el adaptador de recursos relacionales de WebSphere al seleccionar el recuadro Utilizar este origen de datos en la persistencia gestionada por contenedor (CMP) en el área Propiedades generales de la página Origen de datos. No puede modificar o suprimir fábricas de conexiones creadas automáticamente.

Para ver esta página de la consola administrativa, pulse Recursos >Adaptadores de recursos >Adaptador de recursos relacionales de WebSphere > Fábrica de conexiones CMP.
