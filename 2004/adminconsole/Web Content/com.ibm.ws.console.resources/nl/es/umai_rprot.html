Colección de proveedores de protocolo

Utilice esta página para seleccionar o añadir un nuevo proveedor de protocolos para que interactúe con las API de JavaMail y los servidores de correo que se ejecutan en los protocolos correspondientes. Un ejemplo lo constituye SMTP (Simple Mail Transfer Protocol), que es un protocolo de transporte muy popular para el envío de correo. Las aplicaciones JavaMail pueden conectarse a un servidor SMTP y enviar correo a través de éste mediante el proveedor de protocolos.

Para ver esta página de la consola administrativa, pulse Recursos > Proveedores de correo > proveedor_correo > Proveedores de protocolos.

Protocolo
Especifica la configuración del proveedor de protocolo de un determinado protocolo.
Classname
Especifica la clase de implementación del proveedor de protocolos específico (también conocido como proveedor de servicios de JavaMail).
Classpath
Especifica la vía de acceso a la clase de implementación del proveedor de protocolos específico (también conocido como proveedor de servicios de JavaMail).
Tipo
Especifica el tipo de proveedor de protocolos. Opciones válidas son: STORE o TRANSPORT.
