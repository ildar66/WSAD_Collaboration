Valores de configuración de URL

Utilice esta página para configurar los URL (Uniform Resource Locators) que señalan a recursos accesibles electrónicamente, como por ejemplo, un archivo de directorio de una máquina de una red o un documento almacenado en una base de datos.

Para ver esta página de la consola administrativa, pulse Recursos > Proveedores de URL > proveedor_URL > URL > URL.

Pestaña Configuración

Nombre
Especifica el nombre de visualización del recurso.
Nombre JNDI
Especifica el nombre JNDI.
Descripción
Especifica la descripción del recurso.
Categoría
Especifica la serie de categorías que puede utilizarse para clasificar o agrupar el recurso.
Especificación
Especifica la serie con la que se forma un URL.
