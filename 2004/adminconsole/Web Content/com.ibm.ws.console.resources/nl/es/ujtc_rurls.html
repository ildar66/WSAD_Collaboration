Colección de configuración de URL

Utilice esta página para configurar los URL (Uniform Resource Locators) que señalan a recursos accesibles electrónicamente, como por ejemplo, los archivos de directorio de una máquina de una red o un documento almacenado en una base de datos.

Para ver esta página de la consola administrativa, pulse Recursos > Proveedores de URL > proveedor_URL > URL.

Nombre
Especifica el nombre de visualización del recurso.
Nombre JNDI
Especifica el nombre JNDI.
Descripción
Especifica la descripción del recurso.
Categoría
Especifica la serie de categorías que puede utilizarse para clasificar o agrupar el recurso.
