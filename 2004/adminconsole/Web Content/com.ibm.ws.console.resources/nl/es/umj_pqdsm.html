Valores de cola de WebSphere MQ

Utilice este panel para ver o modificar las propiedades de configuración del destino de cola seleccionado que se va a utilizar con el proveedor de JMS de WebSphere MQ.

Un destino de cola se utiliza para configurar las propiedades de una cola JMS. La fábrica de conexiones de cola asociadas del proveedor de JMS de WebSphere MQ crea las conexiones con la cola. Una cola que se vaya a utilizar con el proveedor de JMS de WebSphere MQ tiene las propiedades siguientes.

Nota:

  • Los valores de las propiedades que especifique deben coincidir con los valores especificados al configurar los recursos JMS de WebSphere MQ. Para obtener más información sobre la configuración de recursos JMS de WebSphere MQ, consulte la publicación WebSphere MQ Utilización de Java.
  • En WebSphere MQ, los nombres pueden tener un máximo de 48 caracteres, a excepción de los canales, que tienen un máximo de 20 caracteres.

Para ver esta página de la consola administrativa, pulse Recursos-> Proveedor de JMS de WebSphere MQ-> (en el panel de contenido, en Propiedades adicionales) Destinos de cola de WebSphere MQ-> nombre_destino

Nombre
Nombre por el que se conoce la cola para fines administrativos dentro de IBM WebSphere Application Server.
Tipo de datos String
Nombre JNDI
Nombre JNDI utilizado para enlazar la cola con el espacio de nombres del servidor de aplicaciones.

Como convención, utilice el nombre JNDI plenamente cualificado; por ejemplo, con el formato jms/Nombre, donde Nombre es el nombre lógico del recurso.

Este nombre se utiliza para enlazar la información de enlace de la plataforma. El enlace asocia los recursos definidos en el descriptor de despliegue del módulo con los recursos reales (físicos) enlazados con JNDI por la plataforma.

Tipo de datos String
Descripción
Una descripción de cola, para administración.
Tipo de datos String
Valor por omisión Null
Categoría
Categoría utilizada para clasificar o agrupar esta cola, para sus registros administrativos de IBM WebSphere Application Server.
Tipo de datos String

Especifica si el tiempo de espera de caducidad para esta cola está definida por la aplicación o por la propiedadCaducidad especificada o, de lo contrario, los mensajes de la cola no caducan nunca (tienen un tiempo espera de caducidad ilimitado).

Este CCSID (identificador del juego de caracteres codificado) debe ser uno de los CCSID a los que WebSphere MQ da soporte.
