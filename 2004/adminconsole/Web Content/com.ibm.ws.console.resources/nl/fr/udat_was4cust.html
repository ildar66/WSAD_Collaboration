Collection de propriétés personnalisées (version 4.0)

Utilisez cette page pour visualiser les propriétés d'une source de données version 4.0.

Pour afficher cette page de la console d'administration, cliquez sur Ressources >Fournisseurs JDBC > fournisseur_JDBC > Sources de données (version 4) > source_données > Propriétés personnalisées.

Nom
Nom de la propriété personnalisée.

Assurez-vous que la source de données dispose de la méthode de définition de cette propriété.

Type de données String (chaîne)
Valeur
Valeur de la propriété personnalisée.
Type de données Integer (entier)
Description
Texte décrivant les éventuelles bornes ou valeurs bien définies pour cette propriété.
Type de données String (chaîne)
Obligatoire
Désigne les propriétés qui sont requises pour cette ressource.
Type de données String (chaîne)
