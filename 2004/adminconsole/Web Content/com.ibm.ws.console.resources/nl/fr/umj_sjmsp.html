Collection de fournisseurs JMS

Utilisez cet écran pour afficher la liste des fournisseurs JMS ou sélectionner un fournisseur JMS afin d'afficher ou de modifier ses propriétés de configuration.

Pour afficher cette page, effectuez les étapes suivantes sur la console administrative :

  1. Dans la sous-fenêtre de navigation, développez l'un des chemins suivants :
    • Ressources-> Fournisseur JMS WebSphere
    • Ressources-> Fournisseur WebSphere MQ JMS
    • Ressources-> Fournisseurs JMS génériques

Pour afficher ou modifier les propriétés d'un fournisseur JMS ou de ses ressources, sélectionnez son nom dans la liste affichée.

Pour définir un nouveau fournisseur JMS générique, cliquez sur Nouveau dans la page Ressources-> Fournisseurs JMS génériques.

Pour agir sur un ou plusieurs des fournisseurs JMS affichés, cochez les cases en regard des noms de ces objets, puis utilisez les Boutons de la console administrative fournis.

Nom
Le nom par lequel le fournisseur JMS est connu pour utilisation administrative.
Description
Une description de ce fournisseur JMS pour utilisation administrative.
