Collection de fabriques de connexions CMP

Cette page permet d'afficher les paramètres des fabriques de connexions CMP.

Il s'agit des fabriques de connexions utilisées par un bean CMP pour accéder à n'importe quel magasin de données dorsal. Une fabrique de connexions CMP est utilisée par les entités EJB EJB modèle 2.0 dont la persistance s'appuie sur CMP version 2.x. Les fabriques de connexions listées sur cette page sont créées automatiquement sous l'adaptateur de ressources relationnelles (RRA) WebSphere lorsque vous cochez la case Utiliser cette source de données dans la persistance CMP, dans la section Propriétés générales de la page de la source de données. Il n'est pas possible de modifier, ni de supprimer les fabriques de connexions créées automatiquement.

Pour afficher cette page de la console d'administration, cliquez sur Ressources >Adaptateurs de ressources >Adaptateur de ressources relationnelles WebSphere > Fabriques de connexions CMP.
