Collection de configuration des URL

Cette page permet de configurer des URL qui pointent vers des ressources accessibles par voie électronique (par exemple, des fichiers de répertoire sur une machine réseau ou un document stocké dans une base de données).

Pour afficher la page de la console d'administration, cliquez sur Ressources > Fournisseurs d'URL > nom_fournisseur_url > URL.

Nom
Indique le nom affiché de la ressource.
Nom JNDI
Indique le nom JNDI.
Description
Indique la description de la ressource.
Catégorie
Spécifie la chaîne de la catégorie qui peut être utilisée pour classer ou regrouper la ressource.
