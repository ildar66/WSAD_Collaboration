Paramètres de fournisseur JMS WebSphere

Utilisez cet écran pour afficher les propriétés de configuration du fournisseur JMS WebSphere intégré qui est installé avec WebSphere Application Server. Vous ne pouvez pas modifier ces propriétés.

Pour afficher cette page de la console d'administration, cliquez sur Ressources-> Fournisseur WebSphere JMS

Nom
Nom d'administration sous lequel le fournisseur JMS est connu.
Type de données Chaîne
Valeur par défaut WebSphereJMSProvider
Description
Description du fournisseur JMS dans le cadre de l'administration d'IBM WebSphere Application Server.
Type de données Chaîne
Valeur par défaut Fournisseur JMS WebSphere intégré
