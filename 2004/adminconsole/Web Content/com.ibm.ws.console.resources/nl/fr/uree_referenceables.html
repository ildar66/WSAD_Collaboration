Collection de référençables

Cette page permet de spécifier le nom de classe de la fabrique (factoryClassname) qui servira à convertir les informations de l'espace de nom en une instance de classe pour le type de ressource souhaité.

Pour afficher cette page de la console d'administration, cliquez sur Ressources >Fournisseurs d'environnement de ressources > fournisseur_environnement_ressource > Référençables.

Nom de classe de fabrique
Spécifie un nom de classe d'implémentation de javax.naming.ObjectFactory.
Type de données Chaîne
Nom de classe
Spécifique le type Java auquel un référençable fournit l'accès, pour la validation des liaisons et pour créer la référence.
Type de données Chaîne
