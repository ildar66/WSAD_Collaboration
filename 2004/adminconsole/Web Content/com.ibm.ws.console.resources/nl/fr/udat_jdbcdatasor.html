Collection de sources de données

Utilisez cette page pour créer ou modifier une source de données version 5.0.

Pour afficher cette page de la console d'administration, cliquez sur Ressources > Fournisseurs JDBC > fournisseur_JDBC > Sources de données.

Nom
Indique le nom affiché de cette source de données.
Type de données String (chaîne)
Nom JNDI
Indique le nom JNDI (Java Naming and Directory Interface) de la source de données.
Type de données String (chaîne)
Description
Texte décrivant la source de données.
Type de données String (chaîne)
Catégorie
Chaîne que vous pouvez utiliser pour classifier ou grouper une source de données.
Type de données String (chaîne)
