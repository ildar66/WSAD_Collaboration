Paramètres de fournisseur JMS

Pour utiliser un fournisseur JMS autre que le fournisseur JMS Websphere intégré ou le fournisseur WebSphere MQ JMS, configurez les propriétés du fournisseur JMS sur cet écran.

Pour afficher la page de la console d'administration, cliquez sur Ressources-> Fournisseurs JMS génériques-> nom_fournisseur

Nom
Nom d'administration sous lequel le fournisseur JMS est connu.
Type de données Chaîne
Valeur par défaut Null
Description
Description du fournisseur JMS, pour une utilisation administrative
Type de données Chaîne
Valeur par défaut Null
Chemin d'accès aux classes
Chemin d'accès aux classes Java du fournisseur JMS. La liste des chemins ou des fichiers JAR qui composent l'adresse des classes du fournisseur JMS.
Type de données Chaîne
Valeur par défaut Null
Chemin d'accès aux bibliothèques natives
Chemin d'accès aux bibliothèques natives pour le fournisseur JMS. Chemin facultatif vers toutes les bibliothèques natives requises par le fournisseur JMS.
Type de données Chaîne
Valeur par défaut Null
Fabrique de contexte initial externe
Nom d'accès aux classes Java de la fabrique de contexte initial d'un fournisseur JMS.

Par exemple, pour un fournisseur de services LDAP, cette valeur est au format : com.sun.jndi.ldap.LdapCtxFactory.

Type de données Chaîne
Valeur par défaut Null
URL de fournisseur externe
URL du fournisseur JMS pour les recherches JNDI externes.

Par exemple, une URL LDAP de fournisseur JMS est au format : ldap://hostname.company.com/contextName.

Type de données Chaîne
Valeur par défaut Null
