Paramètres des fournisseurs de messagerie

Cette page permet d'implémenter JavaMail et de créer des sessions de messagerie, qui sont une collection de fournisseurs de protocoles, tels que le fournisseur de messagerie intégré de WebSphere Application Server qui comprend trois fournisseurs de protocoles : SMTP (Simple Mail Transfer Protocol), IMAP (Internet Message Access Protocol) et POP3 (Post Office Protocol).

Pour afficher la page de la console d'administration, cliquez sur Ressources > Fournisseurs JavaMail > nom_fournisseur_messagerie.

Onglet Configuration

Portée
Indique la portée de la ressource configurée. Cette valeur indique l'emplacement de configuration du fichier de configuration.
Nom
Indique le nom du fournisseur de ressources JavaMail.
Description
Indique la description du fournisseur de messagerie.
