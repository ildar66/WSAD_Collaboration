Collection de fournisseurs d'URL

Cette page permet de créer des fournisseurs d'URL pour gérer les protocoles d'URL qui ne sont pas gérés par le JDK (Java Developer Kit), par exemple, les protocoles autres que http, ftp et file. Un fournisseur d'URL implémente la fonctionnalité pour un protocole d'URL donné. Ce fournisseur est composé de deux classes qui étendent java.net.URLStreamHandler et java.net.URLConnection. Un fournisseur d'URL par défaut est inclus dans la configuration initiale du produit. Ce fournisseur utilise le support d'URL disponible dans le JDK. Toute ressource URL dotée de protocoles basés sur Java 2 Standard Edition 1.3.1, tels que HTTP ou FTP, peut utiliser le fournisseur d'URL par défaut.

Pour afficher la page de la console d'administration, cliquez sur Ressources > Fournisseurs d'URL .
