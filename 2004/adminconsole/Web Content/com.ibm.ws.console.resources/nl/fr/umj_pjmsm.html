Paramètres du fournisseur WebSphere MQ JMS

Utilisez cet écran pour afficher les propriétés de configuration du fournisseur WebSphere MQ JMS. Ces propriétés s'appliquent uniquement si vous avez installé WebSphere MQ comme fournisseur JMS sur le fournisseur JMS interne installé avec WebSphere Application Server. Vous ne pouvez pas modifier ces propriétés.

Pour afficher cette page de la console d'administration, cliquez sur Ressources-> Fournisseur WebSphere MQ JMS

Nom
Nom par lequel le fournisseur WebSphere MQ JMS est connu pour une utilisation administrative.
Type de données Chaîne
Valeur par défaut Fournisseur WebSphere MQ JMS (WebSphereMQJMSProvider)
Description
Description du fournisseur JMS dans le cadre de l'administration d'IBM WebSphere Application Server.
Type de données Chaîne
Valeur par défaut Fournisseur WebSphere MQ JMS
Chemin d'accès aux classes
Chemin d'accès aux classes Java du fournisseur JMS. La liste des chemins ou des fichiers JAR qui composent l'adresse des classes du fournisseur JMS.
Type de données Chaîne
Valeur par défaut $MQJMS_LIB_ROOT
Chemin d'accès aux bibliothèques natives
Chemin d'accès aux bibliothèques natives pour le fournisseur JMS. Chemin facultatif vers toutes les bibliothèques natives requises par le fournisseur JMS.
Type de données Chaîne
Valeur par défaut $MQJMS_LIB_ROOT

La propriété Chemin d'accès aux bibliothèques natives est définie au répertoire d'installation de la fonction WebSphere MQ Java.
