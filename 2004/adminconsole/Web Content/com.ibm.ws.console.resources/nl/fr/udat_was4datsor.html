Sources de données (version 4)

Utilisez cette page pour visualiser les paramètres d'une source de données de type version 4.0.

Les sources de données version 4.0 utilisent l'architecture du gestionnaire de connexions de WebSphere Application Server version 4.0. Tous les modules EJB 1.1 doivent utiliser une source de données version 4.0.

Pour afficher cette page de la console d'administration, cliquez sur Ressources > Fournisseurs JDBC > fournisseur_JDBC > Sources de données (version 4.0).

Nom
Indique un identificateur de la source de données.
Type de données String (chaîne)
Nom JNDI
Indique le nom JNDI (Java Naming and Directory Interface) de la source de données.
Type de données String (chaîne)
Description
Texte décrivant la source de données.
Type de données String (chaîne)
Catégorie
Chaîne de texte que vous pouvez utiliser pour classifier ou grouper la source de données.
Type de données String (chaîne)
