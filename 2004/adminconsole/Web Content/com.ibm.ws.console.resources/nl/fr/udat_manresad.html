Collection d'adaptateurs de ressources

Utilisez cette page pour sélectionner un adaptateur de ressources, lequel représente un fichier archive contenant le code qui implémente une bibliothèque pour la connexion à un système d'information d'entreprise dorsal tel que CICS, SAP ou PeopleSoft.

Pour afficher cette page de la console d'administration, cliquez sur Ressources >Adaptateurs de ressources.

Nom
Liste des adaptateurs de ressources disponibles.

Ces adaptateurs de ressources peuvent être fournis par une société tierce (autre qu'IBM). En général, un adaptateur donné ne peut se connecter qu'à un seul type de système dorsal (EIS), mais il peut accepter de nombreuses configurations différentes pour les connexions à ce système. L'adaptateur de ressources possède de nombreuses propriétés de configuration qui sont définies dans la spécification J2C. Les valeurs de ces propriétés sont configurées par la société fournissant l'adaptateur.

Type de données String (chaîne)
