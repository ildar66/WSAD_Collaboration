Paramètres des sessions de messagerie

Cette page permet de configurer des fournisseurs de messagerie spécifiques.

Pour afficher la page de la console d'administration, cliquez sur Ressources > Fournisseurs JavaMail > nom_fournisseur_messagerie > Sessions JavaMail > nom_session_messagerie.

Onglet Configuration

Nom
Indique le nom d'administration de l'objet session JavaMail.
Nom JNDI
Indique le nom JNDI (Java Naming and Directory Interface) pour la ressource, y compris les sous-contextes de nommage.

Ce nom sert de lien entre les informations de liaison de la plateforme pour les ressources définies dans le descripteur de déploiement des applications du client et les ressources réelles confinées dans JNDI par la plateforme.

Description
Spécifie une description facultative des enregistrements d'administration.
Catégorie
Spécifie une collection facultative permettant de classer ou de regrouper des sessions.
Hôte de transfert de courrier
Spécifie le serveur activé lors de l'envoi de courrier.
Protocole de transfert de courrier
Indique le protocole de transfert utilisé lors de l'envoi de courrier.
Utilisateur de transfert de courrier
Indique l'ID utilisateur à utiliser lorsque l'hôte de transfert du courrier requiert une authentification.

Ce paramètre n'est généralement pas utilisé pour la plupart des serveurs de messagerie. Vous pouvez laisser cette zone vide sauf si vous utilisez un serveur de messagerie qui requiert un ID utilisateur et un mot de passe.

Mot de passe de transfert de courrier
Indique le mot de passe à utiliser lorsque l'hôte de transfert du courrier requiert une authentification.

Ce paramètre n'est généralement pas utilisé pour la plupart des serveurs de messagerie. Vous pouvez laisser cette zone vide sauf si vous utilisez un serveur de messagerie qui requiert un ID utilisateur et un mot de passe.

Courrier de
Indique l'émetteur du courrier.

Cette valeur représente une adresse électronique Internet qui, par défaut, affiche le message reçu sous forme d'adresse De ou Répondre à. La réponse du destinataire aboutit à cette adresse.

Hôte de stockage de courrier
Indique le serveur activé lors de la réception de courrier.

Combiné à l'ID utilisation de stockage de courrier et au mot de passe, ce paramètre représente un compte de messagerie correct. Par exemple, si le compte de messagerie est john_william@mon.entreprise.com, l'hôte de stockage de courrier est mon.entreprise.com.

Protocole de stockage de courrier
Indique le protocole utilisé lors de la réception de courrier ; il peut s'agir d'IMAP, de POP3 ou de tout protocole de stockage pour lequel l'utilisateur a installé un fournisseur.
Utilisateur de stockage de courrier
Indique l'ID utilisateur du compte de messagerie donné.

Par exemple, si le compte de messagerie est john_william@mon.entreprise.com, l'utilisateur est john_william.

Mot de passe de stockage de courrier
Indique le mot de passe du compte de messagerie donné.

Par exemple, si le compte de messagerie est john_william@mon.entreprise.com, entrez le mot de passe correspondant à l'ID john_william.

Débogage
Active et désactive le mode de débogage pour cette session de messagerie.
