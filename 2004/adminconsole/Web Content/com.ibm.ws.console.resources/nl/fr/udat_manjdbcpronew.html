Nouveau fournisseur JDBC

Cette page permet de choisir le type de fournisseur JDBC à créer.

Pour afficher cette page de la console d'administration, cliquez sur Ressources > Fournisseurs JDBC > Nouveau.

Fournisseurs JDBC
Noms des types de fournisseurs JDBC supportés.

Si la liste des types de fournisseurs JDBC supportés n'inclut pas le fournisseur JDBC que vous souhaitez utiliser, sélectionnez Fournisseur JDBC définis par l'utilisateur. Pour obtenir des informations sur les propriétés spécifiques requises par ce fournisseur JDBC, consultez sa documentation.
