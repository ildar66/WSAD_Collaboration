Collection de fournisseurs JDBC

Utilisez cette page pour visualiser un fournisseur JDBC.

Pour afficher cette page de la console d'administration, cliquez sur Ressources > Fournisseurs JDBC dans l'arborescence de navigation de la console.

Notez la portée de votre fournisseur JDBC. Si vous choisissez une portée autre que celle par défaut (Noeud), il est possible que le fournisseur ne soit pas disponible dans les autres contextes de portée.

Nom
Définit un identificateur pour ce fournisseur.

Par exemple, vous pouvez indiquer Fournisseur JDBC DB2 (XA) dans cette zone.

Type de données String (chaîne)
Description
Chaîne de texte décrivant ce fournisseur.
