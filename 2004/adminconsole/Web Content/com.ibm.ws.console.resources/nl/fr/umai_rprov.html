Paramètres des fournisseurs de protocoles

Cette page permet de configurer les paramètres des fournisseurs de protocoles.

Pour afficher la page de la console d'administration, cliquez sur Ressources > Fournisseurs JavaMail > nom_fournisseur_messagerie > Fournisseurs de protocole > nom_fournisseur_protocole.

Onglet Configuration

Protocole
Spécifie la configuration du fournisseur de protocole pour un protocole donné.
Nom de classe
Indique la classe d'implémentation du fournisseur de protocoles spécifique (également connu sous le nom de fournisseur de services JavaMail).
Chemin d'accès aux classes
Indique le chemin d'accès à la classe d'implémentation du fournisseur de protocoles spécifique (également connu sous le nom de fournisseur de services JavaMail).
Type
Indique le type de fournisseur de protocoles. Les options correctes sont Stockage et Transfert.
