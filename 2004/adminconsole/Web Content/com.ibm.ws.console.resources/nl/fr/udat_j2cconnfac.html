Collection de fabriques de connexions J2C

Utilisez cette page pour sélectionner une fabrique de connexions, laquelle représente un ensemble particulier de valeurs de configuration des connexions.

Les composants d'application tels que les EJB (Enterprise JavaBeans) ont des descripteurs de références de ressources qui désignent la fabrique de connexions, et non l'adaptateur de ressources. La fabrique de connexions est véritablement un "conteneur" de liste de propriétés de configuration. Outre le jeu arbitraire de propriétés de configuration définies par le fournisseur de l'adaptateur de ressources, il existe plusieurs propriétés de configuration standard qui s'appliquent à la fabrique de connexions. Ces propriétés standard sont utilisées par le gestionnaire de pool de connexions Java 2 Connectors (J2C) dans le contexte d'exécution (runtime) du serveur d'applications ; elles ne sont pas connues par le code de l'adaptateur de ressources fourni par la société tierce.

Pour afficher cette page de la console d'administration, cliquez sur Ressources > Adaptateurs de ressources > adaptateur_ressources > Fabriques de connections J2C.
