Collection de sessions de messagerie

Cette page permet de configurer les propriétés de la session de messagerie définies sous le fournisseur de messagerie parent.

Pour afficher la page de la console d'administration, cliquez sur Ressources > Fournisseurs JavaMail > nom_fournisseur_messagerie > Sessions JavaMail.

Nom
Indique le nom d'administration de l'objet session JavaMail.
Nom JNDI
Indique le nom JNDI (Java Naming and Directory Interface) pour la ressource, y compris les sous-contextes de nommage.

Ce nom sert de lien entre les informations de liaison de la plateforme pour les ressources définies dans le descripteur de déploiement des applications du client et les ressources réelles confinées dans JNDI par la plateforme.

Description
Spécifie une description facultative des enregistrements d'administration.
Catégorie
Spécifie une collection facultative permettant de classer ou de regrouper des sessions.
