Collection de fabriques de connexions JMS

Fabriques de connexions JMS configurées dans le fournisseur JMS associé pour la messagerie point à point et par publication/abonnement. Utilisez cet écran pour créer ou supprimer des fabriques de connexions JMS ou pour sélectionner une fabrique de connexions afin d'afficher ou de modifier ses propriétés de configuration.

Cet écran affiche une liste des fabriques de connexions JMS génériques avec un résumé de leurs propriétés de configuration.

Pour afficher cette page, effectuez les étapes suivantes sur la console administrative :

  1. Dans la sous-fenêtre de navigation, développez Ressources-> Fournisseurs JMS génériques
  2. Dans la sous-fenêtre de contenu, sélectionnez le fournisseur JMS devant prendre en charge la fabrique de connexions JMS.
  3. Dans la liste de propriétés additionnelles du volet de contenu, sélectionnez l'option de fabrique de connexions JMS générique.

Pour afficher ou modifier les propriétés d'une fabrique de connexions JMS, sélectionnez son nom dans la liste affichée.

Pour agir sur une ou plusieurs des fabriques de connexions JMS affichées, cochez les cases à côté des noms de ces objets, puis utilisez les Boutons de la console administrative fournis.
