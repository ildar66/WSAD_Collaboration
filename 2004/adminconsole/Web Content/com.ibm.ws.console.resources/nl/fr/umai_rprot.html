Collection de fournisseurs de protocoles

Cette page permet de sélectionner ou d'ajouter un nouveau fournisseur de protocole afin d'interagir avec les API JavaMail et les serveurs de messagerie s'exécutant sur desprotocoles liés. Par exemple, le protocole SMTP (Simple Mail Transfer Protocol) est un protocole de transfert couramment utilisé pour envoyer du courrier. Les applications JavaMail peuvent se connecter à un serveur SMTP et envoyer du courrier par son intermédiaire, à l'aide du fournisseur de protocoles.

Pour afficher la page de la console d'administration, cliquez sur Ressources > Fournisseurs JavaMail > nom_fournisseur_messagerie > Fournisseurs de protocole.

Protocole
Spécifie la configuration du fournisseur de protocole pour un protocole donné.
Nom de classe
Indique la classe d'implémentation du fournisseur de protocoles spécifique (également connu sous le nom de fournisseur de services JavaMail).
Chemin d'accès aux classes
Indique le chemin d'accès à la classe d'implémentation du fournisseur de protocoles spécifique (également connu sous le nom de fournisseur de services JavaMail).
Type
Indique le type de fournisseur de protocoles. Les options correctes sont Stockage et Transfert.
