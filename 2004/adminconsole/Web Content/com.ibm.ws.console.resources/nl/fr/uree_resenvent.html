Collection d'entrées d'environnement de ressources

Utilisez cette page pour visualiser les entrées d'environnement de ressources.

Une ressource d'environnement peut être de type arbitraire. Pour plus d'informations sur les références d'environnement de ressources et les ressources d'environnement, reportez-vous à la spécification EJB 2.0.

Pour afficher cette page de la console d'administration, cliquez sur Ressources >Fournisseurs d'environnement de ressources > fournisseur_environnement_ressource > Entrées d'environnement de ressources.

Nom
Identificateur littéral qui aide à distinguer cette entrée resource-env des autres.

Par exemple, vous pouvez indiquer Ma ressource comme nom.

Type de données Chaîne
Nom JNDI
Chaîne à utiliser pour la recherche de cette ressource d'environnement à l'aide de JNDI.

Il s'agit de la chaîne à laquelle vous liez les descripteurs de déploiement des références de l'environnement de ressources.

Type de données Chaîne
Description
Texte aidant à identifier ultérieurement cette ressource et à la distinguer des autres.
Type de données Chaîne
Catégorie
Catégorie que vous pouvez utiliser pour grouper des ressources d'environnement en fonction de caractéristiques qu'elles ont en commun.

Il s'agit d'une propriété d'organisation uniquement, qui n'a aucun impact sur la fonction de la ressource d'environnement.
