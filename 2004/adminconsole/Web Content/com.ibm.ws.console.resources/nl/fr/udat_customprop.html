Collection de propriétés personnalisées

Cette page permet d'afficher les propriétés personnalisées.

Les propriétés personnalisées sont propres à chaque ressource. Certaines ressources n'en nécessitent aucune, tandis que d'autres en exigent une ou plusieurs.

Important: Une fois cette page affichée, la première chose que vous devez faire est de cliquer sur la zone Requis pour trier les propriétés par ordre décroissant. Toutes les valeurs requises (true) sont alors triées en début de page. Vous devez répéter cette procédure jusqu'à ce que toutes les propriétés requises soient définies.

Pour afficher cette page de la console d'administration, cliquez sur Ressources >Fournisseurs JDBC > fournisseur_JDBC > Sources de données > source_données> Propriétés personnalisées.

Nom
Indique le nom de la propriété.

Assurez-vous que l'adaptateur de ressources possède le paramètre correspondant à ce nom.

Type de données String (chaîne)
Valeur
Spécifie la valeur de la propriété.
Type de données Integer (entier)
Description
Texte décrivant les éventuelles bornes ou valeurs bien définies pour cette propriété.
Type de données String (chaîne)
Obligatoire
Désigne les propriétés qui sont requises pour cette ressource.
Type de données String (chaîne)
