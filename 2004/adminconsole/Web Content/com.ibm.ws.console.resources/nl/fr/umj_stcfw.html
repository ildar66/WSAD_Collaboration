Collection de fabriques de connexions de rubrique WebSphere

Les fabriques de connexions de rubriques configurées dans le fournisseur JMS WebSphere intégré pour la messagerie par publication/abonnement avec des rubriques JMS.

Cet écran affiche une liste des fabriques de connexions de rubriques WebSphere avec un résumé de leurs propriétés de configuration.

Pour afficher cette page, effectuez les étapes suivantes sur la console administrative :

  1. Dans la sous-fenêtre de navigation, développez Ressources-> Fournisseur JMS WebSphere.
  2. Dans la liste de propriétés additionnelles du volet de contenu, sélectionnez l'option de fabrique de connexions de rubriques WebSphere.

Pour afficher ou modifier les propriétés d'une fabrique de connexions, sélectionnez son nom dans la liste affichée.

Pour agir sur une ou plusieurs des fabriques de connexions affichées, cochez les cases à côté des noms de ces objets, puis utilisez les Boutons de la console administrative fournis.
