Paramètres du fournisseur d'URL

Cette page permet de créer des fournisseurs d'URL.

Pour afficher la page de la console d'administration, cliquez sur Ressources > Fournisseurs d'URL > nom_fournisseur_url.

Onglet Configuration

Nom
Indique le nom d'administration du fournisseur d'URL
Description
Description du fournisseur d'URL destinée à vos enregistrements d'administration.
Chemin d'accès aux classes
Spécifie les chemins ou les noms de fichier JAR qui composent l'emplacement des classes du fournisseur de ressources.
Nom de la classe de gestionnaire de flux
Spécifie le nom complet d'une classe Java définie par l'utilisateur qui étend java.net.URLStreamHandler pour un protocole d'URL donné, tel que FTP.
Protocole
Spécifie le protocole pris en charge par ce gestionnaire de flux. Par exemple, "nntp", "smtp", "ftp".
