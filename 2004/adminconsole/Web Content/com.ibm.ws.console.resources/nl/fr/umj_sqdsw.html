Collection de destinations de files d'attente WebSphere

Les destinations de files d'attente configurées dans le fournisseur JMS WebSphere intégré pour la messagerie point à point avec des files d'attente JMS.

Cet écran affiche une liste des destinations de files d'attente WebSphere avec un résumé de leurs propriétés de configuration.

Pour afficher cette page, effectuez les étapes suivantes sur la console administrative :

  1. Dans la sous-fenêtre de navigation, développez Ressources-> Fournisseur JMS WebSphere.
  2. Dans la liste des propriétés additionnelles du volet de contenu, sélectionnez Destination de file d'attente WebSphere.

Pour afficher ou modifier les propriétés d'une destination de file d'attente, sélectionnez son nom dans la liste affichée.

Pour agir sur une ou plusieurs des destinations de rubrique affichées, cochez les cases en regard des noms de ces objets, puis utilisez les Boutons de la console administrative fournis.
