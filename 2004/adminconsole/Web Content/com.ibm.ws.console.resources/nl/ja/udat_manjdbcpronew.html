新規 JDBC プロバイダー

このページを使用して、作成する JDBC プロバイダーのタイプを選択します。

この管理コンソール・ページを表示するには、「 リソース」>「JDBC プロバイダー」>「新規」をクリックします。

JDBC プロバイダー
サポートされる JDBC プロバイダー・タイプの名前を指定します。

サポートされる JDBC プロバイダー・タイプのリストに、 使用する JDBC プロバイダーが含まれていない場合は、「 User-Defined JDBC Provider」を選択します。 JDBC プロバイダーで必要な具体的なプロパティーについての詳細は、 当該 JDBC プロバイダーの資料を調べる必要があります。
