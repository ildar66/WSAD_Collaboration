データ・ソース (バージョン 4)

このページを使用して、バージョン 4.0 スタイルのデータ・ソース設定を表示します。

このバージョン 4.0 のデータ・ソースは、WebSphere Application Server バージョン 4.0 接続マネージャーのアーキテクチャーを使用します。 すべての EJB 1.1 モジュールは、バージョン 4.0 データ・ソースを 使用する必要があります。

この管理コンソール・ページを表示するには、「 リソース」>「JDBC プロバイダー」>「JDBC_provider」>「データ・ソース (バージョン 4)」をクリックします。
