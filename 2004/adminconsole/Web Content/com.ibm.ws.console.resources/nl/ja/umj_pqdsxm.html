WebSphere MQ キュー設定 (MQ 構成)

このパネルを使用して、WebSphere MQ に定義した、 選択したキュー宛先の構成プロパティーを表示または変更します。

キュー宛先を使用して、JMS キューのプロパティーを構成します。 WebSphere MQ JMS プロバイダーで使用するキューには、 WebSphere MQ に定義された以下の追加プロパティーがあります。

注:

  • 表示される一部のプロパティーは読み取り専用であるため、変更できません。
  • これらのプロパティーを表示または変更するには、 キューが常駐している WebSphere MQ Queue Manager をリモート管理用に構成して、実行する必要があります。
  • 指定するプロパティー値は、WebSphere MQ の JMS リソースを構成する際に指定した値と一致していなければなりません。 WebSphere MQ JMS リソースの構成の詳細については、資料「WebSphere MQ Java の使用 」を参照してください。この資料は、WebSphere MQ マルチプラットフォーム・ライブラリーに関する Web ページ    http://www.ibm.com/software/ts/mqseries/library/manualsa/manuals/crosslatest.html などから入手することができます。
  • WebSphere MQ では、名前には最大 48 文字まで使用できますが、 チャネルの場合は例外で最大 20 文字までになります。

この管理コンソール・ページを表示するには、「 リソース」->「WebSphere MQ JMS プロバイダー」-> (コンテンツ・ペイン内、「追加プロパティー」の下にある)「WebSphere MQ Queue Destinations」->「destination_name」-> (コンテンツ・ペイン内、「追加プロパティー」の下にある)「MQ 構成」をクリックします。

Maximum queue depth
キューに入れることができるメッセージの最大数。
データ型 整数
単位 メッセージ
デフォルト
範囲 ゼロ以上で、下記の値以下の値。
  • キューが OS/390 上にある場合、999 999 999
  • キューが別の WebSphere MQ プラットフォーム上にある場合、640 000

指定可能な最大値についての詳細は、「 WebSphere MQ MQSC Command Reference」を参照してください。
