Referenceables コレクション

このページを使用して、ネーム・スペースの情報を、 必要なリソースのタイプに対応するクラス・インスタンスに変換するファクトリーの factoryClassname を指定します。

この管理コンソール・ページを表示するには、「 リソース」>「リソース環境プロバイダー」>「resource_environment_provider」>「Referenceables」をクリックします。

ファクトリー・クラス名
javax.naming.ObjectFactory インプリメンテーション・クラス名を指定します。
データ型 ストリング
クラス名
妥当性検査のバインドと参照の作成のために、Referenceable がアクセスを 提供する Java タイプを指定します。
データ型 ストリング
