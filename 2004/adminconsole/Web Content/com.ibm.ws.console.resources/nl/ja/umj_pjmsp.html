JMS プロバイダー設定

組み込み WebSphere JMS プロバイダーや WebSphere MQ JMS プロバイダー以外の JMS プロバイダーを使用する場合は、 このパネルを使用して JMS プロバイダーのプロパティーを構成します。

この管理コンソール・ページを表示するには、「リソース」->「一般 JMS プロバイダー」->「provider_name」をクリックします。

名前
管理目的で JMS プロバイダーを使用する際の名前。
データ型 ストリング
デフォルト Null
説明
管理目的で使用する、JMS プロバイダーの説明
データ型 ストリング
デフォルト Null
クラスパス
JMS プロバイダーの Java クラスパス。JMS プロバイダー・クラスのロケーションを形成する、 パスまたは JAR ファイル名のリストです。
データ型 ストリング
デフォルト Null
ネイティブ・ライブラリー・パス
JMS プロバイダーのネイティブ・ライブラリー・パス。JMS プロバイダーが必要とするネイティブ・ライブラリーへのオプション・パスです。
データ型 ストリング
デフォルト Null
外部初期コンテキスト・ファクトリー
JMS プロバイダーの初期コンテキスト・ファクトリーの Java クラス名。

例えば、LDAP サービス・プロバイダーの場合は、 この値は次のような形式になります。 com.sun.jndi.ldap.LdapCtxFactory

データ型 ストリング
デフォルト Null
外部プロバイダー URL
外部 JNDI ルックアップを行うための JMS プロバイダー URL。

例えば、JMS プロバイダーの LDAP URL は次のような形式になります。 ldap://hostname.company.com/contextName

データ型 ストリング
デフォルト Null
