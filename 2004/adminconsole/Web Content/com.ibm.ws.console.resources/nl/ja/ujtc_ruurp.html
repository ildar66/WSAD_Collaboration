URL プロバイダー・コレクション

このページを使用して、 Java Developer Kit (JDK) で処理されない URL プロトコルを処理する新規の URL プロバイダーを作成します。 JDK で処理されないプロトコルは、例えば、http、ftp および file 以外のプロトコルです。 URL プロバイダーは、特定の URL プロトコルのための機能をインプリメントします。 このプロバイダーは、 java.net.URLStreamHandler および java.net.URLConnection を拡張する 2 つのクラスから成ります。 初期製品構成にはデフォルトの URL プロバイダーが含まれています。 このプロバイダーは、JDK によって提供される URL サポートを利用します。HTTP や FTP など、 Java 2 Standard Edition 1.3.1 に基づいたプロトコルを持つ URL リソースは、 デフォルトの URL プロバイダーを使用することができます。

この管理コンソール・ページを表示するには、「 リソース」>「URL プロバイダー」をクリックします。

名前
URL プロバイダーの管理名を指定します。
説明
管理レコードの用の URL プロバイダーについて説明します。
