WebSphere JMS プロバイダー設定

このパネルを使用して、WebSphere Application Server とともにインストールされる組み込み WebSphere JMS プロバイダーの構成プロパティーを表示します。 これらのプロパティーは変更できません。

この管理コンソール・ページを表示するには、「リソース」->「WebSphere JMS プロバイダー」をクリックします。

名前
管理目的で JMS プロバイダーを使用する際の名前。
データ型 ストリング
デフォルト WebSphereJMSProvider
説明
IBM WebSphere Application Server 内で管理目的で使用する、JMS プロバイダーの説明。
データ型 ストリング
デフォルト 組み込み型 WebSphere JMS プロバイダー
