URL プロバイダー設定

このページを使用して、新規 URL プロバイダーを作成します。

この管理コンソール・ページを表示するには、「 リソース」>「URL プロバイダー」>「URL_provider」をクリックします。

「構成」タブ

名前
URL プロバイダーの管理名を指定します。
説明
管理レコードのための URL プロバイダーについて説明します。
クラスパス
リソース・プロバイダー・クラスの場所を表すパスまたは JAR ファイル名を指定します。
ストリーム・ハンドラー・クラス名
特定の URL プロトコル (FTP など) の java.net.URLStreamHandler を拡張する、 ユーザー定義の Java クラスの完全修飾名を指定します。
プロトコル
このストリーム・ハンドラーがサポートするプロトコルを指定します。 例えば、"nntp"、"smtp"、"ftp" などです。
