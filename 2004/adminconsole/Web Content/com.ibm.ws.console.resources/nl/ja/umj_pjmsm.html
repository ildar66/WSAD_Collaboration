WebSphere MQ JMS プロバイダー設定

このパネルを使用して、WebSphere MQ JMS プロバイダーの構成プロパティーを表示します。 これらのプロパティーが適用されるのは、 WebSphere Application Server とともにインストールした内部 JMS プロバイダーに、 JMS プロバイダーとして WebSphere MQ を上書きしてインストールした場合に限られます。 これらのプロパティーは変更できません。

この管理コンソール・ページを表示するには、「リソース」->「WebSphere MQ JMS Provider」をクリックします。

名前
管理目的で使用する、WebSphere MQ JMS プロバイダーの名前。
データ型 ストリング
デフォルト WebSphereMQJMSProvider
説明
IBM WebSphere Application Server 内で管理目的で使用する、JMS プロバイダーの説明。
データ型 ストリング
デフォルト WebSphere MQ JMS プロバイダー
クラスパス
JMS プロバイダーの Java クラスパス。JMS プロバイダー・クラスのロケーションを形成する、 パスまたは JAR ファイル名のリストです。
データ型 ストリング
デフォルト $MQJMS_LIB_ROOT
ネイティブ・ライブラリー・パス
JMS プロバイダーのネイティブ・ライブラリー・パス。JMS プロバイダーが必要とするネイティブ・ライブラリーへのオプション・パスです。
データ型 ストリング
デフォルト $MQJMS_LIB_ROOT

「ネイティブ・ライブラリー・パス」プロパティーは、WebSphere MQ Java フィーチャーのインストール先ディレクトリーに設定されます。
