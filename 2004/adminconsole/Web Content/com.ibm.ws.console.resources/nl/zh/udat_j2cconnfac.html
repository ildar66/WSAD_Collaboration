J2C 连接工厂集合

使用此页面选择连接工厂，其代表一组连接配置值。

应用程序组件（如企业 bean）有一些资源引用描述符，它们引用连接工厂而不是资源适配器。 连接工厂实际上是配置属性列表持有程序。除了由资源适配器供应商定义的任意配置属性集之外，还有几个应用到连接工厂的标准配置属性。这些标准属性由 Java 2 连接器连接池管理器在应用程序服务器运行时中使用，并且不为提供资源适配器代码的供应商所知。

要查看此管理控制台页面，单击资源 > 资源适配器 > resource_adapter > J2C 连接工厂。

名称
指定连接工厂显示名称的列表。
数据类型 String
JNDI 名称
指定此连接工厂的 Java 命名与目录接口（JNDI）名称。
数据类型 String
描述
指定此连接工厂的文本描述。
数据类型 String
类别
指定可用于对这个连接工厂进行分类或分组的字符串。
数据类型 String
