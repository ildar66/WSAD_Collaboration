定制属性（V4）集合

使用此页面查看 V4.0 的数据源属性。

要查看此管理控制台页面，单击资源 > JDBC 提供程序 > JDBC_provider > 数据源（V4） > data_source > 定制属性

名称
指定定制属性名称。

确保数据源有这个属性的设置方法（setter）。

数据类型 String
值
指定定制属性的值。
数据类型 整型
描述
指定描述这个属性的所有边界或明确的值的文本。
数据类型 String
必需的
指定此资源所需的属性。
数据类型 String
