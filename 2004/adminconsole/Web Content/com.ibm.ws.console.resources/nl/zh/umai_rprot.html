协议提供程序集合

使用此页面选择或添加新的协议提供程序以与 JavaMail API 及相关协议上运行的邮件服务器进行交互。示例是简单邮件传输协议（SMTP），这是流行的发送邮件的传输协议。使用协议提供程序，JavaMail 应用程序能连接到 SMTP 服务器并通过它发送邮件。

要查看此管理控制台页面，单击资源 > 邮件提供程序 > mail_provider > 协议提供程序。

协议
为给定的协议指定协议提供程序的配置。
类名
指定特定协议提供程序（也称为 JavaMail 服务供应商）的实现类。
类路径
指定到特定协议提供程序（也称为 JavaMail 服务供应商）的实现类的路径。
类型
指定协议提供程序的类型。有效的选项是 STORE 或 TRANSPORT。
