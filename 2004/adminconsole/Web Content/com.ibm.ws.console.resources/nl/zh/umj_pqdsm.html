WebSphere MQ 队列设置

使用此面板查看或更改所选队列目标的配置属性，以与 WebSphere MQ JMS 提供程序一起使用。

队列目标用于配置 JMS 队列的属性。 到队列的连接由 WebSphere MQ JMS 提供程序的关联队列连接工厂创建。与 WebSphere MQ JMS 提供程序一起使用的队列具有以下属性。

注:

  • 您指定的属性值必须与配置 WebSphere MQ JMS 资源时指定的值相匹配。要获得有关配置 WebSphere MQ JMS 资源的更多信息，请参阅 WebSphere MQ Using Java 一书。
  • 在 WebSphere MQ 中，名称最长可以是 48 个字符，但例外是通道最长为 20 个字符。

要查看此管理控制台页面，单击资源 -> WebSphere MQ JMS 提供程序 ->（在内容窗格中，“其它属性”下）WebSphere MQ 队列目标 -> destination_name

名称
在 IBM WebSphere Application Server 中出于管理目的，用于标识队列的名称。
数据类型 String
JNDI 名称
用于将连接绑定到应用程序服务器的名称空间的 JNDI 名称。

按约定，使用全限定 JNDI 名称；例如，格式为 jms/Name，其中 Name 是资源的逻辑名称。

该名称用于链接平台绑定信息。绑定使模块部署描述符定义的资源与由平台绑定到 JNDI 的实际（物理）资源相关联。

数据类型 String
描述
用于管理目的，队列的描述
数据类型 String
缺省 空
类别
为了您的 IBM WebSphere Application Server 管理记录，用于分类或分组此队列的类别。
数据类型 String
持久性

数据类型 枚举
单位 不适用
缺省 APPLICATION_DEFINED
范围
应用程序定义
目标上的消息由将它们放到队列的应用程序定义持久性。
队列定义
［仅用于 WebSphere MQ 目标］目标上的消息由 WebSphere MQ 队列定义属性定义其持久性。
持久

非持久

指定的优先级
如果优先级属性设置为 Specified，则在这里输入该队列的消息优先级，其范围在 0（最低）到 9（最大）之间

如果优先级属性设置为 Specified，则发送到此队列的消息具有此属性指定的优先级值。

到期
此队列的到期超时由应用程序或指定的到期属性定义，或者队列上的消息从不到期（具有无限到期超时）
数据类型 枚举
单位 不适用
缺省 APPLICATION_DEFINED
范围
应用程序定义

此队列上的消息无到期超时，所以这些消息从不到期。
指定的到期
