连接器模块设置

使用此页面查看连接器模块。

要查看此管理控制台页面，单击应用程序 > 企业应用程序 > 应用程序 > 连接器模块 > connector_module。

“配置”选项卡

Uri
指定到由 WebSphere Application Server 服务的资源的逻辑路径。
数据类型 String
名称
指定连接器模块的显示名称。
数据类型 String
altDD
指定连接器模块的备用 DD。

给定模块的备用 DD URI。

数据类型 String
启动权值
指定连接器模块高于其它模块的启动优先级。

当您的应用程序包含多个模块时，服务器启动期间您指定的启动权值确定此模块的启动优先级高于其它模块。具有较低启动顺序的模块首先启动。

数据类型 String
