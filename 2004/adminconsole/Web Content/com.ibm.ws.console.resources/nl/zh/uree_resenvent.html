资源环境条目集合

使用此页面查看“资源环境条目”。

环境资源可以是任何任意的类型。请参阅 EJB 2.0 规范以获取关于资源环境引用和环境资源的更多信息。

要查看此管理控制台页面，单击资源 >资源环境提供程序 > resource_environment_provider > 资源环境条目。

名称
指定帮助分辨出这个资源环境条目的文本标识。

例如，您可为这个名称使用我的资源。

数据类型 String
JNDI 名称
指定使用 JNDI 查找这个环境资源时使用的字符串。

这是您将其绑定到资源环境引用部署描述符的字符串。

数据类型 String
描述
指定帮助进一步标识和分辨这个资源的信息文本
数据类型 String
类别
指定用于根据一些公共特征对环境资源进行分组的类别。

它是一个严格的组织属性，同时对环境资源功能没有影响。

数据类型 String
