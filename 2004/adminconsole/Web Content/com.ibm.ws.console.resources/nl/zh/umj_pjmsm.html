WebSphere MQ JMS 提供程序设置

使用此面板查看 WebSphere MQ JMS 提供程序的配置属性。仅当您将 WebSphere MQ 安装为内部 JMS 提供程序（与 WebSphere Application Server 一起安装）之上的 JMS 提供程序时，才应用这些属性。您不能更改这些属性。

要查看此管理控制台页面，单击资源 -> WebSphere MQ JMS 提供程序

名称
为了管理目的而用于标识 WebSphere MQ JMS 提供程序的名称。
数据类型 String
缺省 WebSphereMQJMSProvider
描述
在 IBM WebSphere Application Server 中出于管理目的，对 JMS 提供程序的描述。
数据类型 String
缺省 WebSphere MQ JMS 提供程序
类路径
JMS 提供程序的 Java 类路径。路径或 JAR 文件名的列表，它们共同组成 JMS 提供程序类的位置。
数据类型 String
缺省 $MQJMS_LIB_ROOT
本机库路径
JMS 提供程序的本机库路径。JMS 提供程序需要的到任何本地库的可选路径。
数据类型 String
缺省 $MQJMS_LIB_ROOT

设置本机库路径属性为安装了 WebSphere MQ Java 功能部件的目录。
