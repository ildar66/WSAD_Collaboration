JMS 提供程序设置

如果您要使用不同于嵌入式 WebSphere JMS 提供程序或 WebSphere MQ JMS 提供程序的 JMS 提供程序，则使用此面板配置 JMS 提供程序的属性。

要查看此管理控制台页面，单击资源 -> 一般 JMS 提供程序 -> provider_name

名称
为了管理目的用于标识 JMS 提供程序的名称。
数据类型 String
缺省 空
描述
用于管理目的，JMS 提供程序的描述
数据类型 String
缺省 空
类路径
JMS 提供程序的 Java 类路径。路径或 JAR 文件名的列表，它们共同组成 JMS 提供程序类的位置。
数据类型 String
缺省 空
本机库路径
JMS 提供程序的本机库路径。JMS 提供程序需要的到任何本地库的可选路径。
数据类型 String
缺省 空
外部初始上下文工厂
用于 JMS 提供程序的初始上下文工厂的 Java 类名。

例如，对于 LDAP 服务供应商，该值的格式为：com.sun.jndi.ldap.LdapCtxFactory。

数据类型 String
缺省 空
外部提供程序 URL
用于外部 JNDI 查找的 JMS 供应商 URL。

例如，JMS 供应商的 LDAP URL 的格式为：ldap://hostname.company.com/contextName。

数据类型 String
缺省 空
