URL 提供程序集合

使用此页面创建新的 URL 提供程序，用于处理 Java Developer Kit（JDK）不处理的 URL 协议，例如除 http、ftp 和 file 之外的协议。URL 提供程序实现特定 URL 协议的功能性。此提供程序由扩展 java.net.URLStreamHandler 和 java.net.URLConnection 的两个类组成。 缺省 URL 提供程序包含在初始的产品配置中。此提供程序利用 JDK 提供的 URL 支持。所有包含基于 Java 2 Standard Edition 1.3.1 的协议（如 HTTP 或 FTP）可以使用缺省 URL 提供程序。

要查看此管理控制台页面，单击资源 > URL 提供程序。

名称
指定 URL 提供程序的管理名称。
描述
为您的管理记录描述 URL 提供程序。
