WebSphere JMS 提供程序设置

使用此面板查看嵌入式 WebSphere JMS 提供程序（与 WebSphere Application Server 一起安装）的配置属性。您不能更改这些属性。

要查看此管理控制台页面，单击资源 -> WebSphere JMS 提供程序

名称
为了管理目的用于标识 JMS 提供程序的名称。
数据类型 String
缺省 WebSphereJMSProvider
描述
在 IBM WebSphere Application Server 中出于管理目的，对 JMS 提供程序的描述。
数据类型 String
缺省 内置 WebSphere JMS 提供程序
