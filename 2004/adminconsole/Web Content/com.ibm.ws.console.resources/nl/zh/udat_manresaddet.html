资源适配器设置

使用此页面指定资源适配器的设置。

此资源适配器可由除 IBM 外的第三方供应商提供。通常，单个资源适配器只能连接到一种后端系统（EIS）类型，但它可支持很多不同的用于连接到此 EIS 的配置。资源适配器具有很多配置属性，这些配置属性在 J2C 规范中定义，并由提供此代码的供应商设置。

要查看此管理控制台页面，单击资源 >资源适配器 > resource_adapter。

“配置”选项卡

作用域
指定可看到此资源定义的级别 － 单元、节点或服务器级别。

WebSphere 5.0 在多个作用域中允许如 JDBC 提供程序、名称空间绑定或共享库等资源，并且在更多特定作用域中定义的资源将覆盖在更多常规作用域中定义的复制资源。

单元
最常规的作用域。可以从所有节点和服务器看到单元作用域中定义的资源，除非这些资源已被覆盖。要查看单元作用域中定义的资源，不要在作用域选择表单中指定服务器或节点名。
节点
多数资源类型的缺省作用域。节点作用域中定义的资源覆盖单元作用域中定义的所有复制资源，并且同一节点上的所有服务器可以看见它们，除非它们在该节点上的服务器作用域中被覆盖了。要查看节点作用域中定义的资源，不要指定服务器，但是要在作用域选择表单中选择节点名。
服务器
定义资源的最特定作用域。在“服务器”作用域上定义的资源覆盖任何在“单元”作用域或父“节点”作用域上定义的重复资源定义，并且它们仅对于特定服务器是可见的。要查看服务器作用域中定义的资源，在作用域选择表单中指定服务器名和节点名。

创建资源时，它们总是创建到面板中所选的当前作用域。要在其它作用域中查看作用域，在作用域选择表单中指定不同的节点或服务器。

数据类型 String
名称
指定资源提供程序的名称。

没有空格的字符串对资源提供程序来说是非常有意义的文本标识。

数据类型 String
描述
指定资源工厂的文本描述。

描述资源工厂和其目的的自由格式文本字符串。

数据类型 String
归档路径
指定包含这个资源适配器模块的 .RAR 文件的路径。
数据类型 String
类路径
指定共同组成资源提供程序类的位置的路径或 JAR 文件名的列表。

这包含资源适配器需要的所有其它库。资源适配器代码库自己自动添加到类路径，但是如果需要 .RAR 以外的任何资源适配器代码库，可在这里指定它。

数据类型 String
本机路径
指定组成资源提供程序本地库位置的路径列表。

资源适配器代码库自己自动添加到类路径，但是如果需要 .RAR 以外的任何资源适配器代码库，可在这里指定它。

数据类型 String
