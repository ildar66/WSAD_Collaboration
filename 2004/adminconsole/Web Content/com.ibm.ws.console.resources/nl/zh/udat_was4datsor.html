数据源（V4）

使用此页面查看 V4.0 样式的数据源设置。

这些 V4.0 数据源使用 WebSphere Application Server V4.0 连接管理器体系结构。所有 EJB 1.1 模块必须使用一个 V4.0 的数据源。

要查看此管理控制台页面，单击资源 > JDBC 提供程序 > JDBC_provider > 数据源（V4）。

名称
指定数据源的文本标识。
数据类型 String
JNDI 名称
指定数据源的 Java 命名和目录接口（JNDI）名。
数据类型 String
描述
指定描述数据源的文本。
数据类型 String
类别
指定可用于分类或分组数据源的文本字符串。
数据类型 String
