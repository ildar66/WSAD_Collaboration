类属 JMS 目标设置

使用此面板查看或更改所选 JMS 目标的配置属性，以与关联的 JMS 提供程序一起使用。

JMS 目标用于配置关联一般 JMS 提供程序的 JMS 目标的属性。到 JMS 目标的连接由关联的 JMS 连接工厂创建。与一般 JMS 提供程序（不同于嵌入式 WebSphere JMS 提供程序或 WebSphere MQ JMS 提供程序）一起使用的 JMS 目标具有下列属性。

要查看此管理控制台页面，单击资源 -> 一般 JMS 提供程序 -> provider_name-> JMS 目标 -> 目标

名称
在 IBM WebSphere Application Server 中出于管理目的，用于标识队列的名称。
数据类型 String
类型
此 JMS 目标是队列（对于点到点）还是主题（对于发布／预订）。

选择以下选项之一：

队列
点到点消息传递的 JMS 队列目标。
主题
发布／预订消息传递的 JMS 主题目标。

JNDI 名称
用于将连接绑定到应用程序服务器的名称空间的 JNDI 名称。

按约定，使用全限定 JNDI 名称；例如，格式为 jms/Name，其中 Name 是资源的逻辑名称。

该名称用于链接平台绑定信息。绑定使模块部署描述符定义的资源与由平台绑定到 JNDI 的实际（物理）资源相关联。

数据类型 String
描述
用于管理目的，队列的描述
类别
为了您的 IBM WebSphere Application Server 管理记录，用于分类或分组此队列的类别。
数据类型 String
外部 JNDI 名称
用于将连接绑定到应用程序服务器的名称空间的 JNDI 名称。

按约定，使用全限定 JNDI 名称；例如，格式为 jms/Name，其中 Name 是资源的逻辑名称。

该名称用于链接平台绑定信息。绑定使模块部署描述符定义的资源与由平台绑定到 JNDI 的实际（物理）资源相关联。

数据类型 String
