邮件会话设置

使用此页面配置特定邮件提供程序。

要查看此管理控制台页面，单击资源 > 邮件提供程序 > mail_provider > 邮件会话 > mail_session。

“配置”选项卡

名称
指定 JavaMail 会话对象的管理名称。
JNDI 名称
指定资源的 Java 命名和目录接口（JNDI）名，包括所有命名子上下文。

此名称提供了客户机应用程序部署描述符中定义的资源的平台绑定信息，与平台绑定到 JNDI 的实际资源之间的链接。

描述
指定管理记录的可选描述。
类别
指定分类或分组会话的可选集合。
邮件传输主机
指定发送邮件时访问的服务器。
邮件传输协议
指定发送邮件时使用的传送协议。
邮件传输用户
指定邮件传输主机需要认证时的用户标识。

这个设置通常不用于大多数邮件服务器。保留此字段空白（除非您使用需要用户标识和密码的邮件服务器）。

邮件传送密码
指定邮件传输主机需要认证时的密码。

这个设置通常不用于大多数邮件服务器。保留此字段空白（除非您使用需要用户标识和密码的邮件服务器）。

邮件来源
指定邮件创始者。

该值表示因特网电子邮件地址，缺省情况下，在接收的消息中显示为来源或答复地址。收件人的回复发送至此地址。

邮件存储主机
指定接收邮件时访问的服务器。

该设置由邮件存储用户标识和密码组成以表示一个有效的邮件帐户。例如，如果邮件帐户是 john_william@my.company.com，则邮件存储主机是 my.company.com。

邮件存储协议
指定接收邮件时使用的协议；它可以是 IMAP、 POP3 或用户安装了提供程序的存储协议。
邮件存储用户
指定给定邮件帐户的用户标识。

例如，如果邮件帐户是 john_william@my.company.com，则用户是 john_william。

邮件存储密码
指定给定邮件帐户的密码。

例如，如果邮件帐户是 john_william@my.company.com，则输入标识 john_william 的密码。

调试
为该邮件会话打开和关闭切换调试方式。
