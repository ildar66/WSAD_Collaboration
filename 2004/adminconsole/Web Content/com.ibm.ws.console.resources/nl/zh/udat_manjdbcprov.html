JDBC 提供程序集合

使用此页面查看 JDBC 提供程序。

要查看此管理控制台页面，在控制台导航树中单击资源 > JDBC 提供程序。

注意您的 JDBC 提供程序的作用域。如果您选择缺省节点以外的任何其它节点，那么提供程序可能在其它作用域上下文中不可用。

名称
指定这个提供程序的文本标识。

例如，此字段可以是 DB2 JDBC 提供程序（XA）。

数据类型 String
描述
指定描述这个提供程序的文本字符串。
数据类型 String
