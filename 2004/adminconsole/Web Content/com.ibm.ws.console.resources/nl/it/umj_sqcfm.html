Raccolta della produzione di connessioni code di WebSphere MQ

Le produzioni di connessioni code configurate nel provider JMS diWebSphere MQ per la messaggistica point-to-point con code JMS.

In questo pannello viene visualizzato un elenco di produzione connessioni code WebSphere MQ con un riepilogo delle relative proprietà di configurazione.

Per visualizzare questa pagina, utilizzare la console di gestione e completare le operazioni seguenti:

  1. Nel pannello di navigazione, espandere Risorse-> Provider JMS WebSphere MQ.
  2. Nell'elenco Ulteriori proprietà del pannello del contenuto, selezionare Produzione connessioni code WebSphere MQ.

Per visualizzare o modificare le proprietà di una produzione connessioni, selezionarne il nome nell'elenco visualizzato.

Per agire su una o più produzioni connessioni tra quelle elencate, fare clic sulle caselle di controllo accanto ai nomi degli oggetti desiderati, quindi utilizzare i Pulsanti della console di gestione forniti.
