Raccolta destinazione JMS generica

Le destinazioni JMS configurate nel provider di JMS associato per la messaggistica point-to-point e pubblicazione/sottoscrizione. Utilizzare questo pannello per creare o cancellare le destinazioni JMS, oppure selezionare una destinazione JMS per visualizzare o modificare le relative proprietà di configurazione.

Per visualizzare questa pagina, utilizzare la console di gestione e completare le operazioni seguenti:

  1. Nel pannello di navigazione, espandere Risorse-> Provider JMS generici.
  2. Nel pannello del contenuto, selezionare il provider di JMS desiderato per supportare la destinazione JMS.
  3. Nell'elenco Ulteriori proprietà del pannello del contenuto, selezionare Destinazione JMS generica.

Per visualizzare o modificare le proprietà di una destinazione JMS, selezionarne il nome nell'elenco visualizzato.

Per agire su una o più destinazioni JMS tra quelle elencate, fare clic sulle caselle di controllo accanto ai nomi degli oggetti desiderati, quindi utilizzare i Pulsanti della console di gestione forniti.
