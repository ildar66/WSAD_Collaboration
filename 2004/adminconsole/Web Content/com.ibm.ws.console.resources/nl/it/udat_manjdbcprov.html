Raccolta di provider di JDBC

Utilizzare questa pagina per visualizzare un provider di JDBC.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse > Provider JDBC nell'albero di navigazione della console.

Osservare l'ambito del provider di JDBC. Se si seleziona un valore Nodo diverso da quello predefinito, è possibile che il provider non sia disponibile in altri contesti.

Name
Specifica un identificativo di testo per questo provider.

Ad esempio, questo campo può essere DB2 provider JDBC (XA).

Tipo dati Stringa
Descrizione
Specifica una stringa di testo che descrive questo provider.
Tipo dati Stringa
