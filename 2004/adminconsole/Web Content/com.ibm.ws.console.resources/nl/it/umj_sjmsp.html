Raccolta di provider JMS

Utilizzare questo pannello per elencare i provider JMS oppure selezionare un provider JMS per visualizzare o modificare le relative proprietà di configurazione.

Per visualizzare questa pagina, utilizzare la console di gestione e completare le operazioni seguenti:

  1. Nel pannello di navigazione, espandere uno dei seguenti percorsi:
    • Risorse-> Provider JMS WebSphere
    • Risorse-> Provider JMS WebSphere MQ
    • Risorse-> Provider JMS generici

Per visualizzare o modificare le proprietà di un provider JMS o le relative risorse, selezionarne il nome nell'elenco visualizzato.

Per definire un provider JMS generico, nella pagina Risorse-> Provider JMS generici fare clic su Nuovo.

Per agire su uno o più provider JMS tra quelli elencati, fare clic sulle caselle di controllo accanto ai nomi degli oggetti desiderati, quindi utilizzare i Pulsanti della console di gestione forniti.
