Raccolta destinazioni coda di WebSphere

Le destinazioni coda configurate nel provider JMS interno di WebSphere per la messaggistica point-to-point con code JMS.

In questo pannello viene visualizzato un elenco di destinazioni coda WebSphere con un riepilogo delle relative proprietà di configurazione.

Per visualizzare questa pagina, utilizzare la console di gestione e completare le operazioni seguenti:

  1. Nel pannello di navigazione, espandere Risorse-> Provider JMS WebSphere
  2. Nell'elenco Ulteriori proprietà del pannello del contenuto, selezionare Destinazione code WebSphere.

Per visualizzare o modificare le proprietà di una destinazione coda, selezionarne il nome nell'elenco visualizzato.

Per agire su una o più destinazioni coda tra quelle elencate, fare clic sulle caselle di controllo accanto ai nomi degli oggetti desiderati, quindi utilizzare i Pulsanti della console di gestione forniti.
