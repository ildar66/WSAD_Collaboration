Impostazioni del provider JMS di WebSphere

Utilizzare questo pannello per visualizzare le proprietà di configurazione del provider JMS compreso in WebSphere installato con WebSphere Application Server. Non è possibile modificare queste proprietà.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse-> Provider JMS di WebSphere

Nome
Il nome attribuito al provider JMS per scopi di gestione.
Tipo dati Stringa
Predefinito WebSphereJMSProvider
Descrizione
Una descrizione del provider JMS, per scopi di gestione all'interno di IBM WebSphere Application Server.
Tipo dati Stringa
Predefinito Provider JMS incorporato di WebSphere
