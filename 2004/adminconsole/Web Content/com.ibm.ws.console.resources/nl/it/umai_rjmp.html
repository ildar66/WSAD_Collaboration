Raccolta provider di posta

Utilizzare questa pagina per implementare JavaMail e creare sessioni di posta, che rappresentano una raccolta di provider di protocolli come ad esempio il provider di posta incorporato di WebSphere Application Server che comprende tre provider di protocollo: SMTP (Simple Mail Transfer Protocol), IMAP (Internet Message Access Protocol) e POP3 (Post Office Protocol).

Per visualizzare questa pagina della console di gestione, fare clic su Risorse > Provider di posta.

Name
Specifica il nome del provider risorse JavaMail.
Descrizione
Specifica la descrizione del provider di risorsa.
