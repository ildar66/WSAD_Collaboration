Raccolta delle proprietà personalizzate (Versione 4)

Utilizzare questa pagina per visualizzare le proprietà per un'origine dati della Versione 4.0.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse >Provider JDBC > provider_JDBC > Origini dati (Versione 4) > origine_dati > Proprietà personalizzate

Name
Specifica il nome della proprietà personalizzata

Assicurarsi che l'origine dati abbia un metodo setter di questa proprietà.

Tipo dati Stringa
Valore
Specifica il valore della proprietà personalizzata.
Tipo dati Numero intero
Descrizione
Specifica il testo per descrivere qualsiasi collegamento o valore ben definito per questa proprietà.
Tipo dati Stringa
Necessario
Specifica le proprietà obbligatorie per questa risorsa.
Tipo dati Stringa
