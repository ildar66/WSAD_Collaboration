Raccolta di proprietà personalizzate

Utilizzare questa pagina per visualizzare le proprietà personalizzate.

Le proprietà personalizzate sono univoche per ogni risorsa, e possono esse impostate a zero o a un valore superiore a zero.

Nota: dopo aver visualizzato questa pagina, la prima operazione consiste nel fare clic sul campo Obbligatorio per eseguire l'ordinamento in modo decrescente. Tutti i valori obbligatori (true) vengono ordinati all'inizio della pagina. È necessario ripetere questo processo finché non vengono impostate tutte le proprietà.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse >Provider JDBC > provider_JDBC > Origini dati > origine_dati> Proprietà personalizzate.

Name
Specifica il nome della proprietà.

È necessario assicurare che l'adattatore risorse abbia l'impostazione per questo nome.

Tipo dati Stringa
Valore
Specifica il valore della proprietà.
Tipo dati Numero intero
Descrizione
Specifica il testo per descrivere qualsiasi collegamento o valore ben definito per questa proprietà.
