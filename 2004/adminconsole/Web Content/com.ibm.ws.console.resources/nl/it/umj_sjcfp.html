Raccolta produzione connessioni JMS

Le produzioni connessioni JMS configurate nel provider di JMS associato per la messaggistica point-to-point e pubblicazione/sottoscrizione. Utilizzare questo pannello per creare o cancellare le produzioni connessioni JMS oppure selezionare una produzione connessione per visualizzare o modificare le relative proprietà di configurazione.

In questo pannello viene visualizzato un elenco di produzioni connessioni JMS generiche con un riepilogo delle relative proprietà di configurazione.

Per visualizzare questa pagina, utilizzare la console di gestione e completare le operazioni seguenti:

  1. Nel pannello di navigazione, espandere Risorse-> Provider JMS generici.
  2. Nel pannello del contenuto, selezionare il provider di JMS desiderato per supportare la produzione connessioni JMS.
  3. Nell'elenco Ulteriori proprietà del pannello del contenuto, selezionare Produzione di connessioni JMS generiche.

Per visualizzare o modificare le proprietà di una produzione connessioni JMS, selezionarne il nome nell'elenco visualizzato.

Per agire su una o più produzioni connessioni JMS tra quelle elencate, fare clic sulle caselle di controllo accanto ai nomi degli oggetti desiderati, quindi utilizzare i Pulsanti della console di gestione forniti.
