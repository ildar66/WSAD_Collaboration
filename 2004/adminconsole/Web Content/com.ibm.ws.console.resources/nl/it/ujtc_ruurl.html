Impostazioni del provider di URL

Utilizzare questa pagina per creare i nuovi provider di URL.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse > Provider URL > provider_URL.

Scheda Configurazione

Nome
Specifica il nome di gestione per il provider di URL.
Descrizione
Descrive il provider di URL, per i record di gestione.
Classpath
Specifica i percorsi o nomi di file JAR che insieme costituiscono l'ubicazione delle classi dei provider di risorse.
Nome classe gestore flussi
Specifica il nome completo di una classe Java definita dall'utente che estende java.net.URLStreamHandler per un protocollo URL particolare, ad esempio FTP.
Protocollo
Specifica il protocollo supportato dall'handler del flusso specificato. Ad esempio "nntp", "smtp", "ftp".
