Impostazioni della sessione di posta

Utilizzare questa pagina per configurare rovider di posta specifici.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse> Provider di posta > provider_posta > Sessioni di posta > sessioni_posta.

Scheda Configurazione

Name
Specifica il nome di gestione dell'oggetto sessione JavaMail.
Nome JNDI
Specifica il nome JNDI (Java Naming and Directory Interface) della risorsa, compresi eventuali contesti secondari di denominazione.

Questo nome fornisce il collegamento tra le informazioni proprie della piattaforma per le risorse definite nel descrittore di distribuzione dell'applicazione client e le risorse reali collegate a JNDI tramite la piattaforma.

Descrizione
Specifica una descrizione facoltativa dei record di gestione.
Categoria
Specifica una raccolta facoltativa per classificare o raggruppare le sessioni.
Host trasporto posta
Specifica il server a cui si accede per l'invio della posta.
Protocollo trasporto posta
Specifica il protocollo di trasporto utilizzato durante l'invio della posta.
Utente trasporto posta
Specifica l'ID utente quando l'host di trasporto posta richiede l'autenticazione.

Questa impostazione non viene generalmente utilizzata per la maggior parte dei server di posta. Lasciare vuoto questo campo a meno che non si utilizzi un server di posta che richiede un ID utente e una password.

Password di trasporto posta
Specifica la password quando l'host di trasporto posta richiede l'autenticazione.

Questa impostazione non viene generalmente utilizzata per la maggior parte dei server di posta. Lasciare vuoto questo campo a meno che non si utilizzi un server di posta che richiede un ID utente e una password.

Posta da
Specifica il mittente della posta.

Questo valore indica l'indirizzo di posta elettronica Internet che, per impostazione predefinita, viene visualizzato nel messaggio ricevuto, come indirizzo Da o Rispondi a. La risposta al messaggio verrà indirizzata a questo indirizzo.

Host memoria posta
Specifica il server a cui si accede durante la ricezione della posta.

Quest'impostazione, insieme alla password e all'ID utente memoria posta, rappresenta un account di posta valido. Ad esempio, se l'account di posta è john_william@my.company.com l'host della memoria di posta è my.company.com.

Protocollo memoria posta
Specifica il protocollo utilizzato durante la ricezione della posta; potrebbe essere IMAP, POP3 o un qualsiasi protocollo di memoria per cui l'utente ha installato un provider.
Utente memoria posta
Specifica l'ID utente per l'account di posta specificato.

Ad esempio, se l'account di posta è john_william@my.company.com, l'utente è john_william.

Password memoria posta
Specifica la password per l'account di posta specificato.

Ad esempio, se l'account di posta è john_william@my.company.com immettere la password per l'ID john_william.

Debug
Attiva e disattiva la modalità di debug per questa sessione di posta.
