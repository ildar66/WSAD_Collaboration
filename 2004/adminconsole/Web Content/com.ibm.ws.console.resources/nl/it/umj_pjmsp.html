Impostazioni del provider JMS

Se si desidera utilizzare un provider JMS diverso dal provider JMS interno di WebSphere o dal provider JMS di WebSphere MQ, utilizzare questo pannello per configurare le proprietà del provider JMS:

Per visualizzare questa pagina della console di gestione, fare clic su Risorse-> Provider JMS generici-> nome_provider

Nome
Il nome attribuito al provider JMS per scopi di gestione.
Tipo dati Stringa
Predefinito Nullo
Descrizione
Una descrizione del provider di JMS, per scopi di gestione
Tipo dati Stringa
Predefinito Nullo
Classpath
Percorso classe Java per il provider JMS. L'elenco di percorsi o nomi file JAR che insieme costituiscono l'ubicazione delle classi del provider JMS.
Tipo dati Stringa
Predefinito Nullo
Percorso libreria nativa
Percorso libreria nativa per il provider JMS. Un percorso facoltativo ad una qualsiasi delle librerie native di cui ha bisogno il provider JMS.
Tipo dati Stringa
Predefinito Nullo
Produzione del contesto iniziale esterno
Il nome classe Java della produzione del contesto iniziale per il provider JMS.

Ad esempio, per un provider del servizio LDAP il valore è: com.sun.jndi.ldap.LdapCtxFactory.

Tipo dati Stringa
Predefinito Nullo
URL del provider esterno
L'URL del provider JMS per ricerche JNDI esterne.

Ad esempio, un URL LDAP per un provider JMS è: ldap://hostname.company.com/contextName.

Tipo dati Stringa
Predefinito Nullo
