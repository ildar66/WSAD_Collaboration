Impostazioni del provider JMS di WebSphere MQ

Utilizzare questo pannello per visualizzare le proprietà di configurazione del provider JMS di WebSphere MQ. Queste proprietà vengono applicate solo se WebSphere MQ è stato installato come provider JMS sul provider JMS interno installato con WebSphere Application Server. Non è possibile modificare queste proprietà.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse-> Provider JMS di WebSphere MQ

Nome
Il nome attribuito al provider JMS di WebSphere MQ per scopi di gestione.
Tipo dati Stringa
Predefinito WebSphereMQJMSProvider
Descrizione
Una descrizione del provider JMS, per scopi di gestione all'interno di IBM WebSphere Application Server.
Tipo dati Stringa
Predefinito Provider JMS WebSphere MQ
Classpath
Percorso classe Java per il provider JMS. L'elenco di percorsi o nomi file JAR che insieme costituiscono l'ubicazione delle classi del provider JMS.
Tipo dati Stringa
Predefinito $MQJMS_LIB_ROOT
Percorso libreria nativa
Percorso libreria nativa per il provider JMS. Un percorso facoltativo ad una qualsiasi delle librerie native di cui ha bisogno il provider JMS.
Tipo dati Stringa
Predefinito $MQJMS_LIB_ROOT

La proprietà del Percorso libreria nativa è impostata sulla directory in cui è installata la funzione Java WebSphere MQ.
