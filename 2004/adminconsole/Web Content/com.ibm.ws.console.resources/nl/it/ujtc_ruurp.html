Raccolta provider di URL

Utilizzare questa pagina per creare nuovi provider di URL per gestire protocolli URL che non sono gestiti da JDK (Java Developer Kit), ad esempio protocolli diversi da http, ftp e file. Un provider URL implementa la funzionalità di un particolare protocollo URL. Questo provider è costituito da due classi che estendono le classi java.net.URLStreamHandler e java.net.URLConnection. Nella configurazione del prodotto iniziale è compreso un provider di URL predefinito. Questo provider utilizza il supporto URL fornito da JDK. Qualsiasi risorsa URL con protocolli basati su Java 2 Standard Edition 1.3.1, quali HTTP o FTP, possono utilizzare il provider URL predefinito.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse > Provider URL.

Nome
Specifica il nome di gestione per il provider di URL.
Descrizione
Descrive il provider di URL per i record di gestione.
