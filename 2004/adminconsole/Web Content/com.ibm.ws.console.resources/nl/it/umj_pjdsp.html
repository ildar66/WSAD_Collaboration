Impostazioni della destinazione JMS generica

Utilizzare questo pannello per visualizzare o modificare le proprietà di configurazione della destinazione JMS selezionata da utilizzare con il provider JMS associato.

Una destinazione JMS viene utilizzata per configurare le proprietà di una destinazione JMS per il provider JMS generico associato. Le connessioni alla destinazione JMS vengono create dalla produzione connessioni JMS associata. Una destinazione JMS da utilizzare con un provider JMS generico (diverso dal provider JMS interno di WebSphere o dal provider JMS di WebSphere MQ) ha le seguenti proprietà:

Per visualizzare questa pagina della console di gestione, fare clic suRisorse-> Provider JMS generici-> nome_provider-> Destinazioni JMS-> destinazione

Nome
Il nome attribuito alla coda per scopi di gestione all'interno di IBM WebSphere Application Server.
Tipo dati Stringa
Tipo
Specifica se la destinazione JMS specificata indica una coda (per il tipo point-to-point) o un argomento (per il tipo pubblicazione/sottoscrizione).

Selezionare una delle seguenti opzioni:

Coda
Una destinazione coda JMS per la messaggistica di tipo point-to-point.
Argomento
Una destinazione argomento JMS per la messaggistica pubblicazione/sottoscrizione.

Nome JNDI
Il nome JNDI utilizzato per collegare la coda allo spazio nome del server delle applicazioni.

Per convenzione, utilizzare il nome completo JNDI; ad esempio, nel formato jms/Nome, dove Nome è il nome logico della risorsa.

Tale nome viene utilizzato per il collegamento alle informazioni specifiche della piattaforma. Il collegamento associa le risorse definite dal descrittore di distribuzione del modulo alle risorse effettive (fisiche) collegate a JNDI dalla piattaforma.

Tipo dati Stringa
Descrizione
Una descrizione della coda, per scopi di gestione
Categoria
Una categoria utilizzata per classificare o raggruppare questa coda per i record di gestione IBM WebSphere Application Server.
Tipo dati Stringa
Nome JNDI esterno
Il nome JNDI utilizzato per collegare la coda allo spazio nome del server delle applicazioni.
