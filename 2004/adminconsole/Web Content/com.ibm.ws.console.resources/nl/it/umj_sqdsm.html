Raccolta destinazioni coda di WebSphere MQ

Le destinazioni coda configurate nel provider di JMS di WebSphere MQ per la messaggistica point-to-point con code JMS.

In questo pannello viene visualizzato un elenco di destinazioni coda WebSphere MQ con un riepilogo delle relative proprietà di configurazione.

Per visualizzare questa pagina, utilizzare la console di gestione e completare le operazioni seguenti:

  1. Nel pannello di navigazione, espandere Risorse-> Provider JMS WebSphere MQ.
  2. Nell'elenco Ulteriori proprietà del pannello dei contenuti, selezionare Destinazione code WebSphere MQ.

Per visualizzare o modificare le proprietà di una destinazione coda, selezionarne il nome nell'elenco visualizzato.

Per agire su una o più destinazioni coda tra quelle elencate, fare clic sulle caselle di controllo accanto ai nomi degli oggetti desiderati, quindi utilizzare i Pulsanti della console di gestione forniti.
