Raccolta destinazioni argomento di WebSphere

Le destinazioni argomento configurate nel provider JMS interno di WebSphere per la messaggistica pubblicazione/sottoscrizione con argomenti JMS. Utilizzare questo pannello per creare o cancellare le destinazioni argomento, oppure selezionare una destinazione argomento per visualizzare o modificare le relative proprietà di configurazione.

In questo pannello viene visualizzato un elenco di destinazioni argomento WebSphere con un riepilogo delle relative proprietà di configurazione.

Per visualizzare questa pagina, utilizzare la console di gestione e completare le operazioni seguenti:

  1. Nel pannello di navigazione, espandere Risorse-> Provider JMS WebSphere
  2. Nell'elenco Ulteriori proprietà del pannello del contenuto, selezionare Destinazione argomenti WebSphere.

Per visualizzare o modificare le proprietà di una destinazione argomento, selezionarne il nome nell'elenco visualizzato.

Per agire su una o più destinazioni argomento tra quelle elencate, fare clic sulle caselle di controllo accanto ai nomi degli oggetti desiderati, quindi utilizzare i Pulsanti della console di gestione forniti.
