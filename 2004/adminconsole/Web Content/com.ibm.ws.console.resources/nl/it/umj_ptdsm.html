Impostazioni argomenti di WebSphere MQ

Utilizzare questo pannello per visualizzare o modificare le proprietà di configurazione della destinazione argomenti selezionata da utilizzare con il provider JMS di WebSphere MQ.

Una destinazione argomento viene utilizzata per configurare le proprietà di un argomento JMS per il provider JMS associato. Le connessioni all'argomento vengono create dalla produzione di connessioni argomenti associata. Un argomento da utilizzare con il provider JMS di WebSphere MQ interno ha le seguenti proprietà.

Nota:

  • I valori delle proprietà specificati devono corrispondere ai valori specificati durante la configurazione di WebSphere MQ per le risorse JMS. Per ulteriori informazioni sulla configurazione delle risorse JMS di WebSphere MQ, consultare il manuale Using Java di WebSphere MQ.
  • In WebSphere MQ, i nomi possono avere un massimo di 48 caratteri, ad eccezione dei canali, che possono contenere un massimo di 20 caratteri.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse-> Provider JMS di Websphere MQ-> (Nel pannello contenuto, in Ulteriori proprietà) Destinazioni argomenti Websphere MQ-> nome_destinazione

Nome
Il nome attribuito all'argomento per scopi di gestione all'interno di IBM WebSphere Application Server.
Tipo dati Stringa
Nome JNDI
Il nome JNDI utilizzato per collegare l'argomento allo spazio nome del server delle applicazioni.

Per convenzione, utilizzare il nome completo JNDI; ad esempio, nel formato jms/Nome, dove Nome è il nome logico della risorsa.

Tale nome viene utilizzato per il collegamento alle informazioni specifiche della piattaforma. Il collegamento associa le risorse definite dal descrittore di distribuzione del modulo alle risorse effettive (fisiche) collegate a JNDI dalla piattaforma.
