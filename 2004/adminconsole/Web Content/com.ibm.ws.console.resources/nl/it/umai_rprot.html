Raccolta provider di protocollo

Utilizzare questa pagina per selezionare o aggiungere un provider di protocolli nuovo per interagire con le API JavaMail e i server di posta che eseguono i protocolli. Un esempio è SMTP (Simple Mail Transfer Protocol), che è un protocollo di trasporto per inviare la posta. È possibile collegare le applicazioni JavaMail a un server SMTP e inviare la posta tramite questi utilizzando il provider di protocolli.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse > Provider di posta > provider_posta > Provider di protocolli.

Protocollo
Specifica la configurazione del provider di protocollo per un protocollo specificato.
Nome classe
Specifica la classe di implementazione per il provider di protocolli specifico (anche noto come provider del servizio JavaMail).
Classpath
Specifica il percorso alla classe di implementazione per il provider di protocolli specifico (anche noto come provider del servizio JavaMail).
Tipo
Specifica il tipo di provider di protocolli. Opzioni valide sono MEMORIA o TRANSPORTO.
