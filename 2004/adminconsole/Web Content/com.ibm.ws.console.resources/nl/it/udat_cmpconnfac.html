Raccolta di produzioni connessioni CMP

Utilizzare questa pagina per visualizzare le impostazioni di produzioni di connessioni CMP esistenti.

Sono le produzioni connessioni utilizzate da un bean CMP (container-managed persistence) per accedere all'archivio dati backend. Una produzione connessioni CMP viene utilizzata da entità 2.0 del modello EJB con la versione CMP 2.x. Le produzioni connessioni elencate in questa pagina vengono create automaticamente nell'Adattatore risorse relazionali WebSphere quando viene selezionata la casella Utilizzare questo DataSource nella persistenza gestita dal contenitore (CMP) nell'area Proprietà generali della pagina Origine dati. Non è possibile modificare o eliminare le produzioni di connessioni create automaticamente.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse >Adattatori risorse >Adattatore risorse relazionali WebSphere > Produzioni connessioni CMP.
