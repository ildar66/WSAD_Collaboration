Raccolta di origini dati

Utilizzare questa pagina per creare o modificare l'origine dati della versione 5.0.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse > Provider JDBC > provider_JDBC > Origini dati.

Name
Specifica il nome visualizzato di questa origine dati.
Tipo dati Stringa
Nome JNDI
Specifica il nome JNDI (Java Naming and Directory Interface) per questa origine dati.
Tipo dati Stringa
Descrizione
Specifica una descrizione dell'origine dati.
Tipo dati Stringa
Categoria
Specifica una stringa che può essere utilizzata per classificare o raggruppare un'origine dati.
Tipo dati Stringa
