Raccolta configurazione dell'URL

Utilizzare questa pagina per configurare gli URL (Uniform Resource Locators) che puntano a risorse a cui è possibile accedere elettronicamente, come i file in una directory su una macchina in rete o un documento memorizzato in un database.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse > Provider URL > provider_URL > URL.

Nome
Specifica il nome visualizzato della risorsa.
Nome JNDI
Specifica il nome JNDI.
Descrizione
Specifica la descrizione della risorsa.
Categoria
Specifica la stringa di categoria che può essere utilizzata per classificare o raggruppare la risorsa.
