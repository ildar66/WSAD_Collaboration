Raccolta di produzioni connessioni J2C

Utilizzare questa pagina per selezionare una produzione connessioni che rappresenti una serie di valori di configurazione della connessione.

I componenti dell'applicazione, quali i bean enterprise, contengono descrittori dei riferimenti risorse che fanno riferimento ad una produzione connessioni specifica e non all'adattatore risorse. La produzione connessioni rappresenta un supporto dell'elenco delle proprietà di configurazione. Oltre alla serie arbitraria di proprietà di configurazione, definite dal fornitore dell'adattatore risorse, sono presenti diverse proprietà di configurazione standard che si applicano alla produzione di connessioni. Queste proprietà standard vengono utilizzate dal gestore lotti connessioni J2C (Java 2 Connector) nel runtime del server delle applicazioni e non sono note al codice dell'adattatore risorse fornito dal rivenditore.

Per visualizzare questa pagina della console di gestione, fare clic su Risorse > Adattatori risorse > adattatore_risorse > Produzioni connessioni J2C.

Name
Specifica un elenco di nomi visualizzati di produzioni connessioni.
Tipo dati Stringa
Nome JNDI
Specifica il nome JNDI (Java Naming and Directory Interface) di questa produzione connessioni.
Tipo dati Stringa
Descrizione
Specifica una descrizione di questa produzione connessioni.
Tipo dati Stringa
Categoria
Specifica una stringa che è possibile utilizzare per classificare o raggruppare questa produzione connessioni.
Tipo dati Stringa
