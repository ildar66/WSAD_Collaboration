Raccolta produzione connessioni code di WebSphere

Le produzioni connessioni code configurate nel provider JMS interno di WebSphere per la messaggistica point-to-point con code JMS.

In questo pannello viene visualizzato un elenco di produzioni connessioni code WebSphere con un riepilogo delle relative proprietà di configurazione.

Per visualizzare questa pagina, utilizzare la console di gestione e completare le operazioni seguenti:

  1. Nel pannello di navigazione, espandere Risorse-> Provider JMS WebSphere
  2. Nell'elenco Ulteriori proprietà nel pannello del contenuto, selezionare Produzione connessioni code WebSphere.

Per visualizzare o modificare le proprietà di una produzione connessioni, selezionarne il nome nell'elenco visualizzato.

Per agire su una o più produzioni connessioni tra quelle elencate, fare clic sulle caselle di controllo accanto ai nomi degli oggetti desiderati, quindi utilizzare i Pulsanti della console di gestione forniti.
