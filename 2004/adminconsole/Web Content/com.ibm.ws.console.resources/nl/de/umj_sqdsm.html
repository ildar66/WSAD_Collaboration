Zieladressen für WebSphere-MQ-Warteschlangen

Die Zieladressen von Warteschlangen, die im WebSphere-MQ-JMS-Provider für das Point-to-Point-Messaging mit JMS-Warteschlangen konfiguriert sind.

Diese Anzeige enthält eine Liste der Zieladressen der WebSphere-MQ-Warteschlangen mit einer Zusammenfassung der entsprechenden Konfigurationsmerkmale.

Führen Sie in der Administrationskonsole die folgenden Schritte durch, um diese Seite der Administrationskonsole aufzurufen.

  1. Wählen Sie im Navigationsfenster Ressourcen > WebSphere-MQ-JMS-Provider aus.
  2. Wählen Sie im Inhaltsfenster in der Liste "Weitere Merkmale" die Option Zieladresse für eine WebSphere-MQ-Warteschlange aus.

Wenn Sie die Merkmale der Zieladresse für eine Warteschlange anzeigen oder ändern möchten, wählen Sie den Namen der entsprechenden Zieladresse in der Liste aus.

Wenn Sie mit einer oder mehreren aufgelisteten Zieladressen von Warteschlangen arbeiten möchten, klicken Sie die Markierungsfelder neben den Namen der gewünschten Objekte an und verwenden Sie dann die entsprechenden Knöpfe in der Adminstrationskonsole.
