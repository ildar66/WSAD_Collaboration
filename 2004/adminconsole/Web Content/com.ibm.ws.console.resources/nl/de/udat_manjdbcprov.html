JDBC-Provider

Auf dieser Seite können Sie einen JDBC-Provider anzeigen.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, in der Navigationsstruktur der Konsole auf Ressourcen > JDBC-Provider.

Beachten Sie den Bereich des JDBC-Providers. Wenn Sie den Standardwert Knoten nicht auswählen, ist der Provider in anderen Bereichskontexten möglicherweise nicht verfügbar.

Name
Eine Text-ID für diesen Provider.

Dieses Feld kann z. B. den Wert DB2 JDBC Provider (XA) enthalten.

Datentyp String
Beschreibung
Eine Textzeichenfolge, die diesen Provider beschreibt.
Datentyp String
