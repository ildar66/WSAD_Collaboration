JMS-Provider

In dieser Anzeige können Sie die JMS-Provider auflisten oder einen JMS-Provider auswählen, um dessen Konfigurationsmerkmale anzuzeigen oder zu ändern.

Führen Sie in der Administrationskonsole die folgenden Schritte durch, um diese Seite der Administrationskonsole aufzurufen.

  1. Wählen Sie im Navigationsfenster eine der folgenden Optionskombinationen aus:
    • Ressourcen > WebSphere-JMS-Provider
    • Ressourcen > WebSphere-MQ-JMS-Provider
    • Ressourcen > Generische JMS-Provider

Wenn Sie die Merkmale für einen JMS-Provider oder dessen Ressourcen anzeigen oder ändern möchten, wählen Sie den entsprechenden Namen in der Liste aus.

Möchten Sie einen neuen generischen JMS-Provider definieren, klicken Sie auf der Seite Ressourcen > Generische JMS-Provider auf Neu.

Möchten Sie einen oder mehrere der aufgelisteten JMS-Provider bearbeiten, klicken Sie auf die Markierungsfelder neben den Namen der Objekte, die Sie bearbeiten möchten, und verwenden Sie die dann die entsprechenden Knöpfe in der Adminstrationskonsole.

Name
Der Name, mit dem der JMS-Provider für die Administration bezeichnet wird.
Beschreibung
Eine Beschreibung dieses JMS-Providers für die Administration.
