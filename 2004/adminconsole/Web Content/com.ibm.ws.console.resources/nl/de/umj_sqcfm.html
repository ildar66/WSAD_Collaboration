Verbindungs-Factorys für WebSphere-MQ-Warteschlangen

Die Verbindungs-Factorys für Warteschlangen, die im WebSphere-MQ-JMS-Provider konfiguriert wurden und für das Point-to-Point-Messaging mit JMS-Warteschlangen vorgesehen sind.

Diese Anzeige enthält eine Liste der Verbindungs-Factorys für WebSphere-MQ-Warteschlangen mit einer Zusammenfassung der entsprechenden Konfigurationsmerkmale.

Führen Sie in der Administrationskonsole die folgenden Schritte durch, um diese Seite der Administrationskonsole aufzurufen.

  1. Wählen Sie im Navigationsfenster Ressourcen > WebSphere-MQ-JMS-Provider aus.
  2. Wählen Sie im Inhaltsfenster in der Liste "Weitere Merkmale" die Option Verbindungs-Factory für WebSphere-MQ-Warteschlange aus.

Wenn Sie die Merkmale für eine Verbindungs-Factory anzeigen oder ändern möchten, wählen Sie den Namen der entsprechenden Verbindungs-Factory in der Liste aus.

Wenn Sie mit einer oder mehreren der aufgelisteten Verbindungs-Factorys arbeiten möchten, klicken Sie das Markierungsfeld neben den Namen der gewünschten Objekte an und verwenden Sie dann die bereitgestellten Knöpfe in der Adminstrationskonsole.
