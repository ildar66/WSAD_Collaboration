Generische JMS-Zieladressen

Dabei handelt es sich um die JMS-Zieladressen, die im zugeordneten JMS-Provider für das Point-to-Point- und das Pub/Sub-Messaging konfiguriert sind. In dieser Anzeige können Sie JMS-Zieladressen erstellen, löschen oder eine JMS-Zieladresse auswählen, um die entsprechenden Konfigurationsmerkmale anzuzeigen oder zu ändern.

Führen Sie in der Administrationskonsole die folgenden Schritte durch, um diese Seite der Administrationskonsole aufzurufen.

  1. Wählen Sie im Navigationsfenster Ressourcen > Generische JMS-Provider aus.
  2. Wählen Sie im Inhaltsfenster den JMS-Provider aus, der die JMS-Zieladressen unterstützen soll.
  3. Wählen Sie im Inhaltsfenster in der Liste "Weitere Merkmale" die Option Generische JMS-Zieladresse aus.

Wenn Sie die Merkmale einer JMS-Zieladresse anzeigen oder ändern möchten, wählen Sie den Namen der entsprechenden JMS-Zieladresse in der Liste aus.

Möchten Sie eine oder mehrere der aufgelisteten JMS-Zieladressen bearbeiten, klicken Sie die Markierungsfelder neben den Namen der Objekte an, die Sie bearbeiten möchten, und verwenden Sie die entsprechenden Knöpfe in der Adminstrationskonsole.
