Datenquellen (Version 4)

Auf dieser Seite können Sie die Einstellungen einer Datenquelle der Version 4.0 anzeigen.

Diese Datenquellen der Version 4.0 verwenden die Verbindungsmanagerarchitektur von WebSphere Application Server Version 4.0. Alle EJB-1.1-Module müssen eine Datenquelle der Version 4.0 verwenden.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > JDBC-Provider >JDBC-Provider > Datenquellen (Version 4).

Name
Gibt eine Textbeschreibung der Datenquelle an.
Datentyp String
JNDI-Name
Gibt den JNDI-Namen (Java Naming and Directory Interface) dieser Datenquelle an.
Datentyp String
Beschreibung
Gibt eine Textbeschreibung der Datenquelle an.
Datentyp String
Kategorie
Gibt eine Textzeichenfolge an, die zur Klassifizierung oder Gruppierung der Datenquelle verwendet werden kann.
Datentyp String
