Referenzierbare Einträge

Auf dieser Seite können Sie den Klassennamen der Factory angeben, die Informationen im Namespace in eine Klasseninstanz für den gewünschten Ressourcentyp konvertiert.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > Ressourcenumgebungsprovider > Ressourcenumgebungsprovider > Referenzierbare Objekte.

Klassenname der Factory
Enthält den Klassennamen für die Implementierung von javax.naming.ObjectFactory.
Datentyp String
Klassenname
Gibt den Java-Typ an, auf den über ein referenzierbares Objekt zugegriffen werden kann und der zur Validierung der Bindung und zur Erstellung der Referenz verwendet wird.
Datentyp String
