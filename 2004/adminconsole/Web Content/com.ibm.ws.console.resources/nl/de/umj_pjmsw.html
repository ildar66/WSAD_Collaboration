Einstellungen des WebSphere-JMS-Providers

In dieser Anzeige können Sie die Konfigurationsmerkmale des eingebetteten WebSphere-JMS-Providers, der mit dem WebSphere Application Server installiert wird, anzeigen. Sie können diese Merkmale nicht ändern.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Ressourcen-> WebSphere-JMS-Provider.

Name
Der Name, mit dem der JMS-Provider für die Administration bezeichnet wird.
Datentyp String
Standardwert WebSphere-JMS-Provider
Beschreibung
Eine Beschreibung des JMS-Provider für die Administration in IBM WebSphere Application Server.
Datentyp String
Standardwert Integrierter WebSphere-JMS-Provider
