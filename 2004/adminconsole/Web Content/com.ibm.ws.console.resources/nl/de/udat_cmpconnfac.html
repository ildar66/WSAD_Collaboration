CMP-Verbindungs-Factorys

Auf dieser Seite können Sie Einstellungen für vorhandene CMP-Verbindungs-Factorys anzeigen.

Diese Verbindungs-Factorys werden von einer CMP-Bean (Container-Managed Persistence) verwendet, um auf den Zugriff auf Back-End-Datenspeicher zu ermöglichen. Eine CMP-Verbindungs-Factory wird von Entitäten des EJB-Modells 2.0 mit CMP-Version 2.x verwendet. Verbindungs-Factorys, die auf dieser Seite aufgelistet sind, werden automatisch unter dem relationalen WebSphere-Ressourcenadapter erstellt, wenn Sie auf der Seite "Datenquelle" im Bereich "Allgemeine Merkmale" das Markierungsfeld Diese Datenquelle mit CMP verwenden auswählen. Automatisch erstellte Verbindungs-Factorys können nicht geändert oder gelöscht werden.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > Ressourcenadapter > Relationaler WebSphere-Ressourcenadapter > CMP-Verbindungs-Factorys.
