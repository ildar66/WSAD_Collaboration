Einstellungen für URL-Provider

Auf dieser Seite können Sie neue URL-Provider erstellen.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > URL-Provider > URL-Provider.

Register "Konfiguration"

Name
Gibt den Administrationsnamen für den URL-Provider an.
Beschreibung
Beschreibt den URL-Provider für die Administration.
Klassenpfad
Eine Liste von Pfaden bzw. Namen von Jar-Dateien, die zusammen das Verzeichnis der Ressourcen-Provider-Klassen angeben.
Klassenname des Datenstrom-Handler
Vollständig qualifizierter Name einer benutzerdefinierten Java-Klasse, die java.net.URLStreamHandler für ein bestimmtes URL-Protokoll, wie z. B. FTP, erweitert.
Protokoll
Das von diesem Datenstrom-Handler unterstützte Protokoll. Beispiele: "nntp", "smtp", "ftp".
