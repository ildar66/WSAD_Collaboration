Datenquellen

Auf dieser Seite können Sie Datenquellen von WAS 5.0 erstellen oder ändern.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > JDBC-Provider >JDBC-Provider > Datenquellen.

Name
Gibt den Anzeigenamen dieser Datenquelle an.
Datentyp String
JNDI-Name
Gibt den JNDI-Namen (Java Naming and Directory Interface) für diese Datenquelle an.
Datentyp String
Beschreibung
Gibt eine Textbeschreibung der Datenquelle an.
Datentyp String
Kategorie
Gibt eine Textzeichenfolge an, die zur Klassifizierung oder Gruppierung einer Datenquelle verwendet werden kann.
Datentyp String
