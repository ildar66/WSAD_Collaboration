Zieladressen für WebSphere-MQ-Topics

Die Zieladressen von Topics, die im WebSphere-MQ-JMS-Provider konfiguriert wurden und für das Pub/Sub-Messaging mit JMS-Topics vorgesehen sind. In dieser Anzeige können Sie Zieladressen von Topics erstellen, löschen oder die Zieladresse für ein Topic auswählen, um dessen Konfigurationsmerkmale anzuzeigen oder zu ändern.

Diese Anzeige enthält eine Liste der Zieladressen von WebSphere-MQ-Topics mit einer Zusammenfassung der entsprechenden Konfigurationsmerkmale.

Führen Sie in der Administrationskonsole die folgenden Schritte durch, um diese Seite der Administrationskonsole aufzurufen.

  1. Wählen Sie im Navigationsfenster Ressourcen > WebSphere-MQ-JMS-Provider aus.
  2. Wählen Sie im Inhaltsfenster in der Liste "Weitere Merkmale" die Option Zieladresse für WebSphere-MQ-Topic aus.

Wenn Sie die Merkmale der Zieladresse für ein Topic anzeigen oder ändern möchten, wählen Sie den Namen der entsprechenden Zieladresse in der Liste aus.

Wenn Sie mit einer oder mehreren aufgelisteten Topic-Zieladressen arbeiten möchten, klicken Sie die Markierungsfelder neben den Namen der gewünschten Objekte an und verwenden Sie dann die entsprechenden Knöpfe in der Adminstrationskonsole.
