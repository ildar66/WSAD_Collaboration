Bereichseinstellungen in der Administrationskonsole

Mit den Bereichseinstellungen können Sie den Bereich der auf einer Administrationskonsolseite angezeigten Informationen z. B. auf eine bestimmte Zelle, einen bestimmten Knoten oder Server begrenzen. Wenn Sie den Wert für Bereich ändern, können Sie andere Variablen, die sich auf ein Objekt beziehen und den Inhalt der Objektgruppentabelle möglicherweise ändern, anzeigen.

Klicken Sie neben einem Feld auf Durchsuchen, um die Optionen für das Einschränken des Feldbereichs anzuzeigen. Wenn das Feld schreibgeschützt ist, können Sie den Bereich nicht ändern. Wenn z. B. nur ein Server vorhanden ist, können Sie nicht zum Bereich eines anderen Servers umschalten.

Zelle
Begrenzt die Anzeige auf alle Server in der benannten Zelle.
Knoten
Begrenzt die Anzeige auf alle Server auf dem benannten Knoten.
Server
Begrenzt die Anzeige auf den benannten Server.

Der Serverbereich hat Vorrang vor dem Knoten- und dem Zellenbereich, und der Knotenbereich hat Vorrang vor dem Zellenbereich. Beachten Sie, dass Objekte in nur einem Bereich erstellt werden, obwohl sie möglicherweise in mehreren Bereichen sichtbar sind.
