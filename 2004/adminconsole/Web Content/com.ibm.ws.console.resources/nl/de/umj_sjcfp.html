JMS-Verbindungs-Factorys

Die JMS-Verbindungs-Factorys, die im zugeordneten JMS-Provider für das Point-to-Point- und das Pub/Sub-Messaging konfiguriert sind. In dieser Anzeige können Sie JMS-Verbindungs-Factorys erstellen, löschen oder eine Verbindungs-Factory auswählen, um die entsprechenden Konfigurationsmerkmale anzuzeigen oder zu ändern.

Diese Anzeige enthält eine Liste der der JMS-Verbindungs-Factorys mit einer Zusammenfassung der entsprechenden Konfigurationsmerkmale.

Führen Sie in der Administrationskonsole die folgenden Schritte durch, um diese Seite der Administrationskonsole aufzurufen.

  1. Wählen Sie im Navigationsfenster Ressourcen > Generische JMS-Provider aus.
  2. Wählen Sie im Inhaltsfenster den JMS-Provider aus, der die JMS-Verbindungs-Factory unterstützen soll.
  3. Wählen Sie im Inhaltsfenster in der Liste "Weitere Merkmale" die Option Generische JMS-Verbindungs-Factory aus.

Wenn Sie die Merkmale für eine JMS-Verbindungs-Factory anzeigen oder ändern möchten, wählen Sie den Namen der entsprechenden JMS-Verbindungs-Factory in der Liste aus.

Wenn Sie mit einer oder mehreren der aufgelisteten Verbindungs-Factorys arbeiten möchten, klicken Sie die Markierungsfeldern neben den Namen der gewünschten Objekte an und verwenden Sie dann die entsprechenden Knöpfe in der Adminstrationskonsole.
