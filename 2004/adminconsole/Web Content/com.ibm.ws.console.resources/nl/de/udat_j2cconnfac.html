J2C-Verbindungs-Factorys

Auf dieser Seite können Sie eine Verbindungs-Factory auswählen, die eine Gruppe von Konfigurationswerten für Verbindungen beinhaltet.

Anwendungskomponenten, wie z. B. Enterprise-Beans, besitzen Deskriptoren für Ressourcenreferenzen, die auf die Verbindungs-Factory und nicht auf den Ressourcenadapter verweisen. Die Verbindungs-Factory ist eine Komponente, die eine Liste der Konfigurationsmerkmale enthält. Zusätzlich zur willkürlich zusammengestellten Gruppe der Konfigurationsmerkmale, die vom Hersteller des Ressourcenadapters definiert wurden, sind verschiedene Standardkonfigurationsmerkmale vorhanden, die für die Verbindungs-Factory gelten. Diese Standardmerkmale werden vom Manager des J2C-Verbindungspools in der Laufzeit des Anwendungsservers verwendet und vom Ressourcenadaptercode, der vom Hersteller bereitgestellt wird, nicht erkannt.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > Ressourcenadapter >Ressourcenadapter > J2C-Verbindungs-Factorys.

Name
Gibt eine Liste der Anzeigenamen von Verbindungs-Factorys an.
Datentyp String
JNDI-Name
Gibt den JNDI-Namen (Java Naming and Directory Interface) für die Verbindungs-Factory an.
Datentyp String
Beschreibung
Eine Textbeschreibung dieser Verbindungs-Factory.
Datentyp String
Kategorie
Eine Zeichenfolge, die zur Klassifizierung oder Gruppierung dieser Verbindungs-Factory verwendet werden kann.
Datentyp String
