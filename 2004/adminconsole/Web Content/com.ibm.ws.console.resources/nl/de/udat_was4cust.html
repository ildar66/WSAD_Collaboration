Benutzerdefinierte Merkmale (Version 4)

Auf dieser Seite können Sie Merkmale für eine Datenquelle der Version 4.0 anzeigen.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > JDBC-Provider > JDBC-Provider > Datenquellen (Version 4) > Datenquelle > Benutzerdefinierte Merkmale.

Name
Gibt den Namen des benutzerdefinierten Merkmals an.

Vergewissern Sie sich, dass für die Datenquelle die set-Methode dieses Merkmals angegeben ist.

Datentyp String
Wert
Gibt den Wert des benutzerdefinierten Merkmals an.
Datentyp Integer
Beschreibung
Text zur Beschreibung von Grenzwerten bzw. klar definierten Werten für dieses Merkmal.
Datentyp String
Erforderlich
Gibt Merkmale an, die für diese Ressource erforderlich sind.
Datentyp String
