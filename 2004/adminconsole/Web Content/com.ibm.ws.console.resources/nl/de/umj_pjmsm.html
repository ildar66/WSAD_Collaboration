Einstellungen des WebSphere-MQ-JMS-Providers

In dieser Anzeige können Sie die Konfigurationsmerkmale des WebSphere-MQ-JMS-Providers anzeigen. Diese Merkmale sind nur gültig, wenn Sie WebSphere MQ als JMS-Provider über den internen, mit dem WebSphere Application Server installierten JMS-Provider installiert haben. Sie können diese Merkmale nicht ändern.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Ressourcen-> WebSphere-MQ-JMS-Provider.

Name
Der Name, mit dem der WebSphere-MQ-JMS-Provider für die Administration bezeichnet wird.
Datentyp String
Standardwert WebSphereMQJMSProvider
Beschreibung
Eine Beschreibung des JMS-Provider für die Administration in IBM WebSphere Application Server.
Datentyp String
Standardwert WebSphere-MQ-JMS-Provider
Klassenpfad
Der Java-Klassenpfad des JMS-Providers. Die Liste der Pfade bzw. Name von JAR-Dateien, die zusammen die Speicherposition der JMS-Provider-Klassen angeben.
Datentyp String
Standardwert $MQJMS_LIB_ROOT
Pfad der nativen Bibliothek
Der native Bibliothekspfad für den JMS-Provider. Ein optionaler Pfad zu den nativen Bibliotheken, die der JMS-Provider benötigt.
Datentyp String
Standardwert $MQJMS_LIB_ROOT

Als Pfad der nativen Bibliothek ist das Verzeichnis festgelegt, in dem WebSphere MQ Java installiert ist.
