Einstellungen für URL-Konfiguration

Auf dieser Seite können Sie URLs (Uniform Resource Locators) konfigurieren, die auf elektronisch verfügbare Ressourcen, wie z. B. Verzeichnisdateien auf einer Maschine in einem Netz, oder ein in einer Datenbank gespeichertes Dokument verweisen.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > URL-Provider > URL-Provider > URLs > URL.

Register "Konfiguration"

Name
Gibt den Anzeigenamen für die Ressource an.
JNDI-Name
Gibt den JNDI-Namen an.
Beschreibung
Gibt die Beschreibung der Ressource an.
Kategorie
Gibt eine Kategoriezeichenfolge an, die zur Klassifizierung oder Gruppierung der Ressource verwendet werden kann.
Spezifikation
Die Zeichenfolge, aus der ein URL gebildet wird.
