Ressourcenumgebungseinträge

Auf dieser Seite können Sie Ressourcenumgebungseinträge anzeigen.

Eine Umgebungsressource kann einen beliebigen Typ haben. Weitere Informationen zu resource-env-refs und Umgebungsressourcen enthält die EJB-2.0-Spezifikation.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > Ressourcenumgebungsprovider > Ressourcenumgebungsprovider > Ressourcenumgebungseinträge.

Name
Eine Textkennung, mit der dieser Ressourcenumgebungseintrag von anderen Einträgen unterschieden werden kann.

Sie können z. B. Meine Ressource als Namen verwenden.

Datentyp String
JNDI-Name
Die Zeichenfolge, die beim Suchen dieser Umgebungsressource mit JNDI verwendet werden soll.

An diese Zeichenfolge werden Deployment-Deskriptoren (für resource-env-ref) gebunden.

Datentyp String
Beschreibung
Gibt einen Text zur weiteren Kennzeichnung und Abgrenzung dieser Ressource an.
Datentyp String
Kategorie
Gibt eine Kategorie an, die Sie verwenden können, um Umgebungsressourcen in Übereinstimmung mit allgemeinen Merkmalen zu gruppieren.

Die Kategorie ist ausschließlich ein Administrationsmerkmal und hat keine Auswirkungen auf die Funktion der Umgebungsressource.

Datentyp String
