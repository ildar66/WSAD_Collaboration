Mail-Sitzungen

Auf dieser Seite können Sie Merkmale von Mail-Sitzungen, die unter dem übergeordneten Mail-Provider definiert sind, konfigurieren.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > Mail-Provider > Mail-Provider > Mail-Sitzungen.

Name
Gibt den Administrationsnamen des JavaMail-Sitzungsobjekts an.
JNDI-Name
Gibt den JNDI-Namen (Java Naming and Directory Interface) für die Ressource einschließlich aller Subkontexte an.

Dieser Name wird als Verbindung zwischen den Bindungsinformationen der Plattform für die Ressourcen, die im Deployment-Deskriptor der Clientanwendung definiert sind, und den Ressourcen verwendet, die tatsächlich von der Plattform an die JNDI gebunden wurden.

Beschreibung
Eine optionale Beschreibung der Datensätze für die Administration.
Kategorie
Gibt eine optionale Gruppe für das Klassifizieren oder das Zusammenfassen von Sitzungen an.
