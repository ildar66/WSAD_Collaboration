Benutzerdefinierte Merkmale

Auf dieser Seite können Sie die benutzerdefinierten Merkmale anzeigen.

Benutzerdefinierte Merkmale sind für die einzelnen Ressourcen einmalig, es können 0 oder mehr Merkmale erforderlich sein.

Anmerkung: Wenn Sie diese Seite aufgerufen haben, müssen Sie zunächst auf das Feld Erforderlich klicken, um in absteigender Reihenfolge zu sortieren. Alle erforderlichen Werte (Einstellung "true") werden dann am Anfang der Seite aufgelistet. Sie müssen diesen Prozess wiederholen, bis alle erforderlichen Merkmale gesetzt sind.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Ressourcen > JDBC-Provider > JDBC-Provider > Datenquellen > Datenquelle > Benutzerdefinierte Merkmale.

Name
Gibt den Namen des Merkmals an.

Sie müssen sicherstellen, dass für den Ressourcenadapter die Einstellung für diesen Namen angegeben ist.

Datentyp String
Wert
Gibt den Wert des Merkmals an.
Datentyp Integer
Beschreibung
Text zur Beschreibung von Grenzwerten bzw. klar definierten Werten für dieses Merkmal.
Datentyp String
Erforderlich
Gibt Merkmale an, die für diese Ressource erforderlich sind.
Datentyp String
