Collegamenti client dei servizi Web

Utilizzare questa pagina per specificare il nome del file WSDL (Web Service Description Language) e le mappature del tipo di porta predefinita per i servizi Web in un modulo.

Per visualizzare questa pagina, fare clic su Applicazioni >Applicazioni Enterprise > istanza_applicazione > Moduli Web > istanza_modulo >Collegamenti client dei servizi Web.

Per i moduli EJB, fare clic su Applicazioni > Applicazioni Enterprise > istanza_applicazione > Moduli EJB > istanza_modulo>Collegamenti client dei servizi Web
