Raccolta Programmi di valutazione ID sicuri

Utilizzare questa pagina per visualizzare un elenco di programmi di valutazione ID sicuri. Il programma di valutazione ID sicuri determina se rendere affidabile l'autorità di asserzione d'identità. Una volta reso affidabile l'ID, WebSphere Application Server emette credenziali proprie, utilizzate in una chiamata downstream per richiamare le risorse. Il programma di valutazione ID sicuri implementa l'interfaccia com.ibm.wsspi.wssecurity.id.TrustedIDEvaluator.

Per visualizzare questa pagina della console di gestione, completare le seguenti operazioni:

  1. Fare clic su Server > Server delle applicazioni > nome_server.
  2. In Servizi aggiuntivi, fare clic su Servizi Web: collegamenti predefiniti per la sicurezza dei servizi Web (WS-Security, Web Services Security) > Programmi di valutazione ID sicuri.

Fare clic su Nuovo per creare un nuovo programma di valutazione ID sicuri.

Fare clic su Elimina per eliminare un programma di valutazione ID sicuri.

Con questo pannello, completare le seguenti operazioni:

  1. Specificare un nome e una classe per i programmi di valutazione ID sicuri.
  2. Salvare le modifiche facendo clic su Salva nella parte superiore della console di gestione. Facendo clic su Salva, si torna al pannello principale della console di gestione.
  3. Tornare al pannello di raccolta programmi di valutazione ID sicuri per aggiornare il runtime di sicurezza dei servizi Web (WS-Security, Web Services Security) con le informazioni di collegamento predefinite, contenute nel file ws_security.xml.
  4. Fare clic su Aggiorna runtime. Verranno aggiornate anche le modifiche di configurazione apportate agli altri servizi Web nel runtime di sicurezza dei servizi Web (WS-Security, Web Services Security).
