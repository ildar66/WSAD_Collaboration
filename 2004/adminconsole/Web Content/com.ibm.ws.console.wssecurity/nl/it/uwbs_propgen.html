Raccolta di proprietà per la sicurezza dei servizi Web (WS-Security, Web Services Security)

Utilizzare questa pagina per visualizzare un elenco di proprietà aggiuntive per la configurazione.

I modi per visualizzare il pannello Raccolta di proprietà per la sicurezza dei servizi Web (WS-Security, Web Services Security) sono molteplici. Completare le seguenti procedure per visualizzare una di queste pagine della console di gestione:

  1. Fare clic su Server > Server delle applicazioni > nome_server.
  2. In Ulteriori proprietà, fare clic su Servizi Web: collegamenti predefiniti per la sicurezza dei servizi Web (WS-Security, Web Services Security) > Indicatori di posizione della chiave > nome_indicacatoreposizione_chiave.
  3. In Ulteriori proprietà, fare clic su Proprietà.
  4. Fare clic su Nuovo per creare una nuova proprietà.
  5. Fare clic su Elimina per eliminare una proprietà specificata precedentemente.
