Servizi Web: raccolta collegamenti sicurezza server

Utilizzare questa pagina per visualizzare un elenco di configurazioni di collegamento lato server per la sicurezza dei servizi Web (WS-Security, Web Services Security).

Per visualizzare questa pagina della console di gestione, completare le seguenti operazioni:

  1. Fare clic su Applicazioni enterprise > nome_applicazione.
  2. In Elementi correlati, fare clic su Modulo Web > Nome_file_URI> Servizi Web: collegamenti sicurezza server.

Porta
Specifica la porta tramite cui i messaggi vengono ricevuti dal mittente della richiesta
Servizio Web
Specifica il nome del servizio Web.
Collegamento ricevente richiesta
Specifica la configurazione di collegamento utilizzata per ricevere i messaggi di richiesta dal mittente della richiesta stessa.

Fare clic su Modifica per configurare le informazioni sulla firma e sulla codifica e visualizzare un elenco di ancoraggi sicuri, certificati intermedi nel file di memorizzazione delle chiavi, indicatori di posizione della chiave, programmi di valutazione ID sicuri e mappature di login.

Le informazioni sul collegamento (del ricevente della richiesta) specificate per il server devono corrispondere alle informazioni sul collegamento (del mittente della richiesta) specificate per il client.

Collegamento mittente risposta
Specifica la configurazione di collegamento utilizzata per inviare i messaggi di richiesta al ricevente della risposta.

Fare clic su Modifica per configurare le informazioni sulla firma e sulla codifica e per visualizzare un elenco di oggetti indicatori di posizione della chiave per il mittente della risposta.

Le informazioni sul collegamento (del mittente della risposta) specificate per il server devono corrispondere alle informazioni sul collegamento (del ricevente della risposta) specificate per il client.
