Raccolta chiavi

Utilizzare questa pagina per visualizzare un elenco di nomi logici mappati su un alias di chiavi nel file di memorizzazione delle chiavi.

Per visualizzare questa pagina della console di gestione, completare le seguenti operazioni:

  1. Fare clic su Server > Server delle applicazioni > nome_server.
  2. In Ulteriori proprietà, fare clic su Servizi Web: collegamenti predefiniti per la sicurezza dei servizi Web (WS-Security, Web Services Security) > Indicatori di posizione della chiave > nome_indicacatoreposizione_chiave.
  3. In Ulteriori proprietà, fare clic su Chiavi.
  4. Fare clic su Nuovo per creare un nuovo oggetto chiave nel file di memorizzazione delle chiavi.
  5. Fare clic su Elimina per eliminare una mappatura di un oggetto chiave all'interno del file di memorizzazione delle chiavi.

Nome chiave
Specifica il nome dell'oggetto chiave trovato nel file di memorizzazione delle chiavi.
Alias chiave
Specifica un alias per l'oggetto chiave.

L'alias viene utilizzato quando l'indicatore di posizione della chiave ricerca gli oggetti chiave nel file di memorizzazione delle chiavi.
