Impostazioni di configurazione Raccolta archivio certificati

Utilizzare questa pagina per specificare il nome e il provider di un archivio di certificati.

Per visualizzare questa pagina della console di gestione, completare le seguenti operazioni:

  1. Fare clic su Server > Server delle applicazioni > nome_server.
  2. In Ulteriori proprietà, fare clic su Servizi Web: collegamenti predefiniti per la sicurezza dei servizi Web (WS-Security, Web Services Security) > Raccolta archivio certificati > Nuovo.

Nome archivio certificati
Specifica il nome dell'archivio di certificati. Il collegamento dell'applicazione utilizza il nome dell'archivio di certificati per indicare un collegamento predefinito.
Provider archivio certificati
Specifica il provider per l'implementazione dell'archivio di certificati.
Tipo dati Stringa
Predefinito IBM CertPath
