Impostazioni di configurazione del Programma di valutazione ID sicuri

Utilizzare queste informazioni per configurare i programmi di valutazione ID sicuri.

Per visualizzare questa pagina della console di gestione, completare le seguenti operazioni:

  1. Fare clic su Server > Server delle applicazioni > nome_server.
  2. In Ulteriori proprietà, fare clic su Servizi Web: collegamenti predefiniti per la sicurezza dei servizi Web (WS-Security, Web Services Security) > Programmi di valutazione ID sicuri > Nuovo.

È necessario specificare le proprietà nome e valore per il programma di valutazione ID sicuri predefinito per creare un elenco di ID sicuri per la valutazione.

Nome Programma di valutazione ID sicuri
Specifica il nome univoco utilizzato dal collegamento dell'applicazione per indicare un programma di valutazione ID sicuri definito nel collegamento predefinito.

È necessario specificare il nome del programma di valutazione ID sicuri nella forma trustedId_n, dove _n indica il numero intero da 0 a n.

Nome classe Programma di valutazione ID sicuri
Specifica il nome classe del programma di valutazione ID sicuri.
Predefinito com.ibm.wsspi.wssecurity.id.TrustedIDEvaluatorImpl
