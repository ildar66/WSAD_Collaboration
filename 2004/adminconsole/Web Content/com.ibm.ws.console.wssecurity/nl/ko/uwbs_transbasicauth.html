HTTP 기본 인증 콜렉션

이 페이지에서는 이 포트에 대한 전송 레벨 기본 인증 보안용 사용자 ID를 지정할 수 있습니다. 웹 서비스가 다른 웹 서비스에 대한 클라이언트일 때 이 구성을 사용할 수 있습니다.

전송 레벨 보안을 사용하여 기본 인증이 사용 가능하도록 설정할 수 있습니다. 전송-레벨 보안은 메시지 레벨 보안에서 사용 가능 또는 사용 불가능 상태로 만들 수 있습니다. 전송 레벨 보안은 최소 보안을 제공하기 때문에, 웹 서비스 응용프로그램에 보안이 필수적인 경우 메시지 레벨 보안을 사용하십시오.

이 관리 콘솔 페이지를 보려면, 다음 단계를 완료하십시오.

  1. 응용프로그램 > 엔터프라이즈 응용프로그램 > application_name을 클릭하십시오.
  2. 관련 항목 아래에서 웹 모듈 > URI_file_name> 웹 서비스: 클라이언트 보안 바인딩을 클릭하십시오.
  3. HTTP 기본 인증 아래에서 편집을 클릭하십시오.

다음 단계를 완료하여 HTTP SSL 기본 인증을 지정할 수도 있습니다.

  1. 응용프로그램 > 엔터프라이즈 응용프로그램 > application_name을 클릭하십시오.
  2. 관련 항목 아래에서 웹 모듈 > URI_file_name> 웹 서비스: 클라이언트 보안 바인딩을 클릭하십시오.
  3. HTTP SSL 구성 아래에서 편집을 누르십시오.

기본 인증 ID
이 포트의 HTTP 기본 인증에 대한 사용자 ID를 지정합니다.
기본 인증 암호
이 포트의 HTTP 기본 인증에 대한 암호를 지정합니다.
