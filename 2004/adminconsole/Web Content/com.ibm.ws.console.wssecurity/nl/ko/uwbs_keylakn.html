키 구성 설정

키 저장 파일에 논리 이름에서 키 별명으로의 맵핑을 저장하려면 이 페이지를 참조하십시오.

이 관리 콘솔 페이지를 보려면 다음 단계를 완료하십시오.

  1. 서버 > Application Servers > server_name을 클릭하십시오.
  2. 추가 등록 정보 아래에서 웹 서비스: 웹 서비스 보안을 위한 기본 바인딩 > 키 위치 지정자 > key_locator_name을 클릭하십시오.
  3. 추가 등록 정보 아래에서, 키 > 새로 작성을 클릭하십시오.

키 이름
키 오브젝트의 이름을 지정합니다. 이 이름은 키 위치 지정자가 키 저장 파일 내에서 키를 찾는 데 사용됩니다.
키 별명
키 저장 파일에 포함된 키 오브젝트의 별명을 지정합니다.
키 암호
키 저장 파일 내에 키 오브젝트에 액세스하는 데 필요한 암호를 지정합니다.
