키 위치 지정자 콜렉션

이 페이지를 참조하여 사용 가능한 키 위치 지정자의 목록을 보십시오. 키 위치 지정자는 디지털 서명 및 암호화에 필요한 키를 식별합니다. 키 위치 지정자는 com.ibm.wsspi.wssecurity.config.KeyLocator 인터페이스를 구현해야 합니다. 두 가지 기본 구현은 com.ibm.wsspi.wssecurity.config.WSldKeyStoreMapKeyLocator 및 com.ibm.wsspi.wssecurity.config.KeyStoreKeyLocator입니다.

이 관리 콘솔 페이지를 보려면 다음 단계를 완료하십시오.

  1. 서버 > Application Servers > server_name을 클릭하십시오.
  2. 추가 등록 정보 아래에서 웹 서비스: 웹 서비스 보안을 위한 기본 바인딩 > 키 위치 지정자를 클릭하십시오.
  3. 새로 작성을 클릭하여 키 위치 지정자를 작성하십시오. 삭제를 클릭하여 키 위치 지정자를 삭제하십시오.

이 패널을 사용하여 다음 단계를 완료하십시오.

  1. 패널에 키 위치 지정자 이름 및 키 위치 지정자 클래스 이름을 지정하십시오.
  2. 관리 콘솔의 맨 위에 있는 저장을 클릭하여 변경사항을 저장하십시오. 저장을 클릭하면 관리 콘솔 홈 패널로 되돌아 갑니다.
  3. 변경사항을 저장한 후, 키 위치 지정자 콜렉션 패널로 돌아가 ws_security.xml 파일에 있는 기본 바인딩 정보를 사용하여 웹 서비스 보안 런타임을 갱신해야 합니다.
  4. 웹 서비스 보안 런타임을 갱신하려면 런타임 갱신을 클릭하십시오. 런타임 갱신을 클릭하면, 다른 웹 서비스에 작성한 구성 변경사항도 웹 서비스 보안 런타임에 갱신됩니다.
  5. 일단 키 위치 지정자를 정의하고나면, 추가 등록 정보 아래에서 키 위치 지정자 이름을 클릭하여 추가 등록 정보와 키를 지정하십시오.
