키 콜렉션

키 저장 파일의 키 별명에 맵핑된 논리 이름 목록을 보려면 이 페이지를 참조하십시오.

이 관리 콘솔 페이지를 보려면 다음 단계를 완료하십시오.

  1. 서버 > Application Servers > server_name을 클릭하십시오.
  2. 추가 등록 정보 아래에서 웹 서비스: 웹 서비스 보안을 위한 기본 바인딩 > 키 위치 지정자 > key_locator_name을 클릭하십시오.
  3. 추가 등록 정보 아래에서, 키를 클릭하십시오.
  4. 키 저장 파일에 키 오브젝트를 새로 작성하려면 새로 작성을 클릭하십시오.
  5. 키 저장 파일 내의 키 오브젝트 맵핑을 삭제하려면 삭제를 클릭하십시오.

키 이름
키 저장 파일에 있는 키 오브젝트 이름을 지정합니다.
키 별명
키 오브젝트의 별명을 지정합니다.

별명은 키 저장 파일에서 키 위치 지정자가 키 오브젝트를 검색할 때 사용됩니다.
