신뢰 ID 평가자 구성 설정

이 정보는 신뢰 ID 평가자를 구성하는 데 사용합니다.

이 관리 콘솔 페이지를 보려면, 다음 단계를 완료하십시오.

  1. 서버 > Application Servers > server_name을 클릭하십시오.
  2. 추가 서비스 등록 정보에서 웹 서비스: 웹 서비스 보안을 위한 기본 바인딩 > 신뢰 ID 평가자 > 새로 작성을 클릭하십시오.

기본 신뢰 ID 평가자에 대한 이름 및 값 등록 정보를 지정하여 평가할 신뢰 ID 목록을 작성해야 합니다.

신뢰 ID 평가자 이름
응용프로그램 바인딩이 기본 바인딩에 정의된 신뢰 ID 평가자를 참조하기 위해 사용하는 고유 이름을 지정합니다.

신뢰 ID 평가자의 이름을 trustedId_n(여기서, _n은 0 ~ n의 정수) 양식으로 지정해야 합니다.

신뢰 ID 평가자 클래스 이름
신뢰 ID 평가자의 클래스 이름을 지정합니다.
