HTTP 基本鑑別集合

請利用這個頁面來指定這個埠的傳輸層基本鑑別安全的使用者 ID 和密碼。 當 Web 服務是另一個 Web 服務的用戶端時，您可以使用這個配置。

您可以利用傳輸層安全來啟用基本鑑別。 您可以在訊息層次安全之外，獨立啟用或停用傳輸層安全。 由於傳輸層安全只提供最起碼的安全，因此，當 Web 服務應用程式很需要安全時，請使用訊息層安全。

如果要檢視這個管理主控台頁面，請完成下列步驟：

  1. 按一下應用程式 > 企業應用程式 > application_name。
  2. 在「相關項目」之下，按一下 Web 模組 > URI_file_name > Web 服務：用戶端安全連結。
  3. 在「HTTP 基本鑑別」之下，按一下編輯。

請完成下列步驟來指定 HTTP SSL 基本鑑別：

  1. 按一下應用程式 > 企業應用程式 > application_name。
  2. 在「相關項目」之下，按一下 Web 模組 > URI_file_name > Web 服務：用戶端安全連結。
  3. 在「HTTP SSL 配置」之下，按一下編輯。

基本鑑別 ID
指定這個埠的 HTTP 基本鑑別的使用者 ID。
基礎鑑別密碼
指定這個埠的 HTTP 基本鑑別的密碼。
