HTTP SSL 配置集合

請利用這個頁面來配置傳輸層 Secure Socket Layer (SSL) 安全。 當 Web 服務是另一個 Web 服務的用戶端時，您可以使用這個配置。

您可以利用傳輸層安全來啟用 HTTP SSL（或 HTTPS）。 您可以在訊息層次安全之外，獨立啟用或停用傳輸層安全。 由於傳輸層安全只提供最起碼的安全，因此，當 Web 服務應用程式很需要安全時，請使用訊息層安全。

如果要檢視這個管理主控台頁面，請完成下列步驟：

  1. 按一下應用程式 > 企業應用程式 > application_name。
  2. 在「相關項目」之下，按一下 Web 模組 > URI_file_name > Web 服務：用戶端安全連結。
  3. 在「HTTP SSL 配置」之下，按一下編輯。

請完成下列步驟來指定 HTTP SSL 基本鑑別：

  1. 按一下應用程式 > 企業應用程式 > application_name。
  2. 在「相關項目」之下，按一下 Web 模組 > URI_file_name > Web 服務：用戶端安全連結。
  3. 在 HTTP 基本配置之下，按一下編輯。

啟用 HTTP SSL
請選取這個選項來啟用這個埠的 HTTP 傳輸的安全 Socket 通訊。 啟用之後，WebSphere Application Server 會使用「HTTP SSL 配置」設定。
HTTP SSL 配置
指定這個埠的 HTTP 傳輸要用的 SSL 配置別名。

如果您選取已啟用 HTTP SSL，就會使用這個選項。 SSL 別名定義在您可以按一下安全 > SSL 來配置的 Secure Socket Layer 配置儲存庫中。
