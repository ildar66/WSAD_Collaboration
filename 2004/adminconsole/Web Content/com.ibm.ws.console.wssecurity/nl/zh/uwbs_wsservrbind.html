Web 服务：服务器安全性绑定集合

使用此页面查看 Web 服务安全性的服务器端绑定配置列表。

要查看此管理控制台页面，完成以下步骤：

  1. 单击企业应用程序 > application_name。
  2. 在“相关项”下，单击 Web 模块 > URI_file_name > Web 服务：客户机安全性绑定。

端口
指定从请求发送程序接收消息的端口。
Web 服务
指定 Web 服务的名称。
请求接收程序绑定
指定用于从请求发送程序接收请求消息的绑定配置。

单击编辑配置签名信息和加密信息，并查看信任锚、集合证书存储中的中间证书、密钥定位器、可信标识求值程序和登录映射的列表。

为服务器指定的（请求接收程序的）绑定信息必须匹配为客户机指定的（请求发送程序的）绑定信息。

响应发送程序绑定
指定用于发送请求消息到响应接收程序的绑定配置。

单击编辑配置签名和加密信息，并查看响应发送程序的密钥定位器对象列表。

为服务器指定的（响应发送程序的）绑定信息必须匹配为客户机指定的（响应接收程序的）绑定信息。
