HTTP SSL 配置集合

使用此页面配置传输级别安全套接字层（SSL）安全性。当 Web 服务是另一个 Web 服务的客户机时，您可以使用此配置。

您可以使用传输级别安全性以启用 HTTP SSL（或 HTTPS）。可独立于消息级别安全性启用或禁用传输级别安全性。因为传输级别安全性提供最小的安全性，因此当安全性对于 Web 服务应用程序是必不可少的时候，使用消息级别安全性。

要查看此管理控制台页面，完成以下步骤：

  1. 单击应用程序 > 企业应用程序 > application_name。
  2. 在“相关项”下，单击 Web 模块 > URI_file_name > Web 服务：客户机安全性绑定。
  3. 在“HTTP SSL 配置”中，单击编辑。

您也可以通过完成以下步骤指定 HTTP SSL 基本认证：

  1. 单击应用程序 > 企业应用程序 > application_name。
  2. 在“相关项”下，单击 Web 模块 > URI_file_name > Web 服务：客户机安全性绑定。
  3. 在 HTTP 基本配置中，单击编辑。

启用 HTTP SSL
选择此选项以启用此端口的 HTTP 传输的安全套接字通信。当启用时，WebSphere Application Server 使用 HTTP SSL 配置设置。
HTTP SSL 配置
指定用于与此端口的 HTTP 传输一起使用的 SSL 配置的别名。

如果您选择启用 HTTP SSL，则使用此选项。SSL 别名在安全套接字层配置指令表中定义，您可以通过单击安全性 > SSL 来配置该指令表。
