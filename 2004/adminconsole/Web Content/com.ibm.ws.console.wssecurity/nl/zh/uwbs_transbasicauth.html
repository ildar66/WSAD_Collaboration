HTTP 基本认证集合

使用此页面指定此端口的传输级别基本认证安全性的用户标识和密码。当 Web 服务是另一个 Web 服务的客户机时，您可以使用此配置。

您可以使用传输级别安全性以启用基本认证。可独立于消息级别安全性启用或禁用传输级别安全性。因为传输级别安全性提供最小的安全性，因此当安全性对于 Web 服务应用程序是必不可少的时候，使用消息级别安全性。

要查看此管理控制台页面，完成以下步骤：

  1. 单击应用程序 > 企业应用程序 > application_name。
  2. 在“相关项”下，单击 Web 模块 > URI_file_name > Web 服务：客户机安全性绑定。
  3. 在“HTTP 基本认证”中，单击编辑。

您也可以通过完成以下步骤指定 HTTP SSL 基本认证：

  1. 单击应用程序 > 企业应用程序 > application_name。
  2. 在“相关项”下，单击 Web 模块 > URI_file_name > Web 服务：客户机安全性绑定。
  3. 在“HTTP SSL 配置”中，单击编辑。

基本认证标识
指定用于此端口的 HTTP 基本认证的用户标识。
基本认证密码
指定用于此端口的 HTTP 基本认证的密码。
