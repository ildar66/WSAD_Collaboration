管理控制台按钮

下列按钮选项可在管理控制台的各个页面中使用，这取决于您已启用产品的哪些功能部件。

  • 放弃。放弃仍处于未准备好状态的事务。撤销事务完成的所有操作。
  • 添加。把所选的或输入的项添加到列表，或者产生一个把项添加到列表的对话框。
  • 应用。在不退出页面的情况下，将您的更改保存到页面中。
  • 上一页。显示前一个页面或序列中的项。注意，管理控制台不支持使用浏览器的“后退”和“前进”按钮。使用那些按钮会导致间歇性问题。改为使用管理控制台面板上的“后退”或“取消”按钮。
  • 浏览。打开使您能够在系统上查看文件的对话框。
  • 取消。退出当前页面或对话框时，废弃未保存的更改。注意，管理控制台不支持使用浏览器的“后退”和“前进”按钮。使用那些按钮会导致间歇性问题。改为使用管理控制台面板上的“取消”按钮。
  • 更改。在安全性上下文中，让您在用户注册表中搜索一个用户标识，以该用户标识运行应。在容器属性的上下文中，让您更改容器正在使用的数据源。
  • 清除。清除您的更改并恢复最近保存的值。
  • 清除选择。清除此选项卡式页面上的表中的任何所选单元。
  • 关闭。退出对话框。
  • 提交。释放准备好的事务所持有的所有锁并强制此事务提交。
  • 复制。创建所选的应用程序服务器的副本。
  • 创建。把对所有选项卡式页面的更改保存到对话框并退出该对话框。
  • 删除。除去所选的实例。
  • 详细信息。显示有关事务的详细信息。
  • 完成。把对所有选项卡式页面的更改保存到对话框并退出该对话框。
  • 向下。在列表中向下移动。
  • 转储。激活已跟踪的应用程序服务器的转储。
  • 编辑。让您编辑列表中所选的项，或产生一个对话框以编辑该项。
  • 导出。访问页面以导出企业应用程序的 EAR 文件。
  • 导出 DDL。访问页面以导出企业应用程序的 DDL 文件。
  • 导出密钥。将 LTPA 密钥导出到其它域中。
  • 过滤器。产生一个用于指定资源的对话框，以在此选项卡式页面上的表中查看。
  • 完成。强制事务完成，无论其结果是否已报告到所有参与的应用程序。
  • 第一个。显示一系列记录中的第一个记录。
  • 生成密钥。生成新的 LTPA 密钥。当使用 LTPA 认证机制第一次打开安全性，采用在面板中输入的密码自动生成 LTPA 密钥。如果需要生成新的密钥，则在打开安全性的情况下备份服务器后使用此按钮。单击此按钮生成密钥并将这些密钥传递到所有活动的服务器（单元、节点和应用程序服务器）。新密钥可用于加密和解密 LTPA 令牌。在控制台任务栏上单击保存，将新密钥和密码保存到库中。
  • 立即停止。停止服务器，但忽略正常服务器停顿进程，此进程将允许在关闭整个服务器进程之前完成发送中的请求。此关闭方式比正常服务器停止进程更快，但某些应用程序客户机可能会接收到异常。
  • 导入密钥。从其它域导入新的 LTPA 密钥。要支持 WebSphere 中跨多个 WebSphere 域（单元）的单次注册（SSO），LTPA 密钥和密码应该在这些域中共享。将密钥从一个单元导出到文件后，单击此按钮将密钥导入所有活动的服务器（单元、节点和应用程序服务器）。新密钥可用于加密和解密 LTPA 令牌。在控制台任务栏上单击保存，将新密钥和密码保存到库中。
  • 安装。显示“准备应用程序安装”页面，您将使用此页面把应用程序或 EJB 或 Web 组件部署到应用程序服务器上。
  • 安装 RAR。此安装 RAR 按钮打开一个用于安装 JCA 连接器并为其创建一个资源适配器的对话框。
  • 管理事务。显示运行在服务器上的活动事务列表。因为事务资源不可用，您可以强制地完成已停止处理的任何事务。
  • 修改。打开用于更改规范的对话框。
  • 移动。将所选的应用程序服务器移动到管理单元中的另一个位置。提示时，指定目标位置。
  • 新建。显示您用于定义新实例的页面。例如，在“应用程序服务器”页面上单击新建显示您可以配置新的应用程序服务器的页面。
  • 下一步。显示序列中的下一个页面、框架或项。
  • 确定。保存您的更改并退出此页面。
  • Ping。尝试联系所选的应用程序服务器。
  • 上一步。显示序列中的前一个页面、框架或项。
  • 退出。退出对话框，废弃任何未保存的更改。
  • 刷新。刷新当前此选项卡式页面上列出的实例的数据视图。
  • 除去。删除所选的项。
  • 复位。在选项卡或页面上清除您的更改并恢复最新保存的值。
  • 重新启动。停止所选的对象并再次启动它们。
  • 检索新的。检索新记录。
  • 保存。把本地配置的更改保存到主配置中。
  • 选择。对于资源分析，让您选择监控资源的作用域。
  • 设置。把您的更改保存到对话框中的设置。
  • 设置（Settings）。显示编辑与 servlet 相关的资源设置的对 话框。
  • 使用中的设置。显示一个对话框，该对话框显示现在正在使用的设置。
  • 启动。在应用程序服务器的上下文中，启动所选的应用程序服务器。在数据集合的上下文中，开始对选项卡式页面上的表的数据收集。
  • 停止。在服务器组件（例如，应用程序服务器）的上下文中，停止所选的服务器组件。在数据集合的上下文中，停止对选项卡式页面上的表的数据收集。
  • 卸载。从 WebSphere Application Server 配置库中删除部署的应用程序。并从文件系统中删除应用程序二进制文件。
  • 更新。用已更新的应用程序替换部署在服务器上的应用程序。作为更新的一部分，您可能需要在 “准备应用程序安装和更新应用程序”页面上完成步骤。
  • 更新资源列表。更新表上的数据。发现并将新实例添加到表中。
  • 查看。打开关于文件的对话框。
