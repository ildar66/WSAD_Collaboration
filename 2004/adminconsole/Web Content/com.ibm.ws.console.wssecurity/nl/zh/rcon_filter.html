管理控制台过滤器设置

使用过滤器设置指定如何过滤集合视图中显示的过滤条目。

选择要过滤的列，然后输入过滤标准。

要过滤的列
使用下拉列表选择要过滤的列。当您应用该过滤器时，相应地修改集合表。

例如，如果您计划输入过滤应用程序服务器名所遵守的标准，则选择应用程序服务器。

过滤标准
在下拉列表旁的字段中，输入在集合条目的名称中必须找到的字符串，以限定要在该集合表中出现的条目。字符串可以包含 %、* 或 ? 符号作为通配符。例如，输入 *App* 以查找名称中包含字符串 App 的任何应用程序服务器。

使用 \ 作为每个字符 ( ) ^ * % { } \ + $ 的前缀，以便执行搜索的正则表达式引擎能正确匹配所提供的搜索标准。例如，要搜索提供程序名中包含 (XA) 的所有 JDBC 提供程序，如下指定：

*\(XA\)
