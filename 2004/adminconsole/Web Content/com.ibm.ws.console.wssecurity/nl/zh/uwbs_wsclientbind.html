Web 服务：客户机安全性绑定集合

使用此页面查看 Web 服务安全性的客户机端绑定配置列表。当 Web 服务是另一个 Web 服务的客户机时，使用这些绑定。

要查看此管理控制台页面，完成以下步骤：

  1. 单击企业应用程序 > application_name。
  2. 在“相关项”下，单击 Web 模块 > URI_file_name > Web 服务：客户机安全性绑定。

端口
指定用于从服务器发送和接收消息的端口。
Web 服务
指定 Web 服务的名称。
请求发送程序绑定
指定用于发送请求消息到请求接收程序的绑定配置。

单击编辑配置请求发送方的签名信息、加密信息和登录绑定，并查看密钥存储中密钥定位器的列表。

为客户机指定的请求发送方的绑定信息必须匹配为服务器指定的请求接收方的绑定信息。

响应接收程序绑定
指定用于从响应发送程序接收响应消息的绑定配置。

单击编辑配置签名和加密信息，并查看信任锚、集合证书存储中所找到的中间证书，以及响应接收程序的密钥定位器对象列表。

为客户机指定的响应接收方的绑定信息必须匹配为服务器指定的响应发送方的绑定信息。

HTTP 基本认证
指定用于带有 HTTP 传输级别基本认证的此端口的用户标识和密码。您可以独立于消息级别安全性启用传输级别认证安全性。

单击编辑以配置传输级别认证的基本认证标识和密码。

HTTP SSL 配置
启用和配置此端口的传输级别 SSL 安全性。您可以独立于消息级别安全性启用传输级别 SSL 安全性。

单击编辑以指定此端口的传输级别 HTTP SSL 配置的设置。
