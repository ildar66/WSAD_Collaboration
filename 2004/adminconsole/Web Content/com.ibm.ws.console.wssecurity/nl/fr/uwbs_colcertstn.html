Paramètres de configuration du magasin de certificats de collection

Cette page permet d'indiquer le nom et le fournisseur d'un magasin de certificats.

Pour afficher cette page de la console d'administration, suivez la procédure ci-après.

  1. Cliquez sur Serveurs > Serveurs d'applications > nom_serveur.
  2. Sous Propriétés supplémentaires, cliquez sur Services Web : Liaisons par défaut pour la sécurité des services Web > Magasin de certificats de collection > Nouveau.

Nom du magasin de certificats
Indique le nom du magasin de certificats. La liaison d'application utilise le nom du magasin de certificats pour référencer une liaison prédéfinie.
Fournisseur du magasin de certificats
Indique le fournisseur de l'implémentation du magasin de certificats.
Type de données Chaîne
Valeur par défaut IBM CertPath
