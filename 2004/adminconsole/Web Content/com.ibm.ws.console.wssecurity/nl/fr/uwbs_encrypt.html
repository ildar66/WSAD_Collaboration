Collection Informations relatives au chiffrement

Cette page permet d'indiquer la configuration des paramètres de chiffrement et de déchiffrement. Cette configuration permet de chiffrer et de déchiffrer des parties du message, y compris le corps du message et le jeton du nom d'utilisateur.

Pour afficher cette page de la console d'administration, suivez la procédure ci-après.

  1. Cliquez sur Applications d'entreprise > nom_application.
  2. Dans la section Articles liés, cliquez sur Module Web.
  3. Dans la section Propriétés supplémentaires, cliquez sur Services Web : Liaisons de sécurité du serveur.
  4. Dans la section Liaison de récepteur de demande, cliquez sur Editer > Informations de chiffrement.
  5. Cliquez sur Nouveau pour créer une méthode de chiffrement.
  6. Cliquez sur Supprimer pour supprimer une méthode de chiffrement.
