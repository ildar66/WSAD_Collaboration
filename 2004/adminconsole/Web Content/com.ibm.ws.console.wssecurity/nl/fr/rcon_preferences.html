Paramètres des préférences de la console d'administration

Utilisez les paramètres de préférence pour indiquer la façon dont vous aimeriez voir apparaître les informations à l'écran, sur une page de la console d'administration.

Nombre maximum de lignes
Nombre maximal de lignes à afficher par page dans le cas d'une longue liste.
Historique de filtrage
Indique si le même critère de filtrage doit être utilisé pour afficher cette page lors de votre prochaine consultation de la page.

Cochez la case Conserver les critères de filtrage pour conserver le dernier critère de filtrage saisi. Lorsque vous retournez à la page ultérieurement, le critère de filtrage retenu contrôle la collection d'applications qui s'affiche dans la table.
