Paramètres de filtrage de la console d'administration

Utilisez les paramètres de filtrage pour indiquer comment filtrer les entrées qui s'affichent en série.

Sélectionnez la colonne à filtrer puis saisissez les critères de filtrage.

Colonne à filtrer
Utilisez la liste déroulante pour sélectionner la colonne à filtrer. Lorsque vous appliquez le filtre, la tableau des entrées est modifié en fonction.

Par exemple, sélectionnez Serveurs d'applications si vous voulez saisir des critères de filtrage par noms de serveurs d'applications.

Critères de filtrage
Dans la zone qui se trouve à côté de la liste déroulante, saisissez une chaîne qui doit se trouver dans le nom d'une des entrées pour déterminer l'entrée devant apparaître dans le tableau des entrées. La chaîne peut contenir les symboles %, *, ou ? comme caractères génériques. Par exemple, entrez *App* pour rechercher les serveurs d'applications dont le nom contient la chaîne App.

Faites précéder les caractères ( ) ^ * % { } \ + $ du signe \ afin que le moteur d'expression standard utilisé lors de la recherche soit conforme aux critères de recherche fournis. Par exemple, pour rechercher tous les fournisseurs JDBC dont le nom contient (XA), indiquez l'entrée suivante :

*\(XA\)
