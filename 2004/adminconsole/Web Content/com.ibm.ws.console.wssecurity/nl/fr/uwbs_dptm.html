Collection de définitions de mappage de port par défaut

Cette page permet d'afficher et de gérer le mappage des types de port par défaut d'un service Web.

Pour afficher cette page de la console d'administration, cliquez sur Applications >Application d'entreprise > instance_application > Modules Web > instance_module > Liaisons des clients de services Web > Editer > instance_port_par_défaut.

Pour les modules EJB, cliquez sur Applications >Application d'entreprise > instance_application > Modules EJB > instance_module > Liaisons des clients de services Web > Editer > instance_port_par_défaut.

Indiquez le port par défaut d'un service lorsqu'un type de port spécifique est demandé. Le type de port est décrit par son nom local et son espace de nom . Une méthode getPort ne spécifiant que le type de port extrait le port nommé par l'espace de nom et le nom local du port par défaut.
