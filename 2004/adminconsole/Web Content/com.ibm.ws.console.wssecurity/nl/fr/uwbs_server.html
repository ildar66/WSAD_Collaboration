Portée d'implémentation des services Web

Cette page permet d'afficher et de gérer la portée des ports d'un bean Service Web.

Pour afficher cette page de la console d'administration, cliquez sur Applications >Applications d'entreprise > instance_application > Modules Web > instance_module >Portée d'implémentation des services Web.

Port
Indique un nom de port pour un service Web. Un module peut contenir un ou plusieurs services, qui peuvent contenir chacun un ou plusieurs ports.
