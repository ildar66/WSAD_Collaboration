Liaisons des clients de services Web

Cette page permet de spécifier les mappages des noms de fichier WSDL (Web Service Description Language) aux types de port par défaut dans un module.

Pour afficher cette page, cliquez sur Applications >Applications d'entreprises > instance_application > Modules Web > instance_module > Liaisons du client de services Web.

Pour les modules d'EJB, cliquez sur Applications >Applications d'entreprise > instance_application > Modules EJB > instance_module >Liaisons du client de services Web
