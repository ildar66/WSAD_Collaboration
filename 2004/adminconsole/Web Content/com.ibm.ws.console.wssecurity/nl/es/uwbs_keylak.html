Grupo de claves

Utilice esta página para ver una lista de nombres lógicos correlacionados con un alias de clave en el archivo de almacén de claves.

Para ver esta página de la consola administrativa, realice los pasos siguientes:

  1. Pulse Servidores > Servidores de aplicaciones > nombre_servidor.
  2. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security > Localizadores de claves nombre_localizador_claves.
  3. En Propiedades adicionales, pulse Claves.
  4. Pulse Nuevo para crear un objeto de clave nuevo en el almacén de claves.
  5. Pulse Suprimir para suprimir una correlación de un objeto de clave en el archivo de almacén de claves.

Nombre de clave
Especifica el nombre del objeto de clave del archivo de almacén de claves.
Alias de clave
Especifica un alias para el objeto de clave.

El alias se utiliza cuando el localizador de claves busca los objetos de claves en el almacén de claves.
