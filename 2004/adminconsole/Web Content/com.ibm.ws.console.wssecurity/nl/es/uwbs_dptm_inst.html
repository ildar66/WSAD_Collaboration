Valores de propiedades de correlaciones de tipos de puertos por omisión

Utilice esta página para ver y gestionar una correlación de tipos de puertos por omisión para un servicio Web.

Para ver esta página de la consola administrativas, pulse Aplicaciones > Aplicaciones de empresa > instancia_aplicación > Módulo Web > instancia_módulo>Enlaces del cliente de servicios Web > Editar> instancia_puerto_por_omisión.

Para módulos Web, pulse Aplicaciones > Aplicaciones de empresa > instancia_aplicación > Módulos EJB > instancia_módulo>Enlaces del cliente de servicios Web > Editar> instancia_puerto_por_omisión.

Especifique el puerto por omisión de un servicio cuando se solicite un tipo de puerto determinado. El tipo de puerto se describe mediante su nombre local y su espacio de nombre. A un método getPort que solamente especifique el tipo de puerto se le asigna el puerto cuyo nombre sea el nombre local del puerto y el espacio de nombres.

Pestaña Configuración

Especifica el espacio de nombres del tipo de puerto que se ha de correlacionar.
