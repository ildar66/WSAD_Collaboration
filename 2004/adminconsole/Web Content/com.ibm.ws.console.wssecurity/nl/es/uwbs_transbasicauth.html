Colección de autenticación básica HTTP

Utilice esta página para especificar un ID de usuario y una contraseña para la seguridad de autenticación básica a nivel de transporte para este puerto. Puede utilizar esta configuración cuando un servicio Web es un cliente de otro servicio Web.

Puede utilizar la seguridad a nivel de transporte para habilitar la autenticación básica. La seguridad a nivel de transporte puede habilitarse o inhabilitarse independientemente de la seguridad a nivel de mensajes. Como la seguridad a nivel de transporte proporciona una seguridad mínima, utilice la seguridad a nivel de mensajes cuando la seguridad sea esencial para una aplicación de servicios Web.

Para ver esta página de la consola administrativa, realice los pasos siguientes:

  1. Pulse Aplicaciones > Aplicaciones de empresa > nombre_aplicación.
  2. En Elementos relacionados, pulse Módulo Web > nombre_archivo_URI> Servicios Web: Enlaces de seguridad del cliente.
  3. En Autenticación básica HTTP, pulse Editar.

También puede especificar una autenticación básica HTTP SSL realizando los pasos siguientes:

  1. Pulse Aplicaciones > Aplicaciones de empresa > nombre_aplicación.
  2. En Elementos relacionados, pulse Módulo Web > nombre_archivo_URI> Servicios Web: Enlaces de seguridad del cliente.
  3. En Configuración HTTP SSL, pulse Editar.

ID de autenticación básica
Especifica el ID de usuario para la autenticación básica HTTP para este puerto.
Contraseña de autenticación básica
Especifica la contraseña para la autenticación básica HTTP de este puerto.
