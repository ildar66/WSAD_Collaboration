Recogida de información de cifrado

Utilice esta página para especificar la configuración de los parámetros de cifrado y descifrado. Esta configuración se utiliza para cifrar y descifrar partes del mensaje, incluido el símbolo de nombre de usuario y el texto.

Para ver esta página de la consola administrativa, realice los pasos siguientes:

  1. Pulse Aplicaciones de empresa > nombre_aplicación.
  2. En Elementos relacionados, pulse Módulo Web.
  3. En Propiedades adicionales, pulse Servicios Web: Enlaces de seguridad del servidor.
  4. En Enlace del receptor de peticiones, pulse Editar > Información de cifrado.
  5. Pulse Nuevo para crear un método de cifrado.
  6. Pulse Suprimir para suprimir un método de cifrado.
