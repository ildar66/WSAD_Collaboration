Propiedades en el nivel de célula de Nonce

Utilice esta página para especificar la configuración de Nonce en el nivel de célula en un entorno de WebSphere Application Server Network Deployment.

Para ver esta página de la consola administrativa, pulse Seguridad > Servicios Web > Propiedades.

Para definir la configuración de Nonce en el extremo del servidor , pulse Servidores > Servidores de aplicaciones > nombre_servidor. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security.

Nonce es un número criptográfico exclusivo incorporado en un mensaje para facilitar la detención de repetidos ataques no autorizados de símbolos de nombre de usuario. En un entorno de WebSphere Application Server Network Deployment, debe especificar valores para los campos Tiempo de espera de antememoria de Nonce, Antigüedad máxima de Nonce y Desfase horario de Nonce para el nivel de servidor.

Tiempo de espera de antememoria de Nonce
Especifica el valor de tiempo de espera en segundos para la antememoria de Nonce en el servidor. Nonce es un valor generado aleatoriamente.

Debe especificar un mínimo de 300 segundos para el campo Tiempo de espera de antememoria de Nonce. Sin embargo, si no especifica un valor, el valor por omisión es 600 segundos.

El campo Tiempo de espera de antememoria de Nonce para el nivel de célula es necesario en el entorno de WebSphere Application Server Network Deployment.

Nota: Si realiza cambios en el valor de tiempo de espera de antememoria de Nonce, debe reiniciar WebSphere Application Server para que los cambios entren en vigor.

Valor por omisión 600 segundos
Mínimo 300 segundos
Antigüedad máxima de Nonce
Especifica la cantidad de tiempo en segundos antes de que caduque la indicación de la hora de Nonce. Nonce es un valor generado aleatoriamente.

Debe especificar un mínimo de 300 segundos para el campo Antigüedad máxima de Nonce. Sin embargo, el valor máximo no puede exceder el número de segundos especificado en el campo Tiempo de espera de antememoria de Nonce para el nivel de célula. Si no especifica un valor, el valor por omisión es 300 segundos. El valor especificado en este campo del nivel de célula es el valor máximo que puede especificar para el campo Antigüedad máxima de Nonce para el nivel de servidor. Para especificar el valor Antigüedad máxima de Nonce para el nivel de servidor, pulse Servidores > Servidores de aplicaciones > nombre_servidor. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security.

El campo Antigüedad máxima de Nonce para el nivel de célula es necesario en el entorno de WebSphere Application Server Network Deployment.

Valor por omisión 300 segundos
Rango 300 segundos a Tiempo de espera de antememoria de Nonce
Desfase horario de Nonce
Especifica el valor de desfase horario en segundos a tener en cuenta cuando WebSphere Application Server comprueba lo reciente que es el mensaje. Nonce es un valor generado aleatoriamente.

Debe especificar un mínimo de 0 segundos para el campo Desfase horario de Nonce. Sin embargo, el valor máximo no puede exceder el número de segundos especificado en el campo Antigüedad máxima de Nonce del nivel de célula. Si no especifica un valor, el valor por omisión es 0 segundos.

El campo Desfase horario de Nonce para el nivel de célula es necesario en el entorno de WebSphere Application Server Network Deployment.

Valor por omisión 0 segundos
Rango 0 segundos a Antigüedad máxima de Nonce
