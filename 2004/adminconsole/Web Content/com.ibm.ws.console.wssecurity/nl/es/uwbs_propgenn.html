Valores de configuración de propiedades de Web Services Security

Utilice esta página para configurar propiedades adicionales.

Hay varios modos de ver un panel de valores de configuración de propiedades de Web Services Security. Realice los pasos siguientes para ver una de estas páginas de la consola administrativa:

  1. Pulse Servidores > Servidores de aplicaciones > nombre_servidor.
  2. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security > Localizadores de claves nombre_localizador_claves.
  3. En Propiedades adicionales, pulse Propiedades > Nuevo.
