Grupo de localizadores de claves

Utilice esta página para ver una lista de los localizadores de claves disponibles. Los localizadores de claves identifican las claves necesarias para la signatura digital y el cifrado. Un localizador de claves debe implementar la interfaz com.ibm.wsspi.wssecurity.config.KeyLocator. Las dos implementaciones por omisión son: com.ibm.wsspi.wssecurity.config.WSldKeyStoreMapKeyLocator y com.ibm.wsspi.wssecurity.config.KeyStoreKeyLocator.

Para ver esta página de la consola administrativa, realice los pasos siguientes:

  1. Pulse Servidores > Servidores de aplicaciones > nombre_servidor.
  2. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security > Localizadores de claves.
  3. Pulse Nuevo para crear un localizador de claves. Pulse Suprimir para suprimir un localizador de claves.

Mediante este panel, realice los pasos siguientes:

  1. Especifique un nombre de localizador de claves y un nombre de clase de localizador de claves en el panel
  2. Para guardar los cambios, pulse Guardar en la parte superior de la consola administrativa. Cuando pulse Guardar, volverá al panel de inicio de la consola administrativa.
  3. Después de guardar los cambios, vuelva al panel de colecciones Localizador de claves para actualizar el tiempo de ejecución de Web Services Security con la información de enlace por omisión, que se encuentra en el archivo ws_security.xml.
  4. Para actualizar el tiempo de ejecución de Web Services Security, pulse Actualizar el tiempo de ejecución. Al pulsar Actualizar el tiempo de ejecución, los cambios de tiempo de configuración realizados en los demás servicios Web se actualizan durante el tiempo de ejecución de Web Services Security.
  5. Cuando haya definido los localizadores de claves, pulse el nombre del localizador de claves para especificar propiedades y claves adicionales en Propiedades adicionales.

Nombre de clase del localizador de claves
