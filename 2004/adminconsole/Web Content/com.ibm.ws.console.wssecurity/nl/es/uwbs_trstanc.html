Grupo de anclas de confianza

Utilice esta página para ver una lista de los objetos de almacén de claves que contienen los certificados raíz de confianza. Estos objetos se utilizan para validar las vías de acceso de certificados de los símbolos de seguridad con formato X.509 recibidos. Los objetos de almacén de claves de las anclas de confianza contienen los certificados raíz de confianza que utiliza la API CertPath para validar la confianza de una cadena de certificados.

Para crear el archivo de almacén de claves, utilice la herramienta de claves en el directorio dir_instalación\java\jre\bin\keytool.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones nombre_servidor. En Propiedades adicionales, pulse Servicios Web: Enlaces por omisión para Web Services Security > Anclas de confianza.

Pulse Nuevo para crear una nueva ancla de confianza

Pulse Suprimir para suprimir un ancla de confianza.

Si Pulsa Actualizar el tiempo de ejecución, se actualiza el tiempo de ejecución de Web Services Security con la información de enlace por omisión, que se encuentra en el archivo ws-security.xml que se ha guardado previamente. Si realiza cambios en este panel, debe realizar los pasos siguientes:

  1. Para guardar los cambios, pulse Guardar en la parte superior de la consola administrativa. Cuando pulse Guardar, volverá al panel de inicio de la consola administrativa.
  2. Vuelva al panel de colecciones Anclas de confianza y pulse Actualizar el tiempo de ejecución.

    Nota: Al pulsar Actualizar el tiempo de ejecución, los cambios de tiempo de configuración realizados en los demás servicios Web se actualizan durante el tiempo de ejecución de Web Services Security.

Nombre del ancla de confianza
Especifica el nombre exclusivo que se utiliza para identificar el ancla de confianza.
Vía de acceso del almacén de claves
Especifica la ubicación del archivo de almacén de claves que contiene las anclas de confianza.
Tipo de almacén de claves
Especifica el tipo del archivo de almacén de claves.

El valor de este campo es JKS o JCEKS.
