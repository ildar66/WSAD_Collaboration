Schlüssel

Verwenden Sie diese Seite, um um eine Liste der logischen Namen anzuzeigen, die einem Schlüsselaliasname in der Keystore-Datei zugeordnet werden.

Gehen Sie zum Anzeigen dieser Seite der Administrationskonsole wie folgt vor:

  1. Klicken Sie auf Server > Anwendungsserver > Servername.
  2. Klicken Sie unter "Weitere Merkmale" auf Web Services: Standardbindungen für die Sicherheit von Web Services > Key Locator > Name_des_Key_Locator.
  3. Klicken Sie unter "Weitere Merkmale" auf Schlüssel.
  4. Klicken Sie auf Neu, um ein neues Schlüsselobjekt in der Keystore-Datei zu erstellen.
  5. Klicken Sie auf Löschen, um die Zuordnung eines Schlüsselobjekts in der Keystore-Datei zu löschen.

Schlüsselname
Der Name des in der Keystore-Datei gefundenen Schlüsselobjekts.
Aliasname des Schlüssels
Ein Aliasname für das Schlüsselobjekt.

Der Aliasname wird verwendet, wenn der Key Locator die Schlüsselobjekte im Keystore sucht.
