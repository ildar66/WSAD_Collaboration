X.509-Zertifikate

Verwenden Sie diese Seite, um eine Liste der X.509-Zertifikate anzuzeigen.

Gehen Sie zum Anzeigen dieser Seite der Administrationskonsole wie folgt vor:

  1. Klicken Sie auf Server > Anwendungsserver > Servername.
  2. Klicken Sie unter "Weitere Merkmale" auf Web Services: Standardbindungen für die Sicherheit von Web Services > Collection Certificate Store.
  3. Klicken Sie auf der Seite "Collection Certificate Store" unter "Weitere Merkmale" auf X.509-Zertifikate.

Klicken Sie auf Neu, um einen neuen Pfad zu einem X.509-Zertifikat zu erstellen.

Klicken Sie auf Löschen, um einen Pfad zu einem X.509-Zertifikat zu löschen.

Pfad des X509-Zertifikats:
Das Verzeichnis mit dem X.509-Zertifikat.
