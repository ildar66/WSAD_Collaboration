トラステッド ID エバリュエーター・コレクション

このページを使用して、トラステッド識別 (ID) エバリュエーターのリストを表示します。 トラステッド ID エバリュエーターは、識別表明権限を信頼するかどうかを決定します。ID が信頼されると、WebSphere Application Server は適切な信任状を発行します。この信任状は、リソースの起動をダウンストリームに要求する際に使用されます。 トラステッド ID エバリュエーターは、com.ibm.wsspi.wssecurity.id.TrustedIDEvaluator インターフェースをインプリメントしています。

この管理コンソール・ページを表示するには、以下のステップを実行します。

  1. 「サーバー」>「アプリケーション・サーバー」>「server_name 」をクリックします。
  2. 「Additional Services」の下の「Web Services: Default bindings for Web Services Security」>「Trusted ID Evaluators」をクリックします。

トラステッド ID エバリュエーターを作成する場合は、「新規」をクリックします。

トラステッド ID エバリュエーターを削除する場合は、「削除」をクリックします。

このパネルを使用して、次のステップを実行します。

  1. トラステッド ID エバリュエーター名およびトラステッド ID エバリュエーター・クラス名を指定します。
  2. 管理コンソールの上部にある「保管」をクリックして変更を保管してください。「保管」をクリックすると、管理コンソールのホーム・パネルに戻ります。
  3. 「Trusted ID Evaluator collection」パネルに戻り、ws-security.xml ファイルに含まれている デフォルトのバインディング情報で Web サービス・セキュリティー・ランタイムを更新します。
  4. 「Update runtime」をクリックします。他の Web サービスに加えられた構成も、Web サービス・セキュリティー・ランタイムで更新されます。

Trusted ID Evaluator Name
トラステッド ID エバリュエーターの固有の名前を指定します。
Trusted ID Evaluator Classname
トラステッド ID エバリュエーターのクラス名を指定します。
