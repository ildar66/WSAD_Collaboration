HTTP 基本認証コレクション

このページを使用して、このポートに対するトランスポート・レベル基本認証セキュリティーの ユーザー ID およびパスワードを指定します。Web サービスが別の Web サービスのクライアントになっている場合に、この構成を使用できます。

トランスポート・レベルのセキュリティーを使用して、基本認証を使用可能にできます。トランスポート・レベルのセキュリティーは、 メッセージ・レベルのセキュリティーから個別に使用可能とすることも使用不可とすることもできます。トランスポート・レベルの提供するセキュリティーは 最低限のセキュリティーであるため、セキュリティーが Web サービス・アプリケーションに対して必要不可欠である場合は、 メッセージ・レベルのセキュリティーを使用します。

この管理コンソール・ページを表示するには、以下のステップを実行します。

  1. 「アプリケーション」>「エンタープライズ・アプリケーション」>「application_name 」をクリックします。
  2. 「Related Items」の下の「Web モジュール」>「URI_file_name」>「Web Services: Client Security Bindings」をクリックします。
  3. 「HTTP Basic Authentication」の下の「編集」をクリックします。

以下のステップを完了して、HTTP SSL 基本認証を指定することもできます。

  1. 「アプリケーション」>「エンタープライズ・アプリケーション」>「application_name 」をクリックします。
  2. 「Related Items」の下の「Web モジュール」>「URI_file_name」>「Web Services: Client Security Bindings」をクリックします。
  3. 「HTTP SSL Configuration」の下の「編集」をクリックします。

Basic Authentication ID
このポートに対する HTTP 基本認証用のユーザー ID を指定します。
Basic Authentication Password
このポートに対する HTTP 基本認証用のパスワードを指定します。
