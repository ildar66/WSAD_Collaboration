Web サービス・セキュリティー・プロパティー・コレクション

このページを使用して、構成のための追加プロパティーのリストを表示します。

「Web services security property collection」パネルを表示する方法は、 複数あります。以下のステップを完了して、これらの管理コンソールページのいずれか 1 つを表示します。

  1. 「サーバー」>「アプリケーション・サーバー」>「server_name 」をクリックします。
  2. 「追加プロパティー」の下の「Web Services: Default bindings for Web Services Security」>「Key Locators」>「key_locator_name」をクリックします。
  3. 「追加プロパティー」の下の「プロパティー」をクリックします。
  4. 新規のプロパティーを作成する場合は、「新規」をクリックします。
  5. 指定済みのプロパティーを削除する場合は、「削除」をクリックします。

名前
プロパティーの名前を指定します。
値
プロパティーの値を指定します。
