X.509 証明書構成の設定

このページを使用して、X.509 証明書の保管場所を指定します。

この管理コンソール・ページを表示するには、以下のステップを実行します。

  1. 「サーバー」>「アプリケーション・サーバー」>「server_name 」をクリックします。
  2. 「追加プロパティー」の下の「Web Services: Default bindings for Web Services Security」>「Collection Certificate Store」をクリックします。
  3. 「Collection Certificate Store」ページの「追加プロパティー」の下の「X.509 Certificates」>「新規」をクリックします。

X509 Certificate Path
X.509 証明書の保管場所を指定します。
