暗号化情報コレクション

このページを使用して、暗号化パラメーターおよび暗号化解除パラメーターの構成を指定します。この構成は、本文とユーザー名トークンを含むメッセージの各部を暗号化および暗号化解除するために使用します。

この管理コンソール・ページを表示するには、以下のステップを実行します。

  1. 「エンタープライズ・アプリケーション」>「application_name 」をクリックします。
  2. 「Related Items」の下の「Web モジュール」をクリックします。
  3. 「追加プロパティー」の下の「Web Services: Server Security Bindings」をクリックします。
  4. 「Request Receiver Binding」の下の「編集」>「Encryption Information」をクリックします。
  5. 暗号化メソッドを作成する場合は、「新規」をクリックします。
  6. 暗号化メソッドを削除する場合は、削除」をクリックします。

Encryption Information
暗号化情報の名前を指定します。
