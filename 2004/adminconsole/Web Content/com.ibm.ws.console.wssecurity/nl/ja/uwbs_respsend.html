応答送信側のバインディング・コレクション

このページを使用して、Web サービス・セキュリティー応答メッセージを送信するための バインディング構成を指定します。

この管理コンソール・ページを表示するには、以下のステップを実行します。

  1. 「アプリケーション」>「エンタープライズ・アプリケーション」>「application_name 」をクリックします。
  2. 「Related Items」の下の「Web モジュール」>「URI_file_name」>「Web Services: Server Security Bindings」をクリックします。
  3. 「Response Sender Binding」の下の「編集」をクリックします。

Signing Information
署名パラメーターの構成を指定します。

サーバー・レベルの構成において、「認証メソッド」が IDAssertion、「ID タイプ」が X509Certificate になっている場合には、 これらのパラメーターを X.509 証明書検証にも使用できます。そのような場合には、「Certificate Path」フィールドにのみ入力する必要があります。

Encryption Information
暗号化パラメーターおよび暗号化解除パラメーターの構成を指定します。
Key Locators
ディジタル署名および暗号化のための鍵を、鍵ストア・ファイルまたはリポジトリーから取り出す鍵ロケーター・オブジェクトのリストを指定します。 鍵ロケーターは、名前または論理名をエイリアスにマップするか、認証済みの識別を鍵にマップします。この論理名は、 鍵ロケーターのインプリメンテーションに鍵を配置するために使用されます。
