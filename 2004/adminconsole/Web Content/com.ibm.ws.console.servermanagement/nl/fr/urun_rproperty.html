Collection de propriétés personnalisées

Cette page permet d'afficher et de gérer les paires de données nom-valeur arbitraires, où le nom est une clé de propriété et la valeur une chaîne pouvant être utilisée pour définir des propriétés de configuration de système interne.

Pour afficher la page de la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur > Propriétés personnalisées.

Nom
Indique le nom (ou la clé) de la propriété.
Valeur
Spécifie la valeur associée au nom spécifié.
Description
Fournit des informations sur la paire nom-valeur.
Requise
Indique si la zone requiert une valeur. Si cette case est cochée, vous devez fournir une valeur.
Expression valide
