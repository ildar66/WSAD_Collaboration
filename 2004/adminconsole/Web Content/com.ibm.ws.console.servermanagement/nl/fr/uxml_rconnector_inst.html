Paramètres des connecteurs JMX

Cette page permet d'afficher la configuration d'un connecteur JMX (Java Management Extensions).

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > nom_serveur > Services d'administration > Connecteurs JMX > type_connecteur.

Onglet Configuration

Type
Indique le type du connecteur JMX.
Type de données Enum
Valeur par défaut SOAPConnector
Plage
SOAPConnector
Pour les connexions JMX utilisant le protocole SOAP (Simple Object Access Protocol).
RMIConnector
Pour les connexions JMX utilisant RMI (Remote Method Invocation).
HTTPConnector
Pour les connexions JMX utilisant HTTP.
JMSConnector
Pour les connexions JMX utilisant JMS (Java Messaging Service).
