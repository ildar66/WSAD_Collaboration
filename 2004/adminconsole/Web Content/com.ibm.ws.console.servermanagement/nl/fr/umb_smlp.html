Collection de ports d'écoute de messages

Les ports d'écoute de messages configurés dans le domaine administratif

L'écran affiche une liste des ports d'écoute de messages configurés dans le domaine administratif. Chaque port d'écoute est utilisé avec un bean géré par messages pour recevoir automatiquement les messages d'une destination JMS associée. Vous pouvez utiliser cet écran pour ajouter de nouveaux ports d'écoute ou pour modifier les propriétés des ports d'écoute existants. Pour davantage d'informations sur les propriétés des ports d'écoute, reportez-vous à la rubrique Propriétés des ports d'écoute.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs-> serveur_applications-> Service d'écoute des messages-> Ports d'écoute
