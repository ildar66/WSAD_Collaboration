Collection de fournisseurs de MBean d'extension

Cette page permet d'afficher et de modifier la configuration des fournisseurs de MBean d'extension.

Vous pouvez configurer les fournisseurs de MBean d'extension JMX afin qu'ils soient utilisés pour étendre les ressources gérées WebSphere existantes dans le système d'administration de base. Chaque fournisseur de MBean est une bibliothèque contenant une implémentation d'un MBean JMX et le fichier de descripteur XML du MBean.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > nom_serveur > Services d'administration > Fournisseurs de MBean d'extension

Chemin d'accès aux classes dans la bibliothèque du fournisseur contenant le descripteur MBean.
Description
Description arbitraire de la configuration du fournisseur MBean d'extension.
Nom
