Collection de groupes de caches externes

Cette page permet de définir des ensembles de caches externes contrôlées par WebSphere Application Server sur des serveurs Web, tels que IBM Edge Server et IBM HTTP Server.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > serveur > Service de cache dynamique > Groupes de caches externes.

Nom
Indique le nom du groupe de caches externes.

Le nom du groupe de caches externes doit correspondre à la propriété externalcache, telle qu'elle est définie dans le servlet ou le fichier JSP cachespec.xml.

Lorsque la mise en cache externe est activée, la cache met les pages en correspondance avec ses URI et celles qui concordent sont envoyées vers la cache externe. Les entrées peuvent alors être servies à partir de la cache externe plutôt qu'à partir du serveur d'applications.

Type
Indique le type du groupe de caches externes.
