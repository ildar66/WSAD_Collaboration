Détails du service de débogage

Cette page vous permet d'afficher et de modifier les paramètres utilisés par le Service de débogage.

Pour afficher cette page de la console administrative, cliquez sur Serveurs > Serveurs d'applications > serveur > Service de débogage.

Onglet Configuration

Démarrage
Détermine si le serveur tentera ou non de lancer le service de débogage au moment de son démarrage.
Port de débogage JVM
Indique le port que la JVM (Java Virtual Machine) écoutera pour les connexions de débogage.
Arguments du débogage JVM
Spécifie la chaîne d'arguments de débogage utilisée pour démarrer la JVM en mode débogage.
Filtres des classes pour le débogage
Spécifie une table de classes à ne pas prendre en compte lors du débogage. Si vous êtes en mode pas à pas, le programme de débogage ne s'arrêtera pas dans les classes correspondant à une entrée de filtre.
Port de débogage BSF
Indique le port sur lequel le gestionnaire de débogage BSF est à l'écoute.
Niveau de consignation BSF
Indique le niveau de consignation fourni par le gestionnaire de débogage BSF. Les valeurs admises sont comprises entre 0 et 3, 3 étant le niveau de consignation le plus élevé.
