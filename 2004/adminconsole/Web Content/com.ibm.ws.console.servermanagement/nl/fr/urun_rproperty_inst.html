Propriétés personnalisées

Cette page permet de configurer les paires de données nom-valeur arbitraires, où le nom est une clé de propriété et la valeur une chaîne pouvant être utilisée pour définir des propriétés de configuration de système interne. En définissant une nouvelle propriété, cela vous permet de configurer un paramètre de manière plus avancée qu'avec le paramètre disponible dans la console d'administration.

Pour afficher la page de la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur > Propriétés personnalisées > nom_propriété.

Onglet Configuration

Nom
Indique le nom (ou la clé) de la propriété.
Type de données Chaîne
Valeur
Spécifie la valeur associée au nom spécifié.
Type de données Chaîne
Description
