Paramètres de module de connecteur

Utilisez cette page pour visualiser les modules de connecteurs.

Pour afficher cette page de la console d'administration, cliquez sur Applications > Applications d'entreprise > application > Modules de connecteurs > module_connecteur.

Onglet Configuration

URI
Chemin d'accès logique à la ressource qui est servie par WebSphere Application Server.
Type de données String (chaîne)
Nom
Nom complet du module de connecteur.
Type de données String (chaîne)
Autre DD
Spécifie l'autre descripteur de déploiement (DD) du module de connecteur.

URI de l'autre DD d'un module donné.

Type de données String (chaîne)
Pondération de démarrage
Indique la priorité de démarrage du module de connecteur par rapport aux autres.

Lorsque votre application contient plusieurs modules, le poids spécifié ici détermine la priorité de démarrage de ce module par rapport aux autres modules lors du démarrage du serveur. Les modules dont l'ordre de démarrage est le plus faible sont lancés en premier.

Type de données String (chaîne)
