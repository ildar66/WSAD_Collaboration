Paramètres de journaux des processus

Cette page permet d'afficher ou de modifier les paramètres spécifiant les fichiers dans lesquels les flux de sortie standard et d'erreurs standard sont enregistrés.

Pour afficher la page de la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur > Définition des processus > Journaux des processus.

Onglet Configuration

Nom du fichier Stdout
Indique le fichier vers lequel le flux standard de sortie est dirigé. Le nom du fichier peut inclure un nom de chemin symbolique défini dans les entrées de la variable.

Utilisez le champ dans l'onglet de configuration pour spécifier le nom de fichier. Utilisez le champ dans l'onglet d'exécution pour sélectionner un fichier et l'afficher. Cliquez sur Afficher pour afficher ce fichier.

Pour diriger la sortie du serveur vers la console d'administration ou vers le processus ayant lancé le serveur, supprimez le nom de fichier ou spécifiez console dans l'onglet de configuration.

Type de données Chaîne
Unité Chemin d'accès au fichier
