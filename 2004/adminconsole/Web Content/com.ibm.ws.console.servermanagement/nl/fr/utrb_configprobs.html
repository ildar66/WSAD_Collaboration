Problèmes de configuration

Cette page permet d'identifier et d'afficher les incidents qui existent dans la configuration courante.

Pour afficher la page de la console d'administration, cliquez sur Résolution des incidents > Problèmes de configuration dans l'arborescence de navigation de la console.

Cliquez sur un incident de configuration dans le tableau pour afficher sa description.

Validation du document de configuration
Ces zones permettent de spécifier le niveau de validation à effectuer sur les documents de configuration.
Validation interdocuments
Permet d'activer la validation interdocuments de configuration.

L'activation de la validation interdocuments permet la comparaison des documents de configuration pour trouver les paramètres en conflit.

Problèmes de configuration

Affiche les messages d'erreur liés à l'incident de configuration en cours. Cliquez sur un message pour afficher des informations détaillées sur l'incident.

Portée

Trie la liste des incidents de configuration en fonction des fichiers de configuration où les messages d'erreur s'affichent. Cliquez sur un message pour afficher des informations détaillées sur l'incident.

Zones d'explication des incidents de configuration :

Les zones d'information suivantes s'affichent lorsque vous cliquez sur un message d'incident de configuration :

Message
Affiche le message renvoyé par le valideur.
Explication
Rapide explication de l'incident.
Action de l'utilisateur
Indique l'action recommandée pour corriger l'incident.
Objet cible
Identifie l'objet de configuration à l'endroit où l'erreur de validation s'est produite.
Gravité
Indique la gravité de l'erreur de configuration, 1 indiquant une erreur grave. Plus le descripteur de gravité possède une valeur élevée, plus la gravité est faible.
URI local
Indique l'URI local du fichier de configuration dans lequel l'erreur s'est produite.
URI complet
Indique l'URI complet du fichier de configuration dans lequel l'erreur s'est produite.
Nom de classe du valideur
Nom de classe du valideur signalant l'incident.
