Paramètres de configuration de Runtime Performance Advisor

Cette page permet de définir les paramètres du conseiller Runtime Performance Advisor.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > nom_serveur > Configuration de Runtime Performance Advisor.

Onglet Configuration

Activer Runtime Performance Advisor
Indique si Runtime Performance Advisor est en cours d'exécution.
Intervalle de calcul
Des données PMI sont collectées sur un intervalle de temps et la moyenne de ces données est calculée en vue de fournir des conseils. L'intervalle de calcul correspond à la durée pour laquelle les données sont collectées pour un conseil donné. Par conséquent, les détails des messages apparaissent sous forme de moyennes de cet intervalle.
Séquence d'avertissements maximale
Indique le nombre d'avertissements consécutifs émis avant la mise à jour du seuil d'alerte.
Nombre de processeurs
Indique le nombre de processeurs sur le serveur.

Bouton Redémarrer
La sélection du bouton Redémarrer dans l'onglet Contexte d'exécution permet de réinitialiser Runtime Performance Advisor en utilisant les dernières informations de configuration enregistrées sur le disque. Cette action permet également de redéfinir l'état de Runtime Performance Advisor. Par exemple, le nombre actuel d'avertissements prend la valeur zéro pour chaque message.
