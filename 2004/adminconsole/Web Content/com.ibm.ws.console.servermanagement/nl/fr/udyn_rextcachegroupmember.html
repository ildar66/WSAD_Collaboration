Collection de membres d'un groupe de caches externes

Cette page permet de définir des caches spécifiques qui font partie d'un groupe de caches.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > serveur > Service de cache dynamique > Groupes de caches externes > groupe_caches_externes > Membres du groupe de caches externes.

Nom du bean adaptateur
Indique le nom du bean adaptateur.

Voici des exemples de noms de bean adaptateur pris en charge dans WebSphere Application Server :
AFPA
AdapterBeanName : com.ibm.ws.cache.servlet.Afpa
Address : port d'écoute d'afpa
ESI
AdapterBeanName : com.ibm.websphere.servlet.cache.ESIInvalidatorServlet
Address : hôte local
WTE
AdapterBeanName : com.ibm.websphere.edge.dynacache.WteAdapter
Address : nom d'hôte:port (nom d'hôte et port sur lequel WTE effectue l'écoute
