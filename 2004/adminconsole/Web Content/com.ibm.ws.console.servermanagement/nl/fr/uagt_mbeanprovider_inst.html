Paramètres des fournisseurs de MBean d'extension

Cette page permet d'afficher et de modifier la configuration d'un fournisseur de MBean d'extension JMX.

Vous pouvez configurer une bibliothèque contenant une implémentation d'un MBean JMX et le fichier de descripteur XML de ce MBean afin qu'elle soit utilisée pour étendre les ressources gérées WebSphere existantes dans le système d'administration de base

Pour afficher la page de la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur > Services d'administration > Fournisseurs de MBean d'extension > nom_bibliothèque_fournisseur

Onglet Configuration

Chemin d'accès aux classes
Chemin d'accès aux classes dans la bibliothèque du fournisseur contenant le descripteur MBean. Le chargeur de classe a besoin de ces informations pour charger et analyser le fichier de descripteur XML du MBean d'extension.
Type de données Chaîne
Description
Description arbitraire de la configuration du fournisseur MBean d'extension. Cette zone permet de saisir un texte identifiant ou différenciant la configuration du fournisseur.
Type de données Chaîne
Nom
Nom utilisé pour identifier la bibliothèque du fournisseur de MBean d'extension.
