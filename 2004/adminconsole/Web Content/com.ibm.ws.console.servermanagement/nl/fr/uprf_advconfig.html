Paramètres de configuration des conseils

Cette page permet de sélectionner les conseils que vous souhaitez activer ou désactiver.

Pour afficher cette page, cliquez sur Serveurs > Serveurs d'applications > nom_serveur > Configuration de Runtime Performance Advisor > Configuration des conseils.

Onglet Configuration

Nom du conseil
Indique le conseil que vous pouvez activer ou désactiver.
Composant concerné
Indique le composant WebSphere Application Server auquel s'appliquent les conseils de Runtime Performance Advisor.
Etat des conseils
Indique si les conseils ont l'état Arrêté ou Démarré.

Il existe deux valeurs -- Démarré ou Arrêté. Démarré indique que les conseils sont en cours d'exécution, s'ils sont applicables. Arrêté indique que les conseils ne sont pas en cours d'exécution.

Onglet Contexte d'exécution

Nom du conseil
Indique le conseil que vous pouvez activer ou désactiver.
Composant concerné
Indique le composant WebSphere Application Server auquel s'appliquent les conseils de Runtime Performance Advisor.
Etat des conseils
Indique si les conseils ont l'état Arrêté, Démarré ou Non disponible.

L'état des conseils peut prendre l'une des trois valeurs suivantes -- Démarré, Arrêté ou Non disponible. Démarré indique que les conseils sont appliqués. Arrêté indique que les conseils ne sont pas appliqués. Non disponible indique que les conseils ne sont pas applicables dans la configuration actuelle (c'est le cas par exemple du conseil relatif à la taille des sessions persistantes dans une configuration sans sessions persistantes).
