Paramètres des composants de serveur

Cette page permet d'afficher ou de configurer une instance de composant de serveur.

Pour afficher la console d'administration, sélectionnez Serveurs > Serveurs d'applications > nom_serveur > Composants de serveur > nom_composant_serveur.

Onglet Configuration

Nom
Indique le nom du composant.
Type de données Chaîne
Etat initial
Indique l'état du composant lors du démarrage du processus serveur. Les options possibles sont Démarré et Arrêté. La valeur par défaut est Démarré.
Type de données Chaîne
Valeur par défaut Démarré
