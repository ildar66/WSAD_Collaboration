端点设置

使用此页面查看和更改通信端点的配置，通信端点是由与进程共同运行的运行时组件使用的。端点提供服务器的主机和端口规范。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > 端点 > end_point_name

“配置”选项卡

端点名称
指定端点的名称。名称在服务器内必须是唯一的。

注意，此字段仅在定义应用程序服务器端点时显示。

数据类型 String
主机
指定客户机用于请求资源（如 命名服务、管理服务或 JMS 代理）的 IP 地址、带有域名后缀的域名服务器（DNS）主机名或只是 DNS 主机名。

例如，如果主机名为 myhost，则全限定 DNS 名可以是 myhost.myco.com，而 IP 地址可以为 155.123.88.201。

数据类型 String
缺省 *
端口
指定配置服务接受客户机请求的端口。端口值与主机名联合使用。
数据类型 整型
缺省 无
