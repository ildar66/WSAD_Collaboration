HTTP 传输集合

使用此页面查看或管理 HTTP 传输。传输提供 Web 服务器的 WebSphere 插件和应用程序驻留的 Web 模块的 Web 容器之间的请求队列。当您请求 web 浏览器中的应用程序时，请求传递到 Web 服务器，然后一直传输到 Web 容器。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > Web 容器 > HTTP 传输。

主机
指定为传输绑定的主机 IP 地址。如果应用程序服务器位于本地机器上，则主机名可以是 localhost。
端口
指定绑定传输的端口。端口号可以是系统上任何当前号码未使用的端口。端口号对于给定的机器上的每个应用程序服务器实例必须是唯一的。
启用 SSL
指定是否使用安全套接字层（SSL）保护 WebSphere 插件和应用程序服务器之间的连接。缺省为不使用 SSL。
