扩展 MBean 提供程序设置

使用此页面查看和更改 JMX 扩展 MBean 提供程序的配置。

您可配置包含 JMX MBean 及其 MBean XML 描述符文件的库，以用于扩展核心管理系统中现有 WebSphere 管理的资源

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > 管理服务 > 扩展 MBean 提供程序 > provider_library_name

“配置”选项卡

类路径
提供程序库中可找到 MBean 描述符的类路径。类装入程序需要此信息装入和语法分析扩展 MBean XML 描述符文件。
数据类型 String
描述
扩展 MBean 提供程序配置的任意描述性文本。为有助于标识或区分提供程序配置的任何文本使用此字段。
数据类型 String
名称
用于标识扩展 MBean 提供程序库的名称。
数据类型 String
