管理服务设置

使用此页面查看和更改管理服务的配置。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > 管理服务

“配置”选项卡

单机
指定服务器进程是否是一个 Network Deployment 单元中的参与者，或不是。如果为真，服务器不参与分布式管理。如果为假，服务器参与 Network Deployment 系统。

基本 WebSphere Application Server 安装的缺省值为真。当运行 addNode 将服务器合并到 Network Deployment 部署单元，则值切换为假。

数据类型 Boolean
缺省 true
首选连接器
指定首选的 JMX 连接器类型。可用的选项（如 SOAPConnector 或 RMIConnector）是使用“JMX 连接器”页面定义的。
数据类型 String
