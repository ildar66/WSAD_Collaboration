进程执行设置

使用此页面查看或更改启动或初始化 UNIX 进程的命令行信息。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > 流程定义 > 进程执行。

“配置”选项卡

进程优先级
指定进程的操作系统优先级。仅 root 用户可更改此值。
数据类型 整型
缺省 1000 适用于大多数操作系统中的 WebSphere Application Server。在 OS/400 上，缺省为 25。
掩码
指定进程运行所用的用户掩码（文件方式许可权掩码）。
数据类型 整型
运行用户
指定运行进程的用户。
数据类型 String
组运行
指定作为成员之一运行进程的组。

在 OS/400 上，忽略“组运行”设置。

数据类型 String
在进程组中运行
指定进程的特定进程组。此进程组对类似于处理程序分区等有用。系统管理员可以指定运行的进程组，如 12 个处理程序中的 6 个。缺省值（0）不为任何特定组指定进程。

在 OS/400 上，忽略“进程组运行”设置。

数据类型 整型
缺省 0
