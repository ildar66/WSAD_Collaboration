服务器集合

使用此页面查看关于信息和管理应用程序和 JMS 服务器。

应用程序服务器

“应用程序服务器”页面列出单元中的应用程序服务器和应用程序服务器的节点。

Network Deployment 产品也显示应用程序服务器的状态。状态 表明服务器是否运行、已停止或遇到问题。

要查看此管理控制台页面，单击服务器 > 应用程序服务器。

JMS 服务器

每个 JMS 服务器为管理域中的节点提供 JMS 提供程序的功能。在管理域中每个节点上存在至多一个 JMS 服务器，而且域内的任何应用程序服务器可访问域中任何节点上任何 JMS 服务器提供服务的 JMS 资源。

要查看此管理控制台页面，单击服务器 > JMS 服务器。

名称
指定服务器的逻辑名。在节点中，服务器名必须是唯一的。
节点
指定应用程序服务器的节点名。
状态  
表明应用程序服务器已启动或已停止。（仅 Network Deployment）

注意：如果状态是不可用，Node Agent 没有在节点中运行，您必须在可以启动服务器之前重新启动 Node Agent。
