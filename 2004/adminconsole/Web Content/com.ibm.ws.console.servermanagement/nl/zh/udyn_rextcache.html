外部高速缓存组集合

使用此页面定义由 Web 服务器上的 WebSphere Application Server 控制的外部高速缓存的集合，如 IBM Edge Server 和 IBM HTTP Server。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server > 动态高速缓存服务 > 外部高速缓存组。

名称
指定外部高速缓存组名。

外部高速缓存组名需要与 servlet 或 JSP cachespec.xml 文件中定义的 externalcache 属性匹配。

当外部高速缓存启用时，高速缓存用其 URI 与页面匹配，并将匹配的页面推送到外部高速缓存。然后，可以从外部高速缓存服务条目，而不是从应用程序服务器。

类型
指定外部高速缓存组类型。
