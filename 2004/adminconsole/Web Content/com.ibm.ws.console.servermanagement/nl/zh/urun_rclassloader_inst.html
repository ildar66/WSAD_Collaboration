类装入程序设置

使用此页面配置驻留在应用程序服务器上的应用程序的类装入程序。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > 类装入器 >class_loader_ID。

“配置”选项卡

类装入程序标识
表明标识类装入程序实例的服务器的唯一字符串。产品指定标识。
数据类型 String
类装入程序方式
指定应用程序类装入程序策略为 SINGLE 时的类装入程序方式。PARENT_FIRST 导致类装入程序在尝试从本地类路径装入类之前，委托将类装入到父类装入程序。PARENT_LAST 导致类装入程序在委托类装入到它的父之前，尝试从它的本地类路径装入类；这允许应用程序类装入程序重设和提供它自已的，存在于父类装入程序中的类版本。
数据类型 String
缺省 PARENT_FIRST
