定制服务设置

使用此页面配置在应用程序服务器中运行的服务。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > 定制服务 >custom_service_name。

“配置”选项卡

启动
指定当服务器的包含进程（服务器）启动时它是否尝试启动和初始化服务。缺省情况下，当它的包含进程启动时不启用服务。要启动服务，在复选框中选取复选标记。
数据类型 Boolean
缺省 false
外部配置 URL
指定定制服务配置文件的 URL。

如果您的定制服务类要求一个配置文件，指定该值的配置文件的全限定路径名。该文件传递至您的定制服务类中。

数据类型 String
单位 URL
类名
指定服务实现的类名。该类必须实现定制服务接口。
数据类型 String
单位 Java 类名称
显示名
指定服务名称。
数据类型 String
描述
描述定制服务。
数据类型 String
类路径
指定用于定位服务类和 JAR 文件的类路径。
数据类型 String
单位 类路径
