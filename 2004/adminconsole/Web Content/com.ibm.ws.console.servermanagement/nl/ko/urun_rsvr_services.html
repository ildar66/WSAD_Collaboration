사용자 정의 서비스 콜렉션

이 페이지를 참조하여 Application Server가 사용할 수 있는 서비스 목록을 보고 서비스가 사용 가능한지 보십시오. 사용자 정의 서비스는 WebSphere Application Server에 플러그인하고 서버가 시작하거나 시스템 종료할 때 실행하는 코드를 정의하는 기능을 제공합니다.

이 관리 콘솔 페이지를 보려면 서버 > Application Server > server_name> 사용자 정의 서비스를 클릭하십시오.

외부 구성 URL
사용자 정의 서비스 구성 파일에 대한 URL을 지정합니다.

사용자 정의 서비스 클래스가 구성 파일을 필요로 하는 경우, 값이 해당 구성 파일에 대한 완전한 경로 이름을 제공합니다. 이 파일 이름이 사용자의 사용자 정의 서비스에 전달됩니다.

클래스 이름
서비스 구현의 클래스 이름을 지정합니다. 이 클래스는 사용자 정의 서비스 인터페이스를 구현해야 합니다.
표시 이름
서비스의 이름을 지정합니다.
시작
서비스의 포함하는 프로세스(서버)가 시작할 때 서버가 서비스를 시작하고 초기화하려 시도할지 여부를 지정합니다. 기본적으로 프로세스가 포함된 서비스가 시작될 때 서비스가 사용 가능하지 않습니다.
