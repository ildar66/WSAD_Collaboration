Extension MBean Provider 설정

이 페이지를 참조하여 JMX extension MBean provider의 구성을 보고 변경하십시오.

JMX MBean과 MBean XML 설명자 파일의 구현을 포함하는 라이브러리를 구성하여 코어 관리 시스템의 기존 WebSphere 관리 자원을 확장하는데 사용할 수 있습니다.

이 콘솔 페이지를 보려면, 서버 > Application Servers > server_name > 관리 서비스 > Extension MBean Providers > provider_library_name을 클릭하십시오.

구성 탭

클래스 경로
MBean Descriptor가 위치할 수 있는 프로바이더 라이브러리의 클래스 경로.클래스 로더의 경우 Extension MBean XML 설명자 파일을 로드하여 구문 분석하기 위해 이 정보를 필요로 합니다.
데이터 유형 문자열
설명
Extension MBean Provider 구성을 위한 임의의 설명 텍스트. 프로바이더 구성의 식별 또는 차별화에 도움을 주는 임의의 텍스트에 이 필드를 사용하십시오.
데이터 유형 문자열
이름
Extension MBean Provider 라이브러리를 식별하기 위해 사용되는 이름.
