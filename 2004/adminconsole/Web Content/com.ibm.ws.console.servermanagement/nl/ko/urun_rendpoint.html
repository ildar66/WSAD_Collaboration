엔드포인트 콜렉션

이 페이지를 참조하여 프로세스 내에서 실행 중인 런타임 구성요소가 사용한 통신 엔드포인트를 보고 관리하십시오. 엔드포인트는 서버의 호스트 및 포트 스펙을 제공합니다.

이 관리 콘솔 페이지를 보려면 서버 > Application Server > server_name > 엔드포인트를 클릭하십시오.

이 페이지는 Application Server의 엔드포인트에 대해서 작업할 경우에만 표시됩니다.

엔드포인트 이름
엔드포인트의 이름을 지정합니다. 각 이름은 서버 내에서 고유해야 합니다.
