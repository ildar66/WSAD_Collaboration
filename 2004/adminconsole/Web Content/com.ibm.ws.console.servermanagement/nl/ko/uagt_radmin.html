관리 서비스 설정

이 페이지를 참조하여 관리 서비스에 대한 구성을 보고 변경하십시오.

이 관리 콘솔 페이지를 보려면, 서버 > Application Servers > server_name > 관리 서비스를 클릭하십시오.

구성 탭

독립형
서버 프로세스가 Network Deployment 셀의 참여자인지 여부를 지정합니다. True인 경우 서버는 분산 관리에 참여하지 않습니다. False인 경우 서버는 Network Deployment 시스템에 참여합니다.

기본 WebSphere Application Server 설치 설치에 대한 기본값은 true입니다. addNode가 서버를 Network Deployment 셀에 통합하기 위해 실행할 때 값은 false로 전환합니다.

데이터 유형 부울
기본값 true
선호 커넥터
선호하는 JMX 커넥터 유형을 지정합니다. SOAPConnector 또는 RMIConnector 같은 사용 가능한 옵션은 JMX 커넥터 페이지를 사용하여 정의됩니다.
