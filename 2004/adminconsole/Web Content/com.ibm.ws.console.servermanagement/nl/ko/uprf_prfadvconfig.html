런타임 성능 어드바이저 구성 설정

이 페이지를 참조하여 런타임 성능 어드바이저의 설정을 지정하십시오.

이 관리 페이지를 보려면, 서버 > Application Server > server_name > 런타임 성능 어드바이저 구성을 클릭하십시오.

구성 탭

런타임 성능 어드바이저 사용 가능
런타임 성능 어드바이저의 실행 여부를 지정합니다.
계산 간격
시간 간격으로 PMI 데이터를 받아 평균화하여 권고를 제공합니다. 계산 간격은 이 권고에 맞게 데이터를 취합하는 시간을 지정합니다. 그러므로 권고 메시지 내의 세부사항은 이 간격 동안의 평균으로 나타납니다.
최대 경고 순서
임계값이 갱신되기 전에 발행되는 연속 경고 수를 지정합니다.
프로세서 수
서버의 프로세서 수를 지정합니다.

런타임 탭

런타임 성능 어드바이저 사용 가능
런타임 성능 어드바이저의 실행 여부를 지정합니다.
계산 간격
시간 간격으로 PMI 데이터를 받아 평균화하여 권고를 제공합니다. 계산 간격은 이 권고에 맞게 데이터를 취합하는 시간을 지정합니다. 그러므로 권고 메시지 내의 세부사항은 이 간격 동안의 평균으로 나타납니다.

프로세서 수

다시 시작 단추
런타임 탭에서 다시 시작을 선택하면 디스크에 저장된 최종 정보를 사용하여 런타임 성능 어드바이저를 다시 초기화합니다. 이 조치도 런타임 성능 어드바이저의 상태를 재설정함을 참고하십시오. 예를 들어 각 메시지에 대한 현재 경고 계수가 0으로 재설정됩니다.
