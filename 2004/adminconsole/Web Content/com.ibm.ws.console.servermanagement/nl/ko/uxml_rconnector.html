Java 관리 확장 커넥터

이 페이지를 참조하여 JMX(Java Management Extensions) 커넥터 구성을 보고 변경하십시오.

이 관리 콘솔 페이지를 보려면, 서버 > Application Servers > server_name > 관리 서비스 > JMX 커넥터를 클릭하십시오.

JMX(Java Management Extensions) 커넥터는 스크립팅 프로세스 호출 시 WebSphere Application Server와 통신합니다. 커넥터의 유형 및 매개변수에 대한 기본값은 없습니다. wsadmin.properties 파일은 SOAP(Simple Object Access Protocol) 및 적절한 포트 번호를 지정합니다. 또한 RMI(Remote Method Invocation) 커넥터를 사용할 수도 있습니다.

다음 방법 중 하나를 사용하여 커넥터 유형 및 속성을 선택하십시오.

  • 등록 정보 파일에서 등록 정보를 지정하십시오.
  • 명령행에서 옵션을 지정하십시오.

구성 탭

유형
JMX 커넥터의 유형을 지정합니다.
데이터 유형 Enum
기본값 SOAPConnector
범위
SOAPConnector
SOAP(Simple Object Access Protocol)를 사용하는 JMX 연결의 경우.
RMIConnector
RMI(Remote Method Invocation)를 사용하는 JMX 연결의 경우.
HTTPConnector
HTTP를 사용하는 JMX 연결의 경우.
JMSConnector
JMS(Java Messaging Service)를 사용하는 JMX 연결의 경우.
