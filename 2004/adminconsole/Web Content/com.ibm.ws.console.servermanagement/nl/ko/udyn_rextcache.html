외부 캐시 그룹 콜렉션

이 페이지를 참조하여 IBM Edge Server 및 IBM HTTP Server와 같은 WebSphere Application Server가 제어하는 외부 캐시의 세트를 정의하십시오.

이 관리 콘솔 페이지를 보려면, 서버 > Application Servers > server > 동적 캐시 서비스 > 외부 캐시 그룹을 클릭하십시오.

이름
외부 캐시 그룹 이름을 지정합니다.

외부 캐시 그룹 이름은 Servlet 또는 JSP cachespec.xml 파일에 정의되는 externalcache 등록 정보와 일치해야 합니다.

외부 캐싱이 사용 가능할 때 캐시는 그의 URI를 갖는 페이지와 일치하고, 일치하는 페이지를 외부 캐시에 넣습니다. 입력은 Application Server가 아닌 외부 캐시에서 제공될 수 있습니다.

유형
외부 캐시 그룹 유형을 지정합니다.
