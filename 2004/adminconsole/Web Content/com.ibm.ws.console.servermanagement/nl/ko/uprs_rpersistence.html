분산 환경 설정

이 페이지를 참조하여 분산 환경에서 세션을 저장하기 위한 유형을 지정하십시오.

이 관리 콘솔 페이지를 보려면 서버 > Application Servers > server_name > 웹 컨테이너 > 세션 관리 > 분배 환경 설정을 클릭하십시오.

구성 탭

분배된 세션
세션 저장에 사용될 분배 환경 유형을 지정합니다.

없음 서버가 종료되면 세션 관리 기능이 세션 데이터를 버리도록 지정합니다.
데이터베이스 세션 관리 기능이 데이터소스 연결 설정에 지정된 데이터소스에 세션 정보를 저장하도록 지정합니다. 데이터베이스를 클릭하여 데이터 소스 설정을 변경하십시오.
메모리 대 메모리 복제 세션 관리 기능이 메모리의 데이터소스에 세션 정보를 저장하도록 지정합니다. 세션 정보는 장애 복구를 위해 다른 세션 관리 기능으로 복사됩니다. 이 메모리 대 메모리 설정을 사용하고 변경할 복제기를 지정하려면 메모리 대 메모리 복제를 클릭하십시오. (WebSphere Application Server Network Deployment에만 해당.)
