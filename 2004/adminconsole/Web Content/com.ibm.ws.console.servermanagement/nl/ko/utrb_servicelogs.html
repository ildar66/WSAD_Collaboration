IBM 서비스 로그 설정

이 페이지를 참조하여 활동 로그라고도 하는 IBM 서비스 로그를 구성하십시오.

이 관리 콘솔 페이지를 보려면 문제점 해결 > 로그 및 추적 > server > IBM 서비스 로그를 클릭하십시오.

구성 탭

서비스 로그 사용 가능
IBM 서비스 로그에 의한 로그 파일의 작성을 지정합니다.
파일 이름
IBM 서비스 로그에서 사용한 파일의 이름을 지정합니다.
최대 파일 크기
서비스 로그 파일의 최대 크기를 MB 단위로 지정합니다. 기본값은 2MB입니다.

이 크기로 도달하면 서비스 로그는 장소로 랩핑됩니다. 서비스 로그는 JVM 로그와 같이 새 로그 파일로 롤오버되지 않습니다.

메시지 필터링
메시지 필터 레벨을 원하는 상태로 지정합니다. 이 옵션을 사용하여 일부 메시지 카테고리를 필터로 걸러내거나 제한할 수 있습니다. 이 필터 값은 서비스 로그 뿐만 아니라 모든 로그에 적용됩니다.
상관 ID 사용 가능
각각의 메시지와 함께 로그되는 상관 ID의 생성을 지정합니다.

상관 ID를 사용하여 특정 클라이언트 요청으로 활동을 연관시키거나 다중 Application Server에서 활동을 연관시킬 수 있습니다.
