프로세스 정의 설정

이 페이지에서는 프로세스 정의에 따르는 설정을 보거나 변경할 수 있습니다. 프로세스 정의는 시작이나 초기화 프로세스를 하는 데 필요한명령행 정보를 제공합니다.

이 관리 콘솔 페이지를 보려면 서버 > Application Servers > server_name > 프로세스 정의를 클릭하십시오.

구성 탭

실행 파일 이름
프로세스의 실행 가능한 이름을 지정합니다.
데이터 유형 문자열
실행 파일 인수
프로세스가 시작할 때 실행하는 실행 파일 명령을 지정합니다.

예를 들어, 실행 가능한 대상 프로그램은 다음 세 가지 인수일 수 있습니다. arg1 arg2 arg3.

데이터 유형 문자열

작업 디렉토리
프로세스가 실행할 파일 시스템 디렉토리를 지정하십시오.

이 디렉토리는 상대 경로 이름으로 입력 및 출력 파일의 위치를 결정하는 데 사용됩니다.

아카이브된 Enterprise Bean은 Bean이 실행 중인 Application Server의 현재 작업 디렉토리에 놓입니다. 작업 디렉토리가 WebSphere Application Server 제품의 루트 디렉토리 아래에 있는 알려진 디렉토리인지 확인하십시오.
