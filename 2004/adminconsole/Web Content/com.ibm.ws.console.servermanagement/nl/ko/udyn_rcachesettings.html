동적 캐시 서비스 설정

이 페이지를 참조하여 동적 캐시 서비스 설정을 구성하고 관리하십시오.

이 관리 콘솔 페이지를 보려면 서버 > Application Server > server > 동적 캐시 서비스를 클릭하십시오.

구성 탭

시작 상태
동적 캐시의 사용 가능 여부를 지정합니다.
캐시 크기
캐시가 보유하는 최대 항목 수 값에 양의 정수를 지정합니다.

이 필드에 100 - 200,000 범위의 캐시 크기값을 입력하십시오.

기본 우선순위
항목이 꽉 찬 캐시에 머무는 기간을 판별하는 캐시 항목에 대한 기본 우선순위를 지정합니다.
기본값 1
디스크 오프로드
디스크 오프로드의 사용 가능 여부를 지정합니다.

기본적으로 동적 캐시는 메모리에 구성되는 항목 수를 유지보수합니다. 캐시가 가득 찬 상태에서 새 항목이 작성되는 경우 각 캐시 항목에 대해 구성되는 우선순위와 최소한 최근에 사용되는 알고리즘이 캐시에서 항목을 제거하는 데 사용됩니다. 또는 디스크 오프로드를 구성하여 메모리에서 항목을 제거하는 대신 나중에 사용하도록 파일 시스템(위치는 구성 가능)에 복사할 수 있습니다.
