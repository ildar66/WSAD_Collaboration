권고 구성 설정

이 페이지를 참조하여 사용 가능 또는 불가능하게 하려는 권고를 선택하십시오.

이 관리 페이지를 보려면, 서버 > Application Server > server_name > 런타임 성능 어드바이저 구성 > 권고 구성을 클릭하십시오.

구성 탭

권고 이름
사용 가능 또는 불가능하게 할 수 있는 권고를 지정합니다.
구성요소에 적용되는 권고
런타임 성능 권고가 적용되는 WebSphere Application Server 구성요소를 지정합니다.
권고 상태
권고가 중지 또는 시작되어 있는지 여부를 지정합니다.

오직 두 개의 값(시작됨 및 중지됨)이 있습니다. 시작됨은 권고가 적용되는 경우 권고 실행을 의미합니다. 중지됨은 권고 실행 안 함을 의미합니다.

런타임 탭

권고 이름
사용 가능 또는 불가능하게 할 수 있는 권고를 지정합니다.
구성요소에 적용되는 권고
런타임 성능 권고가 적용되는 WebSphere Application Server 구성요소를 지정합니다.
권고 상태
권고가 중지, 시작되어 있는지 또는 사용 불가능한지 여부를 지정합니다.

권고 상태는 세 값(시작됨, 중지됨 또는 사용 불가능) 중 하나입니다. 시작됨은 권고 적용 중임을 의미합니다. 중지됨은 권고가 적용되지 않음을 의미합니다. 사용 불가능은 권고가 현재 구성(지속적 세션 없이 구성에서 지속된 세션 크기 권고와 같음)에 적용되지 않음을 의미합니다.
