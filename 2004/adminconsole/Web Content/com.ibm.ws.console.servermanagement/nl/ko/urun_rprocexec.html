프로세스 실행 설정

이 페이지를 참조하여 UNIX 프로세스 시작 및 초기화에 대한 명령행 정보를 보거나 변경하십시오.

이 관리 콘솔 페이지를 보려면 서버 > Application Servers > server_name > 프로세스 정의 > 프로세스 수행을 클릭하십시오.

구성 탭

프로세스 우선순위
프로세스에 대한 운영 체제 우선순위를 지정합니다. 루트 사용자만 이 값을 변경할 수 있습니다.
데이터 유형 정수
기본값 대부분의 운영 체제에서는 WebSphere Application Server에 대해 1000입니다. OS/400에서 기본값은 25입니다.
UMASK
프로세스가 실행되는 사용자 마스크를 지정합니다(파일 모드 권한 마스크).
데이터 유형 정수
Run As 사용자
프로세스를 실행하는 사용자를 지정합니다.
데이터 유형 문자열
Run As 그룹
프로세스를 실행할 수 있는 자격을 가진 사용자들의 그룹을 지정합니다.

OS/400에서 Run As 그룹 설정이 무시됩니다.

데이터 유형 문자열
프로세스 내 실행 그룹
프로세스에 대한 특정 프로세스 그룹을 지정합니다. 이 프로세스 그룹은 프로세서 파티션과 같은 작업에 대해 유용합니다. 시스템 관리자는 실행할 프로세스 그룹을 지정할 수 있습니다(예: 12 프로세서 중 6). 기본값(0)은 특정 그룹에 프로세스를 지정하지 않습니다.

OS/400에서 프로세스 내 실행 그룹 설정이 무시됩니다.

데이터 유형 정수
기본값 0
