프로세스 로그 설정

이 페이지를 참조하여 표준 출력 및 표준 오류 스트림이 기록하는 파일을 지정하는 설정을 보거나 변경하십시오.

이 관리 콘솔 페이지를 보려면 서버 > Application Server > server_name > 프로세스 정의 > 프로세스 로그를 클릭하십시오.

구성 탭

표준 출력 파일 이름
표준 출력 스트림이 경로 지정되는 파일을 지정합니다. 파일 이름은 변수 항목에 정의되는 기호 경로 이름을 포함할 수 있습니다.

구성 탭에서 필드를 사용하여 필드 이름을 지정합니다. 런타임 탭에서 필드를 사용하여 보기 위한 파일을 선택하십시오. 보기를 클릭하여 파일을 보십시오.

파일 이름을 삭제하거나 구성 탭에서 console을 지정하여 서버 출력을 서버를 시작한 관리 콘솔 또는 프로세스로 지정하십시오.
