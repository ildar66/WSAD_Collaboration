Valores de servidor de nombres

Utilice esta página para configurar los valores de proveedor de servicios de denominación del servidor de aplicaciones.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones >nombre_servidor > Componentes de servidor > Servidor de nombres.

Pestaña Configuración

Nombre
Especifica el nombre de visualización del servidor.
Tipo de datos String
Estado inicial
Especifica el estado de ejecución. Las opciones son: Iniciado y Detenido.
Tipo de datos String
Valor por omisión Iniciado
