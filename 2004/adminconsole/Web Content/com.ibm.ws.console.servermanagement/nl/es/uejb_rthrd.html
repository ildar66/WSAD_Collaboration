Valores de agrupación de hebras

Utilice esta página para configurar una agrupación de hebras que utiliza el servidor de aplicaciones. Las peticiones se envían al servidor por medio de cualquier transporte HTTP. Una agrupación de hebras permite a los componentes del servidor volver a utilizar hebras para eliminar la necesidad de crear nuevas hebras durante el tiempo de ejecución. Durante la creación de nuevas hebras se emplean tiempo y recursos.

Para ver esta página de la consola administrativa, pulse Servidores > Gestionar servidores de aplicaciones >nombre_servidor > Servicio ORB > Agrupación de hebras. (Puede llegar a esta página utilizando diferentes rutas de navegación).

Pestaña Configuración

Tamaño mínimo
Especifica el número mínimo de hebras permitidas en la agrupación.
Tipo de datos Integer
Valor por omisión 10
Tamaño máximo
Especifica el número máximo de hebras permitidas en la agrupación.

Si Tivoli Performance Viewer muestra la medida Percent Maxed (porcentaje alcanzado) para seguir coherentemente en los dígitos dobles, considere aumentar el tamaño máximo. La medida Percent Maxed (porcentaje alcanzado) indica la cantidad de tiempo que están en uso las hebras configuradas. Si hay varios clientes conectando simultáneamente al ORB del servidor, aumente el tamaño para dar soporte hasta 1000 clientes.

Tipo de datos Integer
Valor por omisión 50
Recomendado 50 (25 en sistemas Linux)
Tiempo excedido de inactividad de hebra
Especifica el tiempo en milisegundos de inactividad que debe trascurrir antes de que se reclame una hebra. Un valor igual a 0 indica que no hay espera mientras que un valor negativo (inferior a 0) indica una espera permanente.
Tipo de datos Integer
Unidades Milisegundos
Valor por omisión 3500
Con posibilidad de crecimiento
Especifica si el número de hebras puede aumentar por encima del tamaño máximo configurado para la agrupación de hebras.
Tipo de datos Boolean
Valor por omisión No habilitado (false)
Rango Los valores válidos son Permitir asignación de hebras por encima del tamaño máximo de hebra o No habilitado.
