Valores de anotaciones cronológicas de proceso

Utilice esta página para ver o modificar los valores para especificar los archivos en los que se escriben las corrientes de datos de error estándar y de salida estándar.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Definición de proceso > Anotaciones cronológicas de proceso.

Pestaña Configuración

Nombre de archivo de salida estándar
Especifica el archivo al que se dirige la corriente de salida estándar. El nombre del archivo puede incluir un nombre de vía de acceso simbólico definido en las entradas de variable.

En la pestaña de configuración especifique en el campo el nombre de archivo. En la pestaña de tiempo de ejecución seleccione en el campo un archivo para consulta. Pulse Ver para consultar el archivo.

Para dirigir la salida del servidor a la consola administrativa o al proceso que ejecutó el servidor, suprima el nombre de archivo o especifique consola en la pestaña de configuración.

Tipo de datos String
Unidades Nombre de la vía de acceso del archivo
