Valores del servidor de aplicaciones

Utilice esta página para ver o modificar los valores de una instancia de componente de servidor.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones nombre_servidor.

La pestaña Configuración proporciona campos que se pueden editar, mientras que la pestaña Tiempo de ejecución proporciona información de sólo lectura. La pestaña Tiempo de ejecución sólo está disponible cuando el servidor se está ejecutando.

Pestaña Configuración

Nombre
Especifica un nombre lógico para el servidor. Los nombres de servidor deben ser exclusivos en un nodo.
Tipo de datos String
Valor por omisión servidor1
Estado inicial
Especifica el estado de ejecución de componentes solicitado cuando se inicia el servidor por primera vez.
Tipo de datos String
Valor por omisión Iniciado
Política del cargador de clase de aplicación
Especifica si debe utilizar un único cargador de clases para cargar todas las aplicaciones o utilizar un cargador de clases distinto para cada aplicación.

Las opciones son SINGLE y MULTIPLE. El valor por omisión es utilizar un cargador de clases diferente para cada aplicación (MULTIPLE).

Tipo de datos String
Valor por omisión MULTIPLE
Modalidad del cargador de clases de la aplicación
Especifica si el cargador de clases debe buscar en el cargador de clases padre o en el cargador de clases de la aplicación antes de cargar una clase. El estándar para los cargadores de clases JDK y los cargadores de clases de WebSphere es PADRE_ÚLTIMO. Al especificar PADRE_ÚLTIMO, la aplicación puede alterar temporalmente las clases incluidas en el cargador de clases padre, pero esta acción potencialmente puede dar lugar a ClassCastException o LinkageErrors si ha mezclado clases alteradas temporalmente con no alteradas temporalmente.

Las opciones son PADRE_PRIMERO y PADRE_ÚLTIMO. El valor por omisión es buscar en el cargador de clases padre antes de buscar en el cargador de clases de la aplicación para cargar la aplicación.

Tipo de datos String
Valor por omisión PADRE_PRIMERO
