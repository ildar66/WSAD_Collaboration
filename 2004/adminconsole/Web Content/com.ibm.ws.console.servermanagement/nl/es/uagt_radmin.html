Valores del servicio de administración

Utilice esta página para ver y modificar la configuración de un servicio de administración.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Servicios de administración.

Pestaña Configuración

Autónomo
Especifica si el proceso del servidor participa o no en una célula de Network Deployment. Si el valor es true, el servidor no participa en una administración distribuida. Si el valor es false, el servidor participa en el sistema de Network Deployment.

El valor por omisión de las instalaciones de WebSphere Application Server base es true. Si se ejecuta addNode para incorporar el servidor en una célula de Network Deployment, el valor cambia a false.

Tipo de datos Boolean
Valor por omisión true
Conector preferido
Especifica el tipo de conector JMX preferido. Las opciones disponibles, tales como SOAPConnector o RMIConnector, se definen mediante la página Conectores JMX.
Tipo de datos String
