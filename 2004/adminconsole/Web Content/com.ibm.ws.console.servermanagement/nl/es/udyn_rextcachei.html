Valores de grupo de antememoria externa

Utilice esta página para configurar los grupos de antememorias externas que controla WebSphere Application Server en los servidores Web, como por ejemplo, IBM Edge Server e IBM HTTP Server.

Para ver esta página de la consola administrativa, pulse Servidores > Servidor de aplicaciones > servidor > Servicio de antememoria dinámica > Grupos de antememorias externas > grupo_antememoria_externa.

Pestaña Configuración

Nombre
Especifica el nombre del grupo de antememoria externa.

Es necesario que el nombre de grupo de antememoria externa coincida con la propiedad de antememoria externa tal como se define en el servlet o el archivo de cachespec.xml de JSP (JavaServer Pages).

Cuando se habilita la colocación en antememoria externa, la antememoria hace coincidir las páginas con sus URI y envía las páginas coincidentes a la antememoria externa. Las entradas procederán entonces de la antememoria externa en lugar del servidor de aplicaciones. Esta posibilidad constituye un importante ahorro de rendimiento.

Tipo
Especifica el tipo de grupo de antememoria externa.
