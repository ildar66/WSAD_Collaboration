Valores de parámetros de ajuste

Utilice esta página para establecer los parámetro de ajuste para sesiones distribuidas.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Contenedor Web > Gestión de sesiones > Valores de entorno distribuido > Parámetros de ajuste personalizado.

Pestaña Configuración

Nivel de ajuste
Especifica que el recurso de gestión de sesiones proporcione varios valores predefinidos que afectan al rendimiento.

Seleccione uno de estos valores predefinidos o personalice un valor. Para personalizar un valor, seleccione primero uno de los valores predefinidos, el que más se acerque al valor que desee, pulse Valores personalizados, introduzca los cambios y pulse Aceptar.
