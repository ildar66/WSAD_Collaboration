Valores de entorno distribuido

Utilice esta página para especificar un tipo para guardar una sesión en un entorno distribuido.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Contenedor Web > Gestión de sesiones > Valores de entorno distribuido.

Pestaña Configuración

Sesiones distribuidas
Especifica el tipo de entorno distribuido que se utilizará para guardar sesiones.

Ninguno Especifica que el recurso de gestión de sesiones descarte los datos de sesión cuando el servidor se cierre.
Base de datos Especifica que el recurso de gestión de sesiones almacene la información de sesión en el origen de datos especificado por los valores de conexión del origen de datos. Pulse Base de datos para modificar estos valores de origen de datos.
Duplicación de memoria a memoria Especifica que el recurso de gestión de sesiones almacene la información de sesión en un origen de datos en la memoria. La información de sesión se copia en otro recurso de gestión de sesiones para la recuperación de errores. Pulse Duplicación de memoria a memoria para especificar el duplicador que se utilizará y cambiar estos valores de memoria a memoria. (Sólo para WebSphere Application Server Network Deployment)
