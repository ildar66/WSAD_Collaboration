Valores de preferencias de la consola administrativa

Utilice los valores de Preferencias para especificar la forma en que se quiere visualizar la información en una página de consola administrativa.

Número máximo de filas
Número máximo de filas a visualizar por página cuando la recopilación es grande.
Historial de filtro
Si se quiere utilizar el mismo criterio de filtro para visualizar esta página la próxima vez que la visite.

Seleccione el cuadro de selección Mantener criterio de filtro para mantener el último criterio de filtro entrado. Cuando se vuelve a la página más adelante, el criterio de filtro que se mantiene controla la recopilación de aplicación que aparece en la tabla.
