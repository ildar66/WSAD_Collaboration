Valores de punto final

Utilice esta página para ver y modificar la configuración de un punto final de comunicación utilizado por los componentes de tiempo de ejecución que se ejecutan en un proceso. Un punto final proporciona especificaciones de sistema principal y de puerto para un servidor.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Puntos finales > nombre_punto_final.

Pestaña Configuración

Nombre de punto final
Especifica el nombre del punto final. El nombre debe ser exclusivo en el servidor.

Tenga en cuenta que esta campo sólo se muestra cuando esté definiendo un punto final para un servidor de aplicaciones.

Tipo de datos String
Sistema principal
Especifica la dirección IP, el nombre de sistema principal DNS (servidor de nombres de dominio) con el sufijo de nombre de dominio, o sólo el nombre de sistema principal DNS que el cliente utiliza para solicitar un recurso (por ejemplo, el servicio de denominación, servicio administrativo o intermediario JSP).

Por ejemplo, si el nombre de sistema principal es sistppal, el nombre DNS plenamente cualificado puede ser sistppal.miempresa.com y la dirección IP puede ser 155.123.88.201.

Tipo de datos String
Valor por omisión *
Puerto
Especifica el puerto con el que se configura el servicio para que acepte las peticiones de cliente. El valor de puerto se utiliza junto con el nombre de sistema principal.
Tipo de datos Integer
Valor por omisión Ninguno
