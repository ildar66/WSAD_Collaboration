Conectores Java Management Extensions

Utilice esta página para ver y modificar la configuración de conectores JMX (Extensiones de gestión de Java).

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > nombre_servidor > Servicios de administración > Conectores JMX.

Los conectores JMX (Extensiones de gestión de Java) se comunican con WebSphere Application Server cuando invoca un proceso de scripts. No hay ningún valor por omisión para el tipo y los parámetros de un conector. El archivo wsadmin.properties especifica el conector SOAP (Protocolo de acceso a objetos sencillo) y un número de puerto adecuado. También puede utilizar el conector RMI (Remote Method Invocation).

Utilice uno de los métodos siguientes para seleccionar el tipo de conector y los atributos:

  • Especifique las propiedades en un archivo de propiedades.
  • Indique las opciones en la línea de mandatos.
