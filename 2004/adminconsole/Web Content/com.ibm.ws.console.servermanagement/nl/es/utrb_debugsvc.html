Detalles del servicio de depuración

Utilice esta página para ver y modificar los valores utilizados por el servicio de depuración.

Para ver esta página de la consola administrativa, pulse Servidores > Servidores de aplicaciones > servidor > Servicio de depuración.

Pestaña Configuración

Arranque
Especifica si el servidor intentará iniciar el servicio de depuración cuando se inicie el servidor.
Puerto de depuración de JVM
Especifica el puerto en el que la máquina virtual Java escuchará la recepción de conexiones de depuración.
Argumentos de depuración de JVM
Especifica la serie de argumentos de depuración utilizados para iniciar la JVM en modalidad de depuración.
Filtros de clases de depuración
Especifica una matriz de clases que debe ignorarse durante la depuración. Al ejecutarse en modalidad paso a paso, el depurador no se detendrá en las clases que coincidan con una entrada de filtro.
Puerto de depuración de BSF
Especifica el puerto en el que escucha el gestor de depuración de BSF.
Nivel de registro cronológico de BSF
Especifica el nivel de registro cronológico proporcionado por el gestor de depuración de BSF. El rango válido es 0-3, con el valor 3 como el nivel más alto de registro cronológico.
