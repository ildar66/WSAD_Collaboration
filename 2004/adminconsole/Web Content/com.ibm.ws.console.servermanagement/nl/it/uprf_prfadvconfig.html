Impostazioni di configurazione Runtime Performance Advisor

Utilizzare questa pagina per specificare le impostazioni per Runtime Performance Advisor.

Per visualizzare questa pagina di gestione, fare clic su Server > Server delle applicazioni > nome_server > Configurazione di Runtime Performance Advisor.

Scheda Configurazione

Abilita Runtime Performance Advisor
Specifica se Runtime Performance Advisor è in esecuzione.
Intervallo di calcolo
I dati PMI vengono rilevati da un intervallo di tempo e utilizzati come media per fornire avvisi. L'intervallo di calcolo indica la lunghezza di tempo in base al quale i dati vengono rilevati per questo avviso. Pertanto, i dettagli contenuti nei messaggi di avviso verranno visualizzati come medie di questo intervallo.
Numero massimo di sequenza dell'avviso
Specifica il numero massimo di avvisi consecutivi emessi prima dell'aggiornamento della soglia.
Numero di processori
Specifica il numero di processori sul server.

Scheda Runtime

Abilita Runtime Performance Advisor
Specifica se Runtime Performance Advisor è in esecuzione.
Intervallo di calcolo
I dati PMI vengono rilevati da un intervallo di tempo e utilizzati come media per fornire avvisi. L'intervallo di calcolo indica la lunghezza di tempo in base al quale i dati vengono rilevati per questo avviso. Pertanto, i dettagli contenuti nei messaggi di avviso verranno visualizzati come medie di questo intervallo.
Numero massimo di sequenza dell'avviso
Specifica il numero massimo di avvisi consecutivi emessi prima dell'aggiornamento della soglia.
Numero di processori
Specifica il numero di processori sul server.
Pulsante di riavvio
Se si seleziona Riavvia sulla scheda Runtime viene inizializzato di nuovo Runtime Performance Advisor utilizzando le ultime informazioni salvate sul disco. Notare che questa operazione ripristina lo stato del Runtime Performance Advisor. Ad esempio, per ciascun messaggio l'effettivo conteggio delle avvertenze viene ripristinato su zero.
