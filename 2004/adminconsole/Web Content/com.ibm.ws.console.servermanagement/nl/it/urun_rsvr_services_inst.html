Impostazioni Servizio personalizzato

Utilizzare questa pagina per configurare un servizio in esecuzione in un server delle applicazioni.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Servizi personalizzati >nome_servizio_personalizzato.

Scheda Configurazione

Avvio
Specifica se il server effettuerà o meno un tentativo di avviare e inizializzare il servizio nel momento in cui il rispettivo processo (il server) verrà avviato. Per impostazione predefinita, il servizio non viene abilitato all'avvio del rispettivo processo. Per abilitare il servizio, contrassegnare la casella di controllo.
Tipo dati Valore booleano
Predefinito false
URL di configurazione esterno
Specifica l'URL di un file di configurazione del servizio personalizzato.

Se la classe del servizio personalizzato richiede un file di configurazione, specificare il nome del percorso completo per il file di configurazione, per indicarne il valore. Questo nome file viene trasferito alla classe del servizio personalizzato dell'utente.

Tipo dati Stringa
Unità URL
Nome classe
Specifica il nome classe dell'implementazione del servizio. Questa classe deve implementare l'interfaccia Servizio personalizzato.
Tipo dati Stringa
Unità Nome classe Java
Nome visualizzato
Specifica il nome del servizio.
Tipo dati Stringa
Descrizione
Descrive il servizio personalizzato.
Tipo dati Stringa
Classpath
Specifica il percorso classe utilizzato per individuare le classi e i file JAR del servizio.
Tipo dati Stringa
Unità Percorso classe
