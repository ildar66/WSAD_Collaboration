Impostazioni Politica di monitoraggio

Utilizzare questa pagina per visualizzare o modificare le impostazioni che controllano le modalità di monitoraggio e di riavvio di un processo da parte dell'agente nodo.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Definizione processo > Politica di monitoraggio.

Scheda Configurazione

Numero massimo di tentativi di avvio
Specifica il numero massimo di tentativi da effettuare per avviare il server delle applicazioni prima di rinunciare.
Tipo dati Numero intero
Intervallo di ping
Specifica la frequenza dei tentativi di comunicazione tra il processo parent, come ad esempio l'agente nodo, e il processo che sta tentando di creare, come ad esempio un server delle applicazioni. Regolare questo valore in base ai requisiti per il riavvio del server non avviato. La diminuzione del valore consente di rilevare più rapidamente gli errori; aumentando il valore, si riduce la frequenza dell'esecuzione del ping, diminuendo così il sovraccarico del sistema.
Tipo dati Numero intero
Timeout di ping
Quando un processo parent richiama un processo child, ad esempio quando un gestore del processo richiama un server, il processo parent esegue il ping del processo child per verificare che quest'ultimo sia stato richiamato con esito positivo. Questo valore specifica il numero di secondi che il processo parent deve attendere (dopo il ping del processo child) prima di attestare che il processo child ha avuto esito negativo.
Tipo dati Numero intero
Unità Secondi
Riavvio automatico
Specifica se il processo deve eseguire o meno l'avvio automatico in caso di errore. Il valore predefinito è quello di consentire il riavvio automatico del processo.
Tipo dati Valore booleano
Predefinito vero
Stato riavvio nodo
Specifica lo stato desiderato di un processo dopo la chiusura e il riavvio di un nodo. Le opzioni sono le seguenti: STOPPED, RUNNING, PREVIOUS. Il valore predefinito è STOPPED.
Tipo dati Stringa
Predefinito STOPPED
Intervallo I valori validi sono STOPPED, RUNNING o PREVIOUS.
