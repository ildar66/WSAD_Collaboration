Impostazioni di trasporto HTTP

Utilizzare questa pagina per visualizzare e configurare un trasporto HTTP. Il nome della pagina deve essere quello di un'impostazione SSL, come DefaultSSLSettings.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Contenitore Web > Trasporti HTTP > nome_host.

Scheda Configurazione

Host
Specifica l'indirizzo IP host per collegarsi al trasporto.

Se il server delle applicazioni si trova su una macchina locale, il nome host deve essere localhost.

Tipo dati Stringa
Porta
Specifica la porta a cui collegarsi per il trasporto. Specificare un numero di porta compreso tra 1 e 65535. Il numero di porta deve essere univoco per ogni server delle applicazioni su una determinata macchina.
Tipo dati Numero intero
Intervallo Da 1 a 65535
SSL abilitato
Specifica se proteggere o meno le connessioni tra il server delle applicazioni e il plugin WebSphere con SSL (Secure Sockets Layer). Il valore predefinito è quello di non utilizzare SSL.
Tipo dati Valore booleano
Predefinito false
SSL
Specifica il tipo di impostazioni SSL (Secure Sockets Layer) delle connessioni tra il server delle applicazioni e il plugin WebSphere. Le opzioni includono una o più impostazioni SSL definite su Centro sicurezza; ad esempio, DefaultSSLSettings, ORBSSLSettings o LDAPSSLSettings.
Tipo dati Stringa
Predefinito Un'impostazione SSL definita in Centro sicurezza
