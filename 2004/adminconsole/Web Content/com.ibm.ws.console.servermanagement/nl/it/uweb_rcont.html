Impostazioni del contenitore Web

Utilizzare questa pagina per configurare le impostazioni del contenitore.

Accedere a questa pagina tramite il seguente percorso nella console di gestione:

Server > Server delle applicazioni >
istanza_server > Contenitore Web

Configurazione - Proprietà generali

Host virtuale predefinito
Indica che un host virtuale è una configurazione che permette ad una singola macchina host di comportarsi come più macchine host. Le risorse associate ad un host virtuale non possono condividere i dati con le risorse associate ad un altro host virtuale, anche se gli host virtuali condividono la stessa macchina fisica.

Selezionare un'opzione dell'host virtuale:

Host predefinito
Il prodotto fornisce un host virtuale predefinito con alcuni alias comuni, ad esempio l'indirizzo IP della macchina, il nome abbreviato e completo dell'host. L'alias comprende la prima parte del percorso di accesso ad una risorsa, come un servlet. Ad esempio, può essere localhost:9080 nella richiesta http://localhost:9080/myServlet.
Host di gestione
Questo è un altro nome per il server delle applicazioni; noto anche come server1 nell'installazione base. Questo processo supporta l'uso della console di gestione.
Cache del servlet
Specifica che se un servlet viene richiamato dopo aver generato l'output da memorizzare nella cache, viene creata una voce cache che contiene non solo l'output, ma anche gli effetti secondari della chiamata. Tali effetti possono includere chiamate ad altri servlet o a file JSP (Java Server Pages) così come i metadati relativi alla voce, comprese le informazioni sul timeout e sulla priorità delle voci.

Abilita cache del servlet
Selezionare questa casella per abilitare la memorizzazione del servlet nella cache.
