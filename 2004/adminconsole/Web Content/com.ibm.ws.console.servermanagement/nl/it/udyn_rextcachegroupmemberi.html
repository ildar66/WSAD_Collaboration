Impostazioni del membro gruppo cache esterno

Utilizzare questa pagina per configurare cache specifiche membri di un gruppo cache.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > server > Servizio cache dinamica > Gruppi cache esterni > Gruppo_cache_esterno > Membri del gruppo cache esterni > Membri_gruppo_cache_esterni.

Scheda Configurazione

Address
Specifica una stringa di configurazione utilizzata da un bean adattatore della cache esterna per connettersi alla cache esterna.
AdapterBeanName
Specifica il nome del bean adattatore.
AFPA
AdapterBeanName: com.ibm.ws.cache.servlet.Afpa
Indirizzo: porta su cui è in ascolto afpa
ESI
AdapterBeanName: com.ibm.websphere.servlet.cache.ESIInvalidatorServlet
Indirizzo: host locale
WTE
AdapterBeanName: com.ibm.websphere.edge.dynacache.WteAdapter
Indirizzo: nome host:porta (nome host e porta su cui WTE è in ascolto)
