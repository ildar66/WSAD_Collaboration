Impostazioni di definizione processo

Utilizzare questa pagina per visualizzare o modificare le impostazioni di una definizione processo, che fornisce le informazioni sulla riga comandi per avviare o inizializzare un processo.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Definizione processo.

Scheda Configurazione

Nome eseguibile
Specifica il nome eseguibile del processo.
Tipo dati Stringa
Argomenti eseguibili
Specifica i comandi eseguibili che vengono eseguiti all'avvio del processo.

Ad esempio, il programma di destinazione eseguibile deve prevedere tre argomenti: arg1 arg2 arg3.

Tipo dati Stringa
Unità Argomenti riga comandi Java
Directory di lavoro
Specifica la directory del file system in cui il processo verrà eseguito.

Questa directory viene utilizzata per stabilire le ubicazioni dei file di input e di output con i relativi nomi di percorso.

I bean enterprise disattivati vengono posizionati nella directory di lavoro corrente del server delle applicazioni su cui i bean sono in esecuzione. Assicurarsi che la directory di lavoro sia riconosciuta nella directory root del prodotto WebSphere Application Server.

Tipo dati Stringa
