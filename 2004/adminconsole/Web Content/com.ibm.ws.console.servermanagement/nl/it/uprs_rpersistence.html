Impostazioni ambiente distribuito

Utilizzare questa pagina per specificare un tipo per il salvataggio di una sessione in un ambiente distribuito.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Contenitore Web > Gestione sessioni > Impostazioni ambiente distribuito.

Scheda Configurazione

Sessioni distribuite
Specifica il tipo di ambiente distribuito da utilizzare per il salvataggio delle sessioni.

Nessuna Specifica che la funzione Gestione sessioni deve eliminare i dati della sessione all'arresto del server.
Database Specifica che la funzione Gestione sessioni deve memorizzare le informazioni delle sessioni nell'origine dati specificata dalle relative impostazioni di connessione. Fare clic su Database per modificare queste impostazioni dell'origine dati.
Replica da memoria a memoria Specifica che la funzione Gestione sessioni deve memorizzare le informazioni delle sessioni nell'origine dati in memoria. Le informazioni delle sessioni vengono copiate su altre funzioni Gestione sessioni per l'eventuale ripristino in seguito a malfunzionamenti. Fare clic su Replica da memoria a memoria per specificare il replicatore da utilizzare e modificare queste impostazioni da memoria a memoria. (Solo per WebSphere Application Server Network Deployment.)
