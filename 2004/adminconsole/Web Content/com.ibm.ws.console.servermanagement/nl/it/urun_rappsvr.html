Impostazioni del server delle applicazioni

Utilizzare questa pagina per visualizzare o modificare le impostazioni di un'istanza del server delle applicazioni.

Per visualizzare questa pagina della console di gestione, fare lic su Server > Server delle applicazioni > nome_server.

La scheda Configurazione fornisce campi modificabili mentre la scheda Runtime offre informazioni di sola lettura. La scheda Runtime è disponibile soltanto quando il server è in esecuzione.

Scheda Configurazione

Nome
Specifica un nome logico del server. I nomi server all'interno di un nodo devono essere univoci.
Tipo dati Stringa
Predefinito server1
Stato iniziale
Specifica lo stato di esecuzione del componente, necessario quando il server viene avviato per la prima volta.
Tipo dati Stringa
Predefinito Avviato
Politica del programma di caricamento classi dell'applicazione
Specifica se utilizzare un solo programma di caricamento classi per caricare tutte le applicazioni o usare un programma di caricamento classi diverso per ogni applicazione.

Le opzioni sono SINGOLO e MULTIPLO. Il valore predefinito è quello di utilizzare un programma di caricamento classi separato per ogni applicazione (MULTIPLO).

Tipo dati Stringa
Predefinito MULTIPLO
Modalità programma di caricamento classi dell'applicazione
Specifica se il programma di caricamento classi deve eseguire una ricerca prima nel programma di caricamento classi parent o in quello dell'applicazione per caricare una classe. Lo standard per i programmi di caricamento classi JDK e WebSphere è PARENT_FIRST. Se si specifica PARENT_LAST, l'applicazione può ignorare le classi contenute nel programma di caricamento classi parent, ma quest'azione può potenzialmente generare un'eccezione ClassCastException o un errore LinkageErrors se si utilizzano insieme le classi sovrascritte e le classi non sovrascritte.

Le opzioni sono PARENT_FIRST e PARENT_LAST. Per caricare una classe, il valore predefinito è quello di eseguire una ricerca nel programma di caricamento classi parent prima di effettuare tale operazione in quello dell'applicazione.

Tipo dati Stringa
Predefinito PARENT_FIRST

Scheda Runtime

ID processo  
Specifica una stringa che identifica il processo.
Tipo dati Stringa
Nome cella  
Specifica il nome della cella del server delle applicazioni.
Tipo dati Stringa
Predefinito nome_hostNetwork
Nome nodo  
Specifica il nome del nodo del server delle applicazioni.
Tipo dati Stringa
Stato  
Indica se lo stato del server delle applicazioni è avviato o arrestato.
Tipo dati Stringa
Predefinito Avviato
