Dettagli sul Servizio di debug

Utilizzare questa pagina per visualizzare e modificare le impostazioni utilizzate dal Servizio di debug.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > server > Servizio di debug.

Scheda Configurazione

Avvio
Specifica se il server tenterà di avviare il servizio di debug all'avvio.
Porta debug JVM
Specifica la porta su cui JVM (Java Virtual Machine) sarà in ascolto per le connessioni di debug.
Argomenti di debug JVM
Specifica la stringa degli argomenti di debug utilizzata per avviare la JVM in modalità debug.
Filtri classe di debug
Specifica una serie di classi da ignorare durante il debug. Durante l'esecuzione in modalità passo-passo, il debugger non si arresterà nelle classi che corrispondono ad una voce filtro.
Porta debug BSF
Specifica la porta su cui il gestore di debug BSF è in ascolto.
Livello di registrazione BSF
Specifica il livello di registrazione fornito dal Gestore di debug BSF. L'intervallo di valori valido è 0-3, con 3 che rappresenta il livello più alto di registrazione.
