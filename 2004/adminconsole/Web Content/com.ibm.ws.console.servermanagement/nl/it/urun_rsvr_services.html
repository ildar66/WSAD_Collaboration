Raccolta di servizi personalizzati

Utilizzare questa pagina per visualizzare un elenco di servizi disponibili per il server delle applicazioni e per verificarne l'abilitazione. Un servizio personalizzato consente di effettuare il collegamento ad un server delle applicazioni WebSphere e di definire il codice in esecuzione al momento dell'avvio o della chiusura del server.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server> Servizi personalizzati.

URL di configurazione esterno
Specifica l'URL di un file di configurazione del servizio personalizzato.

Se la classe dei servizi personalizzati richiede un file di configurazione, il valore fornisce un nome del percorso completo per il file di configurazione. Questo nome file viene trasferito alla classe del servizio personalizzato dell'utente.

Nome classe
Specifica il nome classe dell'implementazione del servizio. Questa classe deve implementare l'interfaccia Servizio personalizzato.
Nome visualizzato
Specifica il nome del servizio.
Avvio
Specifica se il server effettuerà o meno un tentativo di avviare e inizializzare il servizio nel momento in cui il rispettivo processo (il server) verrà avviato. Per impostazione predefinita, il servizio non viene abilitato all'avvio del rispettivo processo.
