Impostazioni dei log di processo

Utilizzare questa pagina per visualizzare o modificare le impostazioni per specificare i file su cui vengono scritti i flussi di errore e di output standard.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Definizione processo > Log processo.

Scheda Configurazione

Nome file Stdout
Specifica il file a cui viene indirizzato il flusso di output standard. Il nome del file può includere un nome percorso simbolico definito nelle voci di variabile.

Utilizzare il campo sulla scheda di configurazione per specificare il nome del file. Utilizzare il campo sulla scheda di runtime per selezionare un file per la visualizzazione. Visualizzare il file facendo clic su Visualizza.

Indirizzare l'output del server alla console di gestione o al processo che ha avviato il server, cancellando il valore di questa proprietà o specificando console sulla scheda di configurazione.

Tipo dati Stringa
Unità Nome percorso file
