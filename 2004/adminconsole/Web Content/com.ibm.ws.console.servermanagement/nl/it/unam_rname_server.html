Impostazioni server dei nomi

Utilizzare questa pagina per configurare le impostazioni del provider del servizio dei nomi per il server delle applicazioni.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Componenti server > Server dei nomi.
