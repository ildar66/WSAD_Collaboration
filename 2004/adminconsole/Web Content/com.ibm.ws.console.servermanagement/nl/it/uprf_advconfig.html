Impostazioni della configurazione dell'avviso

Utilizzare questa pagina per selezionare un avviso che si desidera abilitare o disabilitare.

Per visualizzare questa pagina, fare clic su Server > Server delle applicazioni > nome_server > Configurazione Runtime Performance Advisor > Configurazione avviso.

Scheda Configurazione

Nome avviso
Specifica l'avviso che è possibile abilitare o disabilitare.
Avviso applicato al componente
Specifica il componente WebSphere Application Server a cui si applica l'avviso sulle prestazioni di runtime.
Stato dell'avviso
Specifica se l'avviso è arrestato o avviato.

Esistono solo due valori -- Avviato e Arrestato. Avviato indica che l'avviso, se applicato, è in esecuzione. Arrestato vuol dire che l'avviso non è in esecuzione.

Scheda Runtime

Nome avviso
Specifica l'avviso che è possibile abilitare o disabilitare.
Avviso applicato al componente
Specifica il componente WebSphere Application Server a cui si applica l'avviso sulle prestazioni di runtime.
Stato dell'avviso
Specifica se l'avviso è arrestato, avviato o non disponibile.

Lo stato dell'avviso ha solo uno di questi tre valori -- Avviato, Arrestato o Non disponibile. Avviato vuol dire che l'avviso è applicato. Arrestato indica che l'avviso non è applicato. Non disponibile indica che l'avviso non viene applicato alla configurazione corrente (come ad esempio la Dimensione della sessione permanente in una configurazione senza sessioni persistenti).
