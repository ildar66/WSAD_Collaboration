Raccolta gruppo cache esterno

Utilizzare questa pagina per definire i gruppi di cache esterni controllati da WebSphere Application Server sui server Web, quali IBM Edge Server e IBM HTTP Server.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > server > Servizio cache dinamica > Gruppi cache esterni.

Nome
Specifica il nome del gruppo cache esterno.

Il nome del gruppo cache esterno deve corrispondere alla proprietà externalcache come definito nel file cachespec.xml JSP o servlet.

Quando la memorizzazione nella cache esterna è abilitata, la cache associa le pagine alle URI e invia le pagine associate alla cache esterna. Le voci, quindi, possono essere supportate dalla cache esterna, invece che dal server delle applicazioni.

Tipo
Specifica il tipo di gruppo cache esterno.
