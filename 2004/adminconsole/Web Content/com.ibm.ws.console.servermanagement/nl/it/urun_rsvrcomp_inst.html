Impostazioni Componente server

Utilizzare questa pagina per visualizzare o configurare un'istanza del componente server.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Componenti server > nome_componente_server.

Scheda Configurazione

Name
Specifica il nome del componente.
Tipo dati Stringa
Stato iniziale
Specifica lo stato desiderato del componente all'avvio del processo server. Le opzioni sono: Avviato e Arrestato. L'impostazione predefinita è Avviato.
Tipo dati Stringa
Predefinito Avviato
