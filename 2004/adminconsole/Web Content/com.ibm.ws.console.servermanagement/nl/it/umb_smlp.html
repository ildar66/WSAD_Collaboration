Raccolta porte listener di messaggi

Le porte listener di messaggi configurate nel dominio di gestione

Su questo pannello viene visualizzato un elenco di porte listener di messaggi configurate nel dominio di gestione. Ciascuna porta listener viene utilizzata con un bean basato sui messaggi per ricevere automaticamente i messaggi provenienti da una destinazione JMS associata. È possibile utilizzare questo pannello per aggiungere nuove porte listener o per modificare le proprietà di quelle esistenti. Per ulteriori informazioni sui campi delle proprietà delle porte listener, vedere Proprietà delle porte listener.

Per visualizzare questa pagina della console di gestione, fare clic su Server-> server_applicazioni-> Servizio listener dei messaggi-> Porte Listener
