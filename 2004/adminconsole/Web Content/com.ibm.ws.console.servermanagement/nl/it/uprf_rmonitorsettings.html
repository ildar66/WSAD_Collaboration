Impostazioni del servizio di monitoraggio delle prestazioni

Utilizzare questa pagina per specificare le impostazioni di monitoraggio delle prestazioni, compresa l'abilitazione del monitoraggio, la selezione del modulo PMI e l'impostazione dei livelli di monitoraggio.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > server > Monitoraggio prestazioni.

Scheda Configurazione

Avvio
Specifica se il server delle applicazioni tenta di avviare il servizio specificato. Se un server delle applicazioni viene avviato quando il servizio di monitoraggio delle prestazioni è disabilitato, è necessario riavviare il server per abilitarlo.
Livello della specifica iniziale
Specifica una stringa PMI (Performance Monitoring Infrastructure) che memorizza i livelli di specifica PMI di tutti i componenti del server, ad esempio i livelli del modulo.

Impostare i livelli di specifica PMI selezionando la casella di controllo nessuno, standard o personalizzato. Se viene scelto nessuno, tutti i moduli PMI vengono impostati sul livello nessuno. Scegliendo standard, tutti i moduli PMI vengono impostati su alto e tutti i dati PMI vengono abilitati tranne i dati del livello metodo e i dati JVMPI. Scegliendo Personalizzato, è possibile modificare il livello per ciascun modulo PMI. È possibile impostare il livello su N, L, M, H o X (nessuno, basso, medio, alto e massimo). Si noti che si consiglia di non modificare i nomi dei moduli.
