Connettori JMX (Java Management Extensions)

Utilizzare questa pagina per visualizzare e modificare la configurazione dei connettori JMX (Java Management Extensions).

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server > Servizi di gestione > Connettori JMX

I connettori JMX (Java Management Extensions) comunicano con WebSphere Application Server quando viene richiamato un processo di script. Non esistono valori predefiniti per il tipo e i parametri di un connettore. Il file wsadmin.properties specifica il connettore SOAP (Simple Object Access Protocol) e un numero porta appropriato. È possibile inoltre utilizzare il connettore Remote Method Invocation (RMI).

Utilizzare uno dei seguenti metodi per selezionare il tipo e gli attributi del connettore:

  • Specificare le proprietà in un file di proprietà.
  • Indicare le opzioni sulla riga comandi.
