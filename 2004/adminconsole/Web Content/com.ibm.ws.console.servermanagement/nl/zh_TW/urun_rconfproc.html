程序定義設定

請利用這個頁面來檢視或變更程序定義的設定，以提供啟動或起始設定程序的指令行資訊。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > 程序定義。

「配置」標籤

執行檔名稱
指定程序的執行檔名稱。
資料類型 字串
執行檔引數
指定在程序啟動時執行的執行檔指令。

比方說，可執行的目標程式可能會預期三個引數：arg1 arg2 arg3。

資料類型 字串
單位 Java 指令行引數
工作目錄
指定程序執行所在的檔案系統目錄。

這個目錄用來判斷含相對路徑名稱的輸入和輸出檔的位置。

取消的 Enterprise Bean 放在執行 Bean 的應用程式伺服器現行工作目錄中。 請確定工作目錄是 WebSphere Application Server 產品根目錄下的已知目錄。
