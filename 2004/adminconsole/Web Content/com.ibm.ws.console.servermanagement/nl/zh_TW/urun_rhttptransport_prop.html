HTTP 傳輸設定

您可利用這個頁面來檢視及配置 HTTP 傳輸。頁面的名稱可能是 SSL 設定的名稱，如 DefaultSSLSettings。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > Web 儲存區 > HTTP 傳輸 > host_name。

「配置」標籤

主機
指定傳輸要連結的主機 IP 位址。

如果應用程式伺服器位在本端機器，則主機名稱可能是 localhost。

資料類型 字串
埠
指定傳輸要連結的埠。 請指定在 1 和 65535 之間的埠號。 給定機器中的每個應用程式伺服器都必須有專屬的埠號。
資料類型 整數
範圍 1 至 65535
SSL 已啟用
指定要不要利用 Secure Socket Layer (SSL) 來保護在 WebSphere 外掛程式和應用程式伺服器之間的連線。預設不會使用 SSL。
資料類型 Boolean
預設值 false
SSL
指定 WebSphere 外掛程式與應用程式伺服器之間的連線的 Secure Socket Layer (SSL) 設定類型。 選項包括一個或多個定義在「安全中心」中的 SSL 設定； 例如，DefaultSSLSettings、ORBSSLSettings 或 LDAPSSLSettings。
資料類型 字串
預設值 定義在「安全中心」的 SSL 設定
