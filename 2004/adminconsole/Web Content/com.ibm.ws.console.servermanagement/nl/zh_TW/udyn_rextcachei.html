「外部快取群組」設定

請利用這個頁面來配置在 IBM Edge Server 和 IBM HTTP Server 之類的 Web 伺服器中，WebSphere Application Server 所控制的各組外部快取。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server > 動態快取服務 > 外部快取群組 > external_cache_group。

「配置」標籤

名稱
指定外部快取群組名稱。

外部快取群組名稱需要符合 Servlet 或 JavaServer Pages (JSP) cachespec.xml 檔中所定義的 externalcache 內容。

當啟用外部快取時，快取記憶體會比對頁面及其 URI，再將相符的頁面推到外部快取。 之後，就可以從外部快取記憶體提供項目，而不是從應用程式伺服器提供。 這種能力可以明顯地節省效能。

Type
指定外部快取群組類型。
