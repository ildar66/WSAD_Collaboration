建議配置設定

請利用這個頁面來選取您要啟用或停用的建議。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > 執行時期效能顧問配置 > 建議配置。

「配置」標籤

建議名稱
指定您可以啟用或停用的建議。
元件所適用的建議
指定執行時期效能建議所適用的 WebSphere Application Server 元件。
建議狀態
指定建議是已停止或已啟動。

總共只有兩個值：已啟動和已停止。 已啟動表示在建議適用時，就執行建議。 已停止表示不執行建議。

「執行時期」標籤

建議名稱
指定您可以啟用或停用的建議。
元件所適用的建議
指定執行時期效能建議所適用的 WebSphere Application Server 元件。
建議狀態
指定建議是已停止、已啟動或無法使用。

建議狀態有三個值：已啟動、已停止或無法使用。 已啟動表示正在套用建議。 已停止表示不套用建議。 無法使用表示建議不適合現行配置（如不含持續性階段作業之配置中的「持續保存的階段作業大小」建議）。
