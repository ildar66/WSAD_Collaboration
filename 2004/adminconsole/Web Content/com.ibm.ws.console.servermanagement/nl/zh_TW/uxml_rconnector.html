Java 管理延伸規格連接器

請利用這個頁面來檢視和變更 Java 管理延伸規格 (JMX) 連接器的配置。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > 管理服務 > JMX 連接器。

當您呼叫 Scripting 程序時，Java 管理延伸規格 (JMX) 連接器會與 WebSphere Application Server 通訊。連接器的類型和參數沒有預設值。 wsadmin.properties 檔會指定簡式物件存取通訊協定 (SOAP) 連接器和適當的埠號。 您也可以使用遠端方法呼叫 (RMI) 連接器。

請利用下列方法之一來選取連接器類型和屬性：

  • 在內容檔中指定內容。
  • 在指令行中指出選項。

「配置」標籤

Type
指定 JMX 連接器的類型。
資料類型 列舉
預設值 SOAPConnector
範圍
SOAPConnector
用於使用簡式物件存取通訊協定 (SOAP) 的 JMX 連線。
RMIConnector
用於使用遠端方法呼叫 (RMI) 的 JMX 連線。
HTTPConnector
用於使用 HTTP 的 JMX 連線。
JMSConnector
用於使用 Java 傳訊服務 (JMS) 的 JMX 連線。
