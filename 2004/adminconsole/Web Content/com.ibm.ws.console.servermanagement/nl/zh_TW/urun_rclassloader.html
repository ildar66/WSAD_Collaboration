類別載入器集合

請利用這個頁面來管理應用程式伺服器中的類別載入器實例。 類別載入器會決定應用程式類別載入器或母類別載入器要不要尋找和載入應用程式的 Java 類別檔。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > 類別載入器。

類別載入器 ID
指出使伺服器識別類別載入器實例的唯一字串。 產品會指定這個識別碼。
類別載入器模式
指定當應用程式類別載入器原則是 SINGLE 時的類別載入器模式。 PARENT_FIRST 會使類別載入器在試圖從本端類別路徑中載入類別之前，將載入類別的工作委派給它的母項類別載入器。 PARENT_LAST 會使類別載入器在將類別載入工作委派給母項之前，先嘗試從它的本端類別路徑中載入類別。
