IBM 服務日誌設定

請利用這個頁面來配置 IBM 服務日誌，也稱為活動日誌。

如果要檢視這個管理主控台頁面，請按一下疑難排解 > 日誌和追蹤 > server > IBM 服務日誌。

「配置」標籤

啟用服務日誌
指定 IBM 服務日誌的日誌檔建立程序。
檔案名稱
指定 IBM 服務日誌所用的檔案名稱。
檔案大小上限
指定服務日誌檔的大小上限 (MB)。 預設值是 2 MB。

當到達這個大小時，服務日誌會就地折返。 請注意，服務日誌不像 JVM 日誌，它不會輪替使用新的日誌檔。

訊息過濾
將訊息過濾層次指定成所需要的狀態。 您可以利用這個選項來過濾掉或抑制某些訊息種類。 這個過濾值適用於所有日誌，不只是服務日誌。
啟用相互關係 ID
指定產生隨著每個訊息而記載的相互關係 ID。

您可以利用相互關係 ID 來將活動關聯於特定用戶端要求，或將多個應用程式伺服器中的活動關聯起來。
