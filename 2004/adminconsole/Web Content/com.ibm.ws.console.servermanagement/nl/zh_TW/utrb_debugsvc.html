對服務詳細資料進行除錯

請利用這個頁面，來檢視和修改「對服務進行除錯」所用的設定。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server > 對服務進行除錯。

「配置」標籤

啟動
指定伺服器是否要在啟動時，啟動除錯服務。
JVM 除錯埠
指定 Java 虛擬機器會接聽除錯連線的埠。
JVM 除錯引數
指定以除錯模式啟動 JVM 所用的除錯引數字串。
除錯類別過濾器
指定在進行除錯時，所要忽略的一個類別陣列。 在以逐步模式執行時，除錯器不會停在符合過濾器項目的類別。
BSF 除錯埠
指定 BSF 除錯管理程式所接聽的埠。
BSF 記載層次
指定 BSF 除錯管理程式所提供的記載層次。 有效範圍是 0-3，3 是最高的記載層次。
