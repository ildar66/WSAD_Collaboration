分散式環境設定

請利用這個頁面來指定在分散式環境中儲存階段作業的類型。

如果要檢視這個管理主控台頁面，請按一下伺服器 > 應用程式伺服器 > server_name > Web 儲存區 > 階段作業管理 > 分散式環境設定。

「配置」標籤

分散式階段作業
指定用來儲存階段作業的分散式環境類型。

無 指定在伺服器關閉時，階段作業管理機能要捨棄階段作業資料。
資料庫 指定階段作業管理機能要將階段作業資訊儲存在資料來源連線設定所指定的資料來源中。 請按一下資料庫來變更這些資料來源設定。
記憶體到記憶體抄寫 指定階段作業管理機能要將階段作業資訊儲存在記憶體內的資料來源中。 階段作業資訊會複製至其他階段作業管理機能，供失敗回復使用。 請按一下記憶體到記憶體抄寫來指定要用的抄寫器以及變更這些記憶體至記憶體設定。（只適用於 WebSphere Application Server Network Deployment。）
