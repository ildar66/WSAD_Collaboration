Einstellungen für Prozessausführung

Auf dieser Seite können Sie über die Befehlszeile Informationen zum Starten oder Initialisieren eines UNIX-Prozesses anzeigen oder ändern.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Servername > Prozessdefinition > Prozessausführung.

Register "Konfiguration"

Prozesspriorität
Gibt die Priorität des Prozesses im Betriebssystem an. Dieser Wert kann nur vom Benutzer "root" geändert werden.
Datentyp Integer
Standardwert Für Webcontainer ist der Standardwert unter den meisten Betriebssystemen 1000. Unter OS/400 ist der Standardwert 25.
UMASK
Gibt die Benutzermaske (umask) an, unter der der Prozess ausgeführt wird (die Berechtigungsmaske für Dateimodus).
Datentyp Integer
RunAs-Benutzer
Gibt den Benutzer an, unter dem der Prozess ausgeführt wird.
Datentyp String
RunAs-Gruppe
Gibt die Gruppe an, zu der der Prozess gehört und als die er ausgeführt wird.

Unter OS/400 wird die Einstellung für die RunAs-Gruppe ignoriert.

Datentyp String
RunIn-Prozessgruppe
Gibt eine spezifische Prozessgruppe für den Prozess an. Diese Prozessgruppe ist z. B. im Falle der Partitionierung eines Prozessors nützlich. Ein Systemadministrator kann eine Prozessorgruppe für die Ausführung zuordnen, z. B. 6 von 12 Prozessoren. Der Standardwert (0) sieht vor, den Prozess keiner spezifischen Gruppe zuzuordnen.

Unter OS/400 wird die Einstellung für die RunIn-Prozessgruppe ignoriert.

Datentyp Integer
Standardwert 0
