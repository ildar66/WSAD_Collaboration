Endpoint-Einstellungen

Auf dieser Seite können Sie die Konfiguration für einen Übertragungs-Endpoint, der von Laufzeitkomponenten in einem Prozess verwendet wird, anzeigen und ändern. Endpoints stellen eine Host- und eine Port-Angabe für einen Server bereit.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Servername > Endpoints > Name_des_Endpoint.

Register "Konfiguration"

Name des Endpoint
Gibt den Namen des Endpoint an. Der Name muss auf dem Server eindeutig sein.

Beachten Sie, dass dieses Feld nur angezeigt wird, wenn Sie einen Endpoint für einen Anwendungsserver definieren.

Datentyp String
Host
Die IP-Adresse, der DNS-Hostname mit dem Domänennamenssuffix oder einfach der DNS-Hostname, der von einem Client verwendet wird, um eine Ressource (wie z. B. einen Namensservice, Administrationsservice oder JMS-Broker) anzufordern.

Beispiel: Wenn der Hostname meinhost ist, kann der vollständig qualifizierte DNS-Name meinhost.meineco.com und die IP-Adresse 155.123.88.201 sein.

Datentyp String
Standardwert *
Port
Gibt den Port an, für den der Service konfiguriert wurde, um Clientanforderungen zu empfangen. Der Port-Wert wird zusammen mit dem Hostnamen verwendet.
Datentyp Integer
Standardwert Keiner
