Bibliotheksreferenzen

Auf dieser Seite können Sie Bibliotheksreferenzen, die die Verwendung von globalen Bibliotheken definieren, anzeigen und verwalten. Sie können auf dieser Seite z. B. gemeinsam verwendete Bibliotheksdateien einer implementierten (Deployment) Anwendung zuordnen.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Anwendungen > Enterprise-Anwendungen > Anwendungsname > Bibliotheken.

Bibliotheksname
Gibt den Namen für die Bibliotheksreferenz an.
