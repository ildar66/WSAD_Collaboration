Einstellungen des Dynamic Cache Service

Auf dieser Seite können Sie die Einstellungen für den dynamischen Cache-Service konfigurieren und verwalten.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Server > Dynamischer Cache-Service.

Register "Konfiguration"

Startstatus
Gibt an, ob der dynamische Cache aktiviert ist.
Cache-Größe
Eine positive ganze Zahl als Wert für die Maximalanzahl der im Cache gespeicherten Einträge.

Geben Sie in diesem Feld die Cache-Größe ein. Der gültige Wertebereich liegt zwischen 100 und 200.000.

Standadrdpriorität
Gibt die Standardpriorität für Cache-Einträge an und legt fest, wie lange ein Eintrag in einem vollen Cache bleibt.
Standardwert 1
Plattenauslagerung
Gibt an, ob die Plattenauslagerung aktiviert ist.

Standardmäßig verwaltet der dynamische Cache die Anzahl der im Speicher konfigurierten Einträge. Wenn neue Einträge erstellt werden und der Cache voll ist, werden Einträge nach bestimmten Regeln aus dem Cache entfernt. Diese basieren auf der für die einzelnen Cache-Einträge konfigurierten Priorität und einem Algorithmus für den Zeitpunkt der letzten Verwendung. Sie haben auch die Möglichkeit, eine Plattenauslagerung zu konfigurieren. Die Einträge werden dann nicht aus dem Speicher entfernt, sondern in das Dateisystem kopiert (die Speicherposition ist konfigurierbar), so dass ein späterer Zugriff möglich ist.
