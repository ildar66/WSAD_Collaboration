Details zum Debugging Service

Auf dieser Seite können Sie die Einstellungen des Debugging Service anzeigen und ändern.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Server > Debugging Service.

Register "Konfiguration"

Start
Gibt an, ob der Server versuchen soll, den Debugging Service beim Start des Servers zu starten.
JVM-Debug-Port
Der Port, an dem die JVM (Java Virtual Machine) Debug-Verbindungen überwacht.
JVM-Debug-Parameter
Gibt die Zeichenfolge für den Debug-Parameter an, die verwendet wird, um die JVM im Debug-Modus zu starten.
Filter für Debug-Klassen
Gibt einen Bereich von Klassen an, der beim Debugging ignoriert werden soll. Wenn Sie den Debugger schrittweise ausführen, stoppt er nicht bei den Klassen, die einem Filtereintrag entsprechen.
BSF-Debug-Port
Gibt den Port an, den der BSF-Debug-Manager überwacht.
BSF-Protokollstufe
Gibt die vom BSF-Debug-Manager bereitgestellt Protokollstufe an. Der gültige Bereich ist 0-3, wobei 3 die höchste Protokollstufe darstellt.
