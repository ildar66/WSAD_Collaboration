Cookie-Einstellungen

Auf dieser Seite können Sie Cookie-Einstellungen für die Sitzungsverwaltung konfigurieren.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Server > Anwendungsserver > Servername > Webcontainer > Sitzungsverwaltung > Cookies aktivieren.

Register "Konfiguration"

Cookie-Name
Gibt einen eindeutigen Namen für den Cookie der Sitzungsverwaltung an. Laut Servlet-Spezifikation muss dieser Name JSESSIONID lauten. Aus Flexibilitätsgründen kann dieser Wert jedoch konfiguriert werden.
Sichere Cookies
Gibt an, dass die Sitzungs-Cookies das Feld "Sicher" enthalten. Das Aktivieren der Funktion begrenzt den Austausch von Cookies ausschließlich auf HTTPS-Sitzungen.
Cookie-Domäne
Gibt das Feld "Domäne" eines Cookie für Sitzungsüberwachung an. Dieser Wert legt fest, ob ein Browser ein Cookie an bestimmte Server sendet. Wenn Sie beispielsweise eine bestimmte Domäne angeben, werden Session-Cookies an Hosts in dieser Domäne gesendet. Die Standarddomäne ist der Server.
Cookie-Pfad
Gibt an, dass ein Cookie an den im Pfad angegebenen URL gesendet wird. Geben Sie eine Zeichenfolge an, die einem Pfad auf dem Server entspricht. "/" steht für das Stammverzeichnis. Geben Sie einen Wert ein, um die Pfade, an die das Cookie gesendet wird, einzuschränken. Auf diese Weise können Sie verhindern, dass das Cookie an bestimmte URLs auf dem Server gesendet wird. Wenn Sie das Stammverzeichnis angeben, wird das Cookie immer gesendet, unabhängig davon, auf welchen Pfad auf dem angegebenen Server der Zugriff erfolgt.
Maximale Lebensdauer des Cookie
Gibt den Zeitraum an, für den das Cookie im Client-Browser aktiv bleibt. Sie können festlegen, dass das Cookie für die aktuelle Browser-Sitzung oder für eine maximale Lebensdauer aktiv sein soll. Wenn Sie die Option für maximale Lebensdauer auswählen, müssen Sie die Lebensdauer in Sekunden angeben. Dieser Wert entspricht der Lebensdauer (Time To Live, TTL), die in der Cookie-Spezifikation beschrieben ist.

Der Standardwert ist "-1" und bezeichnet die aktuelle Browser-Sitzung.
