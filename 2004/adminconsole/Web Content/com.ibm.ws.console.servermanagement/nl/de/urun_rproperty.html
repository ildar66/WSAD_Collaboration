Benutzerdefinierte Merkmale

Auf dieser Seite können Sie arbiträre Name/Wert-Paare von Daten anzeigen und verwalten. Dabei ist der Name ein Merkmalschlüssel und der Wert ein Zeichenfolgenwert, der verwendet werden kann, um interne Merkmale der Systemkonfiguration zu definieren.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Servername > Benutzerdefinierte Merkmale.

Name
Gibt den Namen (oder Schlüssel) für das Merkmal an.
Wert
Gibt den Wert zusammen mit dem angegebenen Namen an.
Beschreibung
Stellt Informationen zum Name/Wert-Paar bereit.
Erforderlich
Gibt an, ob im Wertefeld ein Wert eingegeben werden muss. Wenn dieses Feld markiert ist, müssen Sie einen Wert eingeben.
