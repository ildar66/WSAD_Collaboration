Einstellungen für Member externer Cache-Gruppen

Auf dieser Seite können Sie spezifische Caches definieren, die Member einer Cache-Gruppe sind.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Server > Dynamischer Cache-Service > Externe Cache-Gruppe > Externe Cache-Gruppe > Member externer Cache-Gruppe > Member_externer_Cache-Gruppe.

Register "Konfiguration"

Adresse (Address)
Eine Konfigurationszeichenfolge, die von der externen Cache-Adapter-Bean verwendet wird, um eine Verbindung zum externen Cache herzustellen.
Name der Adapter-Bean (AdapterBeanName)
Gibt den Namen der Adapter-Bean an.
AFPA
AdapterBeanName: com.ibm.ws.cache.servlet.Afpa
Address: Port, an dem AFPA empfangsbereit ist.
ESI
AdapterBeanName: com.ibm.websphere.servlet.cache.ESIInvalidatorServlet
Address: lokaler Host
WTE
AdapterBeanName: com.ibm.websphere.edge.dynacache.WteAdapter
Address: Hostname:Port (Hostname und Port, an dem WTE empfangsbereit ist)
