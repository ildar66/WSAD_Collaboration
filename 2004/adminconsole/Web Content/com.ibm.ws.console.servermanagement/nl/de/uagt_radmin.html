Einstellungen für Administrationsservices

Auf dieser Seite können Sie die Konfiguration eines Administrationsservices anzeigen und ändern.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Servername > Administrationsservices.

Register "Konfiguration"

Standalone
Gibt an, ob der Serverprozess an der Zelle von Network Deployment beteiligt ist. Ist dies der Fall, nimmt der Server nicht an der verteilten Administration teil. Ist dies nicht der Fall, ist der Server am Network-Deployment-System beteiligt.

Der Standardwert für Basisinstallationen des WebSphere Application Server ist "true". Wenn addNode ausgeführt wird, um den Server in eine Zelle von Network Deployment einzubinden, wird der Wert in "false" geändert.

Datentyp Boolescher Wert
Standardwert true
Bevorzugter Connector
Gibt den bevorzugten Typ des JMX-Connector an. Verfügbare Optionen, wie z. B. SOAPConnector oder RMIConnector, werden über die Seite "JMX-Connector" definiert.
Datentyp String
