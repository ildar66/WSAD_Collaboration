Einstellungen für Prozessdefinitionen

Verwenden Sie diese Seite, um die Einstellungen für eine Prozessdefinition anzuzeigen oder zu ändern, die Befehlszeileninformationen für das Starten oder Initialisieren eines Prozesses enthält.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Servername > Prozessdefinition.

Register "Konfiguration"

Name der ausführbaren Datei
Gibt den Namen der ausführbaren Datei des Prozesses an.
Datentyp String
Ausführbare Argumente
Gibt die ausführbaren Befehle an, die beim Starten des Prozesses ausgeführt werden.

Beispielsweise kann das ausführbare Zielprogramm drei Parameter erwarten: arg1 arg2 arg3.

Datentyp String
Einheiten Java-Befehlszeilenparameter
Arbeitsverzeichnis
Gibt das Verzeichnis des Dateisystems an, in dem der Prozess ausgeführt wird.

Dieses Verzeichnis wird zur Angabe der Verzeichnisse der Eingabe- und Ausgabedateien mit relativem Pfad verwendet.

Inaktivierte Enterprise-Beans werden in das aktuelle Arbeitsverzeichnis des Anwendungsservers, auf dem die Beans ausgeführt werden, gestellt. Vergewissern Sie sich, dass das Arbeitsverzeichnis unterhalb des Stammverzeichnisses von WebSphere Application Server vorhanden ist.

Datentyp String
