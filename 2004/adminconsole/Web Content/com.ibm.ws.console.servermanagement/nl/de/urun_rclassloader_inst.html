Einstellungen des Klassenladers

Verwenden Sie diese Seite, um einen Klassenlader für Anwendungen, die auf einem Anwendungsserver installiert sind, zu konfigurieren.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Server > Anwendungsserver > Servername > Klassenlader >Klassenlader-ID.

Register "Konfiguration"

Klassenlader-ID
Eine eindeutige Zeichenfolge, mit der der Server die Klassenladerinstanz identifizieren kann. Das Produkt ordnet die ID zu.
Datentyp String
Klassenladermodus
Gibt den Klassenladermodus an, wenn die Policy des Klassenladers der Anwendung SINGLE lautet. Bei Angabe von PARENT_FIRST delegiert der Klassenlader das Laden von Klassen an den übergeordneten Klassenlader und versucht dann, die Klasse vom lokalen Klassenpfad zu laden. Bei Angabe von PARENT_LAST versucht der Klassenlader, Klassen vom lokalen Klassenpfad zu laden, bevor der Ladevorgang an den übergeordneten Klassenlader delegiert wird. Auf diese Weise kann der Klassenlader einer Anwendung seine eigene Version einer Klasse, die im übergeordneten Klassenlader vorhanden ist, überschreiben und bereitstellen.
Datentyp String
Standardwert PARENT_FIRST
