Einstellungen für Serverkomponenten

Auf dieser Seite können Sie eine Serverkomponenteninstanz anzeigen oder konfigurieren.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Servername > Serverkomponenten > Name_der_Serverkomponente.

Register "Konfiguration"

Name
Gibt den Namen der Komponente an.
Datentyp String
Ausgangsstatus
Gibt den gewünschten Status der Komponente an, wenn der Serverprozess gestartet wird. Die Optionen lauten Gestartet und Gestoppt. Der Standardwert ist Gestartet.
Datentyp String
Standardwert Gestartet
