Externe Cache-Gruppen

Auf dieser Seite können Sie Gruppen externer Caches auf vom WebSphere Application Server gesteuerten Webservern, wie z. B. IBM Edge Server und IBM HTTP Server, definieren.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server Anwendungsserver > Server > Dynamischer Cache-Service > Externe Cache-Gruppen.

Name
Gibt den Namen der externen Cache-Gruppe an.

Der Name der externen Cache-Gruppe muss mit dem Merkmal externalcache, wie in der Datei cachespec.xml des Servlet oder der JSP definiert, übereinstimmen.

Bei Aktivierung des externen Caching gleicht der Cache Seiten mit seinen URIs ab und sendet übereinstimmende Seiten an den externen Cache. Die Einträge können dann aus dem externen Cache bereitgestellt werden und müssen nicht vom Anwendungsserver abgerufen werden.

Typ
Gibt den Typ der externen Cache-Gruppe an.
