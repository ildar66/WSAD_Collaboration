Einstellungen des JMX-Connector

Verwenden Sie diese Seite, um die Konfiguration eines JMX-Connector (Java Management Extensions) anzuzeigen.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Server > Anwendungsserver > Servername > Administrationsservices > JMX-Connector > Connector-Typ.

Register "Konfiguration"

Typ
Der Typ des JMX-Connector.
Datentyp Enum
Standardwert SOAPConnector
Bereich
SOAPConnector
Für JMX-Verbindungen, die Simple Object Access Protocol (SOAP) verwenden.
RMIConnector
Für JMX-Verbindungen, die Remote Method Invocation (RMI) verwenden.
HTTPConnector
Für JMX-Verbindungen, die HTTP verwenden.
JMSConnector
Für JMX-Verbindungen, die Java Messaging Service (JMS) verwenden.
