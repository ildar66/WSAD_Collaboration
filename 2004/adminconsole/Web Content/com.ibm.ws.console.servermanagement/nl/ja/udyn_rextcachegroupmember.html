外部キャッシュ・グループ・メンバー・コレクション

このページを使用して、 キャッシュ・グループのメンバーである特定キャッシュを定義します。

この管理コンソール・ページを表示するには、「サーバー」>「アプリケーション・サーバー」>「server」>「動的キャッシュ・サービス」>「外部キャッシュ・グループ」>「external_cache_group」>「External cache group members」をクリックします。

AdapterBeanName
アダプター Bean 名を指定します。

WebSphere Application Server でサポートされているアダプター Bean 名の例は以下のとおりです。
AFPA
AdapterBeanName: com.ibm.ws.cache.servlet.Afpa
Address: afpa が listen するポート
ESI
AdapterBeanName: com.ibm.websphere.servlet.cache.ESIInvalidatorServlet
Address: ローカル・ホスト
WTE
AdapterBeanName: com.ibm.websphere.edge.dynacache.WteAdapter
Address: hostname:port (WTE が listen する ホスト名とポート)
