プロセス定義設定

このページを使用して、プロセス定義の設定値を表示または変更します。これにより、プロセスの始動または初期化 のためのコマンド行情報を提供します。

この管理コンソール・ページを表示するには、「 サーバー」>「アプリケーション・サーバー」>「server_name」>「プロセス定義」をクリックします。

「構成」タブ

実行可能名
プロセスの実行可能名を指定します。
データ型 ストリング
実行可能の引き数
プロセスの始動時に実行する実行可能コマンドを指定します。

例えば、実行可能なターゲット・プログラムは、arg1 arg2 arg3 の 3 つの引き数を取ります。

データ型 ストリング
単位 Java コマンド行の引き数
作業ディレクトリー
プロセスが稼働するファイル・システム・ディレクトリーを指定します。

このディレクトリーを使用して、 相対パス名を持つ入出力ファイルのロケーションを判別します。

非活性化された Enterprise Bean は、 Bean が実行されるアプリケーション・サーバーの現行作業ディレクトリーに置かれます。 作業ディレクトリーは、WebSphere Application Server 製品のルート・ディレクトリーの下にある 既知のディレクトリーになるようにしてください。

データ型 ストリング
