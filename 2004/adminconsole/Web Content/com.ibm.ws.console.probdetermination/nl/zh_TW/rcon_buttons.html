管理主控台按鈕

以下是管理主控台各頁面中的按鈕選項，這會隨著您啟用的產品特性而不同。

  • 中止。中止還沒有進入準備狀態的交易。交易已完成的所有作業都會還原。
  • 新增。將選取或輸入的項目新增至清單中，或產生新增項目至清單中的對話框。
  • 套用。將您的變更儲存至頁面中，但不結束頁面。
  • 上一步。顯示序列中的前一個頁面或項目。請注意，管理主控台不支援使用瀏覽器的「向後」和「向前」按鈕。 使用這些按鈕可能會造成間歇性的問題。 請改用管理主控台中的「向後」或「取消」按鈕。
  • 瀏覽。開啟一個對話框，讓您在系統中尋找檔案。
  • 取消。結束現行頁面或對話框，捨棄未儲存的變更。請注意，管理主控台不支援使用瀏覽器的「向後」和「向前」按鈕。 使用這些按鈕可能會造成間歇性的問題。 請改用管理主控台中的「取消」按鈕。
  • 變更。在安全環境定義中，可讓您搜尋應用程式執行時所採用的使用者 ID 的使用者登錄。 在儲存區內容的環境定義中，可讓您改變儲存區正在使用的資料來源。
  • 清除。清除您的變更，還原最近儲存的值。
  • 清除選項。清除這個標籤頁中各表格內所選取的任何資料格。
  • 關閉。結束這個對話框。
  • 確定。釋出準備好的交易所保留的所有鎖定，強迫交易確定。
  • 複製。建立所選應用程式伺服器的複本。
  • 建立。儲存對話框中所有標籤頁的變更，並結束對話框。
  • 刪除。移除選取的實例。
  • 詳細資料。顯示交易的詳細資料。
  • 完成。儲存對話框中所有標籤頁的變更，並結束對話框。
  • 向下。在清單中向下移動。
  • 傾出。啟動所追蹤應用程式伺服器的傾出。
  • 編輯。讓您編輯清單中所選取的項目，或產生對話框以編輯項目。
  • 匯出。存取用來匯出企業應用程式 EAR 檔的頁面。
  • 匯出 DDL。存取用來匯出企業應用程式 DDL 檔的頁面。
  • 匯出金鑰。將 LTPA 金鑰匯出至其他領域中。
  • 過濾。產生一個對話框來指定這個標籤頁中的各表格所要檢視的資源。
  • 完成。強迫交易完成，不論交易的輸出有沒有報告給所有參與的應用程式都是如此。
  • 第一個。顯示記錄序列中的第一個記錄。
  • 產生金鑰。產生新的 LTPA 金鑰。 當第一次開啟安全並以 LTPA 作為鑑別機制時，會自動以畫面中輸入的密碼來產生 LTPA 金鑰。 如果需要產生新的金鑰，請在開啟安全且備份好伺服器之後，使用這個按鈕。 按一下這個按鈕會產生金鑰，且會將它們延伸至所有作用中的伺服器（Cell、節點和應用程式伺服器）。 新的金鑰可用來加密和解密 LTPA 記號。 請按一下主控台工作列中的儲存，將新的金鑰和密碼儲存在儲存庫中。
  • 立即停止。停止伺服器，但在關閉整個伺服器程序之前，略過讓進行中的要求完成的正常伺服器靜止程序。 這個關閉模式的速度比正常伺服器停止處理程序快，但有些應用程式用戶端可能會收到異常狀況。
  • 匯入金鑰。從其他領域中匯入新的 LTPA 金鑰。 如果要跨越多個 WebSphere 領域 (Cell) 來支援 WebSphere 中的單一登入 (SSO)，這些領域應該共用 LTPA 金鑰和密碼。 從其中一個 Cell 將金鑰匯出至檔案中之後，請按一下這個按鈕，將金鑰匯入所有作用中的伺服器（Cell、節點和應用程式伺服器）。 新的金鑰可用來加密和解密 LTPA 記號。 請按一下主控台工作列中的儲存，將新的金鑰和密碼儲存在儲存庫中。
  • 安裝。顯示「準備安裝應用程式」頁面，讓您將應用程式、EJB 或 Web 元件部署在應用程式伺服器中。
  • 安裝 RAR。安裝 RAR 按鈕會開啟一個對話框，讓您能夠安裝 JCA 連接器及建立它的資源配接器。
  • 管理交易。顯示一份清單來列出在伺服器中執行的作用中交易清單。 您可以強迫完成任何因無法使用交易資源而停止處理的交易。
  • 修改。開啟用來變更規格的對話框。
  • 移動。將選取的應用程式伺服器移至管理 Cell 的不同位置。當出現提示時，請指定目標位置。
  • 新建。顯示用來定義新實例的頁面。 比方說，按一下「應用程式伺服器」頁面的新建會顯示一個可用來配置新應用程式伺服器的頁面。
  • 下一步。顯示序列中的下一頁、頁框或項目。
  • 確定。儲存您的變更，結束頁面。
  • 連通測試。試圖聯絡已選取的應用程式伺服器。
  • 上一頁。顯示序列中的上一頁、頁框或項目。
  • 退出。結束對話框，捨棄任何未儲存的變更。
  • 重新整理。重新整理這個標籤頁目前列出的實例之資料視圖。
  • 移除。刪除選取的項目。
  • 重設。清除您在標籤或頁面中的變更，還原最近儲存的值。
  • 重新啟動。停止選取的物件，再重新啟動它們。
  • 擷取新記錄。擷取新的記錄。
  • 儲存。將本端配置中的變更儲存在主要配置中。
  • 選取。在資源分析上，可讓您選取資源的監視範圍。
  • 設定。儲存對話框中之設定的變更。
  • 設定值。顯示用來編輯 Servlet 相關資源設定值的對話框。
  • 使用中的設定值。產生一個對話框來顯示目前在使用中的設定值。
  • 啟動。在應用程式伺服器的環境定義中，啟動選取的應用程式伺服器。 在資料收集的環境定義中，啟動這個標籤頁的表格之資料收集。
  • 停止。在應用程式伺服器之類的伺服器元件之環境定義中，停止選取的伺服器元件。 在資料收集的環境定義中，停止收集標籤頁表格的資料收集。
  • 解除安裝。從 WebSphere Application Server 配置儲存庫中，刪除已部署的應用程式。 另外，也從檔案系統中刪除應用程式二進位檔。
  • 更新。以更新的應用程式來取代部署在伺服器的應用程式。 在更新過程中，您可能需要完成「準備安裝應用程式」和「更新應用程式」頁面中的步驟。
  • 資源資源清單。更新表格中的資料。 發現新實例，並將新實例新增到表格中。
  • 檢視。開啟檔案的對話框。
