IBM サービス・ログ設定

このページを使用して、IBM 保守ログ (アクティビティー・ログ ともいう) を構成します。

この管理コンソール・ページを表示するには、「トラブルシューティング」>「Logs and Trace」>「server 」>「IBM 保守ログ」をクリックします。

「構成」タブ

保守ログを使用可能にする
IBM 保守ログによるログ・ファイルの作成を指定します。
ファイル名
IBM 保守ログで使用するファイルの名前を指定します。
最大ファイル・サイズ
保守ログ・ファイルの最大サイズを MB 単位で指定します。 デフォルト値は、2 MB です。

この最大サイズに達すると、保守ログは決まった所で循環します。 保守ログは JVM ログのように新しいログ・ファイルにロールオーバーすることはありません。

メッセージのフィルター
メッセージ・フィルター・レベルを希望の状態に指定します。 これは、メッセージの一部のカテゴリーをフィルターに掛けたり、 非表示にしたりする場合に使用することができます。 このフィルター値は、保守ログだけでなくあらゆるログに適用されます。
相関 ID を使用可能にする
各メッセージと共に記録される相関 ID の生成を指定します。

相関 ID を使用すれば、アクティビティーを特定のクライアント要求に関連付けたり、 複数のアプリケーション・サーバー上のアクティビティーを関連付けたりすることができます。
