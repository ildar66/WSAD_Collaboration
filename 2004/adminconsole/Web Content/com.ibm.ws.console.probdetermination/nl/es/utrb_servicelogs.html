Valores de las anotaciones cronológicas de servicio de IBM

Utilice esta página para configurar las anotaciones cronológicas de servicio de IBM, también conocidas como anotaciones cronológicas de actividad.

Para ver esta página de la consola administrativa, Determinación de problemas > Anotaciones cronológicas y rastreo > servidor > Anotaciones cronológicas de servicio de IBM.

Pestaña Configuración

Habilitar anotaciones cronológicas de servicio
Especifica la creación de un archivo de anotaciones cronológicas por las anotaciones cronológicas de servicio de IBM.
Nombre de archivo
Especifica el nombre del archivo que utiliza las anotaciones cronológicas de servicio de IBM.
Tamaño máximo de archivo
Especifica el tamaño máximo en megabytes del archivo de anotaciones cronológicas de servicio. El valor por omisión es 2 megabytes.

Cuando se alcanza este tamaño, el archivo de anotaciones cronológicas de servicio se acomoda en su sitio. Tenga en cuenta que las anotaciones cronológicas de servicio no se recuperan en un nuevo archivo de anotaciones cronológicas como el de anotaciones cronológicas de JVM.

Filtro de mensajes
Especifica el nivel de filtro de mensajes con el estado deseado. Puede utilizar esta opción para filtrar o suprimir algunas categorías de mensajes. Este valor de filtro se aplica a todas las anotaciones cronológicas, no sólo a las anotaciones cronológicas de servicio.
Habilitar ID de correlación
Especifica la generación de un ID de correlación que se anota con cada mensaje.

Puede utilizar el ID de correlación para correlacionar la actividad con una petición de cliente determinada o para correlacionar actividades en varios servidores de aplicaciones.
