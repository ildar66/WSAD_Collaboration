Métrica de petición de PMI (Performance Monitoring Infrastructure)

Utilice esta página para habilitar PMI (Performance Monitoring Infrastructure) Request Metrics, habilitar ARM (Application Response Measurement) de Request Metrics y establecer niveles de rastreo.

Para ver esta página de la consola administrativa, pulse Resolución de problemas > PMI Request Metrics.

Pestaña Configuración

Request Metrics
Habilita PMI Request Metrics.

Cuando se inhabilita, la función de Request Metrics se inhabilita.

ARM (Medida de respuestas de aplicación)
Habilita PMI Request Metrics para llamar un agente ARM subyacente.

Antes de habilitar ARM, instale una implementación ARM adecuada en todos los nodos de WebSphere Application Server. Verifique con el proveedor del agente ARM que la implementación del agente ARM dé soporte a Request Metrics. El soporte de ARM depende del soporte de Request Metrics.

Nivel de rastreo
Especifica la cantidad de datos de rastreo que se puede acumular para una determinada petición.

Debe incluir uno de los siguientes valores: NONE - sin rastreo; HOPS - sólo hay acumulación en los paquetes de proceso principales; PERF_DEBUG - se permite información adicional en los paquetes, pero el rendimiento no es tan intensivo como con DEBUG; DEBUG - rastreo completamente detallado.
