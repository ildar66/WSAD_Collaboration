Valores de filtro PMIRM

Utilice esta página para especificar los filtros que determinan si se debe habilitar o no el rastreo para la petición a medida que ésta se desplaza por WebSphere Application Server.

Para ver esta página de la consola administrativa, pulse Resolución de problemas > PMI Request Metrics > filtros > tipo_filtro.

Pestaña Configuración

Tipo
Especifica el tipo de filtro de Request Metrics.
Habilitar
Especifica si este filtro está habilitado.
