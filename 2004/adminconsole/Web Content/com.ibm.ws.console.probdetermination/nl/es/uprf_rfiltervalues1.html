filterValues (colección)

Utilice esta página para especificar los valores de filtros de IP de cliente o de Request Metrics de EJB o URI.

Para ver esta página de la consola administrativa, pulse Resolución de problemas > PMI Request Metrics > filtros > tipo_filtro > filterValues.

Pestaña Configuración

Valor
Especifica un valor de URI o nombre IP basado en el tipo de filtro.

Por ejemplo, para filtros URI, el valor podría ser "/servlet/snoop".

Habilitar filtro
Especifica si un valor de filtro está habilitado.
