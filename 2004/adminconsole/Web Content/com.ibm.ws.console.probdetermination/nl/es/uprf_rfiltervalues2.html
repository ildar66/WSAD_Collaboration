filterValues (valores)

Utilice esta página para especificar los valores de filtros de IP de cliente o de Request Metrics de EJB o URI.

Para ver esta página de la consola administrativa, pulse Resolución de problemas > PMI Request Metrics > filtros > filtro > filterValues > valor_filtro.

Pestaña Configuración

Valor
Especifica un valor de URI o nombre IP basado en el tipo de filtro.

Por ejemplo, para filtros URI, el valor puede ser "/servlet/snoop".

Habilitar
Especifica si este valor de filtro está habilitado.
