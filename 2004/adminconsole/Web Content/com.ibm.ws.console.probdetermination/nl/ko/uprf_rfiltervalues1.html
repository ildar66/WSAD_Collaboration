필터 값 콜렉션

이 페이지를 참조하여 클라이언트 IP, URI 또는 EJB 요청 매트릭스 필터에 대한 값을 지정하십시오.

이 관리 콘솔 페이지를 보려면 문제점 해결 > PMI 요청 매트릭스 > 필터 > filter_type > 필터 값을 클릭하십시오.

구성 탭

값
필터 유형에 기초하여 URI 값 또는 IP 이름을 지정합니다.

예를 들어, URI 필터에 대해 값은 "/servlet/snoop"일 수 있습니다.

필터 사용 가능
필터 값이 사용 가능한지 여부를 지정합니다.
