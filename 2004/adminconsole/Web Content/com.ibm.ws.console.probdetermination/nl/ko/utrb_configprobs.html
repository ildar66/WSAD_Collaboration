구성 문제 설정

이 페이지를 참조하여 현재 구성의 문제점을 식별하고 보십시오.

이 관리 콘솔 페이지를 보려면 콘솔 탐색 트리에서 문제점 해결 > 구성 문제를 클릭하십시오.

구성 문제 테이블에서 구성 문제를 클릭하여 문제점에 대한 자세한 정보를 보십시오.

구성 문서 유효성 검증
이러한 필드를 사용하여 구성 문서에서 수행할 유효성 검증 레벨을 지정하십시오.
상호 유효성 검증
구성 문서에서 교차 유효성 검증을 사용 가능하게 합니다.

교차 유효성 검증을 사용 가능하게 하면 구성 문서에서 충돌하는 설정을 비교할 수 있습니다.

구성 문제

현재 구성 문제 오류 메시지를 표시합니다. 문제점에 대한 자세한 정보는 메시지를 클릭하십시오.

범위

각 오류가 발생하는 구성 파일별로 구성 문제 목록을 분류하십시오. 문제점에 대한 자세한 정보는 메시지를 클릭하십시오.

각 구성 문제를 설명하는 필드:

구성 문제 메시지를 클릭하면 다음 정보 필드가 나타납니다.

메시지
유효성 검증 프로그램에서 리턴된 메시지를 표시합니다.
설명
문제점에 대한 간단한 설명.
사용자 조치
프로그램을 정정할 수 있는 권장 조치를 지정합니다.
대상 오브젝트
유효성 검증 오류가 발생한 구성 오브젝트를 식별합니다.
심각도
심각한 오류 1을 가진 구성 오류의 심각도를 표시합니다. 심각도는 심각도 설명자가 증가할수록 감소합니다.
로컬 URI
오류가 발생한 구성 파일의 로컬 URI를 지정합니다.
전체 URI
오류가 발생한 구성 파일의 전체 URI를 지정합니다.
유효성 검증기 클래스 이름
문제점을 보고하는 유효성 검증기의 클래스 이름.
