PMI 요청 매트릭스

이 페이지를 참조하여 PMI(Performance Monitoring Infrastructure) 요청 매트릭스 및 요청 매트릭스 ARM(Application Response Measurement)을 사용 가능하게 하고 추적 레벨을 설정하십시오.

이 관리 콘솔 페이지를 보려면 문제점 해결 > PMI 요청 매트릭스를 클릭하십시오.

구성 탭

요청 매트릭스
PMI 요청 매트릭스를 사용 가능하게 하십시오.

사용 불가능한 경우, 요청 매트릭스 기능을 사용할 수 없습니다.

ARM(Application Response Measurement)
PMI 요청 매트릭스를 사용 가능하게 하여 기본 ARM 에이전트를 호출하십시오.

ARM을 사용 가능하게 하기 전에, 모든 WebSphere Application Server 노드에 적절한 ARM 구현을 설치하십시오. ARM 에이전트 프로바이더를 사용하여 ARM 에이전트 구현이 요청 매트릭스를 지원하는지 확인하십시오. ARM 지원은 요청 매트릭스 지원에 종속적입니다.

추적 레벨
지정된 요청에 대한 누적을 위해 추적할 데이터 양을 지정합니다.

다음 중 하나 포함: NONE - 추적하지 않음; HOPS - 주요 프로세스 홉에 대한 누적만; PERF_DEBUG - 홉에 대한 추가 정보를 사용 가능하게 하지만 DEBUG만큼 성능 강화 형식은 아님; DEBUG - 완전한 세부사항 추적.
