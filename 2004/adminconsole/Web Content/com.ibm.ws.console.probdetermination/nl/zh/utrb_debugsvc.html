调试服务详细信息

使用此页面查看和修改调试服务使用的设置。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server > 调试服务。

“配置”选项卡

启动
指定服务器启动时服务器是否将尝试启动调试服务。
JVM 调试端口
指定 Java 虚拟机将为调试连接侦听的端口。
JVM 调试自变量
指定用于以调试方式启动 JVM 的调试自变量字符串。
调试类过滤器
指定调试期间要忽略的类数组。当以循序渐进的方式运行时，调试器不会在匹配过滤器条目的类中停止。
BSF 调试端口
指定 BSF 调试管理器侦听的端口。
BSF 记录级别
指定 BSF 调试管理器提供的记录级别。 有效的范围是 0-3，用 3 来表示记录的最高级别。
