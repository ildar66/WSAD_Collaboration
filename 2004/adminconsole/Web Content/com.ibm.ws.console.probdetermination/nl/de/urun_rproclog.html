Einstellungen für Prozessprotokolle

Auf dieser Seite können Sie Einstellungen für die Dateien des Standardausgabe- und des Standardfehlerdatenstroms anzeigen oder ändern.

Klicken Sie, um diese Seite der Administrationskonsole anzuzeigen, auf Server > Anwendungsserver > Servername > Prozessdefinition > Prozessprotokolle.

Register "Konfiguration"

Name der Datei für Standardausgabe
Die Datei, an die der Standardausgabedatenstrom gesendet wird. Der Dateiname kann einen symbolischen Pfad enthalten, der in den Variableneinträgen definiert ist.

Geben Sie in dem Feld auf der Registerseite "Konfiguration" den Dateinamen ein. Verwenden Sie das Feld auf der Registerseite "Laufzeit", um eine Datei zur Anzeige auszuwählen. Klicken Sie auf Anzeigen, um die Datei anzuzeigen.

Sie können die Serverausgabe an die Administrationskonsole oder an den Prozess, der den Server gestartet hat, senden. Dazu löschen Sie den Dateinamen oder geben auf der Registerseite "Konfiguration" console an.

Datentyp String
Einheiten Vollständiger Pfadname
