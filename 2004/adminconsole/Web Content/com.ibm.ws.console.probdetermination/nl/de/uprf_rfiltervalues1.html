filterValues

Auf dieser Seite können Sie die Filterwerte für Request Metrics bezüglich Client-IP, URI oder EJB angeben.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Fehlerbehebung > PMI Request Metrics > Filter > Filtertyp > filterValues.

Register "Konfiguration"

Wert
Gibt einen URI-Wert oder einen IP-Namen auf der Grundlage des Filtertyps an.

Beispielsweise kann für URI-Filter der Wert "/servlet/snoop" angegeben werden.

Filter aktivieren
Gibt an, ob ein Filterwert aktiviert ist.
