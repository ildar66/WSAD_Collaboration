Einstellungen für PMIRM-Filter

Auf dieser Seite können Sie Filter angeben, die definieren, ob das Tracing für eine auf dem WebSphere Application Server ausgeführte Anforderung aktiviert wird.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Fehlerbehebung > PMI Request Metrics > Filter > Filtertyp.

Register "Konfiguration"

Typ
Der Typ des PMI-Filters.
Aktivieren
Gibt an, ob dieser Filter aktiviert ist.
