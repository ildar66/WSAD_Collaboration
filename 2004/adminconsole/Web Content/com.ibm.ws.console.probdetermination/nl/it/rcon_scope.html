Impostazioni ambito della console di gestione

Utilizzare le impostazioni Ambito per filtrare i contenuti di una tabella di raccolta della console di gestione ad esempio per una cella, nodo o server particolare. La modifica al valore di Ambito consente di visualizzare le altre variabili che si applicano a un oggetto e può modificare i contenuti della tabella di raccolta.

Fare clic su Sfoglia, accanto a un campo, per visualizzare le opzioni disponibili per limitare l'ambito del campo. Se un campo è a sola lettura, non è possibile modificare l'ambito. Ad esempio, se è presente solo un server, non è possibile trasferire l'ambito a un altro server.

Cella
Limita la visibilità a tutti i server sulla cella specificata.
Nodo
Limita la visibilità a tutti i server sul nodo specificato.
Server
Limita la visibilità al server specificato.

L'ambito server ha la precedenza sugli ambiti nodo e cella mentre l'ambito nodo ha la precedenza su quello della cella. Notare che gli oggetti sono creati in un solo ambito ma potrebbero essere visibili anche in altri ambiti.
