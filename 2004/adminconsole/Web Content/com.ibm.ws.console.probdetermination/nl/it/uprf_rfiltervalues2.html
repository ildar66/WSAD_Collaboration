Impostazioni filterValues

Utilizzare questa pagina per specificare i valori per i filtri delle metriche di richiesta EJB, URI o IP client.

Per visualizzare questa pagina della console di gestione, fare clic su Risoluzione dei problemi > Metriche di richiesta PMI > filtri > filtro > filterValues > valore_filtro.

Scheda Configurazione

Valore
Specifica un valore URI o un nome IP in base al tipo di filtro.

Ad esempio, per i filtri URI, il valore può essere "/servlet/snoop".

Attivazione
Specifica se il valore del filtro è abilitato.
