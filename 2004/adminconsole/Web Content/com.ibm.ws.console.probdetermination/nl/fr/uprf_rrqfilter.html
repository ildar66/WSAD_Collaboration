Paramètres des filtres PMIRM

Cette page permet de spécifier des filtres déterminant si la trace est activée ou non pour la demande lors de son cheminement dans WebSphere Application Server.

Pour afficher cette page de la console d'administration, cliquez sur Résolution des incidents > PMI Request Metrics > filters > type_filtre.

Onglet Configuration

Type
Indique le type du filtre Request Metrics.
Activer
Indique si le filtre est activé.
