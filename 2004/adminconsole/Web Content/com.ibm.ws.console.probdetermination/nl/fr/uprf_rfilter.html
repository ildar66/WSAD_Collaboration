Collection de filtres PMIRM

Cette page permet d'afficher la liste des filtres Performance Monitoring Infrastructure (PMI) Request Metrics.

Pour afficher cette page de la console d'administration, cliquez sur Résolution des incidents > PMI Request Metrics > Filtres.

Type
Spécifie le type de filtre de mesures de demande.
Activer
Indique si le filtre est activé.
