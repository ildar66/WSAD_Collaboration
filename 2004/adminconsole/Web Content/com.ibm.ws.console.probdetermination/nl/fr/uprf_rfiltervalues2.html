Valeurs de filterValues

Cette page permet d'indiquer les valeurs des filtres Request Metrics d'EJB, d'URI et d'IP client.

Pour afficher cette page de la console d'administration, cliquez sur Résolution des incidents > PMI Request Metrics > filters filtre > filterValues > valeur_filtre.

Onglet Configuration

Valeur
Indique une valeur d'URI ou un nom IP, en fonction du type de filtre.

Par exemple, pour les filtres d'URI, la valeur peut être "/servlet/snoop".

Activer
Spécifie si cette valeur de filtre est activée.
