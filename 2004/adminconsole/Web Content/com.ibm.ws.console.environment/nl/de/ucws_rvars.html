WebSphere-Variablen

Auf dieser Seite können Sie eine Liste der Substitutionsvariablen mit deren Werten und deren Bereich anzeigen und ändern.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Umgebung > WebSphere-Variablen verwalten.

Wenn Sie auf eine Variable klicken, wird das Feld Beschreibung mit Informationen zur Variable angezeigt.

Name
Gibt einen symbolischen Namen an, der einen Dateipfad, eine Webadresse oder einen anderen Wert darstellt.
Wert
Gibt den Wert des symbolischen Namens an, z. B. den absoluten Dateipfad.
Bereich
Gibt an, ob der symbolische Name auf eine Zelle, einen Knoten oder Server angewendet wird.
