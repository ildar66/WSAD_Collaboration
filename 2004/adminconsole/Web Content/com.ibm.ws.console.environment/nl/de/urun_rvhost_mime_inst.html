Einstellungen für MIME-Typ

Auf dieser Seite können Sie einen MIME-Objekttyp (Multi-Purpose Internet Mail Extensions) konfigurieren.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Umgebung > Virtuelle Hosts > Name_des_virtuellen_Host > MIME-Typen > MIME-Typ.

Register "Konfiguration"

MIME-Typ
Gibt einen MIME-Typ an, z. B. "text", "image" oder "application".

Ein Beispielwert für einen MIME-Typ ist text/html. Ein Standardwert wird nur angezeigt, wenn Sie die Konfiguration einer vorhandenen Instanz anzeigen.

Datentyp String
Erweiterungen
Listet die Erweiterungen auf, die dem MIME-Typ zugeordnet werden.

Dateierweiterungen für den MIME-Typ text/html sind z. B. .htm, .html und .txt. Ein Standardwert wird nur angezeigt, wenn Sie die Konfiguration einer vorhandenen Instanz anzeigen.
