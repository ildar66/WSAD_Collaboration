Einstellungen für virtuelle Hosts

Auf dieser Seite können Sie die Instanz eines virtuellen Host konfigurieren.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Umgebung > Virtuelle Hosts > Name_des_virtuellen_Host.

Register "Konfiguration"

Name
Ein logischer Name, der für die Konfiguration von Webanwendungen für einen bestimmten Hostnamen verwendet wird. Der virtuelle Standard-Host ist in der Regel für die meisten einfachen Konfigurationen geeignet.
Datentyp String
Standardwert Standardhost (default_host)
