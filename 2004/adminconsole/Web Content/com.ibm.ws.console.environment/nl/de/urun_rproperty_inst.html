Einstellungen für benutzerdefinierte Merkmale

Auf dieser Seite können Sie arbiträre Name/Wert-Paare von Daten anzeigen und verwalten. Dabei ist der Name ein Merkmalschlüssel und der Wert ein Zeichenfolgenwert, der verwendet werden kann, um interne Merkmale der Systemkonfiguration zu definieren. Durch die Definition eines neuen Merkmals können Sie eine Einstellung konfigurieren, die über die Einstellungen der Administrationskonsole hinausgeht.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Server > Anwendungsserver > Servername > Benutzerdefinierte Merkmale > Merkmalsname

Register "Konfiguration"

Name
Gibt den Namen (oder Schlüssel) für das Merkmal an.
Datentyp String
Wert
Gibt den Wert zusammen mit dem angegebenen Namen an.
Datentyp String
Beschreibung
Stellt Informationen zum Name/Wert-Paar bereit.
Datentyp String
