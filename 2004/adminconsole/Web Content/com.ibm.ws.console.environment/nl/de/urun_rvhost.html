Virtuelle Hosts

Auf dieser Seite können Sie virtuelle Hosts verwalten.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Umgebung > Virtuelle Hosts.

Name
Ein logischer Name, der für die Konfiguration von Webanwendungen für einen bestimmten Host-Namen verwendet wird. Der virtuelle Standard-Host ist in der Regel für die meisten einfachen Konfigurationen geeignet.

Die Konfiguration eines virtuellen Host ermöglicht einer einzelnen Host-Maschine, wie mehrere Host-Maschinen aufzutreten. Ressourcen, die einem virtuellen Host zugeordnet sind, können mit Ressourcen, die einem anderen virtuellen Host zugeordnet sind, keine Daten gemeinsam benutzen, selbst wenn die virtuellen Hosts auf derselben Maschine installiert sind.
