Einstellungen für Hostaliasnamen

Auf dieser Seite können Sie einen Hostaliasnamen anzeigen und konfigurieren.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Umgebung > Virtuelle Hosts > Name_des_virtuellen_Host> HostaliasnamenHostaliasname.

Register "Konfiguration"

Hostname
Die IP-Adresse, der DNS-Hostname mit dem Domänennamensuffix oder der DNS-Hostname, der von den Clients verwendet wird, um eine Webanwendungsressource, z. B. ein Servlet, eine JSP-Datei oder eine HTML-Seite anzufordern.

Beispielsweise wird für den Hostalianamen meinHost der DNS-Name meinHost:8080 verwendet, wobei 8080 der Port ist. Ein URL verweist wie folgt auf diesen DNS: http://meinHost:8080/servlet/snoop.

Für vorhandene Instanzen gibt der Standardwert den Wert wieder, der bei der Konfiguration des Produkts angegeben wurde. Für neue Instanzen kann der Standardwert * angegeben werden, damit jeder Wert zulässig ist bzw. keine Spezifikation festgelegt wird.

Datentyp String
Standardwert *
Port
Der Port, an dem der Webserver Clientanforderungen akzeptiert. Geben Sie einen Port zusammen mit dem Hostnamen an.

Der Standardwert gibt den Wert wieder, der bei der Konfiguration des Produkts angegeben wurde. Der Standardwert kann 80, 81, 9080 oder ein ähnlicher Wert sein.

Datentyp Integer
Standardwert 80
