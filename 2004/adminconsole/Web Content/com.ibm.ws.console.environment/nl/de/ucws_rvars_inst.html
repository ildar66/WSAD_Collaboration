Einstellungen für Variablen

Auf dieser Seite können Sie den Namen und den Wert einer WAS-Substitutionsvariablen definieren.

Klicken Sie zum Anzeigen dieser Seite der Administrationskonsole auf Umgebung > WebSphere-Variablen verwalten > Name_der_WebSphere-Variablen.

Register "Konfiguration"

Name
Gibt einen symbolischen Namen an, der einen Dateipfad, eine Webadresse oder einen anderen Wert darstellt.

WebSphere Application Server ersetzt den symbolischen Namen überall dort, wo er im System erscheint.

Beispiel: "CONFIG_ROOT" ist der symbolische Name für den Konfigurationsverzeichnispfad "C:\WebSphere\AppServer\Config" für das WAS-Basisprodukt auf einem Windows-System.

Datentyp String
Wert
Gibt den Wert des symbolischen Namens an, z. B. den absoluten Dateipfad.

Beispielsweise könnte es sich bei dem Wert im WAS-Basisprodukt auf einem Windows-System um den absoluten Dateipfad "C:\WebSphere\AppServer\Config" handeln. Der zugehörige symbolische Name könnte "CONFIG_ROOT" lauten.

Datentyp String
Beschreibung
Gibt den Zweck einer Variablen an.
