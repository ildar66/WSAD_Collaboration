변수 설정

WebSphere Application Server 대체 변수의 이름 및 값을 정의하려면 이 페이지를 참조하십시오.

이 관리 콘솔 페이지를 보려면 환경 > WebSphere 변수 관리 > WebSphere_variable_name을 클릭하십시오.

구성 탭

이름
파일 경로, 웹 주소 또는 기타 값을 표시하는 기호 이름을 지정합니다.

WebSphere Application Server는 시스템에 값이 나타나는 기호 이름을 대체합니다.

예를 들어, "CONFIG_ROOT"는 Windows 시스템의 기본 WebSphere Application Server 제품에 대한 구성 디렉토리 경로("C:\WebSphere\AppServer\Config")를 나타내는 기호 이름일 수 있습니다.

데이터 유형 문자열
값
기호 이름이 나타내는 값(예: 절대 파일 경로)을 지정합니다.

예를 들어, 값은 Windows 시스템에 있는 기본 WebSphere Application Server 제품의 절대 파일 경로("C:\WebSphere\AppServer\Config")일 수 있습니다. 해당 기호 이름은 "CONFIG_ROOT"일 수 있습니다.

데이터 유형 문자열
설명
변수의 목적을 문서화합니다.
