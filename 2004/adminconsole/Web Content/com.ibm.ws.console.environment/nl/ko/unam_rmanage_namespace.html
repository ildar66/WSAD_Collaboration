네임 스페이스 바인딩 콜렉션

이 페이지를 참조하여 EJB, CORBA CosNaming NamingContext, CORBA 리프(leaf) 노드 오브젝트, JNDI를 사용하여 찾을 수 있는 오브젝트 또는 상수 문자열 값의 이름 바인딩을 구성하십시오.

구성된 바인딩에 대한 바인딩 정보는 구성에 저장되고 바인딩의 범위 안에 있는 각 서버에 대한 네임 서버의 시작 시에 적용됩니다.

네임 스페이스 바인딩 설정 관리 페이지를 보려면 환경 > 이름 지정 > 네임 스페이스 바인딩을 클릭하십시오.

선택란을 클릭하여 사용자 콜렉션에 있는 사용자 중 하나 이상을 선택하십시오. 단추를 사용하여 선택된 사용자를 제어하십시오.

이름
이러한 구성된 바인딩을 고유하게 식별하기 위해 지정된 이름을 지정합니다.
범위
구성된 바인딩의 범위를 표시합니다. 이 값은 namebindings.xml 파일의 구성 위치를 표시합니다. 이 필드는 정보를 위해서만 제공되며 갱신할 수 없습니다.

구성된 바인딩이 셀 범위인 경우, 시작 컨텍스트는 셀 지속 루트 컨텍스트입니다. 구성된 바인딩이 노드 범위인 경우, 시작 컨텍스트는 노드 지속 루트 컨텍스트입니다. 구성된 바인딩이 서버 범위인 경우, 시작 컨텍스트는 서버의 서버 루트 컨텍스트입니다.

바인딩 유형
바인딩 구성의 유형을 표시합니다. 가능한 선택사항은 String, EJB, CORBA 및 Indirect입니다. 이 필드는 정보를 위해서만 제공되며 갱신할 수 없습니다.
