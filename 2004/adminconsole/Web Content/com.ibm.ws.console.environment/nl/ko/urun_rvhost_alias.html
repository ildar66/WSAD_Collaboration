호스트 별명 콜렉션

가상 호스트에 정의된 호스트 이름 별명을 관리하려면 이 페이지를 참조하십시오. 별명은 클라이언트가 웹 응용프로그램 자원에 대한 URL 요청을 형성하기 위해 사용하는 DNS 호스트 이름 및 포트 번호입니다.

이 관리 콘솔 페이지를 보려면 환경 > 가상 호스트 > virtual_host_name > 호스트 별명을 클릭하십시오.

호스트 이름
IP 주소, 도메인 이름 접미부를 갖는 DNS 호스트 이름 또는 웹 응용프로그램 자원(예: Servlet, JSP 또는 HTML 페이지)을 요청하기 위해 클라이언트가 사용하는 단순한 DNS 호스트 이름을 지정합니다. 예를 들어, myhost:8080 DNS 이름에서 호스트 별명 이름은 myhost입니다.
포트
웹 서버가 클라이언트 요청을 승인하도록 구성된 포트를 ' 지정합니다. 예를 들어, myhost:8080 DNS 이름에서 포트 지정은 8080입니다. URL은 이 DNS를 : http://myhost:8080/servlet/snoop으로 참조합니다.
