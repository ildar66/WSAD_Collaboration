호스트 별명 설정

이 페이지를 참조하여 호스트 별명을 보고 구성하십시오.

이 관리 콘솔 페이지를 보려면 환경 > 가상 호스트 > virtual_host_name > 호스트 별명 > host_alias_name을 클릭하십시오.

구성 탭

호스트 이름
클라이언트가 웹 응용프로그램 자원(예: Servlet, JSP 파일 또는 HTML 페이지)을 요청하기 위해 사용하는 IP 주소, 도메인 이름 확장자를 갖는 DNS 호스트 이름 또는 DNS 호스트 이름을 지정합니다.

예를 들어, 호스트 별명이 myhost이면 DNS 이름은 myhost:8080입니다. 여기서 8080은 포트입니다. URL은 이 DNS를 http://myhost:8080/servlet/snoop로 참조합니다.

기존 인스턴스의 경우, 기본값은 제품 설치 시 지정된 값을 반영합니다. 새 인스턴스의 경우, 모든 값 또는 스펙 없음을 허용하도록 기본값 *을 사용할 수 있습니다.

데이터 유형 문자열
기본값 *
포트
웹 서버가 클라이언트 요청을 승인하는 포트를 지정합니다. 호스트 이름과 함께 포트 값을 지정하십시오.

기본값은 제품 설치 시 지정한 값을 반영합니다. 기본값은 80, 81, 9080 또는 이와 유사한 값일 수 있습니다.

데이터 유형 정수
기본값 80
