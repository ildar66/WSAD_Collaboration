이름
웹 응용프로그램 구성에 필요한 논리 이름을 특정 호스트 이름으로 지정합니다. 기본 가상 호스트가 대부분의 단순 구성에 적합합니다.

가상 호스트 구성은 단일 호스트 시스템이 여러 호스트 시스템을 닮을 수 있게 합니다. 가상 호스트가 동일한 물리적 시스템을 공유하더라도 한 가상 호스트와 연관된 자원은 다른 가상 호스트와 연관된 자원과 데이터를 공유할 수 없습니다.
