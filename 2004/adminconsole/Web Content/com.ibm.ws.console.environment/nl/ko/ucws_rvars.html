WebSphere 변수 콜렉션

이 페이지를 참조하여 값 및 범위가 있는 대체 변수 목록을 보고 변경하십시오.

이 관리 콘솔 페이지를 보려면 환경 > WebSphere 변수 관리를 클릭하십시오.

변수에 대한 정보를 보려면 변수를 클릭한 후 설명 필드에 있는 값을 읽으십시오.

이름
파일 경로, 웹 주소 또는 기타 값을 표시하는 기호 이름을 지정합니다.
값
기호 이름이 나타내는 값(예: 절대 파일 경로)을 지정합니다.
범위
기호 이름이 셀, 노드 또는 서버에 적용되는지 여부를 지정합니다.
