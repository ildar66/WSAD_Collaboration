Impostazioni del collegamento EJB

Utilizzare questa pagina per configurare un collegamento EJB nuovo o per visualizzare o modificare un collegamento EJB esistente.

Per visualizzare questa pagina della console di gestione, fare clic su Ambiente > Denominazione > Collegamenti spazio nome > collegamento_spazionome_EJB.

Scheda Configurazione

Ambito
Indica l'ambito del collegamento configurato. Questo valore indica l'ubicazione di configurazione del file namebindings.xml. Questo campo viene fornito a scopi informativi e non può essere aggiornato.

Se il collegamento configurato è nell'ambito della cella, il contesto iniziale è il contesto root persistente della cella. Se il collegamento configurato è nell'ambito del nodo, il contesto iniziale è il contesto root persistente del nodo. Se il collegamento configurato è nell'ambito del server, il contesto iniziale è il contesto root del server.

Tipo di collegamento
Indica il tipo di collegamento configurato. Le opzioni possibili sono String, EJB, CORBA e Indirect. Questo campo viene fornito a scopi informativi e non può essere aggiornato.
Identificatore collegamento
Specifica il nome che identifica in modo univoco questo collegamento configurato.
Nome nello spazio nome
Specifica il nome utilizzato per questo collegamento nello spazio nome. Può essere un nome semplice o composto a seconda della parte dello spazio nome in cui è configurato questo collegamento.
Ubicazione del bean enterprise
Specifica se il bean enterprise è in esecuzione sul cluster di server o su un unico server. Se viene specificato Server unico, immettere il nome del nodo.
Server
Specifica il nome del server singolo o in cluster su cui è configurato il bean enterprise.
Nome JNDI
Specifica il nome JNDI del bean enterprise distribuito (il nome JNDI del bean presente nei collegamenti del bean enterprise -- non il nome java:comp)
