Raccolta di host virtuali

Utilizzare questa pagina per gestire gli host virtuali.

Per visualizzare questa pagina della console di gestione, fare clic su Ambiente > Host virtuali.

Nome
Specifica un nome logico per configurare le applicazioni Web con un particolare nome host. L'host virtuale predefinito si adatta alla maggior parte delle configurazioni semplici.

Una configurazione host virtuale consente ad un'unica macchina host di avere le caratteristiche di più macchine host. Le risorse associate a un host virtuale non possono condividere i dati con le risorse associate ad un altro host virtuale, anche quando gli host virtuali condividono la stessa macchina.
