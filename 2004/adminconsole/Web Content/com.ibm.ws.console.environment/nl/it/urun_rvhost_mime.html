Raccolta di tipi MIME

Utilizzare questa pagina per visualizzare e configurare i tipi di oggetti MIME (Multi-Purpose Internet Mail Extension) e le estensioni nomi file corrispondenti.

L'elenco contiene una raccolta di mappature delle estensioni del tipo MIME definite per un host virtuale. Le voci MIME dell'host virtuale vengono applicate quando non vengono specificate voci MIME al livello del modulo Web.

Per visualizzare questa pagina della console di gestione, fare clic su Ambiente > Host virtuali > nome_host_virtuale > Tipi MIME.

Tipo MIME
Specifica un tipo MIME, che potrebbe essere testo, immagine o applicazione. Un valore di esempio per il tipo MIME è text/html.
Estensioni
Elenca le estensioni dei file definite sul tipo MIME. Esempi di estensioni di tipi MIME text/html sono .htm, .html e .txt.
