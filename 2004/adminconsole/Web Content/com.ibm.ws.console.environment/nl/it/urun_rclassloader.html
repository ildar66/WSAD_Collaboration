Raccolta programmi di caricamento classi

Utilizzare questa pagina per gestire le istanze del programma di caricamento classi su un server delle applicazioni. Un programma di caricamento classi determina se il programma di caricamento classi parent o quello dell'applicazione può individuare e caricare i file di classe Java di un'applicazione.

Per visualizzare questa pagina della console di gestione, fare clic su Server > Server delle applicazioni > nome_server> Programma di caricamento classi.

ID programma di caricamento classi
Dichiara una stringa univoca al server che identifica l'istanza del programma di caricamento classi. Il prodotto assegna l'identificativo.
Modalità del programma di caricamento classi
Specifica la modalità del programma di caricamento classi quando le politiche del programma di caricamento classi dell'applicazione sono impostate su SINGOLO. PARENT_FIRST indica che il programma di caricamento classi delegherà il caricamento delle classi al corrispondente programma parent prima di tentare il caricamento della classe dal percorso classe locale. PARENT_LAST indica che il programma di caricamento classi tenterà di caricare le classi dal percorso classe locale prima di delegare il caricamento al parent corrispondente; ciò consente al programma di caricamento classi di un'applicazione di sovrascrivere e di fornire la propria versione di una classe che si trova nel programma di caricamento classi parent.
