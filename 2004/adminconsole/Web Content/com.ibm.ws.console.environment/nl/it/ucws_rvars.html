Raccolta di variabili WebSphere

Utilizzare questa pagina per modificare un elenco di variabili di sostituzione con i relativi valori e ambiti.

Per visualizzare questa pagina della console di gestione, fare clic su Ambiente > Gestisci variabili di WebSphere.

Per le informazioni relative a una variabile, fare clic sulla variabile stessa e leggere il valore del campo Descrizione.

Nome
Specifica un nome simbolico che rappresenta un percorso file, un indirizzo Web o un altro valore.
Valore
Specifica il valore rappresentato dal nome simbolico, ad esempio il percorso file assoluto.
Ambito
Specifica se il nome simbolico è applicabile nelle varie celle, nodi o server.
