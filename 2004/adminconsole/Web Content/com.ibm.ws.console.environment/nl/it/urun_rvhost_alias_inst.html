Impostazioni alias dell'host

Utilizzare questa pagina per visualizzare e configurare un alias dell'host.

Per visualizzare questa pagina della console di gestione, fare clic su Ambiente > Host virtuali > nome_host_virtuale > Alias host > nome_alias_host.

Scheda Configurazione

Nome host
Specifica l'indirizzo IP, il nome host DNS con il suffisso nome dominio o il nome host DNS utilizzato dai client per richiedere una risorsa dell'applicazione Web, come un servlet, un file JSP o una pagina HTML.

Ad esempio, quando il nome alias dell'host è myhost, il nome DNS è myhost:8080, dove 8080 è la porta. Un URL fa riferimento a tale DNS nel seguente modo: http://myhost:8080/servlet/snoop.

Per le istanze esistenti, il valore predefinito riflette il valore specificato durante l'impostazione del prodotto. Per le nuove istanze, il valore predefinito può essere * per consentire qualsiasi valore e nessuna specifica.

Tipo dati Stringa
Predefinito *
Porta
Specifica la porta su cui il server Web accetta le richieste client. Specificare un valore porta in congiunzione con il nome host.

Il valore predefinito riflette il valore specificato durante l'impostazione del prodotto. Il valore predefinito potrebbe essere 80, 81, 9080 o un valore simile.

Tipo dati Numero intero
Predefinito 80
