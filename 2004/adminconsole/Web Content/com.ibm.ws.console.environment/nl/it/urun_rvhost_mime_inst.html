Impostazioni del tipo MIME

Utilizzare questa pagina per configurare un tipo di oggetto MIME (Multi-Purpose Internet Mail Extensions).

Per visualizzare questa pagina della console di gestione, fare clic su Ambiente > Host virtuali > nome_host_virtuale > Tipi MIME > tipo_MIME.

Scheda Configurazione

Tipo MIME
Specifica un tipo MIME, che potrebbe essere testo, immagine o applicazione.

Un valore di esempio per il tipo MIME è text/html. Un valore predefinito compare solo quando si visualizza la configurazione di un'istanza esistente.

Tipo dati Stringa
Estensioni
Elenca le estensioni dei file definite sul tipo MIME.

Esempi di estensioni file per un tipo MIME text/html sono .htm, .html e .txt. Il valore predefinito compare solo quando si visualizza la configurazione di un'istanza esistente.
