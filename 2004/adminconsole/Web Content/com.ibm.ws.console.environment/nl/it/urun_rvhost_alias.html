Raccolta di alias host

Utilizzare questa pagina per gestire gli alias di nomi host definiti per un host virtuale. Un alias rappresenta il numero di porta e il nome host DNS utilizzato da un client per formare la richiesta URL per una risorsa dell'applicazione Web.

Per visualizzare questa pagina della console di gestione, fare clic su Ambiente > Host virtuali > nome_host_virtuale > Alias host.

Nome host
Specifica l'indirizzo IP, il nome host DNS con il suffisso nome dominio o soltanto il nome host DNS, utilizzato da un client per richiedere una risorsa dell'applicazione Web (come una pagina HTML, un JSP o un servlet). Ad esempio, il nome alias di host è myhost in un nome DNS myhost:8080.
Porta
Specifica la porta per cui il server Web è stato configurato per accettare richieste client. Ad esempio, l'assegnazione di porta è 8080 in un nome DNS myhost:8080. Un URL fa riferimento a tale DNS nel seguente modo: http://myhost:8080/servlet/snoop.
