Raccolta collegamenti spazio nome

Utilizzare questa pagina per configurare un collegamento nome di un EJB, di un CosNaming NamingContext CORBA, di un oggetto nodo dipendente CORBA, di un oggetto accessibile con JNDI o di un valore stringa costante.

Le informazioni relative ai collegamenti configurati vengono memorizzate nella configurazione e applicate all'avvio del server dei nomi per ciascun server all'interno dell'ambito del collegamento.

Per visualizzare la pagina Gestisci impostazioni dei collegamenti spazio nome, fare clic su Ambiente > Denominazione > Collegamenti spazio nome.

Fare clic sulle caselle di controllo per selezionare uno o più utenti nella raccolta. Utilizzare i pulsanti per controllare gli utenti selezionati.

Nome
Indica i nomi assegnati per identificare i collegamenti configurati in modo univoco.
Ambito
Indica l'ambito del collegamento configurato. Questo valore indica l'ubicazione di configurazione del file namebindings.xml. Questo campo viene fornito a scopi informativi e non può essere aggiornato.

Se il collegamento configurato è nell'ambito della cella, il contesto iniziale è il contesto root persistente della cella. Se il collegamento configurato è nell'ambito del nodo, il contesto iniziale è il contesto root persistente del nodo. Se il collegamento configurato è nell'ambito del server, il contesto iniziale è il contesto root del server.

Tipo di collegamento
Indica il tipo di collegamento configurato. Le opzioni possibili sono String, EJB, CORBA e Indirect. Questo campo viene fornito a scopi informativi e non può essere aggiornato.
