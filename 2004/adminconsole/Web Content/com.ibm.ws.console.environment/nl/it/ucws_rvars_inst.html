Impostazioni delle variabili

Utilizzare questa pagina per definire il nome e il valore di una variabile di sostituzione di WebSphere Application Server.

Per visualizzare questa pagina della console di gestione, fare clic su Ambiente > Gestisci variabili di WebSphere > nome_variabile_WebSphere.

Scheda Configurazione

Nome
Specifica un nome simbolico che rappresenta un percorso file, un indirizzo Web o un altro valore.

WebSphere Application Server sostituisce il nome simbolico in ogni punto del sistema in cui ne viene visualizzato il valore.

Ad esempio, "CONFIG_ROOT" è il nome simbolico che rappresenta il percorso della directory di configurazione "C:\WebSphere\AppServer\Config" per il prodotto di base WebSphere Application Server su un sistema Windows.

Tipo dati Stringa
Valore
Specifica il valore rappresentato dal nome simbolico, ad esempio il percorso file assoluto.

Ad esempio, il valore potrebbe essere il percorso file assoluto, "C:\WebSphere\AppServer\Config," nel prodotto di base WebSphere Application Server su un sistema Windows. Il nome simbolico corrispondente potrebbe essere "CONFIG_ROOT."

Tipo dati Stringa
Descrizione
Documenta lo scopo di una variabile.
