Valores de enlace de objeto CORBA

Utilice esta página para configurar un nuevo enlace de nombre de un enlace de objeto CORBA o para ver o editar un enlace de objeto CORBA existente.

Para ver esta página de la consola administrativa, pulse Entorno > Denominación > Enlaces de espacio de nombres > enlace_espacio_nombres_CORBA.

Pestaña Configuración

Ámbito
Muestra el ámbito del enlace configurado. Este valor indica la ubicación de configuración del archivo namebindings.xml. Este campo se proporciona para fines informativos solamente y no se puede actualizar.

Si el enlace configurado tiene un ámbito de célula, el contexto inicial es el contexto de raíz persistente de célula. Si el enlace configurado tiene un ámbito de nodo, el contexto inicial es el contexto de raíz persistente de nodo. Si el enlace configurado tiene un ámbito de servidor, el contexto inicial es el contexto de raíz de servidor del servidor.

Tipo de enlace
Muestra el tipo de enlace configurado. Las posibles opciones son Serie, EJB, CORBA, y Indirecto. Este campo se proporciona para fines informativos solamente y no se puede actualizar.
Identificador de enlace
Especifica el nombre que identifica de forma exclusiva este enlace configurado.
Nombre de espacio de nombres
Especifica el nombre utilizado para este enlace en el espacio de nombres. Este nombre puede ser simple o compuesto dependiendo de la parte del espacio de nombres donde se configura este enlace.
URL de nombre CORBA
Especifica la serie de URL de nombre CORBA que identifica el punto de enlace del objeto en un servidor CosNaming.
Contexto federado
Especifica si el destino es un contexto CosNaming (true) o un objeto de nodo inferior (false).
Valor Resultado
true El objeto de destino se enlaza con un tipo de enlace CORBA de objeto. Si el URL de nombre CORBA no se puede resolver en un NamingContext, se produce un error cuando se utiliza el enlace por primera vez (esto ocurre cuando el URL se resuelve por primera vez).
false El objeto de destino se enlaza con un tipo de enlace CORBA de objeto.
