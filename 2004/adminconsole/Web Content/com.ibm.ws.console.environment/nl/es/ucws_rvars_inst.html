Valores de variable

Utilice esta página para definir el nombre y el valor de una variable de sustitución de WebSphere Application Server.

Para ver esta página de la consola administrativa, pulse Entorno > Gestionar variables de WebSphere > nombre_variable_WebSphere.

Pestaña Configuración

Nombre
Especifica un nombre simbólico que representa una vía de acceso del archivo, dirección Web u otro valor.

WebSphere Application Server sustituye el nombre simbólico siempre que aparece el valor correspondiente en el sistema.

Por ejemplo, "CONFIG_ROOT" es el nombre simbólico para la vía de acceso del directorio de configuración "C:\WebSphere\AppServer\Config" para el producto WebSphere Application Server base en un sistema Windows.

Tipo de datos String
Valor
Especifica el valor que el nombre simbólico representa, como por ejemplo la vía de acceso absoluta del archivo.

Por ejemplo, el valor podría ser la vía de acceso absoluta del archivo "C:\WebSphere\AppServer\Config," del producto WebSphere Application Server base en un sistema Windows. El nombre simbólico correspondiente podría ser "CONFIG_ROOT."

Tipo de datos String
Descripción
Documenta la finalidad de una variable.
