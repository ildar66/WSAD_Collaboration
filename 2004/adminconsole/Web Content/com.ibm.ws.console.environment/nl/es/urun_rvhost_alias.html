Colección de alias de sistema principal

Utilice esta página para gestionar los alias de sistema principal definidos para un sistema principal virtual. Un alias es el nombre del sistema principal DNS y el número de puerto que un cliente utiliza para formular la petición de URL de un recurso de aplicaciones Web.

Para ver esta página de la consola administrativa, pulse Entorno > Sistemas principales virtuales > nombre_sistppal_virtual > Alias de sistemas principales.

Nombre del sistema principal
Especifica la dirección IP, el nombre del sistema principal DNS con un nombre de dominio como sufijo o el nombre del sistema principal DNS sin sufijo que un cliente utiliza para solicitar un recurso de aplicaciones Web (como un servlet, un JSP o una página HTML). Por ejemplo, el nombre de alias de sistema principal es sistppal en un nombre DNS de sistppal:8080.
Puerto
Especifica el puerto con el que se ha configurado el servidor Web para que acepte las peticiones de cliente. Por ejemplo, la asignación de puerto es 8080 en un nombre DNS de sistppal:8080. Un URL se refiere a este DNS como: http://sistppal:8080/servlet/snoop.
