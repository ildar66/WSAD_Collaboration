Colección de variables de WebSphere

Utilice esta página para ver y modificar una lista de variables de sustitución acompañadas de los valores y el ámbito correspondientes.

Para ver esta página de la consola administrativa, pulse Entorno > Gestionar variables de WebSphere.

Para obtener información sobre una variable, pulse la variable y lea el valor en el campo Descripción.

Nombre
Especifica un nombre simbólico que representa una vía de acceso del archivo, dirección Web u otro valor.
Valor
Especifica el valor que el nombre simbólico representa, como por ejemplo una vía de acceso absoluta del archivo.
Ámbito
Especifica si el nombre simbólico se aplica en toda una célula, nodo o servidor.
