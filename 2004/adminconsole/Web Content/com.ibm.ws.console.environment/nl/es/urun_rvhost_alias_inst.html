Valores del alias de sistema principal

Utilice esta página para ver y configurar un alias de sistema principal.

Para ver esta página de la consola administrativa, pulse Entorno > Sistemas principales virtuales > nombre_sistppal_virtual > Alias de sistemas principales > nombre_alias_sistppal.

Pestaña Configuración

Nombre del sistema principal
Especifica la dirección IP, el nombre del sistema principal DNS con un nombre de dominio como sufijo o el nombre del sistema principal DNS sin sufijo que los clientes utilizan para solicitar un recurso de aplicaciones Web ,como un servlet, un archivo JSP o una página HTML.

Por ejemplo, cuando el nombre de alias de sistema principal es sistppal, el nombre DNS es sistppal:8080, donde 8080 es el puerto. Un URL se refiere a este DNS como: http://sistppal:8080/servlet/snoop.

En las instancias existentes, el valor por omisión indica el valor especificado durante la configuración del producto. En las instancias nuevas, el valor por omisión puede ser * para permitir cualquier valor o ninguna especificación.

Tipo de datos String
Valor por omisión *
Puerto
Especifica el puerto donde el servidor Web acepta las peticiones de cliente. Especifique un valor de puerto junto con el nombre del sistema principal.

El valor por omisión indica el valor especificado durante la configuración del producto. El valor por omisión podría 80, 81, 9080 o un valor similar.

Tipo de datos Integer
Valor por omisión 80
