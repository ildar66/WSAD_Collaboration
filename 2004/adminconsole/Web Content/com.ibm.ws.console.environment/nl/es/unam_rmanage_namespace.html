Colección de enlace de espacio de nombres

Utilice esta página para configurar un enlace de nombre de un EJB, un NamingContext de CosNaming de CORBA, un objeto de nodo inferior CORBA, un objeto que se pueda buscar mediante JNDI o un valor de serie constante.

La información de enlace de los enlaces configurados se almacena en la configuración y se aplica al arrancar el servidor de nombres de cada servidor en el ámbito del enlace.

Para ver la página Gestionar valores de enlaces de espacio de nombres, pulse Entorno > Denominación > Enlaces de espacio de nombres.

Pulse los recuadros de selección para seleccionar uno o más de los usuarios que aparecen en la colección. Utilice los botones para controlar los usuarios seleccionados.

Nombre
Muestra los nombres asignados para identificar de forma exclusiva estos enlaces configurados.
Ámbito
Muestra el ámbito del enlace configurado. Este valor indica la ubicación de configuración del archivo namebindings.xml. Este campo se proporciona para fines informativos solamente y no se puede actualizar.

Si el enlace configurado tiene un ámbito de célula, el contexto inicial es el contexto de raíz persistente de célula. Si el enlace configurado tiene un ámbito de nodo, el contexto inicial es el contexto de raíz persistente de nodo. Si el enlace configurado tiene un ámbito de servidor, el contexto inicial es el contexto de raíz de servidor del servidor.

Tipo de enlace
Muestra el tipo de enlace configurado. Las posibles opciones son Serie, EJB, CORBA, y Indirecto. Este campo se proporciona para fines informativos solamente y no se puede actualizar.
