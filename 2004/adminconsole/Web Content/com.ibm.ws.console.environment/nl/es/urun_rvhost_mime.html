Colección de tipos MIME

Utilice esta página para ver y configurar los tipos de objeto MIME (Multi-Purpose Internet Mail Extensions) y las extensiones de nombre de archivo correspondientes.

En la lista se muestra una colección de correlaciones de extensiones tipo MIME para un sistema principal virtual. Las entradas MIME del sistema principal virtual se aplican cuando no se especifican entradas MIME en el nivel de módulos Web.

Para ver esta página de la consola administrativa, pulse Entorno > Sistemas principales virtuales > nombre_sistppal_virtual > Tipos MIME.

Tipo MIME
Especifica un tipo MIME, que puede ser un texto, imagen o aplicación. Un ejemplo de valor de tipo MIME es text/html.
Extensiones
Enumera las extensiones de archivo de los archivos que se correlacionan con el tipo MIME. Entre los ejemplos de extensiones de un tipo MIME text/html se incluyen .htm, .html, y .txt.
