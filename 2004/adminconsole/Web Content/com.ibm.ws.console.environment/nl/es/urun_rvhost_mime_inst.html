Valores de tipos MIME

Utilice esta página para configurar los tipos de objeto MIME (Multi-Purpose Internet Mail Extensions).

Para ver esta página de la consola administrativa, pulse Entorno > Sistemas principales virtuales > nombre_sistppal_virtual > Tipos MIME > tipo_MIME.

Pestaña Configuración

Tipo MIME
Especifica un tipo MIME, que puede ser un texto, imagen o aplicación.

Un ejemplo de valor de tipo MIME es text/html. Sólo si se está visualizando la configuración de una instancia existente, aparece un valor por omisión.

Tipo de datos String
Extensiones
Enumera las extensiones de archivo de los archivos que se correlacionan con el tipo MIME.

Entre los ejemplos de extensiones de archivo de un tipo MIME text/html MIME se incluyen .htm, .html y .txt. Sólo si se está visualizando la configuración de una instancia existente, aparece un valor por omisión.

Tipo de datos String
