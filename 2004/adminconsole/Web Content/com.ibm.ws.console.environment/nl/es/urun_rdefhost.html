Valores del sistema principal virtual

Utilice esta página para configurar una instancia de sistema principal virtual.

Para ver esta página de la consola administrativa, pulse Entorno > Sistemas principales virtuales > nombre_sistppal_virtual.

Pestaña Configuración

Nombre
Especifica un nombre lógico para configurar aplicaciones Web para un nombre de sistema principal determinado. El sistema principal virtual por omisión es válido para la mayoría de las configuraciones simples.
Tipo de datos String
Valor por omisión default_host
