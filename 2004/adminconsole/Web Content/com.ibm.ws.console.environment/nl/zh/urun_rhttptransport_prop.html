HTTP 传输设置

使用此页面查看和配置 HTTP 传输。页面名称可以是 SSL 设置的页面名称，如 DefaultSSLSettings。

要查看此管理控制台页面，单击服务器 > 应用程序服务器 > server_name > Web 容器 > HTTP 传输 > host_name。

“配置”选项卡

主机
指定绑定传输的主机 IP 地址。

如果应用程序服务器位于本地机器上，则主机名可以是 localhost。

数据类型 String
端口
指定绑定传输的端口。指定 1 到 65535 之间的端口号。端口号对于给定的机器上的每个应用程序服务器必须是唯一的。
数据类型 整型
范围 1 到 65535
启用 SSL
指定是否使用安全套接字层（SSL）保护 WebSphere 插件和应用程序服务器之间的连接。缺省为不使用 SSL。
数据类型 Boolean
缺省 false
SSL
指定 WebSphere 插件和应用程序服务器之间连接的安全套接字层（SSL）设置。选项包含在安全中心中定义的一个或多个 SSL 设置，例如 DefaultSSLSettings、ORBSSLSettings 或 LDAPSSLSettings。
数据类型 String
缺省 在安全中心中定义的一个 SSL 设置
