主机别名集合

使用此页面管理为虚拟主机定义的主机名别名。别名是由客户机用于构成 Web 应用程序资源的 URL 请求的 DNS 主机名和端口号。

要查看此管理控制台页面，单击环境 > 虚拟主机 > virtual_host_name > 主机别名。

主机名
指定客户机用于请求 Web 应用程序资源（如 servlet、JSP 或 HTML 页面）的 IP 地址、带有域名后缀的 DNS 主机名或仅 DNS 主机名。 例如，在 myhost:8080 的 DNS 名称中主机别名是 myhost。
端口
指定已配置 Web 服务器以接受客户机请求的端口。例如，在 myhost:8080 的 DNS 名称中端口赋值是 8080。URL 按以下引用此 DNS：http://myhost:8080/servlet/snoop。
