MIME 类型集合

使用此页面查看和配置多用途因特网邮件扩展（MIME）对象类型和其文件扩展名。

列表显示为虚拟主机定义的 MIME 类型扩展名映射的集合。当您未在 Web 模块级别指定 MIME 条目时应用虚拟主机 MIME 条目。

要查看此管理控制台页面，单击环境 > 虚拟主机 > virtual_host_name > MIME 类型。

MIME 类型
指定 MIME 类型，可以是 text、image 或 application。MIME 类型的示例值是 text/html。
扩展名
列出映射 MIME 类型的文件的文件扩展名。text/html MIME 类型的示例扩展名包括 .htm、.html 和 .txt。
