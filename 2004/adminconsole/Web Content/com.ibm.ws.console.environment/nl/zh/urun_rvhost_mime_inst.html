MIME 类型设置

使用此页面配置多用途因特网邮件扩展（MIME）对象类型。

要查看此管理控制台页面，单击环境 > 虚拟主机 > virtual_host_name > MIME 类型 > MIME_type。

“配置”选项卡

MIME 类型
指定 MIME 类型，可以是 text、image 或 application。

MIME 类型的示例值是 text/html。缺省值仅在查看现有实例的配置的情况下显示。

数据类型 String
扩展名
列出映射 MIME 类型的文件的文件扩展名。

text/html MIME 类型的示例文件扩展名包括 .htm、.html 和 .txt。缺省值仅在查看现有实例的配置的情况下显示。

数据类型 String
