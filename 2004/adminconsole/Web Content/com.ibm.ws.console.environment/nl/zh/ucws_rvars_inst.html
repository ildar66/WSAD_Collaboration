变量设置

使用此页面定义 WebSphere Application Server 替换变量的名称和值。

要查看此管理控制台页面，单击环境 > 管理 WebSphere 变量 > WebSphere_variable_name。

“配置”选项卡

Name
指定表示文件路径、Web 地址或其它值的符号名称。

WebSphere Application Server 在系统中任何出现符号名称的值的地方替换符号名称。

例如，“CONFIG_ROOT”是表示 Windows 系统上的 Base WebSphere Application Server 产品的配置目录路径“C:\WebSphere\AppServer\Config”的符号名称。

数据类型 字符串
值
指定符号名称表示的值，例如绝对文件路径。

例如，值可能是 Windows 系统上的 Base WebSphere Application Server 产品中的绝对文件路径“C:\WebSphere\AppServer\Config”。相应的符号名称可能是“CONFIG_ROOT”。

数据类型 字符串
描述
文档记录变量的用途。
数据类型 字符串
