主机别名设置

使用此页面查看和配置主机别名。

要查看此管理控制台页面，单击环境 > 虚拟主机 > virtual_host_name > 主机别名 > host_alias_name。

“配置”选项卡

主机名
指定客户机用于请求 Web 应用程序资源（如 servlet、JSP 文件或 HTML 页面）的 IP 地址、带有域名后缀的 DNS 主机名或仅 DNS 主机名。

例如，当主机别名是 myhost 时，DNS 名称是 myhost:8080，其中 8080 是端口。URL 按以下引用此 DNS：http://myhost:8080/servlet/snoop。

对于现有实例，缺省值反映产品安装时指定的值。对于新的实例，缺省值可以是 * 以允许任何值或无规范。

数据类型 String
缺省 *
端口
指定 Web 服务器接受客户机请求的端口。 指定与主机名一起的端口值。

缺省值反映产品安装时指定的值。缺省值可能是 80、81、9080 或相似值。

数据类型 整型
缺省 80
