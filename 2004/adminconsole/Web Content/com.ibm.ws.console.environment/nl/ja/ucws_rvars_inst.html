変数設定

このページを使用して、WebSphere Application Server の置換変数の名前と値を定義します。

この管理コンソール・ページを表示するには、「環境」>「WebSphere 変数の管理」>「WebSphere_variable_name」をクリックします。

「構成」タブ

名前
ファイル・パス、Web アドレス、またはその他の値を表すシンボル名を指定します。

WebSphere Application Server は、シンボル名の値がシステムに出現すると、 必ずその値の代わりにシンボル名を使用します。

例えば、"CONFIG_ROOT" は、Windows システム上の WebSphere Application Server 基本製品の構成のディレクトリー・パスである "C:\WebSphere\AppServer\Config" を表すシンボル名になります。

データ型 ストリング
値
シンボル名が表す値 (絶対ファイル・パスなど) を指定します。

例えば、この値は、Windows システム上の WebSphere Application Server 基本製品の絶対ファイル・パス "C:\WebSphere\AppServer\Config" である場合があります。これに対応するシンボル名は、"CONFIG_ROOT" になります。

データ型 ストリング
説明
変数の目的を文書化します。
データ型 ストリング
