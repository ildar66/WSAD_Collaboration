ホスト・エイリアス設定

このページを使用して、ホスト・エイリアスを表示および構成します。

この管理コンソール・ページを表示するには、「環境」>「仮想ホスト」>「virtual_host_name」>「ホスト・エイリアス」>「host_alias_name」をクリックします。

「構成」タブ

ホスト名
クライアントが Web アプリケーション・リソース (サーブレット、JSP ファイル、 HTML ページなど) を要求するために使用する、IP アドレス、ドメイン・ネーム・サフィックスの付いた DNS ホスト名、 またはサフィックスの付かない DNS ホスト名を指定します。

例えばホスト・エイリアス名が myhost の場合、DNS 名は myhost:8080 で、この場合 8080 はポートです。この DNS を参照する URL は http://myhost:8080/servlet/snoop になります。

既存インスタンスの場合のデフォルトは、製品のセットアップ時に指定された値を反映しています。例えば、 デフォルトを * として、すべての値を許可、または指定なしにすることができます。

データ型 ストリング
デフォルト *
ポート
Web サーバーがクライアント要求を受け入れるポートを指定します。ポート値をホスト名とともに指定します。

デフォルトは、製品のセットアップ時に指定された値を反映しています。デフォルトは 80、81、9080 またはこれに類似する値です。

データ型 整数
デフォルト 80
