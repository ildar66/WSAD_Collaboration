管理コンソールのフィルター設定

フィルター設定を使用して、コレクション・ビューに表示されるエントリーのフィルター操作の方法を指定します。

フィルターに掛ける列を選択して、フィルター基準を入力します。

フィルターに掛ける列
ドロップダウン・リストを使用して、フィルター対象の 列を選択します。フィルターを適用すると、これに応じてコレクション表が変更されます。

例えば、 アプリケーション・サーバー名をフィルターに掛ける場合の基準を入力するには、「 アプリケーション・サーバー」を選択します。

フィルター基準
ドロップダウン・リストの隣のフィールドにストリングを入力します。 このストリングは、コレクション表に現れる項目を修飾するコレクション項目の名前に使用されるストリングでなければなりません。 このストリングには、ワイルドカード文字として、%、*、または ? 記号を 使用することができます。 例えば、*App* と入力すると、名前にストリング App を含むアプリケーション・サーバーがすべて検出されます。

( ) ^ * % { } ¥ + $ の文字の各文字の前に ¥ を付けると、 検索を実行する正規表現エンジンが、提供された検索基準と正確に一致します。 例えば、プロバイダー名に (XA) が含まれているすべての JDBC プロバイダーを検索する場合は、 次のように指定します。

*¥(XA¥)
