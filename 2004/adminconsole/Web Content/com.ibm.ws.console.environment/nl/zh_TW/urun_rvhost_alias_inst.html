主機別名設定

請利用這個頁面來檢視和配置主機別名。

如果要檢視這個管理主控台頁面，請按一下環境 > 虛擬主機 > virtual_host_name > 主機別名 > host_alias_name。

「配置」標籤

主機名稱
指定用戶端用來要求 Web 應用程式資源（如 Servlet、JSP 檔或 HTML 頁面）的 IP 位址、含領域名稱字尾的 DNS 主機名稱，或 DNS 主機名稱。

比方說，當主機別名是 myhost 時，DNS 名稱就是 myhost:8080，其中 8080 是埠。URL 會依照下列方式來參照這個 DNS：http://myhost:8080/servlet/snoop。

如果是現有實例，預設值會反映產品設定時所指定的值。 如果是新實例，預設值可以是接受任何值的 *，也可以是完全不指定預設值。

資料類型 字串
預設值 *
埠
指定 Web 伺服器用來接受用戶端要求的埠。 請指定結合主機名稱來指定埠值。

預設值會反映產品設定時所指定的值。 預設值可能是 80、81、9080，或某個類似值。

資料類型 整數
預設值 80
