變數設定

您可利用這個頁面來定義 WebSphere Application Server 替代變數的名稱和值。

如果要檢視這個管理主控台頁面，請按一下環境 > 管理 WebSphere 變數 > WebSphere_variable_name。

「配置」標籤

名稱
指定代表檔案路徑、Web 位址或其他值的符號名稱。

每當 WebSphere Application Server 本身的值出現在系統中，WebSphere Application Server 都會用它來取代符號名稱。

比方說，"CONFIG_ROOT" 是代表 Windows 系統中的基本 WebSphere Application Server 產品之配置目錄路徑 "C:\WebSphere\AppServer\Config" 的符號名稱。

資料類型 字串
值
指定符號名稱所代表的值，如絕對檔案路徑。

比方說，這個值可能是 Windows 系統中的基本 WebSphere Application Server 產品的絕對檔案路徑 "C:\WebSphere\AppServer\Config"。 對應的符號名稱可能是 "CONFIG_ROOT"。

資料類型 字串
說明
說明變數的用途。
