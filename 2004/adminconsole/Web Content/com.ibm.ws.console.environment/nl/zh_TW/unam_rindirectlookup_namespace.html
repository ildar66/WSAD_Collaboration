間接查閱連結設定

請利用這個頁面來配置新的間接查閱名稱連結，或檢視或編輯現有的間接查閱連結。

如果要檢視這個管理主控台頁面，請按一下環境 > 命名 > 名稱空間連結 > indirect_lookup_namespace_binding。

「配置」標籤

範圍
顯示已配置的連結之範圍。 這個值指出 namebindings.xml 檔的配置位置。這個欄位只供參考，無法更新。

如果配置的連結以 Cell 為範圍，起始環境定義就是 Cell 持續性根環境定義。 如果配置的連結不以節點為範圍，起始環境定義就是節點持續性根環境定義。 如果配置的連結以伺服器為範圍，起始環境定義就是伺服器的伺服器根環境定義。

連結類型
已配置的連結之類型。 可能選擇是「字串」、EJB、CORBA 及「間接」。這個欄位只供參考，無法更新。
連結 ID
指定用來唯一識別這個已配置的連結之名稱。
名稱空間中的名稱
指定名稱空間中的這個連結所用的名稱。 隨著這個連結配置所在的名稱空間部分而不同，這個名稱可以是簡式或複合名稱。
提供者 URL
指定取得 JNDI 起始環境定義所需要的提供者 URL 字串。
JNDI 名稱
指定用來查閱起始環境定義中的目標物件之名稱。
