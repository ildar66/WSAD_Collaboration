主機別名集合

請利用這個頁面來管理定義給虛擬主機的主機名稱別名。別名是用戶端用來形成 Web 應用程式資源之 URL 要求的 DNS 主機名稱和埠號。

如果要檢視這個管理主控台頁面，請按一下環境 > 虛擬主機 > virtual_host_name > 主機別名。

主機名稱
指定 IP 位址、含領域名稱字尾的 DNS 主機名稱或僅 DNS 主機名稱，用戶端會使用它們來要求 Web 應用程式資源（如 Servlet、JSP 或 HTML 頁面）。 比方說，在 DNS 名稱 myhost:8080 中，主機別名是 myhost。
埠
指定 Web 伺服器所配置用來接受用戶端要求的埠。 比方說，在 DNS 名稱 myhost:8080 中，指派的埠是 8080。URL 會依照下列方式來參照這個 DNS：http://myhost:8080/servlet/snoop。
