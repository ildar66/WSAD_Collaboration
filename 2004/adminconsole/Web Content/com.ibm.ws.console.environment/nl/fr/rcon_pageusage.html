Caractéristiques de la page de la console d'administration

Cette rubrique fournit des informations sur les éléments de base d'une page de la console d'administration, tels que les divers onglets que l(on peut s'attendre à rencontrer.

Les pages de la console d'administration sont arrangées selon quelques schémas de base. Comprendre leur présentation et leur comportement permet de les utiliser plus aisément.

Pages de collection

Utilisez les pages de la collection pour gérer une collection d'objets d'administration existants. Généralement, ce type de page contient un ou plusieurs des éléments suivants :

Portée, Filtre et Préférences
Ces éléments sont décrits dans les rubriques Paramètres de la portée de la console administrative, Paramètres de filtrage de la console administrative et Paramètres des préférences de la console administrative.
Table d'objets existants
La table présente les objets d'administration existants du type spécifié par la page de la collection. Les colonnes de la table résument les valeurs des principaux paramètres pour ces objets. Si aucun objet n'existe encore, une table vide est affichée. Utilisez les boutons disponibles pour créer un nouvel objet.
Boutons d'actions
Les boutons disponibles sont décrits dans la rubrique Boutons de la console administrative. Dans la plupart des cas, vous devez sélectionner un ou plusieurs objets dans la table, puis cliquer sur un bouton. L'action est alors appliquée aux objets sélectionnés.
Boutons à bascule pour le tri
Les en-têtes de colonnes suivants figurant dans la table sont des icônes représentant l'ordre croissant (^) et l'ordre décroissant (v). Par défaut, les éléments tels que les noms sont classés par ordre décroissant (par ordre alphabétique). Pour activer un autre ordre de tri, cliquez sur les icônes correspondant aux colonnes à trier.

Pages de détails

Utilisez ces pages pour configurer des objets d'administration spécifiques, tels qu'un serveur d'applications. Généralement, ce type de page contient un ou plusieurs des éléments suivants :

Page de configuration à onglets
Cette page composée de différents onglets permet de modifier la configuration d'un objet d'administration.
Page d'exécution à onglets
Ce type de page affiche la configuration en cours, utilisée pour l'objet d'administration. Dans la plupart des cas, ces pages sont accessibles en lecture seulement.
Boutons d'actions
Les boutons disponibles sont décrits dans la rubrique Boutons de la console administrative.

Pages d'assistant

Utilisez ces pages pour effectuer une procédure de configuration, composée de plusieurs étapes. N'oubliez pas que les assistants affichent ou masquent certaines étapes de la procédure en fonction des caractéristiques de l'objet spécifique que vous configurez.
