Paramètres des alias d'hôtes

Cette page permet d'afficher et de configurer un alias d'hôte.

Pour afficher la page de la console d'administration, sélectionnez Environnement > Hôtes virtuels > nom_hôte_virtuel > Alias d'hôte > nom_alias_hôte.

Onglet Configuration

Nom d'hôte
Indique l'adresse IP, le nom d'hôte DNS avec suffixe de nom de domaine ou nom d'hôte DNS seul, utilisés par un client pour demander une ressource de l'application Web, telle qu'un servlet, un fichier JSP ou une page HTML.

Par exemple, lorsque l'alias d'hôte est monhôte, le nom DNS est monhôte:8080, 8080 étant le port. Une adresse URL fait référence à ce DNS de la façon suivante : http://monhôte:8080/servlet/snoop.

Pour les instances existantes, la valeur par défaut reflète la valeur spécifiée lors de l'installation du produit. Pour les nouvelles instances, la valeur par défaut peut être * de façon à autoriser n'importe quelle valeur ou aucune spécification.

Type de données Chaîne
Valeur par défaut *
Port
Indique le port sur lequel le serveur Web accepte les demandes des clients. Indiquez une valeur de port avec le nom d'hôte.

La valeur par défaut reflète la valeur spécifiée lors de l'installation du produit. La valeur par défaut peut être 80, 81, 9080 ou une valeur similaire.

Type de données Entier
Valeur par défaut 80
