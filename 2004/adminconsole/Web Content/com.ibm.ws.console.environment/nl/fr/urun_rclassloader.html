Collection du chargeur de classe

Cette page permet de gérer les instances du chargeur de classe sur un serveur d'applications. Un chargeur de classe détermine si un chargeur de classe d'application ou un chargeur de classe parent cherche et charge des fichiers de classe Java pour une application.

Pour afficher cette page de la console d'administration, cliquez sur Serveurs > Serveurs d'applications > nom_serveur> Chargeur de classe.

ID chargeur de classe
Définit une chaîne unique sur le serveur qui identifie l'instance du chargeur de classe. Le produit attribue l'identifiant.
Mode du chargeur de classe
Indique le mode de fonctionnement du chargeur de classe lorsque la règle de chargeur de classe d'application est SINGLE. Lorsque vous utilisez PARENT_FIRST, le chargeur de classe délègue les classes à son chargeur de classe parent avant de tenter de charger la classe à partir de son chemin d'accès aux classes local. Lorsque vous utilisez PARENT_LAST, le chargeur de classe tente de charger les classes à partir de son chemin d'accès aux classes local avant de déléguer le chargement des classes à son parent ; ceci permet à un chargeur de classe d'application de remplacer une classe existante dans le chargeur de classe et de fournir sa propre version de la classe.
