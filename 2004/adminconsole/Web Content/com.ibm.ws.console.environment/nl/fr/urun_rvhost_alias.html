Collection d'alias d'hôte

Cette page permet de gérer des alias de noms d'hôte définis pour un hôte virtuel. Un alias correspond au nom d'hôte DNS et au numéro de port utilisés par un client afin de créer la demande d'URL pour une ressource d'application Web.

Pour afficher la page de la console d'administration, cliquez sur Environnement > Hôtes virtuels > nom_hôte_virtuel > Alias d'hôte.

Nom d'hôte
Indique l'adresse IP, le nom d'hôte DNS avec suffixe de nom de domaine ou nom d'hôte DNS seul, utilisés par un client pour demander une ressource de l'application Web (telle qu'un servlet, une page JSP ou une page HTML). Par exemple, un alias d'hôte peut être monhôte dans un nom DNS monhôte:8080.
Port
Spécifie le port pour lequel le serveur Web a été configuré de façon à accepter les demandes des clients. Par exemple, l'affectation de port peut être 8080 dans un nom DNS monhôte:8080. Une adresse URL fait référence à ce DNS de la façon suivante : http://myhost:8080/servlet/snoop.
