Paramètres des variables

Cette page permet de définir le nom et la valeur d'une variable de substitution WebSphere Application Server.

Pour afficher la page de la console d'administration, sélectionnez Environnement > Gestion des variables WebSphere > nom_variable_WebSphere.

Onglet Configuration

Nom
Spécifie un nom symbolique représentant un chemin d'accès à un fichier, une adresse Web ou une autre valeur.

WebSphere Application Server substitue le nom symbolique chaque fois que sa valeur apparaît dans le système.

Par exemple, "CONFIG_ROOT" correspond au nom symbolique représentant le chemin d'accès au répertoire de configuration "C:\WebSphere\AppServer\Config" du produit WebSphere Application Server de base, sous Windows.

Type de données String
Valeur
Spécifie la valeur représentée par le nom symbolique (par exemple, un chemin d'accès complet à un fichier).

Cette valeur peut par exemple correspondre au chemin d'accès complet du fichier, "C:\WebSphere\AppServer\Config", dans le produit WebSphere Application Server de base, sous Windows. Le nom symbolique correspondant peut être "CONFIG_ROOT."

Type de données String
Description
Précise l'objet d'une variable.
