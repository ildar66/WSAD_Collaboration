Paramètres de liaison de l'EJB

Cette page permet de configurer une nouvelle liaison d'EJB ou d'afficher ou de modifier une liaison d'EJB existante.

Pour afficher cette page de la console d'administration, cliquez sur Environnement > Nommage > Liaisons de l'espace de nom > liaison_espacenom_EJB.

Onglet Configuration

Portée
Affiche la portée de la liaison configurée. Cette valeur indique l'emplacement de la configuration pour le fichier namebindings.xml. Cette zone contient des informations et ne peut pas être mise à jour.

Si la portée de la liaison configurée est celle de la cellule, le contexte de démarrage est le contexte de racine persistante de la cellule. Si la portée de la liaison configurée est celle du noeud, le contexte de démarrage est le contexte de racine persistante du noeud. Si la portée de la liaison configurée est celle du serveur, le contexte de démarrage est le contexte de racine serveur du serveur.

Type de liaison
Affiche les types de liaison configurés. Les choix possibles sont : Chaîne, EJB, CORBA et Indirect. Cette zone contient des informations et ne peut pas être mise à jour.
Identificateur de liaison
Indique le nom qui identifie de manière unique cette liaison configurée.
Nom dans l'espace de nom
Définit le nom utilisé pour cette liaison dans l'espace de nom. Il peut s'agir d'un nom simple ou d'un nom composé en fonction de la partie de l'espace de nom dans laquelle cette liaison est configurée.
Emplacement du bean enterprise
Indique si le bean enterprise est en cours d'exécution dans un cluster de serveurs ou dans un serveur unique. Si un serveur unique est indiquée, entre le nom de noeud.
Serveur
Indique le nom du serveur du serveur en cluster ou non dans lequel le bean enterprise est configuré.
Nom JNDI
Indique le nom JNDI du bean enterprise déployé (nom JNDI du bean situé dans les liaisons du bean enterprise--et non le nom java:comp)
