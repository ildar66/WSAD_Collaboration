Collection de types MIME

Cette page permet d'afficher et de configurer des types d'objets MIME (Multi-Purpose Internet Mail Extensions) et leurs extensions de noms de fichiers.

La liste présente une collection de mappages d'extensions de types MIME définis pour un hôte virtuel. Les entrées MIME de l'hôte virtuel s'appliquent lorsque vous ne spécifiez pas d'entrées MIME au niveau du module Web.

Pour afficher la page de la console d'administration, cliquez sur Environnement > Hôtes virtuels > nom_hôte_virtuel > Types MIME.

Type MIME
Indique un type MIME qui peut être texte, image ou application. text/html est un exemple de valeur de type MIME.
Extensions
Répertorie les extensions des fichiers qui correspondent au type MIME. Par exemple, les extensions de fichier pour un type MIME text/html peuvent correspondre à .htm, .html et .txt.
