Paramètres de type MIME

Cette page permet de configurer un type d'objet MIME (Multi-Purpose Internet Mail Extensions).

Pour afficher la page de la console d'administration, sélectionnez Environnement > Hôtes virtuels > nom_hôte_virtuel > Types MIME > type_MIME.

Onglet Configuration

Type MIME
Indique un type MIME qui peut être texte, image ou application.

text/html est un exemple de valeur de type MIME. Une valeur par défaut n'est fournie que si vous affichez la configuration d'une instance existante.

Type de données Chaîne
Extensions
Répertorie les extensions des fichiers qui correspondent au type MIME.

Par exemple, les extensions de fichier pour un type MIME text/html peuvent correspondre à.htm, .html et .txt. Une valeur par défaut n'est fournie que si vous affichez la configuration d'une instance existante.
